# 1

今年度終於又搞定了合購活動,有收到影片的朋友大概正在觀看影片吧!說實在的GEN的影片有時還真是高興跟失望一半一半,不知道大家的看法如何,從巨大女子高生3開始扮演的那個女生看起來還不錯,只是演技上還有點生疏的感覺,GEN是喜歡比較沒表情的扮演嗎?真是的!看了巨大女子高生3跟巨大OL最後面攝影休息中時的片段,那位女生很自然的笑,看起來還蠻可愛的,只是扮演時比較少笑是可惜的地方!

接下來就談談個影片感想吧!當然有部分是瀏覽過找時間再仔細看!

１．巨大娘襲来②

這個女生看起孩還尚可,只是皮膚有些黑,至少比其他幾部還好看些,及膝長筒襪的搭配不錯,踩踏的人也很多,算還可以接受的影片

２．破壊 ! 巨大女

這個我確定就是扮演巨大女護士的那位,所以有點失望

３．オンラインPACK

算遇到地雷,又是那位熟女扮演的,頭痛,不予置評

４．巨大な婦警さん

這次的劇情比較特殊,還拿出一把槍,哇靠!太猛了!但是,女生還是差強人意,敗筆

５．巨大女子高生③

這一部是這位女生的處女作,看起來可愛些,這一部開始GEN加入高速攝影,但畫質真的有差,還不錯看,但感覺很拖時間,而這位女生我感覺那個髮型不太適當是女子高中生,比較像時下流行的髮型

６．ストレス・THE 巨大OL

扮演的GTS是巨大女子高生3那位,這次的髮型還不錯看,只是裙子感覺較長些,為啥OL都是這副打扮?這次扮演比較好些,但表情還是很少,最後的那個畫面在消救看起來很可愛,如果拍攝時就那樣子就不錯了

最後一部
GIANTESS-X

連續三部都是這位女生拍攝的,這位女生在GEN的留言板獲得不錯的評價,但是這一部為啥要加入踩狗,手要抓人的特寫還不錯,但是為啥是丟下而不是吃掉?

其實全部看起來就是新人演的3部還不錯看,巨大娘襲來2還可以,其他,不想多說什麼,新人看來很瘦GEN的器重,聽說她還要扮演巨大女警的一部短片,期待GEN的選角都是那樣子不錯的女生,但劇情的確還是要加強

剛下完且看完某些片段。
說一下自己的感想。
在「OL」那部，雖然女主角的穿著有點老派。但是，POV的視角捉的還不錯，最起碼壓破感與對比的感覺有捉出來，而且感覺非常好，此部份在影片的02:30至03:00之間，OL準備稿死警察時。至於其他的部份，因為則一直延用舊手法，故沒啥特別的感覺。有沒有壓迫感、對比，是決定我看影片爽快程度(Katelyn的影片，多會營造這小人與她的對比)。

至於，「破壞巨大女」。我只能說他的穿著非常合我的口味，即使我知道他人不好看，但還是耐著性子把該片看完。就女主角來說，我只能說。她非常非常不適合演護士，因為會將容貌的缺點完全爆露出來。他在此部中，我倒又覺得還好，可能鏡頭刻意跳過吧 ^^.。該片手法覺得大同小異，沒有啥感覺。

「X系列」坦白說，我只看了2分鐘就關了。沒有感覺。

總結：此次合購最起碼有看到Gen有營造小人與女主角的對比，雖然只有30秒。但是，這否是曡花一現，仍有待後續觀察。希望Gen先生能多去營造對比感，進而創造出觀看者的壓迫感，這點在GTS影片中，應該還蠻重要的。
