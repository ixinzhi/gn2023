# 1

今天登上了最新的世界第一高觀景台，上海環球金融中心
具體資料百度。。。。。這裡只談實際體驗

比起許多世界級高樓，環球金融中心最大的優點在於它是實實在在的492米，而不是靠著長長的電線杆子硬撐上去的。。。。。給人的感覺很實惠
頂樓的觀景台設計是半透明的，就是說，牆壁都是玻璃，地板也是玻璃，只有兩條不透明的道路可以安全行走（心理上的），恐高者慎入

然後，最重要的，在這個高度往下看，確實很難分辨出人類
由於角度問題（從上向下），站立行走的路人幾乎只有一到二毫米大小，根本看不清
這還是以人類視角來看的，若是GTS視角，由於眼睛也等比放大，所以會感覺地面的路人更加不顯眼

記得以前有個帖子說百多米的高度上看人很清晰，所以一百多米的GTS如果殺人必然是故意的；那麼500米高度的GTS就很有可能【無意踩踏】
