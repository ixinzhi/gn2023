# 5

雨的形成是一個大空間內凝結的許多水滴，總重量大，但面積也大，就算不看空氣阻力壓力也會被分攤
而一團完整的大水滴打下去重量會集中許多，考慮到高空的分散，破壞力可能會下降一點
這時候就要把黏稠度考慮進來了
黏稠度代表硬度，最極端的例子就是玻璃，雖然是液體只是流動超慢
[https://www.pornhub.com/view_video.php?viewkey=ph5af33697da045](https://www.pornhub.com/view_video.php?viewkey=ph5af33697da045)
如你所見，愛液可以稠到跟保麗龍膠一樣能掛在手上，這個東西巨大化砸到普通人的世界絕對很有破壞力
就算是最稀的分泌液，完整一團從高空下墜，怕

不過那有個前提是水珠的大小(表面張力強度)有跟巨人一起巨大化
如果水珠的物理不是跟著巨人而是普通人的世界走，在分泌出來沒多久就會承受不住自身的重量，
在能累積成大水珠之前就會紛紛跳樓，而能累積到多大又要回到黏稠度的問題
如果黏稠度夠高，那量產愛液海從天而降意義跟冰雹差不多
如果濃度接近於水，只要分泌速度不夠快就沒辦法有多驚人的破壞力
