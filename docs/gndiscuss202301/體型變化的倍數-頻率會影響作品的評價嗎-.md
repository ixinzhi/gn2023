# 4

入圈尚浅，但也想提供些浅薄的看法。

**个人认为：体型的倍率十分重要。**

咱在圈子里所能看到的萌点……诸如或浅或深的受虐倾向，或者是对强大异性的依赖。有点像是从温柔中体现的母性，或者从残忍行为中体现的“被支配”的感觉。而在这些主观感受之外的，是客观上文中比例差距的存在。

对于读者来说，能否对上喜爱的体型差距是很重要的。如果对不上比例，或许点开文章之后读了几段后就不会接着读下去，更甚者不会点开，自然也就不会体会到后面那些有可能会戳到自己萌点的剧情。

对于写手来说，体型的改变或许会让人难以下笔。像是前辈提到的“节奏”的问题，如何合理的继续拉大体格的差距需要对剧情仔细斟酌，这份斟酌本身就会消磨掉写手的很多精力。如果考虑不周就会让整个剧情崩掉……所以是很难办的事情。

而对我个人而言，能把握好频率和倍数的机会可遇而不可求，在平衡之中只能不断的试错，然后退而求其次。

最后咱的解决方案（逃课手段）就是，争取把所有的倍率都尝试一遍，总有人会喜欢其中某一部分，在不断地尝试中总会遇到能够拿捏好平衡的点，届时再努力把握机会咯。
