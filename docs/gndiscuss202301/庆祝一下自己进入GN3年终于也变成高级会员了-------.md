# 1

记得是08，09年左右看了个什么性感女巨人开始入gts的坑，当时还只是个混迹gts贴吧的萌新，贴吧被和谐后就没地方去了，然后就过着翻墙去油土鳖找视频的日子。还顺便了解了不少影响人生观的东西。。
**后来看了但丁的DCU系列文章直接被查毒![](img/16647e444edc27fc17737a89e534b73f.png)**，但大的文真的是有毒几天没看浑身难受。
拖到13年底才注册了个账号来到GN论坛，成为一名合格的真。潜水党。终日看着大佬们的文章图片视频各种一本满足。直到这个暑假才真正的想到去写点东西想去摆脱自己一直是伸手党的身份。
作为之前从没写过任何小说的渣渣来说自己写文才真正的体会到文豪们写文还能长期更新的不易。才发现每个人的回复对于作者来说才是莫大的鼓励。这里只能再次感谢下大触们的无私奉献![](img/afaf29bab8e0191f31485625eda6c801.png)
回想起以前看完文章就潜水走的行为真是羞耻的不行。以后得养成看文回复的好习惯。![](img/e165e72abfe717aae40508dc244b1023.png)

总之在晚上例行上线时竟然发现自己积分有了2W成为了高级会员的一员。忍不住单曲循环起[Uptown Funk](http://www.bilibili.com/video/av2195243/)

现在真是high的不行，混了3年GN终于也是高级会员了。一颗赛艇！
