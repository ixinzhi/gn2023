# 3

楼主提到了“人类文明”，我也是看giga的作品比较多，就只从giga级别的角度谈谈。
简单来说，巨大少女做出的除了用城市自慰以外的任何行为都仅仅是为了取得精神上的快感。举个例子，巨大少女踩扁一座城市，就如同踩到一片砂砾上一样，你不会因为踩到了一片砂砾而感到快乐，因为从人的生理结构上来说，人并没有从脚底的刺激获取快感的机能；但对于内心有施虐倾向（即有情感）的巨大少女来说，是通过“仅仅靠‘踩’这一对自己而言简单的动作就能导致一座城市的灭亡”这一行为所获取的对小人所产生的优越感和凌辱感而伴随着快感。可以看出，巨大少女要想通过羞辱小人来得到其精神快感的话，成立条件有三，一是行为主体的精神需求，一是被作用主体，一是行为过程。

先说行为主体。要满足“内心有施虐倾向”，或者说“可以通过施虐/羞辱的行为来获得快感”，这是典型的精神活动，如果没有感情的话，可以说从这点上就已经不满足条件了。人不会去主动做一件既不能满足生理需求也不能满足精神需求的事情。
其次，是被作用主体。如果巨大少女所在的地方没有人类，她仅仅去破坏自然环境或是各种地形是没有意义的，除非她是个喜欢玩泥巴的三岁小孩。这点应该比较清楚的。这和巨大少女是否具有情感是没有关系的。
最后是行为过程，这要抽象一点，其内在逻辑关系大概就是：“对自己来说轻松/简单/下等的事情”可以对“对他人来说困难/珍贵的事情”有着巨大影响。这和抖S在一定程度上是共通的，但又不完全一样。个人对sm不是很了解，所以略去不谈。其实大家看giga级别的文的时候会发现里面或多或少都会涉及小人的活动。虽然到了这个级别已经无法产生互动，但是仍然可以写巨大少女“轻微的某个动作”就导致了“小人的城市如何如何”。实际上就是通过对比的写作手法，通过小人的费力和挣扎以及巨大少女的轻松悠闲来进一步衬托巨大少女的地位和力量。从巨大少女的角度来看也是如此，大概就是“哈哈我只用一脚就把小人们建了不知道多久的城市踩扁了他们真弱啊我真强大啊”这种感觉。如果你让巨大少女和小人比赛算数学题，就完全没有意思了。
至于自慰这个行为则是话题外的。因为自慰在一定程度上是由于性欲导致的。性欲是一种生理需求而非心理需求。但这里又有区别。譬如，“仅仅是为了满足性欲而自慰”和“为了羞辱小人们而自慰”是不一样的，后者可以参照上面的叙述。在没有情感的前提下，除非是有人强制命令（对于巨大少女来说可以无视这种可能性），一般而言都是会为了生理需求而行动的。

同时，我们还应从另一个角度上考虑，也即“小人”的角度上考虑。我们在写文或是读文的的时候一般有三种视角：巨大少女视角、小人视角、上帝视角。上面的内容是根据巨大少女视角而言的。实际上小人视角和上帝视角是差不多的，区别就在于是“自己感受到被羞辱被压迫”还是旁观着他人“被羞辱被压迫”。对于用小人视角处理的文来说，巨大少女有无情感都是可取的，因为在这里我们将自己代入了小人，因此行为主体变成了我们，而我们无疑是有感情的，因此无论巨大少女是否是有意地进行破坏，我们都可以看做或者说是一定会下意识地觉得是羞辱。譬如，一个没有情感的巨大少女无意识地踩踏了一座城市，你身为城市里的一个居民自然会因为“对自己来说很大的城市”被那个少女“轻易地踩扁了”而感到被羞辱感和恐惧，而实际上人家并没有这个意思。而许多的无意识文正是采用这样的写作思路，放大了上面的行为过程，因为对方甚至只是下意识地活动就能造成如此巨大的影响，将自己代入到小人视角或上帝视角反而比其他的文要更刺激。

最后正式地回答楼主的问题，其实这个问题仅仅从我们自身的角度来讲是很简单的，巨大少女如果没有感情的话肯定是不会主动想要做出“羞辱小人”的举动的。而且，从上面的“行为主体—行为过程—被作用主体”的链条来说，既然情感是和“行为主体”直接相关的，那么缺少了行为主体的条件，链条自然会断裂。不过除此之外，可以说巨大少女的活动对小人来说都是带有“羞辱”的意味所在的。
说点现实的话，从创作者的角度来说，我们要提供给读者的仅仅是“施虐或者受虐”带来的心理上的快感，那么无论巨大少女是否具有情感，只要我们能让读者产生代入感以获得他们想要的东西，就已经足够了。
大清早的语言可能有些混乱，若有欠缺的地方，还望各位不吝赐教。

简单来说，当然不会。前提既然是“没有情感”，而“快感”又是“情感”的一种，既然没有情感，何来的谋取快感呢？
实际上，我认为大多数GTS文里的巨大少女虽然表面上是强势的一方，但是又在为小人的欲望服务，为代入小人取得“羞辱感”从而获得“刺激”的角度服务
打个比方好了，我们大多数人，无论男女都接触过蚂蚁吧。遇到蚂蚁的时候，不同情况下的我们会采取不同的行为，如果只是路过看到地上有只蚂蚁，大多数人就看一眼，“哦，有只蚂蚁。”然后就去做自己该做的事情了。而特别无聊，或者心情特别烦躁的情况下，可能会把蚂蚁按死或者踩死。
但是无论什么时候，你都不会蹲下来，盯着蚂蚁看，然后再说点中二台词，之后慢慢把它折磨死，除非你有一些奇怪的幻想。
在GTS文中则很不一样，巨大少女经常会有这些举动，甚至是更奇怪的举动，比如哈哈大笑然后追着小人踩，还故意踩在他们身边吓唬他们。如果我们从比较现实的角度来看的话，
普通少女变成巨大少女，应该是惊慌，等待政府来救援（在绝大多数正常人思维里，变成巨大少女很可能是一种病或者变异，甚至时日无多了，想办法活下去才比较重要）
比较调皮，天真，呆，或者搞不清楚状况的巨大少女可能会这里看看，那里碰碰，然后发现自己没在做梦，小人又是自己同类的情况下，回归普通人的行动模式，等政府救援
足够理智，心理强大的性格，则会搞清楚情况之后，想办法如何利用自己的体型争取资本
如果是心理极度扭曲，或者接近崩溃的性格，可能会开始报复社会，报复自己讨厌的人
但是这一切都是建立在情感上的，害怕也好，恶毒也好，都是情感的延伸，如果没有情感（我甚至想象不出来没有情感的人是什么样的），则以上行为都不成立了，因为没有情感，不会害怕，不会调皮，不会分析，也不会崩溃
因此，我觉得很难推断巨大少女会有什么样的行为，不如如此假设，巨大少女来自于一个其他的世界，她对于人类这样小的生物早习以为常了，因此不会特意对人类做些什么
那么在这种情况下，人类文明在巨大少女来说就和人类看蚂蚁一样，并没有什么特殊的地方
这里就出现了很有趣的事情，即为人类和巨大少女之间的认知偏差。
比如对于人类来说，踩死一只蚂蚁并不是羞辱它，而是心血来潮或者发泄的渠道，但是如果蚂蚁有足够的思维和情感，这就是羞辱了
同样的，对于巨大少女来说，踏平一座城市可能只是因为它正好在巨大少女去买奶茶的路上，甚至可能没看到就踩过去了，在巨大少女的角度而言，这没什么特殊的，甚至就算注意到了，过两分钟也会忘掉，但是在人类看来，自己努力构建的城市被踩烂了，至少也是值得愤怒的。
并不排除部分恶趣味的巨大少女会有一些行为，比如和人类故意冲着蚂蚁窝小便一样，但是主动的，持之以恒的，以羞辱为目的接触人类文明，大概是不会存在的吧
