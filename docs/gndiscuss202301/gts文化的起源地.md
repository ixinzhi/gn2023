# 2

不存在什么起源地的问题，这个特殊的喜好完全是自发的，天生的，通常跟外界没什么太多关联。

所谓GTS文化起源，其实应该就是我们这些GTS爱好者第一次所见到和熟知的作品。
在西方，应该是格列佛游记。
日本的民间传说有一寸法师。
中国的有女娲造人。
不过原作者都不是GTS同好，但他们的作品却是我们的重要启蒙。

西方GTS爱好者很多，可能主要原因是女权运动的发达，西方女性也比较开放，男人崇拜女性的权力和力量，进而发展成GTS同好。
日本GTS文化发达，应该跟日本人的生存环境有关，发展空间狭小，所以对巨大的事物充满了崇拜，进而产生大量的GTS同好。
至于中国，就目前的情况来看，大多是SM+恋足=GTS，谈不上特别正统。

以上是我的个人看法，欢迎交流。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
