# 3

你提的問題幾乎都取決於角色設定啊……

女方：含蓄內斂 男方：被寵壞而很囂張
可能劇情：男生不停得寸進尺，於是某天女孩爆氣反殺

女方：活潑開朗 男方：極度自卑
可能劇情：女方想盡各種方法哄男主角開心，最後哄到床上

或者男女性格調換，一樣都能寫，憑藉雙方不同的行為給予對方的刺激能發展無限可能性
我以前寫了一堆方法，後來我才知道那種寫作手法叫Character Driven，跟Plot Driven相對，是以角色行動當劇情發展基準
最重要的就是去設想「一個角色在特定心境下面對了特定刺激會產生什麼反應」，讓劇情自然而然的發展
最近最值得參考的應該就是《輝夜姬想讓人告白》，
去翻翻看漫畫，女主角在不同心境下面對刺激會有差超多的反應，但又能說服讀者這都是同一個角色的行動
連作者都在剛連載的時候說「我也不知道這個故事會怎麼發展」，因為角色會怎麼成長連作者都無法預料
當然回歸老話一句，如果你見識過的人不夠多，想像力不夠就不可能同時寫出各種有說服力的角色
先拿自己當範本試試看，試著想像自己養老鼠、自己是被正妹養的老鼠會有什麼感覺，然後再依照這種感覺讓角色行動

總之我覺得你選的題材夠有趣了，背景設定寫好以後就試著放棄控制角色讓他們自然而然發展吧，總有一天會不小心撞見對方自慰的(x)
