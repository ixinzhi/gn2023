# 1

恩……之前看預告的時候真的很期待的說。
但是看了論壇上的同好放出的本體成品之後，只能暗自慶幸當時沒有很衝動的跑去搜索附近的電影院是否有上映。

首先，先吐槽下劇情，不知道是不是編導這货是靈感枯竭，整出了這種東拼西湊的破爛劇情。（因為如果真讲劇情會涉及到劇透，故暫且按下不提），很多場景其實完全可以做的很精彩，結果卻被糟糕的處理手法給糟蹋了。
其次，關於互動的部分，諸位不用特別期待，基本可以無視掉了，完全沒有任何看起來比較自然的身高差對比，比N年之前的幾部經典GTS片（復仇的女巨人 etc.）效果差的太多了，甚至還不如國外GZ之類的早期作品的效果來的好。對於本猫這種特攝片愛好者來講，這部分完全可以打0分。我下載的還是低清版的，仍然可以用肉眼看到很多的處理不完全的場景。因此，我只能把這個理解成為，這部作品純粹是一個超低預算+成本的粗製濫造的骗錢片。

唯一值得我多提一句的就是，完全不穿衣服的時候也沒有聖騎士在片裡刷聖光術或者是馬賽克君出沒，這個應該對於某些魔法師们是個擼點吧。。。（菸~
