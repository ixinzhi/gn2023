# 4

实用性啊，对于在圈子里混久了的人来说，单纯的快餐文很多时候千篇一律，毕竟人体就那么大，写文的人都避免不了去描述gts的某几个身体部位，这个时候，作者的文笔就是拉开差距的关键了，这就像在面对同一道题各位给出不同的解法一样，自然是有好有坏。
很多时候，gts文章会将重点聚焦在互动上（握溃、踩死、vore之类），大家的兴奋点无非是，gts利用自身的强势怎么样去对待小人，当把想写的东西聚焦在”巨人会做出什么样的动作“的时候，其实是限制性很强的（人的动作也是有限的啊），绝大多数的互动情形都被写过了。
我认为，一篇优秀的gts文章，如果没有好的剧情或者设定支撑的话，是很难让人眼前一亮的。看重复的东西会让人审美疲劳，但你要是想写一篇gts文，就跳不开写前人已写过的东西。
这个时候，”新“很大程度上成了能不能俘获读者的一点（能不能让读者感受到对未知的兴奋，而不是千篇一律的展开），也即成为实用性衡量的标准，剧情和设定可以很好的拟补这一点。 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

说实话，作为作者和读者真的是完全不一样的感觉。对于读者来说，因为我那种很燥热的那种人，看文章贼冲动，有时候冲着冲着就贤者了。但牛逼的文章是光看，就让人非常的燥热，脸像初恋的小女生一样发红发烫，呼吸急促，大脑和被电击了一样。那种一定是剧情和尺寸差本身结合的很好的，非常能够带动情绪的。

但是作为作者，就真的特么很奇怪了。我是那种写东西纯粹为了自己爽的那种作者，但是我写的时候xp说实话完全是第二位甚至是附带的，因为写的时候很需要思考和理智，所以思考时候就没法太执着性癖了。我往往是把性癖以及各种人设，当做想象的一个支点，然后脑子和屁股被烙铁烫了的马一样瞎鸡儿乱跑，把这些想到的东西组合在一起，形成一个我自我感觉贼良好自己觉得贼牛逼的框架。
