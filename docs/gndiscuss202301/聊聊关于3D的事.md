# 1

      近期一直在和play club周旋奋战，说实话刚知道刚下载刚接触时异常兴奋，感觉挖到宝，因为一直想摆脱文字束缚，进军图片及视频领域。多年前机缘巧合也曾短时间接触使用过poser，无奈那时硬件和条件相当有限，还没怎么实践就被迫卸载了。

      直到不久前接触到play club，总算是圆了一些梦想，也做出一些东西，大家有目共睹。那么问题来了，随着作品是增多，初始的兴奋和激动在慢慢消退，因为越深入，我就发觉局限性越大，细节部分越受阻碍，无法实现我心中所想。我不知道还能以相当有限的场景及道具坚持多久，即便每次出场的妹纸都不一样(妹纸长相身材等随意捏)，但背景及tiny man次次雷同的话，久如嚼蜡。

      所以想再进一步，据我所知目前常用的3D软件有poser和daz，细节呈现无敌，各种道具无数，阻碍就是好像目前只有鸟语版，但也还能克服。除此之外还有别的人物软件吗？不求操作难易，做这个没有三两下出成品的。但还是很希望有大师简单教授基础技法。不奢望短期内能像各位高手一样有精彩作品呈现，但总能慢慢熟练，最后将很多想法以这种形式问世。我确实是很想学会这一块。

建模

    3dsmax人物一般，场景不错（建筑物建模好用）
          maya人物建模好用，当然也可以做场景
          如果可以的话当然用z-brush雕刻人物了（最为精确）
          c4d制作场景，动画，渲染都不错（而且使用最好上手，软件很有逻辑性，亲测），现在常作为影视后期
          lumion可以快速出效果，算是很方便快捷的软件，且自带gpu快速渲染。（虽然多是为场景效果而使用）

         渲染         

模型做好了的话，就是苦逼的渲染，渲染的话不是很清楚，但是一般用vray吧（基于什么软件就看自己的了，vray是一款渲染插件），渲染需要调整质，光影等等，还有就是非常的吃cpu！！！（是个大坑）

     如果是作图的话，ps是不可缺少的东西。
     如果是视频，那就顺便请入ae深坑吧。
