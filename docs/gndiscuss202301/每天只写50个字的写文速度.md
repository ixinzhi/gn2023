# 1

*本帖最後由 西呱 於 2020-12-23 21:10 編輯*

最近在写新的文，每天50字左右，这还是有大佬帮忙的情况下
感觉自己如同扶不起来的烂泥，除非自然风干变成泥乾糊墙上一样的不堪
但是这次的灵感真的很棒棒，是自我感觉良好的自嗨文，每天嗨到不行的那种
很少有用整时间写文，大多是每天早晨写个2分钟，或者和大佬聊天的时候写那么一点
有什么破局方法吗，一下就提高产量的那种，感觉一直咸鱼不是个事，我还是想普天同嗨的
目前进度3600字了
而且还有一个问题就是碎片化，写的都是很零散的，有很多是接近结局的内容，可以说是在跳着写一篇好几万字的文

啊是的，构思大纲和世界观的时候是最爽的，等到了细节就变得无聊了，尤其是文章一长，简直要人老命。
如果脑子里有具体的场景但是不想写的话，那就只是单纯的懒罢了。这时候想想读者们期待的小眼神就好(*❦ω❦)
如果脑子里空空如也的话……嗯……我不太清楚别人都是怎么解决的，因为我平时是个看书看动画看电影比较多的人，碰到一个场景后我还没开写，那些场景就会以动画或电影般的情景在我脑子里自动演一遍，我只需要描写出来就好了。
而且我是个特别以写小说为乐趣的人，平时几乎所有的类似于走路或坐车等不能看手机的时间里，我脑子里都会去想这些东西。所以如果你的问题是脑子里不肯演的话……专门抽出一个小时来，专门去构思，想到什么了就随手记下来，过后再进行详细扩写。
