# 1

我們知道，大部分生物都具有被動或主動去學習的能力，特別在困境下，這個能力更強。
而人類天生具有比動物更加強大的學習能力。
那麼，當不處於困境時進行的非主動學習，是否是一種本能的學習？ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 2

      楼主又想到什么很独特的点子了吗。事实上我并不认同人类的学习能力强，身体上的能力自然是已经停止进化（学习）了；而知识只是因为结构问题，或者说没有相同的智能生物等条件类比，所以才觉得强。
      此外，我觉得楼主讨论的问题答案是“是”，这应该是一种本能的学习。

# 3

处于困境中的非主动学习应该属于本能吧。
就像为什么猴子搞得学人那样，原则上说不清动机（有趣？讨食？），但是事实上确实有这样的各种好处。
但感觉问样的问题更像是进化，被迫学习象征着某项方面的改变或是行为习惯的养成过程…… <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 6

如果「學習=嘗試解決困境」這個前提成立的話，那就是看你對「困境」的定義
有些人覺得不會死的都不算困境，有些人只要情緒會受影響就覺得是該解決的問題了
所以，因為對「困境」的定義落差，A認為不是困境的情況很可能也能促進B的學習

實際上人學習是沒在考慮算不算困境的，所以你還是給個具體的例子吧，不然根本只是在玩文字遊戲...... <title>7</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
