# 1

就是你写了一篇文章或者做了（剪辑或制作）一部片子，然后又在上面花了跟多功夫去斟酌修改，精益求精，力求完美，然后当你好不容易完成它，想要拿出来和大家分享时……它却因为种种原因（磁盘烧毁、或者存在U盘里不小心丢了之类的）再也找不到了……留下的只有你对它记忆……

因为当你完成它时，会有一种如释重负的感觉（不知道大家有没有，反正我有），这种感觉会造成你对它的记忆往往是很模糊的，而当它刚刚丢失，你还稍微清晰的记得某些细节的时候，你的心情却是很沮丧的，并不想再重复的去写一遍或者做一遍，所以当多日后，你又要想办法去再次重现它时，才发现那几乎是不可能的，在你的记忆中，总觉得之前的那一次是不可被超越的，于是总是幻想奇迹会发生，走在大街上能捡到你丢失的U盘，被烧毁的磁盘能奇迹般的再读取一次……

啰啰嗦嗦说这些话就是发发牢骚，最近被类似的事情搞得心烦意乱，很想把之前写的几篇东西一字不差的复原出来，但是又做不到，因为大概是几个月之前的事了，现在连当时文章里的剧情顺序也记不清了，更别提细节了，好烦心……

最后顺便再问一下，虽然已经在各个问题网站上问了很多遍了：手机的SD卡貌似是被烧毁（外表没有变化，金手指也正常），现在已经读取不能（根本连识别都识别不了，就是说插在手机上和读卡器上都没有反应），请问一下里面存的东西还有可能能想办法找回来吗？ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
