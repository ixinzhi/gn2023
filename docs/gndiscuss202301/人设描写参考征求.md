# 5

我觉得性格可以用一些行为、语气上的细节描写凸显，而外貌一般只能从发型、衣服着手，因为好看的五官一般都差不多，如果有对比五官可以描写得详细一些，例如：眼睛比XX稍大。其实外貌只要有至少一样能代表角色的特征就可以了，只要提到该特征，读者就会知道说的是那一个角色，这样不但能给人深刻的印象，还能凸显的个性。就算性格、外貌（多胞胎）差不多，只要装扮、身材不同，给人的印象自然会有所不同。就我来看，能写四种性格的女性角色已经很利害了，比我这种连词汇量也比别人少，只能依靠度娘的好多了。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

天然呆，傲娇，甚至阿库娅（形容词）都可以试试

各种play就可以是无意中发生的
甚至男主变小/女主变大是莫名其妙的发生
这样就更有神秘感，也多了一条分线（寻找变小/大的原因），就有事可写

性格描写就可以从变小/大前通过一些小事衬托出来
也可以通过描写衣服衬托性格
比如阿库娅（形容词）
就可以一脚踩到香蕉皮——滑倒
