# 17

写任何事情不要碰政治，以现实中的我国为背景，只要出现残忍系内容，一定会涉及到政治（因为我国军/警性质的原因，与之相关必然触及政治红线），因为残忍系无论如何必然涉及犯罪，写到这一步触及政治无可避免。

所以我建议不要出现国名，不要出现地名。

背景完全可以以现实为背景，但是不要出现真实的专有称呼。

其实涉及政治不会有法律问题，主要是读者心理是否接受。干脆把专有名词换掉，和文章本身没有任何关系，所有人都开心，何乐而不为。
