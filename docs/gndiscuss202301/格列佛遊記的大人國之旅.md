# 3

哇，感謝原PO讓我重溫當年看見這些影片的感覺。格列佛的故事還真的是很普遍的兒童讀物，又有繪本又有影片，恐怕就跟「愛麗絲夢遊仙境」一樣，是最初讓人產生GTS幻想的媒體吧？

不過原PO對這些影片的細節記得還真是清楚，我幾乎已經把大綱以外的東西都忘光了。唯一還有印象的一幕也只剩下巨人國的少女把格列佛連人帶小船一起捧在手上並開始跑步。
