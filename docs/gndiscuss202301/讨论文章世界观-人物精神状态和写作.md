# 1

论坛里的文章有时候都不会去深究文章当前世界观当地会对文章的角色会有什么样的影响，因为文章的主要目的都是娱乐，次要目的是为了让人能够有性的刺激。又或者是发表作者的对某件或多件事情的看法，并且希望观众能够认可他们的理念。很少有彻底探讨到底世界观会对人物心理的详细描写，因为人的思维是复杂的，文章里的人物的性格很容易就能推测出来，并没有那么的写实和复杂。文章里的人物心里和行为往往都很有具有论坛文章人物的特色，可以被清晰的分为几个大类，人物的成长有些时候较快，有些时候较慢，有些时候甚至根本没有成长，导致说文章的剧情很快就会被读者所厌倦。
现实的世界会比文章里的世界更加不可预测，论坛文章里的世界往往会有很清晰的表面逻辑，或者是一眼看上去就很合理的剧情，基本上不会有太过非常规的东西，而现实世界往往是很荒谬的，需要深究才能发现多个人物与事件之间的内在联系或者他的合理性，而不是很迅速的就能看出非常合理的逻辑。

我可能还是要求太高了，以后说不定自产自销的时候写的文章，还不如我口中的论坛文章![](img/f86b7f0892ce62ea45ff58e417f53dfa.png)。

事实就是大伙儿大多数是来手冲的，然后相当一部分作者写文是欲望发作了，再然后那些想写剧情的作者很容易面临一个问题就是我写这个剧情想表达这个故事么问题，但是加入gts这个元素的意义在哪儿？然后就会发现这篇文章写一个普通的科幻/奇幻/蛋疼爱情小说好像也没什么问题，然后就不想写了。
至于牛的那当然也有，就是围绕gts这个元素本身来去发散剧情（甚至朝着合理不尴尬的情色方面去写），但有个这个想法的作者不能说很多，又具有这个能力的更不多，还有时间足够完成写出来的就更更不多（就连站长大人的dgu也是海量坑没有填）。毕竟文字（都不说文学了）的创作还是需要一点想象力和灵感的。 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 4

事实上，论坛里所写的，或者说世界上大多数gts文章，都是我们认识的爽文这一类别，这样的文章实际上并不需要太多逻辑支撑，它只需要去不断地刺激感官就好，这是这类文章让读者读下去的动力，如果楼主想写的更有逻辑，那么这些很多没来由的刺激会减少，那么吸引读者的点就很难写，支撑文章的矛盾，悬念，至少对于我来说是难以想出的。而且现在也是快节奏的时代，我也已经很久没有静下心来读完书了，大家都在追求这种快速的刺激，这样写也挺符合大众的，忙了一天，晚上回家，还是有点刺激更能使人开心 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 6

不过楼主说的我觉得有一天特合理但同时也有不合理的地方，就是真实感。
说合理是因为，即使纯纯的爽文，真实感也是很必要的，因为文字完全没有任何直观的信息，完全是是靠读者动用自己的脑子里的记忆来形成画面yy的，比如你写丝袜那我脑子里就开始相信丝袜啥样子，然后哎呦我操好性感冲冲冲。
但是如果真实感有很大欠缺，冲都冲不爽，我觉得这点对上论坛时间比较久或者写文时间比较长的坛友比较明显，一来是gts的描写方式撑死也就那种样子，看多了容易脱敏，比如我经常觉得有些问题一千倍的压迫感尚不如有些文30多米高的压迫感强，而且由于gts巨大的体型差距，有些大家司空见惯的敲断会显得很尴尬，比如超大号gts因为身上有小人什么的突然好激动好高潮水流一地，或者突然就开始自慰。（我自己看的时候就会想，啊，为啥啊，为啥突然就开始脱裤子了啊？）就像做爱需要前戏，哪怕爽文也要铺垫，有时候gts和小人互动时候的不真实感让人容易突然不想冲了。

但我又看了看楼主，楼主认为论坛里的文章逻辑清晰围绕某个脉络，现实世界多线联系错综复杂经常明暗之间规律不清错综复杂。问题是能写的和现实世界一样，莫说在这个论坛，就算在世界文坛，也算的上比较稀有的能力吧。。。而且gts本身幻想成分大，基石可能就不太真实其实。 <title>7</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
