# 1

作为一个在GTS界内混了很久的老人，想阐述一点理论性的东西，在此供大家参考，希望能抛砖引玉。

考虑到目前我们华人GTS界内的能力，以下举例暂且限定在文字类和图片类。（当然，其他类型的作品或资源也可以代入）

—————————————————————————————————————————————————————————

要素一：美

“美”是每一个人都在追求的，而在我们圈内，狭隘的概念就是GTS得是个美女，广义的概念也包括文笔优美、语句的通顺流畅、图像唯美、色泽搭配等。
我想大部分GTS同好在这一要素上的态度都是一致的（欧美风重口味除外）。

首先，要做到这个“美”就是一件不容易的事情。

对文字类作品来说，女主角如果只是一笔带过“一个美女”或者“长得很像某某女星”，甚至干脆就用“某某女星”，基本上就可以断定为快餐YY文了。

真正认真写文的人会把自己心目中的女主角由内而外细致的描绘出来，外表如样貌、穿着甚至头发的颜色，内在的性格、脾气到心态的变化都能面面俱到，能写出这样文章的人才是真正努力过、思考过、酝酿过的作者。写文章就像酿一壶酒，越陈越香。

图片类的作者在这一要素上则相对容易一些，合成图关键在于素材的选择，3D图主要是模型的选择，手绘图就要看作者有无绘画功底了，有功底的人想画出美女不是难事，没有功底的就算是照着线条描也很难达到理想的效果。

要素二：真实性

这里的“真实性”主要目的就是为了达到“身临其境”的效果。毕竟GTS本来就是幻想的产物，若毫无真实性可言，则观众的感受要大打折扣。

真实性包含两部分内容：主观感受和客观效应。

人有五种感官：视觉、听觉、味觉、嗅觉、触觉，外加一个更重要的心理感受。通过对这些感官和心理的细致描绘可以很大幅度的提升作品的质量。

大部分快餐YY文往往都会忽略这些感官感受和内心描绘，单纯的以“第三人称”的视角去叙述主人公事情的经过，就像是流水账。作者创作的时候都没有全身心的投入其中，自然不可能写出拥有真实感受的文章。

客观效应主要是指尽可能的让幻想的故事情节符合情理，同时还要注意自然效应。
能上天入地的不死小强往往就是快餐YY文的主角，十几米高的GTS就能徒手毁坏大楼赤脚踩扁坦克车也让人觉得十分可笑。

对于作图者来说：
合成图要做到真实性主要依赖素材的质量，原作者要尽可能的选择合适的素材，选材上尽量真实自然不做作，技术上尽量天衣无缝融为一体。
那么平日就要有意识的搜集大量的素材，同时要掌握一些基本的PS技术。

3D图的自由度和真实性都是最高的（但是能做的人比较少，主要是电脑硬件和渲染时间的限制），需要注意的是镜头的捕捉（尤其是第一人称POV镜头），以及3D故事情节的流畅性。

手绘图自由度最大，但是要做到真实性难度也是最大的，需要原作者有深厚的绘画功底、立体透视能力和镜头捕捉能力。

要素三：互动内容

GTS同好和非GTS同好作品的最大区别就在于此了。

充分的互动内容构成了所有GTS作品的主要内容。相信在这一点上，每个同好心中都有无数的想法。我们之所以去创作就是为了把自己心中理想的情节展现出来，与大家分享自己的快乐。

在这一要素上，我们所要追求的就是互动的“充分”和“到位”。

“充分”讲究的是多样性，避免让观众感觉生硬和乏味，单一而重复的踩踩踩的动作并不能为你的作品增添更多的魅力，相反会让人联想到强迫症和偏执狂（或者是一个由男性在GTS空空的大脑中驾驶的巨大人形机器人）。换句话说，一个真实的充满活力和各种想法的女孩子也不会一直跟你玩同样花样的“游戏”。

“到位”讲究的不是直入主题，关键在于氛围的渲染和前期的铺垫，而这一点也是快餐文和精品文最大的区别之一。事实上，做好“氛围的渲染和前期的铺垫”比一上来就“踩踏”或“插入”能带给人更多的感受。这，就是“到位”。

而图片类的作品作为一个瞬间的定格，则相对容易一些，原作者需要注意的便是这一瞬间的定格如何尽量的做到饱满和贴切，能充分展现互动的魅力。

我相信，GTS不单单是用来YY和SY的，完全可以上升到更高层次的艺术性。（成人影片也可以很艺术啊）

最后，对于原创作者还是一句话：细节决定成败，刻画好细节就是成功的一半。

———————————————————————————————————————————————————————————

以上三大要素，不论是否是GTS圈内人士的作品：
能满足其中任意一项的，属于“沾边”的GTS作品。
能满足其中任意两项的，属于“合格”的GTS作品。
能满足所有三项的，属于“优秀”的GTS作品。

其实我还有一条理论是3+1的：三大基本要素+一个特殊要素。
特殊要素是指“每个人不同的兴趣和口味”，满足所有3+1要素的GTS作品，对于各人来讲就属于“极品”。

以上，供大家参考，希望能抛砖引玉，谢谢！

# 2

以下是我个人依据以上理论的吐槽：

GTS圈外人士的作品（比如广告、电影、插画、宣传品之类）：
第一要素和第二要素做的比较好，唯一遗憾的就是第三要素，很多时候都有骚到痒处却得不到满足的痛苦。
还有的就是为了保证“真实性”而不得不牺牲掉“互动内容”（比如《LAND OF THE GIANTS》这部美国电视剧）。

但是我还是会留意去搜集的，毕竟前两个要素比圈内人士要专业的多，giantesscity上的webfind开头的帖子我通常也都会点进去看的，一不小心也能挖掘到一些好料。

很多日本的动漫作品也有巨大化或缩小的内容，但往往也都是点到为止，或一闪而过，没有充分的互动情节，这是一大遗憾。
最让我不满的是奥特曼系列的作品，为什么怪兽都这么丑，变成美女也有魅惑功效啊，而且最关键的是没有互动呀！

GTS圈内人士的作品：
只要是有心去做精品的，第三要素往往都能做得不错，但是前两个要素还非常的不成熟。
（可是某先生的影片这一点做得不够“充分”和“到位”，具体可以对照以上我所说的“第三要素”）

图片类：
日本某画家和DA上的某画家，作品内容都还是很不错的，有充实而多样的互动内容，但是最让人遗憾的是他们没有手绘的功底，日本那位GTS画家作品产量很大，但是进步甚微，DA上的那位GTS画家经常照着别人的作品描绘（我都能看出来），但是仍然没有达到理想的效果。

不过不管怎样，我相信他们都已经很努力了，比那些灌水者和伸手党都要强，毕竟他们都能把自己的喜好通过创作的作品与大家分享，这一点就非常的不容易。
（顺便说下，日本那位产量很大的GTS画家的原创角色有在《缩小学园》中客串，我想这也是对他努力成果的一种肯定！）

某著名的华人GTS画家（不知道他人现在何处，反正是很有些年头的老人了，控巨乳），有较强的绘画功底，作品内容也非常有张力，很震撼、很暴力。但是最大的遗憾就是第一要素“美”做得不够到位，很多女主角的镜头感觉像男人（我是这么觉得啦），也许是每个人的审美观不同吧，其实这种写实的风格也可以很唯美啊，论坛里的nyfj118兄在这一点上就做得很好。

关于做商业GTS影片的： 女主角就是一个很大的问题，而这个前提必须是有足够的受众和市场需求，考虑到我们小众群体的人数，不知道何时才能等到大发展的那一天。（其实人数什么的也是可以推算出来的，算法可以参照科学家寻找外星智慧生命的公式，需要大量的统计数据，以后有机会可以做做看）

第二要素“真实性”主要还是在于技术，随着电脑3D技术的发展和圈内技术宅的增多，在这一点上对未来还是很有信心的，或许未来我们更多的还是要依赖虚拟的视觉感官效果。

先啰嗦这么多，以后有想到的再补上。

# 14

今天看到XNR朋友发表的一番见地本人很有感触，这么长时间以来很少有人如此系统地总结当前GTS作品之中存在的问题以及详细归纳，在此表示热烈支持。我们每个同好都能如此诚恳、客观、科学地看待问题，而不是为了鸡毛蒜皮争执不休，我们这个论坛将非常健康美好！借着XNR的理论我继续提炼一下自己的观念，希望与大家分享并给予指正。
首先，这个三要素总结的很好，关于【美】这个概念确实因人而异，无法用一个定义来统一观念。尽管这是个见仁见智的问题，不过我们大家仍然可以有一个共性来界定美的标准，那就是一个事物无论怎样让人看起来非常舒服、使人心情愉快，那这就是美好的。如果一个形象让人看起来很倒胃口、心里难受，那么不可能认定为【美】(至少在大多数眼里)。由此看来，一个美丽女子无论相貌可以千差万别，只要她的外观端庄协调、符合自然规律、整体和谐，就有美感。因为大自然的鬼斧神工刻画出来的风景才是最美的，人工合成永远无法取代浑然天成。
其次，在作品创作中我们如何把握，这就是看每个创作者的艺术功力了。对于图画来说相对简单，只要表现一个动作或状态就可以了，而对于文字创作则需要投入比较多的精力去思考故事情节，小说情节中要符合逻辑关系，前后一定要对称，否则就形成一锅粥前言不搭后语。正如XNR所述我们很多人创作时没有刻画出任务心理活动和思维状态，只顾着破坏和杀戮，这样最后就成了网络游戏。作品内容也要符合自然，这点很重要，你们可以想想一个作品如果所有情节违背自然规律读者就会很难受，比如非要人吃屎或者违反生理做事那样内容看多了就会让我们逐渐厌恶这种爱好，久而久之会使我们淡忘原先的美好感觉。
第三，关于互动性，这个全凭每个人思维能力的控制，你对真实世界观察的多，在作品中写的自然真实，如果你没有设身处地实践过类似的情景，那是无法想象出这样一种互动游戏的过程，即使写出来也极为不真实，这点我们必须遵循自然界物理规律原则，如果乱写必然出错。你不能描写一个比你高大数十倍的女巨人体重却比你轻许多，那不成棉花了么？这里一个很重要的原则就是角色定位准确，什么类型的女巨人会相应做出哪些事情，她不能超出本属的范围，你非要一个身高百米女巨人去过正常人类生活，与普通男子性交，那么她永远无法如愿；再譬如一个超大体型女巨人(数公里高)强行在人类城市中来回闲逛甚至来杀戮，这根本就不是那样的女巨人应该做的事情，即使强行安排这样的场景也是极为别扭的感觉。像但丁的作品就是这样，尽管他把故事描绘的十分精彩，可以说惟妙惟肖生动十足，还是觉得故事有些畸形难以消化，而且他翻译的故事内容超长，千篇一律的写作特点让人很难坚持读完，我一般持续阅读一个章节之后便跳跃着浏览。
最后，我要谈论一下核心问题，真正制约我们创作GTS作品的因素是什么？我想这个问题要涉及到我们心中潜意识的层面了。其实我们可以这样思考，我们究竟为什么对GTS形象情有独钟？我不完全赞同有人分析的年幼时受到母亲严厉管教及外界刺激，我个人认为GTS的兴趣本身是一种美好的情节，是人类一种自然淳朴的情感，也许是基于对母亲的崇拜或对某位美丽女子的追求，有些是心中的臆想，但归根结底应该是我们每个人心中对美好事情的向往。不过由于不同的人在意念中最终表现形式不同，而我们这些特殊爱好群体共同选择了【美丽女巨人】这样的终极形象，实际上她代表着我们所有人心中最底层的愿望。为什么我们看见一个美丽姑娘手里抓握这小人儿感觉是那么舒服和自然，看到一个男巨人(譬如金刚)掌控着小女人是那么难以接受，这是我们心中底层潜意识对女人弱小的同情和对美好事物的认定造成的。我上初中时开始接触色情片，当时对于那样情节感到非常刺激的，但是观后又在为片中女子的遭遇感觉扼腕而难过，尤其是那么美丽的女子不应该受到那样的待遇，因此在心理层面的表象中就为这样的女子打抱不平，可是我心中当时又渴望得到色情片这样的刺激，后来我又看到老电影【金刚】那样的场面，于是我就在表意识中把这样情节的角色颠倒过来，让美丽的女子做强做大，开始是一个女霸王后来成为女巨人，就这样逐渐形成了我心中的模式。可是后来我才发现原来自己内心深处的意愿就是要公平合理地对社会分配，在这里就是要合理安排角色位置，不要让美丽的事物备受摧残，要把丑陋的东西埋葬起来，最后又形成了等级观念，让越美丽的越高大，越丑陋的越渺小。所以话说回来，我们的GTS情节就是要满足心中的愿望，而非在乎那些吞噬与杀戮，如果我们在创作中总是不断堆积那些非常消极、残酷和令人难受的情节，最终就背离我们心中的原始愿望，逐渐地我们将脱离原先热衷的这份GTS情节。何去何从希望每个人心中都有数！/font] <title>15</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
