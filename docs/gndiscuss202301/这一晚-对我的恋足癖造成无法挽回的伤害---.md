# 1

当然我首先是GTS爱好者，同时也是恋足爱好者。这几天闲着没多少事，决定去一个群里一直在推荐的会所去尝试一下被女人踩的感觉。
犹豫了3天，我拨通了那个号码。号码中的女人让我决定好了到指定地点再打电话约。于是又过了一天，也就是前天。我到了指定的地点打了过去。然后对方首先一再确定我是要什么，她们是干什么的，可能担心我是Pol ice之类的吧。然后告诉我从某个小门进去上多少楼。
上楼的时候我还在犹豫，毕竟幻想归幻想，真正到了这种时候还是有些尴尬。
到了那个门前，其实也就是一般的居民楼，结果刚刚到门就开了……一个大概30来岁化妆妖艳的女人让我快点进去。然后到了客厅那女的就开门见山了。“我，她，和她，选哪个。”客厅除了跟我对话的女人，还坐着一个抽烟的女人和一个穿黑丝的看上去较为年轻的女人。我觉得跟我说话的这个有点年纪，又不喜欢抽烟的女人，加上我比较喜欢黑丝，就选择了那个年轻的女人。长得也还算不错。
那个黑丝女孩就把我带到一个卧室，让我把空调打开，问好我的套餐项目。我选择的是丝袜恋足。付完费。就叫我脱衣服，然后就开始一口一个宝贝叫得很亲密……
脱完衣服后叫我躺在床上，她脱掉高跟鞋，上床就往我脸上一踩。其实最开始的那一下感觉还是很不错，毕竟从来没有经历过而且YY了这么久的被美女踩在脚下，虽然没有被缩小，但是感觉还是很不错。味道不大，有一点淡淡的味道。
然而噩梦开始了，当我闻到她右脚前脚掌时，她叫我伸出舌头舔，舔了几下，感觉口干并不是想象那样好，就跟舔一块布一样乏味，最致命的是，舔了几下后发现她脚下有一股像是jing子臭掉的那种味道。瞬间变得恶心起来，甚至反胃，因为我认为她之前有踩过其他男人 ，也许其他男人出来后留在了袜子上。也许袜子都没有洗就来踩我。瞬间没了心情，叫她换一双新的黑丝袜。
她问宝贝为什么，觉得她味道太大了不喜欢？我说你脚上有精味儿。她说那不是精味儿就是正常的脚出汗的味道。但是我不信。坚持让她换。
结果她换了之后，发现那种味道仍然存在。之后还有丝袜套头等等一系列服务我都是勉强做完的毕竟交了钱。
完成后，我发现以前的幻想到了现实真的很糟糕。除了第一下被踩的瞬间。
再往后这几天每次看到GTS的脚后就会想起那一晚的遭遇，那种恶心的味道。。。
往后我该怎么办？
