# 1

熟悉的木屋，熟悉的人物，熟悉的音乐
少年与少女，再次回到了一切开始的地方

作为我们论坛第一款真正意义上的游戏大作，【外传】当然有许多缺点，比如战斗太过简单，比如逃跑太过困难，比如魔兽森林太过复杂
更有人认为GTS含量太少

但是，这是英雄传说，这是空之轨迹
是我们美好的回忆，是我们再也难以握在手中的老时光
所以，如果只是为了打飞机，请不要下载，这是对自己的折磨，也是对游戏制作者的不尊重

空轨为何会成为超强大作？
——因为它的细腻，它的流畅，它对每一个细节近乎偏执的追求，它对每个角色不遗余力地刻画
很欣慰，【外传】中我看到了这一点

开场，那段熟悉的星之所在，那充满了“空轨感”的推理，让我忘记了自己面对的只是一款用RPGMAKER制作的同人，产生了时间回溯的错觉
回到了FC的那一年

从布莱特家到洛连特，再到马鲁加山道，我看到了满地藏着的【荧光菇】，袭击工人的小岩蟹，以及永远插在路口的告示牌

威尔特桥上的士兵依然懒散，下水道的入口同样存在，总是会有大型魔兽霸占着道路来给我们送钱和BP值......

SC开场时凯文不怀好意尾行小艾孤男寡女共处木屋的邪恶行为终于被揭穿，怪盗B始终如一地辛苦奔忙，为推动主线绞尽脑汁

从主角到NPC，每个人的台词都是经得起推敲的，简简单单几句话，体现出的是各自的性格习惯，不做作，不矫情，平淡如水，流入我们的心扉

细节出彩，主线更是不俗

这个外传故事真正平稳地衔接了原著的结尾，展开了新的故事，玲的去向？她与怪盗在一起吗？小约如何恢复原状？结社是不是知道现在发生了什么？这都是一个个悬念，让人迫不及待想要知道结局。

说实在的，这短短的体验版，就剧情而言，实在胜过太多原创文章了！

而隐藏剧情中基尔巴特大爷的华丽登场令人为之振奋，不愧是最伟大的生存系猎人！永远能在主角无法到达的地方出现，永远能靠着人品脱离任何危险！这位才是隐藏大BOSS吧！！！！

向游戏制作者们致敬！！！！！这是我们自己的游戏！！！！！！！！！ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

不过，请大家不要高兴得太早，因为只是试玩版而已。真正的正式版出来才是值得高兴的时刻。
前几天兔子想帮我编剧，从我们讨论的这几天看来，发现制作团队之间的磨合确实困难。这在xnr发起的第一个游戏制作计划的时候已经得到了验证。
看了同好们写的评论，顺便也了解了我所不知道的《永无乡之梦》的历史……和我们的境遇是多么相似呵！我们比他们少的，只是那个骂游戏的家伙而已！
看来如果想顺利合作，还是如我之前说过的：必须要真人见面，在网络上合作实在困难。

我打算开个像LuLu那样的网络交易平台，也是为了能有时间来做这个正式版，现在已经开始画柏斯地图。如果大家真想帮我，就和我交流做网络生意的经验吧。这至少要比我做工薪的时间充裕，能让我有精力继续制作这部游戏。
目前……预计是一年时间完成这个正式版，这个体验版将在我确信100%能完成正式版的情况下在百度贴吧公开。 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
