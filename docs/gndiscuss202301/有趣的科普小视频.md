# 3

这个团队制作的系列视频从科普角度都挺不错的，不过为了动画效果，考究程度略有让步，不要太当真。至少在我比较专业的领域里，它的内容不能说完全错，只能说提出了一些新的适用范围很窄的经验规律
人的缩小这件事，硬要往科学上解释是解释不了的，如果说原子缩小了，那直接跟科学不挨边。如果说原子量减少了，那维持不了人类作为多细胞生物的存在本身。如果说原子间距缩小了，按逻辑我不该提这个的，但曾经看见一篇文真的这么解释，我有点好奇作者在哪上的学，重量是原子间距决定的？
视频里提到了呼吸和视觉问题，呼吸这条很好理解，都到原子核大小了还怎么吸入氧分子。视觉是人的眼球想实现人的视觉必须有这个大小，昆虫的复眼很小，视觉差到近乎没有
从放大的角度来讲，首先考虑一下质能守恒定律，不加入新的质量不能凭空放大。人类身体所使用的血肉材料，骨骼结构，韧带连接方式都是给人类当前的体型设计的，放大了会因为内部新陈代谢积热过高，直接热死。再放大都走不到这一步，重量就把放大的人和稀泥一样压烂了，这是被地球引力弄死了。再放大到地球尺寸呢，自己的引力杀死自己，向内塌陷成一个不可名状的“熔岩”球体
我写比较严谨的文时，给gts的设定是舰娘，其实是我不知道中文里怎么起名合适，本来想的词是spacecraft girl。舰娘的身体结构是给这么大的体积设计的，材料强度超过现实理论的上限，在维持巨大体型的同时，实际上也近乎跟三体的水滴一样刀枪不入了。因为结构本质的不同，体内产热量在静息状态下远远远远低于同体型的人类，才能让巨大化的体温维持在近似人类的程度。对于行动起来的发热，这是航天器散热方法的问题，我在文章里没有正面提及这个问题的答案，就当成在地球上靠空气对流散热，在太空靠引擎喷射吧。
空气阻力一直很大，逆风跑步就能明显感觉到，对F1赛车有爱好的人应该对它有不少了解。至于小人的逃跑，高科技外星人的战斗机不管是钢铁战斗机的外形，还是可爱的人形，作战效能一样，小人逃跑的结果就是一样的，被GTS盯上了还想着逃跑本身就没有任何意义。从残酷系表现来说，因为我本人比较偏温柔系，我觉得被巨大的鞋底压死和被导弹炸死是一样的，没有美感，GTS体现不出来S这个字母的女性特征，换成东风41导弹也是一个表现力。
