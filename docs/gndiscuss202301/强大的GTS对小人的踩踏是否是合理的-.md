# 1

这个问题我本来想仔细想想再发再问，但是这个问题其实也比较复杂不是一两句可以说清楚
不全面的说，先举个例子，如果有成百上千的婴儿被机器抚育大，完全由他们自行发展语言和文化，文明，那么在这样的一个实验性质的沙盘社会中，踩踏这一行为的社会概念究竟是什么样的？
我们可以分为3种例子进行考虑，第一种，踩踏被看作是种侮辱
第二种，踩踏被看作没有引申意义的动作
第三种，踩踏被看作是对对方的尊重而表达的礼貌行为
这时，我想到的是远古人类裸足生活，就像是纪录片里的求生演员那样，脚底是唯一很脏的地方，也像非洲那种贫穷部落那样，脚底是很脏的
那么，在生产力低下，社会财富贫乏的情况下，踩踏不可能是第三种尊重性质的行为，因此答案仅仅限制在了前两种，但是这一结论又是不严谨的，因为缺乏一个严谨的推理逻辑
就我本人的话，向往被女孩踩的初衷确实就是自卑感，但是这里的道理却没有在反思中得到一个结果，而且多年以来不断地困扰我
最近也有人说我在钓鱼炸鱼，我也把自己的想法表述了一下，说明我对这个问题严谨认真的研究态度，以及对人的心态和对行为的研究的极大兴趣，希望有大佬讲解，指点迷津

想個最簡單的問題：眼前有一隻螞蟻，最簡單的處死法是甚麼?
當然是踩死，最省事、最不費力，因為把腳挪到螞蟻上面放下是物理上最快速有效的，這是人體構造決定的
這種舉動放出來的訊息是「連多花點力氣來虐待你都嫌懶」
就像電影裡有些傢伙會把槍橫著拿，用來表現「我根本懶得好好瞄準你」，跩爆
同樣的邏輯放到「嘲諷」上也一樣，拿腳摩擦挑逗是最不花力氣的
一個很懶的人會想只用腳把鞋穿好，會想用腳把遙控器或橡皮擦拉到身邊，把對象換成生物就代表了徹底的藐視不是嗎?
對GTS來說，光是「把抱在胸前的手鬆開」就是何等的勞駕了啊，
所以寫到「羞辱」時用踩的比其他動作都還合理，因為最省力(也可能依據當時姿勢而改變)
如果想要讓其他動作的羞辱可以保持GTS的威嚴感和壓迫感，可以讓她蹲下或坐在地上，
這麼一來「伸手」也會變成小動作的事情(不用彎腰→做完動作把腰挺直)，可以繼續延續之前的「藐視」邏輯
要特別注意的是「抬起來」這個舉動，要GTS花費力氣把手違抗地心引力抬起來是個非常重視小人的行為，
用這種角度來講，拿到嘴巴前舔的羞辱是遠遠不及拿到跨下摩擦的，單純是物理耗能考量來說

如果想理解「用最小動作完成事情」是甚麼概念，去健身房一趟吧，隔天早上一定會有靈感 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

感觉楼主说的这几种情况都有可能...
先不说gts，现实中腿部就是力量很大的地方，正常男性单靠臂力也很难抗衡女性的腿脚。只是因为通过腿部的攻击会限制自身的灵活性(重心不稳等)，起冲突时才往往不将腿作为第一选择，这并不会掩盖腿部力量之大。而踩踏就相当于直接把对方纳入自己大威力武器的射程范围里。所以把人踩在脚下就和把刀架在脖子上的感觉差不多，只是少了一些致命感，多了些羞辱感（具体同5L）。例子就是可以想象一个小人踩在gts的头顶，因为小人的力量太小，不构成威胁，自然谈不上侮辱了。所以踩踏的侮辱是建立在力量基础之上的。
而这种威胁的动作往往和实际效用相联系。在远古，即使在现在，侮辱行为的目的无非就是领地范围和食物等资源的掠夺。包括侮辱他人带来的快感也是从这些实在的东西里得到的。所以如果资源足够丰富，gts的欲望不需小人就都能得到满足，小人不会给gts带来必需的东西，有没有小人无关紧要，那么这就可以看作没什么意义的动作（第二种），被踩住就像开车遇到红灯一样最多发几句牢骚就过去了。2.而如果资源有限，gts需要小人的服务，那么踩踏作为一种显示力量的象征，就可以看作侮辱了（第一种）。
关于第三种..如果小人需要gts的保护，而大部分gts将小人看作累赘那种麻烦的东西的话。那么踩踏也有可能被引申为“我（gts）需要你”，这时踩踏对小人来说应该就算是礼节（恩宠？）了吧...
