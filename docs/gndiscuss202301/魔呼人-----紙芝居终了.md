# 5

既然是屬於 神社 的 縮小女 區 ,我就大略翻譯一下吧

好像是作者發生了什麼情況(不清楚狀況)而這個情況又處理不好,然後因為無力感跟挫折感讓作者疲倦了,喪失創作意念,作者選擇隱退,然後決定封閉這個網站

然後 把作品全部都散出給各位下載...

雖然這個作者實在不怎麼熟,每次也只有 神社 上標示著日期而去觀看而已...

也沒去他的BBS或是獨言看他說話,總而言之對他的愛算是比較偏少的...

部分是因為 他主要的內容 都是吞食,雖然歸類在 縮小女 區,但說實在的GTS吞食畫面還比較多,排泄跟嘔吐以及在嘔吐物裡面被腐蝕掉等重口屬性都有...

然後精靈只有到後面才分男女,前面一概看都像是個小男孩精靈...

雖然精靈有縮小粉,不過都用在精靈自己身上,或是人類身上,用在人類身上也只有把人類縮小到精靈大小,對這位 畫師 的期盼就是希望那些老是被吞掉的精靈把人類們縮小到比他們小然後吞掉...

一口氣看完第15話...深深的感覺到被腰斬的感覺,原本無論怎麼被吃怎麼被吞,都可以順著大便排出來的 精靈,就這麼ENDING了... <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
