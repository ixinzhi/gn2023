# 15

人的欲望是复杂的，就连试图理解欲望本身都可以被所谓“求知欲”所解释
历史，文化之所以可以追究是因为我们能通过历史唯物过程看到一件事物明显的运动轨迹
比如说，两片面包，夹上芝士火腿，就变成了三明治
但是我们却很难去寻找gts的历史渊源，我们难以看到xp所规模性形成的历史过程

单纯的使用符号学试图划线是徒劳的，符号终究是我们抽象提取出来的非运动性标签，而不能以辩证的存在角度来看待一个事物

我们能理解的只不过是事物的【此在】，也就是达到事物本身的方法，这就是【到那事物本身去】

而事物的【原由】则完全可以是无限复杂的，所以观察事物的基础只能建立在【ontology】之上

所以当我们把这一切都带回现实中来，以辩证唯物主义的基础做出实践，我们就会发现
本质上检验对GTS理论的研究就是通过

发展GTS艺术的生产力！
