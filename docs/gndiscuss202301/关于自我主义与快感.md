# 1

gts给我们带来快感，但是就我来说，自己思考自己如果处于那种被置于绝对弱势的境地中。自己会快乐多久。
有时候看文，遇到类似“后悔也没有用，你就一辈子做女生脚下的虫子吧”“你是我用远的足部用品”的句子，确实是兴奋点，当我切身遇到这种情况，我就不会那么兴奋了吧，是的我喜欢女生的足鞋袜，喜欢足的压力和气味，喜欢如此给我带来的屈辱感快感，但是如果一辈子都这样呢，真的这样，我会接受么？
所以，我自己可能真的会觉得90%恐惧10%兴奋。但是看到的是并非自己的男主，被罚做一辈子只能生活在足底，却是100%的屈辱感传达，感到兴奋。只想到了乐趣并没有风险呀！了解这个，也是能更多的写出真实感的来源吧。

# 8

恐惧源于求生（避免伤害）本能，而快感源于“痛感”（精神抑或心理）
两者此时会冲突，也会相合（恐惧属于精神痛感），虽然并不知道最后的结果，但求生的成功也可以带给人莫大的快感，
只是如果只是单纯的奴役而无危险时，你感到的应该只是一种压抑的痛苦（常见于现代社会的社畜身上）
以上仅为我个人想法
