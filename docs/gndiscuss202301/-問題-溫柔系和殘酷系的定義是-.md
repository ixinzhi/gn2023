# 6

這個跟你自己的感覺比較有關呢。

以單純的外在表現來說，那的確就像3樓步兄說的，以巨大少女對待人類的行為來作劃分。

把小人當朋友的、重視小人生命的、不管什麼遊戲總之到了最後都沒人死的，算是溫柔系。
而用言語污辱小人、把小人當玩具或寵物、到了故事的最後總是有人掛掉的，算是殘酷系。

不過你自己的感覺也很重要。如果是討論自己喜歡的是溫柔、還是殘酷的巨大少女的話，也許就不能用這樣簡易的方式來分類了。
畢竟這圈子裡也有很大一部分人是被欺負但卻會覺得有快感的，對吧？
（更有不少像我這種看見別人被欺負而會覺得有快感的……囧）

當然啦，在不想多費唇舌解釋的情況下，粗略地分溫柔系與殘酷系就可以了啦。

--
其實以前有人提出來討論過，說這樣的二分法根本就不嚴謹，應該要以「行動」或「迷戀的身體部位」來區分才對。
不過這基本上是源自英語圈的分類而來的，溫柔系就是「Gentle」，殘酷系就是「Evil」或「Cruel」。
但實際上這只是一種最粗略的分法，至少日語圈就沒有這種說詞，而英語圈也並不是完全這樣分的。

基本上呢，溫柔系是「Gentle」這個是沒問題的；
但「Gentle」基本上只是整個廣大英語GTS圈裡的一個分類，英語GTS圈的主流是「破壞城市」、「踐踏小人」與「體型差下的性行為」……等各種符合「殘酷」這個字眼的行動。
因此也可以說，西方GTS文化圈的主流，完全是偏向殘酷系的。

（實際上呢，日本也是偏好「壞一點」的巨大少女，但比例沒這麼懸殊就是。至於我們…不知道∼∼）

--
