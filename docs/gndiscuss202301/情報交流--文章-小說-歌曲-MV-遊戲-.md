# 8

下面的算是个方式吗？
==========
我很早以前是这样做的：
第一步，先设置google搜索全部数据，然后用google搜索：giantess
第二步，进入google搜索出的网页、论坛、专卖店，然后记下这些站点的名字或者站点里提到的各种类型的关键词和影片、漫画、作者、系列等等的关键词
第三步，找到几个牛逼点儿的种子或者资源的特殊搜索工具站点，然后搜索那些收集好的关键词
第四步，筛选然后下载。。。。
第五步，共享出去，不管用什么方式什么论坛，然后收集回帖的善意的同好，设法进行联系，建立资源共享的圈子。
第六步，互通有无，再根据圈子里的兄弟们互相拓展圈子，与更多的共享者建立联系。

=========
这是我以前获取资源的方式，回想起来真TMD的崇拜自己，还是学生时代的事情呢。 <title>9</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
