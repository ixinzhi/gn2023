# 1

杀生院羁绊礼装
【掌中的楼阁】
此为令人难以想象到是此时之物的世外桃源。
研钵状的螺旋海沟。
如龙宫城一般埋藏在深不见底的大海。

『嘿嘿，欢迎来到我的寝宫。
　看样子，比起迷茫于苦乐之中的地面，
　你选择了沉溺于梦幻之中的地狱吧？』

如风，如梦，如空、
优美的声音响彻着。
对女人来说是狭小的闺房、
对蝼蚁来说是宽广梦幻的地下城。
已经没有回到地面的路可走了，也没有回去的必要了。

『来，抓住小拇指的指尖吧？
　让我抓着你，带你前往你所追求的梦幻
    那充满欢声笑语的楼阁去―――』

身心皆融化进梦幻的海市蜃楼中。
没有痛苦，也没有自由，
女人随心所欲地消耗着虫子的喜怒哀乐。

这太棒了，想再多待一会儿。
我被骗了，快放我出去。
真是可怕，想要更多。
求求你了，让我离开吧。

这里是一切都能被原谅的，手掌之间的极乐净土。

『―――没错。
　不管来多少，都只是蝼蚁的无力反抗。
　接下来，请成为我永无止境的无聊午睡中
　那一丢丢的刺激吧？』

娇声与绝望，都是同等无力的海中泡沫。
是实现女人愿望的，并非玩具的众生愿具。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
