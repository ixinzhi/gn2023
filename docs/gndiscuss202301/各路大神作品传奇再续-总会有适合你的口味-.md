# 5

序章
“我知道，我的能力是什么了！”郑然决然道。
“什么？是什么？能救我们吗？”殷龙似乎看到了希望的曙光。
“能不能救，我不知道，但最起码能改变现状，可以将不同时空的人进行转化移动，至于到另外的时空能不能变成常人，我也不是很清楚，但就我们现在的情况，我觉得我们可以搏一把！”郑然解释道。
殷龙略作迟疑，权衡利弊之下还是认同了这个方法，“那，现在就行动吧！”
就在郑然准备执行代码之时，一个苍老的声音突兀地出现在两人的耳中：
“停下！你们这样会导致时空的错乱的！”
然而，这代码，已经不可逆的执行了……
…………
…………
我躺在赵雪放内衣的抽屉里，早已失去了时间的概念，那天在大学女生更衣室里自己主动去变成赵雪内裤的情景还历历在目。
我记不清我的名字了，好像是叫周斌还是什么的，我到底仅仅是从人变成了一条内裤，还是仅仅是一条在梦中变成过人的内裤呢？
就好比庄周梦蝶：“昔者庄周梦为蝴蝶，栩栩然蝴蝶也，自喻适志与，不知周也。俄然觉，则蘧蘧然周也。不知周之梦为蝴蝶与，蝴蝶之梦为周与？周与蝴蝶，则必有分矣。此之谓物化。”
我现在已经不想去想那么多了，我感觉我就是属于赵雪的一条内裤，对于她的花瓣，是我这些年来面对最多的东西了，我感觉她才是我最亲密的朋友。
什么姜颖，什么人类，也许只是一场梦罢了。
忽然我感觉世界动了起来，外面的光也照了进来，接着赵雪就带着我进了浴室。
我又可以见到主人的花瓣了。
我被放置在窗台的铁架上，主人的胴体我早已看过了无数若干遍，如果我刚开始变我肯定会很有性趣的去欣赏，而现在，虽然仍是一道美丽的风景，但看到她的身体就和看到自己的身体一样，没什么感觉了。
于是我便看向了窗外的天空，隐约觉得天上暗云涌动，阴气逼人，天上居然有两轮圆月，一轮是暗淡的黄色，另一轮，则是妖异的血红。我记忆的某个部分似乎有所触动。
我似乎在无数年前曾经在别的时空变成过小人，那时候好像有姜氏的祖先告诉过我有关时空异象的信息：“双月同天，必有大乱”此乃时空错位的征兆啊！
我下意识回头，开到了开始涂肥皂的赵雪，不禁苦笑，时空错位，纵使世界末日又与我何干。
当赵雪涂到脚的时候，一只手自然的就搭在了我所在的铁架上，我也没有在意，依旧盯着窗外，似乎有一道妖异的光向着这个方向飞来……
是闪电！
等我反应过来时，我已经感觉到一股强大的电流涌入我纤薄的身体，赵雪浑身也发出白光。
“我被雷到了？”这是我失去意识前最后的想法。
…………
…………
我已经不知道在这电脑桌下呆了多久了，自从上次刘娇一脚把我踢到电脑桌下，估计几个星期都有了，刘娇还有她的美女舍友都放假了，那个不愿意透露姓名的女外星人告诉我要晒到太阳才能变回去，才能把明老师和胡蕊她们就出来，还能捞个大美女胡蕊当女朋友，可是在电脑桌下面终年不见天日，这让我怎么变回去啊！我是有点小变态，但让我一直以一条丝袜的形态呆在这个脏不拉几的地方我也受不了啊！我倒希望现在宿舍里某个漂亮妹子能发现我，穿穿我，这样我都能好受些。
忽然，我的脑海里传出了另一个男人的声音：
“不是吧，这么衰，还不是个人！”
嘿，这话我可不爱听了：“你谁啊！说谁不是人呢！啊！你再说一遍，说谁不是人呐！！”
“你……你怎么有意识！你也是人变的？”
对方似乎很惊讶，但我也不能放松警惕，问清楚再说，“你是谁？为什么能在我的脑子里讲话！”
“我？我叫郑然，是……”
“你，你也叫郑然？！”虽说我爹妈给我起的名字还比较大众化，但哪有这么巧的事情，我叫郑然，他也叫郑然！
“唔，那就没错”，那人看起来似乎很淡定，“我是来自另一个时空的你，你先听我说……”
接着，我脑海中的那个人给我讲了一个很长，也很离谱的故事，但是我连外星人这样的设定都能接受，还有什么不能相信的呢？
于是，我也把我的故事告诉了他，当然，被穿时心中那点小兴奋自然被我省略掉了，可不能让另一个时空的我感觉我是个变态。
对方听完了我的故事，沉默了一会儿，说：
“我应该可以让你变回去。”
………………
………………

不得不说，看到您的回复真是让我怦然心动。还记得我初中时候家里刚买电脑时候自己偷偷上gts吧的那种感觉（现在我已经本科毕业了），实在是美好的时光啊（笑）。
依稀记得几篇短文，最早的一篇启蒙文好像叫做什么《帝国公主》之类的（那个公主很怕脏），不知道还有人能否记起了。另外就是转的论坛某位大大的《履鸟传》，只有部分，贴吧ID叫做“在她脚下”某位写的关于老师的文，具体名字也忘记了，还有几篇都是短的福利文。
当时看的是非常非常开心的，但现在，从个人角度来讲，我是十分赞同您的观点的，个人比较喜欢长篇，情节上有完整的设定的那种，逻辑最好能够严谨，而不是纯粹的为了福利而福利。我觉得持我们这种意见的人还是少数的，至少在论坛里，根据文章的回复来看，还是有很多人喜欢您所谓的“搞搞搞，舔舔舔”的（笑），对此我也并不排斥，毕竟能进行创作就是好事。
但换个角度来讲，是不是我们开始厌倦了呢，对于我，确实是无法保持原来对gts的激情了，从而导致看现在的文没有之前的感觉了，举个例子，很早很早之前，偶然找到的一个擦边的房地产广告都能让我兴奋半天，但现在就算看到专业的gts杂志也没有之前的那种冲动了，也许人都是习惯把过去当成经典，而否定当下的吧。
不知不觉的废话了很多，但您的回复确实勾起了我小时候的美好记忆，谢谢！
