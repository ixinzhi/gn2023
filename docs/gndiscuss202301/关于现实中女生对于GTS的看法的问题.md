# 3

嘛~~文章的取材确实是很重要啦，但如果对作者本身体会的真实性太过看重的话，还是有些本末倒置。

我想起了许多侦探小说，大部分作者其实也并不是侦探之类的工作不是么。他们对于人物与案件的细致描写，并非来自于自身的经历，而是在于多方面的精准信息收集，以及对材料的认真考察。才能构建出一套完善且缜密的犯罪逻辑。

对于跨性别描写，我想也是如此。虽然女生亲自写的文，个人脑补很有代入感，我也很喜欢。不过男性也同样可以描写出，非常细腻与真实的女生心理与行为。或者说，对任何角色的心理描写，都非常考验作者的内涵、阅历。

因此我相信，男性作者在描写女性GTS的时候，也会同样的生动。
