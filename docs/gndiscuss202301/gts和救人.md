# 3

有啊，我之前發文問的就有想過「把男主角縮小從警局帶出來」這種橋段
不過這種比一般虐殺文還難寫，是因為在這裡角色們互相之間的利益關係很重要
間諜、人質、機關，每個都有自己的考量，互動也會有不同的目的性
像是，如果我方人質有重要到會被敵方擄走，那交際花去要求見面也會被全程監控
就算單獨見面了，走出房間時牢房人質不見了，間諜就一定會被搜身，躲過搜查也會被暫時囚禁
還要考慮人質的身分，他/她會不會覺得所謂來救人的只是另一個反派組織在搶人或是要滅口……
當然這些不是什麼會死機的bug，不過如果敵人的立場和做法被隨便呼弄過去，劇情張力就會很弱

有一種玩法是，間諜A走到深處時丟另一個小人B鑽進去，小人B找到人質C，把C縮小後B帶出來跟A會合
要在哪裡放人跟會合呢，那就設定個廁所好了，就算有女兵全程監視也可以偷偷進行任務
那為什麼廁所通風管可以連到犯人房間呢，就說是對方所在的建築是我方蓋/設計的
為甚麼呢，可能以前是我方堡壘卻被占領嘛，可能是對方建築設計師也是我方間諜
那對方到底對我方有多少掌握，如果對方實力很弱那位什麼救人還要大費周章……

嗯，要寫是寫得出來，可是太麻煩了，情報戰要做的賽局推演非常燒腦
而且我可以跟你保證，福利性和故事性看的不是題材，是個人的發揮能力
結論：花錢請你能信任的作者寫，沒有那種人就自己去寫 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
