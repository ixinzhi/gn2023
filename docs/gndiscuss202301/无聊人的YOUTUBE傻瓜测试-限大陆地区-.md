# 1

本机使用的是北京地区ADSL2兆带宽的小区包月宽带，没有任何附加增值服务，但位于伟大的墙内的同志们
应该晓得电信和网通的服务商是多么不靠谱，所以以下测试只属于个人意见，如有不同意见。。。。
那就对了！

顺便说一下，本人是傻瓜级脑盲，所以请不要用诸如带速，修改代码，代理服务器端口之类的火星文回复本人。

本机下载速度，电驴种子500情况下一般200K左右，无限制的迅雷大约150，举例的话原来的本站专用下载区中  MEGAUP网站下载  500M电影需要5小时左右（速度不稳定）

使用无界的话早时段（8：00--11：00）连接率大约60%，午时段（12：00--3：00）50%，晚时段（6：00--10：00）几乎没有，深夜（12：00--3：00）回复到60%
个人感觉使用无界是最简便的，尤其适合我这种脑盲，但无界的一个特性是很多稍微时间长一点的片子在播放到一半的时候卡住了，哪怕屏幕上显示的影片缓存（红色播放条）满了的情况下还是看不到片子的结尾，有点郁闷。

自由门
早中晚深夜连接率都同样低得惊人，而且操作上也比无界麻烦一点。而最可怕的就是网站主页上的”不河蟹“的信息了。

以上两种是不需要安装，直接点击使用的，也不需要自行设置代理服务器端口，反正我是总弄不明白关于IP地址的一切。
但不太习惯的就是不管我怎么调整，都只能在IE上看YOUTUBE，每次出现我们即将取消你的浏览器字样总是让我提心吊胆的。

火凤凰
任何时间都可以连接，速度很强大，用它的时候看片子几乎不需要等待，但必须安装软件进电脑，而且我原先用的瑞星和现在的诺顿总是杀毒报警。慎重吧！前一段时间服务器不稳，所以删掉了，现在据说重开，速度未知。

HOSP
我最早用的一款代理，特点就是广告多，纯英文。。。。在某次大乳妹出现被我父母发现后就删掉了。。。
09年初测试数据，除晚时段80%外，其余时段均可连接，速度一般，最大的好处是支持各种浏览器，只要你不在乎不停的关闭广告窗口的话。。。。。
