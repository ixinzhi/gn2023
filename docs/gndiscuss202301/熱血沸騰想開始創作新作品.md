# 1

許久但丁都沒發表新作品,一發表真是一鳴驚人,但丁不虧是中文界GTS大師級人物,而且這陣子也有一些同好陸續張貼收藏的作品,也有些同號自己也開始創作新作品,讓我心情真是興奮,越多人創作中文GTS作品,那是最好的一件事!

最近也因為如此,讓我也開始想重新創作新作品了,離上次發表的小說美少女超人至今大概有一年多的時間沒發表作品了吧!這幾天我也開始動電腦稍微描寫一部小說,進度還不多,等進度比較多時我會貼上來分享給各位的,知道我的同好應該我的作品都是那種純巨大化破壞城市殺戮人類的小說,我喜好從人類的角度及感受來描寫巨大少女的動作以及從旁觀的角度來描寫,所以有時再寫一個地方的描述就寫了一大堆有的沒的,雖然我自己寫的很爽,但是就不知道同好是否能接受我這種有時一個地方寫得又臭又長的方式,還有之前有人批評過我的作品文字排序很亂,的確,太亂啦!就像現在我發表的留言,整個字擠在一起,看起來會很累!

所以這點我稍微注意別人的作品文字排序,文字超多但看起來還不會太亂,嘿嘿!眼前就有一個!就是但丁的小說文字排序的方式!

但丁的方式的確不錯,文字不會太擠,接下來我發表的新作品,排序能就學但丁囉!但描寫方式還是我這個獨特的boysgts風格啦!

我這段時間在寫這部新作,只是剛好下禮拜我就是要期中考啦!快30了還在讀夜二技,而且如果再找到工作,那可能我時間會很少,那麼可能又容易出現一個作品隔好久在做出新作品,那我可能就是喇賽的人啦!

所以近期目標我會把這部作品完成分享給各位,以我所設的目標,新作其實是特別篇,如果能順利陸續推出新作,那麼新作的女主角就是會一直以她為作品的主軸陸續推出作品,希望別像上次寫美少女超人後就沒下文啦!

還有我還會把之前日本GTS的連環圖畫及漫畫改編成圖文小說,這部份也是我想創作的作品之一,有些幾張的連環圖適合創作短篇方式的小說,到時希望大家會滿意唷!也希望現在GTS新作家能陸續推出中文GTS作品,而且越來越多人參與創作唷!那麼各位一定會欣賞到很爽的啦! <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
