# 1

假设一物体（巨大娘）高度为 h, 此物体的一般高度（在此年龄下其高度）为 h_0,
定义 1：对于这一物体（巨大娘），其放大倍数为 n=h/h_0。
例 1：
h=15.3, h_0=1.42, 求 n=?
解答：n=h/h_0=15.3/1.42=10.8，即此物体的放大倍数为 10.8。

公理 1：物体的放大符合空间上的相似，即当一物体放大倍数为 n_0 时，其在 xyz 三个方向上均放大 n_0 倍。
假设 1：物体放大符合密度不变原则。

推论 1：物体放大倍数为 n 时，其质量放大倍数为 n^3
证明：
根据密度的定义，有 ρ=dm_0/(dx_0dy_0dz_0)，
移项，得 ρdx_0dy_0dz_0=dm_0 ①
由公理 1 与假设 1 得，dm=ρdxdydz=ρndxndyndz=n^3ρdx_0d_y0d_z0
将①带入，得 dm=n^3dm_0
积分，得 m=n^3m_0
推论 1 得证。

例 2：
h=117832, h_0=1.53, m_0=35, 求 m ？
解答：n=h/h_0=77,014.38, m=n^3m_0=1.60*10^16。

假设 2：时间不会随着物体的放大而放大。
即一个放大倍数为 n 的物体经过时间 t ，一个未经放大的物体经过时间 t_0 ，在任意情况下总有 t=t_0。

这里，我们已经得到了 L(n)T(1)M(n^3) 的放大倍数关系，即可得到所有动力学量的放大倍数关系。

推论 2：
如一个物理量的单位为 L^αT^βM^γ，
则一个放大倍数为 n 的物体所具有的这一物理量表达的状态或是在同一过程中发出的物理量的倍数指数为 α+γ*3。
证明略（参照推论 1）

例 3:
一未经放大，高度为 h_0=1.24 的物体在某一过程中做功 W_0=10，
如果一高度为 h=12312 的同种物体重复这一过程，求其做功 W。
n=h/h_0=9929.03。观察功的单位，为 L^2T^-2M^1，
