# 1

事实：
本人前一段上传nami的《杰西卡和维姬》第五章被人举报；
小a前一段时间翻译的《我的缩小前男友》在ray里也被举报；
其他一些网站（当然不是那么那么正当的网站）的一些资源也有这种情况……
不解：
本人就是想不通，为什么会被举报呢？
压缩文件带密码，而且隐藏文件名……
以我个人对网络的认识程度，有几种解释：
1.论坛里的人干的……也就是所谓的“同好”，干的……
    可是为什么这么干呢？仅仅是不让别人下载？自己有什么甜头呢？理解不了。
2.纳米盘官方自己“做”的……
2.1现在已经有不需要解压缩密码就可以解压缩的软件了，于是……暴露……被封
    如果是我火星了，谁能告诉我在哪可以下到这种软件？挺有用的
2.2官方为了节省资源，打这个幌子删去~
3.谁无聊，看哪个文件不顺眼就点“举报"按钮……尽管不知道内容是什么……够无聊
本人网龄尚小，不过3、5年，一时还真是想不透，不知道哪位大大了解其中“内幕”，指导一下在下……
