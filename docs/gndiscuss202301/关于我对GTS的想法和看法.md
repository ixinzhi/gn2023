# 1

最近看到很多很沉重的帖子，比如[这个](http://giantessnight.com/gnforum/viewthread.php?tid=11914&extra=&page=1)。
  我也曾经想过，我是不是很不正常？我这种嗜好是不是很变态？
  自从上课看到这个帖子，我就一直想到今天早上。
  其实，我们是不是过度敏感了？在我看来，GTS只是一种爱好，也许与众不同。就好比，当所有人都喜欢一样的事物，而一个人和大家不一样选择了另外一样。
  但是，当这种事情发生的时候，那个不一样的人就会差生离群感，甚至被称为怪胎。这就是我们人类的心理，当一个人与大家不一样时，我们就会去排斥他。
  记得以前看过一本书，是田中芳树的作品。名字已经记不清了，但是里面的一些话让我印象很深刻。比如其中一段提到了，因为自己人数多，而其他人和自己不同，便可以任意欺凌他们，这就是现在很多年轻人的心理。
  我不想成为哲学家，因此也不过多评论。
  我想表达的是，我们可以与人不同，偌大的华文社会，我们这个论坛却只有几千人，可能我们是另类，但更可能其他人只是没发现自己这种爱好。
  其实，我们换一个角度来看，换在60年前，如果你对别人说“我喜欢女仆”，那么只会惹来异样的眼光，会被当成彻底堕落的败类吧。但是现在呢？公开的女仆店，女仆已经成为了一种属性。虽然有些人不会接受，但是反应与表现和以往可谓天壤之别。
  也许会有那么一天，我们自我介绍时，可以说“我的爱好是巨大娘”，就像现在我们说“我的爱好是集邮”一样自然。
  另外一点，关于“反社会”问题。
  其实，所谓反社会，到底什么是反社会？
  难道拍摄“哥斯拉”，“后天”，“独立日”，“2012”等电影的导演是反社会人格？
  只是因为我们的爱好中有破坏的【部分】，就是反社会了吗？在许多GTS作品中，描写的并不是毁灭全人类的情景，我们中也不会有人去希望世界毁灭（最好是如此）。相较之下，‘人类’‘社会’之类的字眼只是来衬托GTS，来反映出GTS的强大，而不是利用GTS来毁灭全人类的情节。我个人认为，如果真的有人写出这样一篇作品，是一个女性巨大化后一味杀戳，毁灭，按照XNR大大的论点，只有GTS出现，却没有互动，没有美感。至少在我看来，一味的毁灭破坏并展现不出任何美感。我们更希望看到的，是类似EOE的作品，里面GTS不是全部，而是透过GTS讲述了许多，但是GTS成分却又站在主流。
  先之说这么多吧，我想说的很多，但是看了看写的这些，又似乎没什么内容。组织下语言先
  希望各位也能发表你们的看法
