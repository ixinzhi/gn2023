想要计算的话需要知道以下较为基本的数值
身高
体重
然后……
知道三围和罩杯的话就更完善
不知道三围的，按照这个来算
胸围=身高 X 0.53( 如:身高160cm的标准胸围=160cm X 0.53=84.8cm )
腰围=身高 X 0.37 ( 如:身高160cm的标准腰围=160cm X 0.37=59.2cm )
臀围=身高 X 0.54 ( 如:身高160cm的标准臀围=160cm X 0.54=86.4cm )
E.G
舒淇
身高：168cm
胸围=身高 X 0.53(168cm X 0.53=89.04cm )
腰围=身高 X 0.37(168cm X 0.37=62.16cm )
臀围=身高 X 0.54(168cm X 0.54=90.72cm )
当然这其实是按照完美比例来算的，人嘛，都是不一样的。凑合着用吧
