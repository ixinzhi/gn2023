# 1

接着，陈羽把小雅变回了正常大小，然后把已经被踩了个半死的雯雯拿了起来。
“羽哥，要怎么——”
“看我的。”陈羽坏笑着，把雯雯变得小到2毫米那么高，又把在鞋子里眼巴巴看着的思思拿了出来。
“思思，你来把这个小东西，塞到我的炮膛里。”
“炮膛？”思思有些茫然，然后自己便被男神抓着送到了胯下的巨炮前面，思思恍然大悟，小心翼翼地从男神的另一只手掌里捏起雯雯，一点一点塞进了男神的马眼里面。
此时的雯雯已经吓疯了，一向高高在上被男生追求的她，居然先是被踩在脚下，被当做拖鞋一般在地上蹭来蹭去，又被像虫子一样踩扁，此刻居然要被当做自己的敌人亲热的助兴工具了，可是她无能为力——即便是在陈羽眼中像虫子一样的思思，此刻对她来说也像女神一样不可反抗。
小雅好奇地看着，然后就见到陈羽把思思甩手丢到了书桌上，跟着便抱起小雅也放到书桌上——小雅立刻感觉到自己丰满浑圆的臀部下面有一个小东西在挣扎，她“噗嗤”一笑，故意扭了扭屁股，隐约听到下面传来一声思思的惨叫，仿佛屈服了一般不再挣扎了，小雅便心安理得地将这个小东西坐在自己的屁股下面。
还没来得及享受，陈羽便粗暴地将巨炮直接送进了小雅的小穴中。小雅惊呼了一下，紧接着就变成了娇喘……（实在不会写这种片段，跳过……）

不知道过了多久，已经爽过了的陈羽将自己的巨炮拔了出来，又把已经瘫软的小雅抱到了床上。小雅又羞又兴奋地问：“羽哥，那个……贱人呢？”
陈羽笑着在小雅的两腿间寻找了一会儿，找到了随液体流出来的雯雯，轻轻粘在指间，送到小雅的面前：“看，在这。”
小雅问：“你打算怎么处置她？踩死吗？”
“那太便宜她了吧？”陈羽嘿嘿一笑，把雯雯变大了一些，“小贱人，你知道自己现在的身份吗？”
雯雯此刻已经六神无主了，刚才的过程对她来说太过恐怖——先是被夹在巨炮里，然后剧烈的地动山摇之后，被一股又热又粘的液体冲出了巨炮，落在了自己的竞争对手的肉壁上，又被喷涌而出的液体冲出了体外——自己仿佛已经不在是一个人，而是一个微不足道的寄生虫，或者没有生命的玩物……
“身份？”雯雯愣愣地抬头望去，仿佛已经灵魂出窍——“不知道啊……”
“那我来告诉你，你是从我的体内被射出去的，射进了小雅的身体，然后又从她的下面出来，现在你知道了吗？”
“啊——”结果是小雅先明白过来，“羽哥你好坏。”接着她也很兴奋地看向那个小东西：“小贱人，以后要喊爸爸妈妈，知道吗？”
