# 1

最近在使用C4D+VR+poser制图，但是从poser导出到C4D之后人物就固定啦没办法调整关节，还有皮肤的质感总也做不好，求指教。还有体液的喷溅的感觉如何做？感觉不会的好多！ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 2

首先欢迎有同好加入3D制作
关于模型导入：
模型从poser导入C4D后是无法进行调整的，插件中有一个reload按钮，你需要在poser中修改，修改完毕后保存，再按reload按钮，才能实现修改。
<title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 3

关于皮肤质感
VR渲染比较适合室内装潢一类的渲染，不太适合人物皮肤渲染，如果用VR做皮肤渲染，会感觉塑料质感比较浓，C4D自带的渲染人物皮肤质感很好，但是场景效果不如VR，自己取舍。 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 4

关于喷溅
C4D自带一个叫做bodypaint3D的工具，十分强大，也十分简单，自己看看网上教程很快就能学会了。 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
