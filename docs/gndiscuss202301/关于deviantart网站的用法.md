# 3

D站不像booru，是鼓勵創作的發圖平台，作品種類繁多，限制級的圖文還要權限才看的到。

建議用giantesd搜索，找到一張對味的圖後，由作者的收藏發掘其他作者，或是加入對方為關注清單，以後有類似關鍵字的圖出來,D站還會通知你 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
