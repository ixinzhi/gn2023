# 1

首先版权是应当得到尊重的

但是真正从头到尾完完全全尊重版权的人非常的不多

论坛里游戏区、视频区、图片区里的资源，有不少其实是 付费/禁止转载 的资源

完全杜绝盗版是一个不可能完成的任务。

如果完全的，一刀切式的在大众平台上面禁止版权资源的传播，只会导致私下里资源倒卖的猖獗

目前来说的话，就GP而言，除非有人公开发在论坛或者大群里，否则我不会上传到公共线路；除非和我私下共享的人明确表示可以上传至公共线路，否则我只会在私密线路中进行上传。
同时如果我知道作者不愿意作品被共享或者作者告知我希望删除，我也会将其从公共线路中山删除。

我很好奇各位对于所谓版权资源的看法。
（共享指发在论坛或者大群，不包括私下小范围分享）

以图片为例，基本上 免费资源随意传播，只要注明作者就行，付费收看的资源不要在公共场合共享，自己约稿的，想共享  可以署名作者就可以。
一般这样没有任何问题 也不会侵害他人利益

除非是买断制的资源，任何资源拿来卖 基本上是侵犯对方版权的。 主要是因为18禁物品 本身国内禁止，而且没有保护，不然真的会“干”死你

# 11

我是个做游戏的，我认为版权很重要，作者不允许传播的就不传播，学生党没能力的可以合购。不过国内的这种东西可不受保护，所以国产的版权就显得不重要，国内作者大多数也不会拿去卖或者声明禁止转载，大多数国产游戏都是用爱发电。
我认为最好的办法就是提高国内道德水平，圈内水平也行。
但有一点很重要，就是自己做的游戏被拿去卖的话会很难过
