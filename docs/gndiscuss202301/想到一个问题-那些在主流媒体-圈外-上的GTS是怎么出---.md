# 4

你先定義什麼叫「圈外人士」? 需要怎樣的認知或活躍度才算圈內?
如果只要知道這種癖好就叫圈內，會掛EX變態的人基本上都是圈內了喔
所以我覺得你想問的是，不知道尺寸癖好的人怎麼會用到這種題材?

如果尺寸癖好可以形塑出一個個的社群，我們可以假設這種東西是刻在基因裡的慾望
所以，不是「因為有了尺寸癖好而有了巨人/小人這種想法」，而是「由於巨人/小人這種想法而有了尺寸癖好」
把順序調成這樣就很好解釋，「即使創作者不在尺寸癖好社群中，也可能自然而然產生類似想法」

舉個超簡單的例子，我一天到晚叫別人去吃屎，但我從來沒有跟Scat社群的人互動過
如果隨機從會喊「吃大便啦」的人中抽樣，說不定也會找到不少對於食用排遺物感到性興奮的案例呢
但是對絕大部分的人來說，這件事情頂多到搞懂「吃屎是什麼意思?」就不用再繼續思考了對吧?
所以，你想去分析「用女巨人當題材的人在想什麼」，研究精神值得嘉獎，但他們自己也不知道啊
