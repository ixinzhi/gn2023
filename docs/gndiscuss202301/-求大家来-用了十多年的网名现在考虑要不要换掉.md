# 1

我想，也许是时候和用了十多年的网民告别了
我的qq网名，五叶草，本来是但大的文章中出现的代表奇迹的草，我在但大连载的时候看到，心情不能平复，就把网名取为五叶草了
这个名字也算是GTS相关了，即使出处比较偏僻
但是我最近百度五叶草的时候却发现这个词完全不符合我的预期
我搜到了五叶草的三种概念，爱情悲剧故事，牛饲料，清热补虚药。。
我感到了极大的打击，这个网名看来是时候换掉了，可是已经用了十多年了，又是但大文章中的神草，也象征了我对GTS的执着信念，换掉又十分可惜
不知道大家有什么看法？
