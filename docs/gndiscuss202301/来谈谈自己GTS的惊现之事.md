# 27

我就悲剧了，我在初一的时候就有了这方面的癖好了吧？但是那时候网络才刚刚普及，各种不全面。所以我没有办法找到类似的资源来满足我的幻想，只有自己没事的时候用我一个单独并且很厚的竖式计算本来画画。嗯……到底画了多少我也忘了，都是采用局部式画的，现在想起来我那画风真是惨不忍睹，但是当时很快乐似得。
那个计算本跟了我有3年之久吧？反正是到处藏（个人隐私这点我是很悲剧的，几乎没有），终于在暑假里的一天还是被我老娘给发现了，问我这是什么东西？我现在都不知道当时是怎么挺过来的，我说：“是放假的时候收拾东西不知道是谁的给装回来了。”我老娘就再也没问什么，虽然东西没拿走，但是后来我赶紧处理掉了。这是第一次……
嗯，第二次是在前两年吧？我刚参加工作，那时候我已经找到组织啦！在组织的滋润下我收藏了不少资源，在一次加班时我将家里电脑中的经典2次元图片用移动硬盘带到单位，因为晚上没人，我就用单位的彩色打印机一顿狂打（遗憾的是当时有点SB，为毛没用A3的纸打呢？），然后找了个带锁的文件夹装起来拿回了家，唉……当时好激动啊。后来过了有一年多吧？也就是去年，老娘收拾房子又给发现了这个文件夹，但是好像她不知道是什么东西，当时问我是什么。因为我已经不是以前的小孩了，所以很厌烦的说：“工作用的东西。”老娘非要看是什么，我当时就发火了，道：“烦不烦啊？我什么东西都要让你看么？我还有没有个人隐私了？”然后就不了了之。唉…… 这就是我的GTS之路。 <title>28</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
