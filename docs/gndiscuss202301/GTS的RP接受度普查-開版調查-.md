# 1

RP的全稱是「Rule Playing」
字面上的意思就是「進行有規則的遊戲」
是種與角色扮演是很接近的一種東西

在開始一個RP之前
負責開頭的人會先寫下一些條件
用來讓不合自身口味的人做些迴避的動作
同時也能吸引臭味相投的人一同玩樂
接著,一個引子被樓主寫出來之後
必須由第二個人或是樓主接續
簡單而言就是多人利用文字敘述
來完成一個完整的GTS故事流程

在國外的大站,這種玩法已經行之有年
但丁前輩曾想引入這種風氣,但最後卻不疾而終
然而我卻想再試著引入一次這種玩法
讓更多有所需要的人能夠去參與遊戲內容

我沒什麼其他的想法
只是想讓大家一起同樂而已
因此才弄出了這則問卷調查
好知道大家的想法
如有疑問請在此回帖 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

觀察了數天，看起來，對RP有興趣的人固然會投票，但沒興趣的人也不至於會去投反對票。

也就是說，儘管多數人都對RP沒有興趣，但也不覺得開個專版有哪裡礙眼。
基本上這是一個很好的訊息。
雖然感興趣的人數少，但只要這一小部份人能夠玩得很愉快，那我就樂意幫忙開版。
就算個把月後大家失去了興致，如果能夠留下什麼精彩的紀錄與主題，那版面也還是會持續保留下來。
