# 5

234纯粹是来捣乱的...
想写文章首先得有一个故事想法，比如你就是想写出那最精彩的一幕或者一个什么样的故事，就思考着往上添加设定
但是你一点想法都没有，那么就靠插标签来激发想法了，也就是开脑洞的一个过程

倍数（十倍？百倍？）、人数（恋爱中的2人？冒险团队4人？）、故事背景（现代？科幻？剑与魔法？超能力？）等等等等

想到一定程度，想出来一个【我想写一个巨人少女和大叔在拯救世界的路上打怪升级的冒险故事】之类“诶，这个不错，看起来很好玩的样子"，然后就可以想人物设定、比如【少女虽然是巨人族但是是个牧师】，【大叔一开始一定是为了偷少女的家传宝石而遇到的，那么它就应该是个盗贼】等等等等，最后弄好了一切设定，只要把这两个人放进故事中，那么就可以从脑洞中源源不断的流出来故事情节了。

最后的问题...也是我写文到现在没写出来一篇的问题...一定要把所有脑洞设定都记下来再开始挑有趣的设定写...不然3分钟热度啊不对，写了3天之后你就开始上下文冲突最后恼火万分丢笔不写了... <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
