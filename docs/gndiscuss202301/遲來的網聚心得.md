# 1

總算抽出時間來寫網聚的心得了

基本上
這次的網聚非常順利
大家實在是太熱情、太大方了

在這之前
先感謝但大和蚱蜢大
把他們自己收集的GTS物拿出來分享
讓我這個伸手黨好慚愧

但大帶來了不少日本當地才有的GTS物
我最中意的就是那本介紹GTS文化的書
內容超豐富
只可惜我看不懂日文
希望有出版社能代理進來
好讓更多人能瞭解GTS啊

至於蚱蜢大帶來的GTS漫都有一定的年代了
有不少本的年紀甚至比我還大
真佩服蚱蜢大的收藏功力

現在開始說網聚的情形

我大約在一點五十分左右的時候到
但到了之後
發現有幾位比我還早到
而桌上已經有不少GTS物
趁這段時間稍微看了一下

等大家都到齊後
開始做基本的自我介紹
至於內容就不透露了
免得造成他人困擾

在這之後的一個小時當中
都是在翻閱但大和蚱蜢大帶來的GTS物
不過我有偷玩一下D3 XD

接著大家開始討論起GTS的話題
像是當初如何發覺自己是GTS控
還有一些業界的情況等等

做為網聚壓軸的抽獎
愛版主手氣真好
做為第一回第一個抽的人
居然被他抽中
愛版主你不考慮去買樂透嗎?

而第二回的抽獎好刺激啊
我和另外兩位一直沒抽到
看到牌一張張的減少
超緊張的
最後則由Pliod抽到

希望明年還能再舉辦一次網聚
要是能有更多人前來的話
就太棒了。
下次我一定會抽到獎品的(握拳)

PS
實際上還有不少趣事
只不過我記性不太好
麻煩哪位在場的幫我補一下
