# 1

这个问题，我想了很久，总感觉文章的分类有点简易，但是我也知道，制作这些分类非常复杂，首先，抖m比较多，但是很多人明显反感调教系，除此之外，哪些毁童年的文章也是让许多人反感(奥特曼这个无数人的童年在gts文章中不知道被黑了多少次）gts包括巨大和缩小，同时还分成温柔 残忍 无意识 吞食和变物。相信许多人都有一种经历，看到标题是温柔系，点进去是调教系，表面上写残忍，点进去是吞食......为什么不能在多些分类呢？在此我提一个建议，在文章上标注属性，我知道这样有些麻烦，但大家如果都清楚的标注属性，我们便可以轻松的找到我们想要的文章。gn的完善，靠的是全体会员的努力，每人献出一份力，gn会变得更加好。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
