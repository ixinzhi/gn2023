# 1

好像有人跟我一樣已經裝了vista然後不能執行apploc. 不過有解決方法啦--by a computer expert---not meXD
我的windows是英文版的,所以可能有我翻的翻譯名詞不對,不過我以前也用過中文版的,所以應該記的一點![](img/b89eecf7b90d70d85e9777f7c0216efe.png)
按'**開始**'>>**全部程式**>>**accessories**(我猜是**附屬應用程式**,因為裡面有記事本XD)>>**command prompt**(考倒我了,command是'電腦指令'的意思,prompt是'準時的'或是'提示')
反正他圖樣是一個小視窗裡面寫'c:\' 注意先別按左鍵,在上面按右鍵點'run as administrator'(用管理人樣式執行) 然後再輸入你applocale.msi檔放哪
我極建議放c或d碟就好 這樣的話放c碟人打c:softsapploc.msi     放d碟則打d:softsapploc.msi
一定要打.msi,不可漏掉囉! 然後再去安裝就可以囉! 我自己試是成功了.
雖然可能整個論壇只有一兩個人用vista.........................![](img/1751a37041f23d672f2071488c42b774.png)
