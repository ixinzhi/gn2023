# 1

从小一起玩大的表姐昨天从美国回来了，我今天去找她玩。舅舅和舅妈都出去了，我一眼就看中了表姐的皮靴……
是我喜欢的那种齿轮底短靴，踩过的地方会留下漂亮的脚印。

这个游戏从小表姐就和我玩的，一般都是表姐踩在我身上跳舞，有时候也穿鞋。现在我们都大了，也知道这意味着什么，不过表姐还是满足了我的要求。
