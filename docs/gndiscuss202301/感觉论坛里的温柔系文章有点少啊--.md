# 6

因为作为gts的核心要素，体格差给温柔系的描写限制实在太大了。
由于体格差的存在，女方对男方施加的动作显然会随着倍率的增加而愈发受限，甚至女方一些不经意间的举动都会对男方产生伤害。所以温柔系基本都是mini，到了mega、giga的程度就基本都是区别对待了，这样本质又回归到残忍系上了。
在这样的前提下动作描写的范围就会越来越窄，各位写手产出的温柔系文本也会趋向同质化。等到能写的花样都写完了，灵感枯竭了，温柔系的文章就越来越少了。

# 11

GTS本身就是一种对绝对力量的崇拜，尤其是megagiga级别的爱好者多少都有点M，mini级别也大多都是力量的碾压，而写作是满足xp的，所以温柔系会很少
从我个人感觉和观察来看，温柔系爱好者以高女、Mini、growth为主，大概分三种，一种是反差萌，即绝对强大/高大/巨大的外表与温婉可人的内心构成的强烈对比与反差；一种是母性向往，就是字面意思上的，这种文一般都是母子文而不是情侣文；还有一种是xp放大，主要将巨乳/臀/腿等特殊xp放大形成长身巨乳/臀/腿等，文章强调幻想出的特殊身体特征，而不会涉及残忍系的内容。
一方面，由于每一种xp的诞生都与实践密切相关，不同人不同的成长经历和生活环境注定了xp的不同，温柔系爱好的形成更是纷繁复杂，相比残忍系单纯的权力崇拜难多了；另一方面，随着年纪增长更多人会选择在幻想里追求刺激，温柔系并不能带给他们足够的满足，也就是“冲不起来”；并且温柔系好就好在绵长的回甘，这是需要长久的沉淀才能形成的，论坛大家恐怕都力有未逮，还不如直接整点简单粗暴的刺激活。
总的来说，就是需求不足，创作难度高，所以相对来说占比很少，但绝对数量还是很可观的 <title>12</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
