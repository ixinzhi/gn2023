# 20

我現在覺得不發表我的意見是不行的...

首先,我對於高跟鞋,靴和運動鞋沒太大興趣,因為那幾乎只能用踏的...而且把腳弄的沒有原的 美了. gts其實不用高跟鞋及靴也可以增高的...

我對於涼鞋是感興趣的,因為它可以很自由的被穿著,還有與gts成正比的破壞力.(上面有一朵花的最好)另外我也對那個叫鬆榚涼鞋的東西有興趣

赤腳就更好了,第一是行動便利,第二是赤腳對腳而言是最好的.而且巨大的赤腳踏起來總是比較好的.

但是我最喜歡的是穿襪子的,絲襪和白襪還可以,我最喜歡的是以下幾個:

縮小校園女生的黑襪

garden這星期,jukes作品後半部的gts的襪子

類似聖誕節襪子的東西(也就是有聖誕節氣氛的圖案,或者原本的紅色)

這三種是我的最愛...... <title>21</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
