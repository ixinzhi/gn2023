# 1

昨天在上海和平影都看了，因为IMAX没上映只好看普通3D版本的
闲话少说，先谈谈GTS相关：
  GTS方面，真的是轻度的很，基本是SW，爱丽丝缩小到十分之一左右和疯帽子互动（当然是GT！）而GTS则只有MINI级别，大概两米多点（一开始有长到3倍左右大小，但没有他人在旁）。不过MINI级别的互动不少，比如和疯帽子的小感情戏，有点感觉。另外戏中有一位GTS控（？）就是红心骑士，他把两米多的爱丽丝按在墙上，说“我就喜欢大个的女孩！”算萌点？

剧情方面，很标准的日系RPG，但在爱丽丝的背景下就很扯淡：穿越的主角，神谕的被选中者，指点道路的隐士（会笑的猫），打败了看守神剑的怪物，然后拿着神剑杀死恶龙，拯救白皇后打倒红皇后——我真的很想吐槽，这不是勇者斗恶龙爱丽丝版么。。。。。

总体评价这电影（非GTS角度）比较一般，亮点在于疯帽子的杰出表演，至于3D。。。。。。我想说这电影比较适合看2D版的，别浪费那3D的钱了 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
