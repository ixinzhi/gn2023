# 1

在新中國建立之後，姓再也不具有高低貴賤之分，在現代社會，人人平等，雖然可能從某些方面來說並不平等，但總體來說不會出現一些非常可笑的歧視問題了。
姓氏，姓，即用來知道這個人出自什麼人的後代，如周文王姓姬，黃帝初姓公孫，后改姬姓，由此可以推出周文王大致是黃帝後代，甚至公孫姓也有可能是黃帝的後代。
氏，即為自己取的標記，也就是圖騰，如炎帝和蚩尤都姓姜（《路史·蚩尤傳》），但炎帝稱神農氏，而蚩尤屬九黎氏，雖然兩人同出同一個姜姓祖先，但氏卻有了不同；氏在遠古時期還可以用來稱呼身份尊貴的人士，如上面提到的神農氏、軒轅氏。
似乎至今日本仍然保留著類似的習慣，也就是在某人的毛字后加上“氏”來表現對那人的尊重。
然而大約春秋戰國之後，氏失去了其本身的作用，因為人們不再單獨形成于某個部族，平頭老百姓也擁有了姓，因此姓氏在秦漢以後合二為一，稱為姓氏。
大部分作者會在自己的作品中為角色取一些聽上去好聽又具有特點的名字來凸顯出主角的與眾不同。
而今天，我就為大家整理一些有據可考，較為古老而且聽上去有“高貴”感覺的姓氏。
一、上古時代
**①“炎黃子孫”**
黃帝，姓公孫，后改姬姓，號軒轅氏，故稱姬軒轅，又號有熊氏，而軒轅這個氏也在發展中逐漸變成了姓氏，在我國多處地區均有分佈。
炎帝，姓姜，號神農氏，又號魁隗氏、连山氏、烈山氏等，據說其母也是有蟜氏氏族的女性。
有熊氏可以指兩位，相傳是黃帝父親的少典和前者，但我們這裡討論的是少典所在的整個有熊氏氏族。
有熊氏後來延伸出羋、熊等姓氏，如春秋戰國時期楚國始祖季連以及楚國的君主，如屈原等。
傳說中黃帝是少典和附寶的後代，附寶為有蟜氏，姒姓。
從炎帝黃帝的關係中，我們可以推出這樣數個比較具有“高貴”感覺的姓氏：**公孫、姬、軒轅、姜、烈、羋、熊、姒**
**②“三皇五帝”**
三皇指的是天皇地皇人皇或天皇地皇和泰皇，而五帝一般指太昊、炎帝、黄帝、少昊、颛顼（白帝、青帝、黄帝、炎帝、黑帝）或太昊與炎帝改為堯、舜
三皇有數種說法，如：
伏羲氏、神農氏、軒轅氏（典出《世本》）
燧人氏、伏羲氏、神農氏（典出《尚書大傳》）
伏羲氏、女媧氏、神農氏（典出《呂氏春秋》）
天皇氏、人皇氏、泰皇（人皇）氏（典出《史記》）
伏羲氏、神農氏、祝融氏（典出《白虎通》）
伏羲氏、神農氏、共工氏（典出《通鑒外記》）
由於神農氏和軒轅氏與之前提到的黃帝炎帝重複，顓頊和少昊均姓姬，因此只挑出燧人氏、伏羲氏、女媧氏、天皇氏、地皇氏、人皇（泰皇）氏、祝融氏和共工氏、太昊、堯、舜進行分析。
燧人氏，風姓，因為據說發明了鑽木取火，於是被稱為“火祖”，燧人氏也是伏羲氏和女媧氏的父親。
燧人氏娶妻華胥氏，華胥、伏羲、女媧皆為風姓。
天皇氏，一般指舊石器時代古越族首領據說姓望。
地皇氏，一般指地母後土或天皇氏子地皇氏，傳說其姓岳。
人皇氏，一般指地皇氏之子，傳說姓恺（典出《路史》）。
祝融氏，火神，有數種說法，一說姓祝融，為祝融氏，一說姓牟，一說姓姜，為炎帝後代，又一說姓龔，又一說妘。
共工氏，水神，據說姓姜，為炎帝後裔，又一說姓龔。
太昊，一說為伏羲氏，即風姓，一說為伏羲女媧時期帝王，暫不清楚姓氏。
帝堯，伊祁姓，現已少有伊祁姓，大多改為伊姓或祁姓，因號陶唐氏，其後代多改唐姓。
帝舜，姓姚，有虞氏，故延伸出虞姓。
從三皇五帝中，我們也可以推出以下比較有“尊貴、高貴”感覺的姓氏：**風、望、岳、恺、祝融、牟、姜、龔、妘、伊祁、伊、祁、唐、姚、虞**
二、歷代帝王
自古以來，上位者非富即貴，而帝王更是其中集大成者，因此第二條分析便從帝王開始
**①秦**
秦始皇，**嬴**姓，秦氏或趙氏，故延伸出**秦**姓和**趙**姓。
**②漢**
漢高祖及其後代皆以**劉**姓。
**③晉**
兩晉帝王皆為**司馬**姓
**④南北朝**
南朝宋齊梁陳
南朝宋宋武帝及其後代皆為劉姓。
南朝齊以及南朝梁統治者皆為**蕭**姓。
南朝陳陳武帝及其後代皆為**陳**姓。
北朝三魏周齊
魏自孝文帝后改**元**姓，此前為**拓跋**姓。
北周為**宇文**姓，北齊為**高**姓。
**⑤隋唐**
隋文帝及其後代為**楊**姓。
唐高祖及其後代均為**李**姓。
武周時期武曌為**武**姓。
**⑥五代十國**
這裡指特重點提到“大政權”
後梁——**朱**姓
後唐——李姓
後晉——**石**姓
後漢——劉姓
後周——**柴**姓
**⑦兩宋**
兩宋時期君主皆姓趙。
**⑧明**
統治者及其後代均為朱姓。
**⑨元、清**
因為元清爭議過大，所以不予考慮
故此整理出以下姓氏：**嬴、趙、秦、劉、司馬、蕭、陳、元、拓跋、宇文、高、楊、李、武、朱、石、柴**。
因時間、精力等比較有限，暫只整理出以上可以給人以“高貴或尊貴”感覺的姓氏，如還有其他想法，如外國姓，也可以在樓下補充。
