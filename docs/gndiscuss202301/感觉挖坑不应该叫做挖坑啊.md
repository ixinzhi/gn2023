# 1

挖坑就是一铲子一铲子填平的过程，那么结果就是没有任何改变，反而浪费了时间——除非有人喜欢看着你一铲子一铲子填坑的过程。

我觉得挖坑更像是造山的过程，因为坑是可以填平的，但是一座山怎么造肯定还是有瑕疵，肯定还是有不协调的地方，而造山的过程和结果自然有人喜欢看，因为填坑可以一股脑，造山一股脑就容易崩塌。
