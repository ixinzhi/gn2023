# 1

原提问：如果你拥有以一己之力碾压全地球战力的能力，你会做什么？原回答者id为 风染丛林，起点作家，作品有《吞噬世界之龙》

回答如下：最大的概率就是世界毁灭，怪物呆在废墟之上统治一群彻底沦为奴隶的家畜。

能够碾压全世界的力量意味着什么？
很多人可能根本没有搞明白，意味着你可以一巴掌拍碎大陆架、把地球撕裂开地核。
无论多聪明的人，都是无法控制这么一个怪物的，因为你的智力在它的力量面前根本微不足道。
就好像是天上的太阳，太阳连智慧都没有，但没有任何聪明人能够去操纵它，相反，反而要战战兢兢，因为太阳倘若一次剧烈的太阳风，都足以令整个世界为之毁灭。

人类是没有办法去威胁这个一个怪物的，因为你的所作所为，都最多只能激怒它。

“弱者需要智慧才能生活在这个世上，而强者，纵然天真的活着，也依然能够肆无忌惮的活着。”

人类的权势是建立在多数人的支持上的，所以人类才需要去发展智慧和谋略，但这个怪物不需要，因为它的力量并不是来源于谁的支持，而是它自己的力量。
所以，它纵然没有任何的智慧和谋略，行事再怎么简单粗暴也无所谓，它并不需要去介意它人的看法。

对于这种怪物而言，人类能威胁它什么？先一巴掌拍碎美洲大陆，然后每天悠哉悠哉的在地球仪上挑选陆地，今天毁灭一个加拿大，明天毁灭一个不列颠，后天再毁灭一个俄罗斯……
全世界的国家太多了，甚至不需要多做什么，仅仅只需要第一次的出手，人类就应该明白该怎么做了。
竭尽全力的去讨好这个怪物、让它感到高兴，尽可能的不让它有任何不满，因为这个怪物动怒之后的结果，是人类所无法承受的。

是的，你可以利用它，可以愚弄它，但是要记住，你千万不能让它发现自己被利用，更不能让它被利用的时候还会心生不满，时刻都保持一个很高兴的状态。

在这个怪物的面前，人类里最聪明的人也不过是皇帝身边的“弄臣”而已，用它们的滑稽和可笑来开怀大笑。
皇帝可以允许“弄臣”欺骗自己，但弄臣绝不能让皇帝发现，更不能让皇帝感到不满。

在这个怪物面前，它就是想当地球皇帝也不过是一句话的事。
纵然那只是它的一念之间，一个小小的玩笑，全世界也必须战战兢兢的去准备皇帝的登基仪式。
任何的国家元首，都必须在怪物面前俯首贴耳，大气都不敢喘。
它不需要去管人类的什么政治和经济，那都是它的仆从和附庸去管理的事情，它所需要做的只是享受成果即可。

如果你问人类反抗怎么办？
这个很容易。
杀。

一座城市一座城市的杀过去，一个国家一个国家的杀过去，随心所欲的杀。

愿意跪在它脚下表示臣服的便姑且可以放过，也不需要去管这人是不是真心的，因为没必要。只要对方跪下了，就意味着它畏惧了，恐惧了，不是什么为了自由能够悍不畏死的硬汉。

“一个人死你们不怕，十个人死你们不怕，一百个人死你们也不怕，那么一万、一百万、一亿、十亿人死呢？”

人类是很多的，当全世界八十亿人口被杀到十不存一、甚至百不存一、整条街道都未必能够活下一个人、到处都是尸体的时候，也依然能够存活数千万的人口，而这个时候，就算是最愚蠢的人也知道该怎么做了。

纵然你们认为人类一百个里，九十九个都是死都不怕的硬汉，但只要剩下的那一个是怕死的软骨头，那么，仅仅只是幸存的这么几千万人，几千万选择在怪物面前跪下的奴隶、家畜，也足够这个怪物玩的了。

你问我为什么写的这么极端？
因为拥有这种力量的人，已经是个怪物了，其自身最初的想法已经没有意义了，从它拥有这种力量的时候开始，它就注定不可能再把这些它打个喷嚏就能震死的脆弱生命视为同类了。
生理上的剧烈变化，最终将导致心理上的彻底异变，人类在它眼里，和猴子、蚂蚁，不会有多大区别。

最后修改，既然还有人没有搞懂我说的话，那我就说的再明白一点:
你们认为自己是一个心无波澜的圣人吗？
是一个能够永远不变质的觉者吗？
你此刻认为的东西，你真的能够永远坚定不移的坚守下去而不异化吗？
你能否肯定的说，你能够完全以自己的理性来掌控自己的欲望、且绝对不会被欲望操纵吗？

怪物曾经也可以是一个人类，但怪物终将会是怪物。

# 2

人之所以存在自我约束，是因为每当你逾越规则的时候，社会与大自然就会形成一种力量来对抗你，无论是受伤还是死亡还是谩骂都是一种力量，让你感受到自己被惩罚了。 问题是，当人类成为了连你打个哈欠都会被吹死的东西时，你便已经不可能再被约束了。 无论其最初的想法是什么，当生命在你面前脆弱的好像泡沫一样，你最终会失去对于生命的那种敬畏感。 无法被约束的怪物势必不会再被规则所约束，无论身体还是心灵，尽管这个时间长度可能会很长，但终将会到来。 你要明白，它根本不需要在意人类的反抗，甚至可能会形成这么一个局面: 畏惧、恐惧于它的力量的人类自发形成一个庞大的国家，竭力试图讨好它。 而不甘于此的人类则在世界各地掀起反叛，但它根本不会去管，任由那些人类叛乱，只有当那些人类形成了一定规模之后，才悠闲的走过去，一巴掌毁灭整个反叛势力。 反叛，于它也是一种有趣的游戏。  你要明白，只要你愿意，那些跪倒在在你面前的人类就会倾尽一切的去构建一个你想要的世界。 不管什么样的世界都可以，只要人类做得到的就都会满足你，以换取自己生存的权力。 整个废土一般的世界之上，也照样能够搭建起一个虚假的繁华都市……只要你能够无视都市之外的人吃人、无尽尸体和死亡的话，这个世界依然会很愉快的。  至于亲人等等，人是社会性动物，人依赖于社会生存，但一个可以无视社会而生存的人类，无论其秉性如何，其社会性已经注定被剥夺了。 伪装的社会性，是无法长久存在的，终有一天会彻底扭曲异变。 常人还能有人劝导、告诫、乃至社会舆论、法律制度能让它修正，但一个无人能改变的生物，其很容易偏离出“通常人类的社会观念”，在自认为的“社会”里越走越远。 举个例子，在这个回答下，很多人表示要努力让这个世界更快的迈向繁荣昌盛的社会当中，这自然是出于正义。 但是，如何算是繁荣昌盛？谁繁荣昌盛？哪个国家？哪个民族？如何对待纠纷？诸如等等的问题，最后显然就是由它自己“自由心证”了。 而偏偏没有人能够劝阻它，所以……它个人自认为的“社会”里的“正义”，最后会变成什么样子？ 答案不言而喻。 我要说的是，当存在这么一个怪物的时候，你就已经不能指望用“常人社会”里的观念去约束它了，能约束它的只有它自己，能困住它的也只有它自己的观念，世界于它，也只是“它眼中的世界”。 而一个只属于“一个人的社会”，其显然不会和“多数人的社会”相同。 执意而为的怪物，最终依然是怪物。 所以，整个人类世界都将会按照你的需求被迫发生转变，原有秩序的破坏是无可避免的事情。 是废土还是美好新世界还是其他，对于人类而言没有意义，反正社会是什么样子，已经由不得它们主宰了。  想太多，人类做不到在不激怒对方的情况下让对方心甘情愿的自杀。 人类的力量终究是有限的，智慧也是有限的，只是一个脑容量1450ml的碳基生物，不是智慧无法估量的怪物。 你口中的这种人物，世界上还从来没有诞生过。 当你试图用语言蛊惑这个怪物的时候，最大的概率就是你把他激怒了，然后他把你连同周围数百公里内的城市群一同拍成粉碎。  是的，我从来没有否认过会有反抗，所以呢？然后呢？ 一座城、一座城的屠过去，一个国家一个国家的灭过去，你反抗一万人就杀十万人，反抗一亿人就杀十亿人，杀到全世界处处都是死尸、杀到处处荒无人烟、杀到人口百不存一…… 你真的相信所有人类就都会为了自由连死都不怕？ 别傻了，更多的人只会在恐惧之下自发形成怪物的仆从军、去清剿那些胆敢激怒这个怪物的反抗势力，形成一个以怪物为核心的庞大势力。  我说了，我没有否认会存在反抗，但之后呢？ 人类会为此而反抗到死绝最后一个人，全人类彻底死绝也要为自由而战吗？ 现在有一把枪指在你的头上，让你跪下，你跪为了自由和尊严而宁死不屈吗？或许你会，但大多数人会屈服于对死亡的恐惧之下。 一个是当你反抗之后必然会死，一个是你不反抗，在濒死境地当中、在废土之上挣扎求生，对于常人来说，选择哪个是再清晰不过的了。 古代王朝，造反的固有，从来没有断绝过，宋朝有多达八百多起有记载的起义，清朝超过三千多起，但绝大多数依然还是镇压了之后，剩下的人继续选择苟且偷生。 有造反，不代表是全员造反，全员造反的唯一可能性就是政府彻底失去了力量，造反已经不需要付出什么代价了。 但对于这么一个怪物而言，它的力量并不是基于民意而诞生的。 秦汉会因为造反而灭绝，法国王室会因为造反而断绝，但这个怪物不会。 你反抗多少次，最后都只是人类自己死去，没有赢的希望。 怪物不在意你反抗还是不反抗，是真心臣服还是假意臣服，那些东西对于怪物没有意义。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

哲學問題呀~！
如果說世界上有這個無敵的傢伙存在，那麼唯一能對抗這傢伙的，就是讓他不高興。
就如同所有哲學問題的答案一樣，需要幾個前提：1.這個無敵的傢伙要能溝通，至少是人類的行為能影響他的行為。
2.存在會令這傢伙不高興的事。

或許大家會覺得很好笑，這些前提不就是答案本身嗎？那幹嘛還提？都給你說就好啦！

其實哲學問題在提問時，總是有表面的"前提"與隱藏在提問者或潛在回答者心中的"前提"，透過這些前提，才能進行討論。

原提問：如果你拥有以一己之力碾压全地球战力的能力，你会做什么？
根據回答，可能的潛在前提有：
1.自己有不須依靠他人的存活能力，甚至把地球弄壞也無所謂
2.自己能接收其他人類的訊息，或者會被人類的行為影響
3.自己曾經是個人類，或是知道人類是什麼樣的生物
4.這些人類在無敵的力量下最後都喪失理智，變成家畜般絕對不會反抗的存在

所以由這些前提，最後就是：
世界毀滅(因為怪物不需要依靠他人)
統治人類(因為怪物理解人類，人類也不會反抗)

然而，每個人心中所想的前提各有不同，因此會有不同的答案，所以這個問題也是沒有絕對的對錯，或許過了百年又會有不同解釋。
