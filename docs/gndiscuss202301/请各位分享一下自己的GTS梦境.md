# 6

做过至少三个，
第一个是梦见我自己在一个不知道什么地方，反正当时我在闲逛，突然，出现了个女孩，挺漂亮的，我过去撩她，然后她就炸毛了。她对我说了一些狠话，我不信，然后她就变大了，说什么一根毛都能毁灭我。然后我开始逃跑，她在后面追（我不知道为什么我一跑她就没追上），后来我慢慢跑不动了，停下来。她就把我抓到高处丢下，我就醒了。
第二个是外星虫族入侵地球的，我以第三人称观看全程，看着主角们死守避难所，偷溜进废弃医院拿药，回来却发现避难所都被一种能吃金属的虫子吃完了（在那之前其实还有两头大象逃跑，也被它们席卷而过，枯骨不留）。整个梦境我是以第三人称看的，主角团队有男有女，梦境的结束是在主角团队看到避难所被吃转身就要逃难，我就看着女主沾血的运动鞋覆盖了我的视角，故事戛然而止，醒来后想了半天才明白我原来只是虫群大军里小小的一员……
第三个是最贴近gts的，我梦见我回到了高中，那天很大的雾，然后老师到了上课时间也没来，几名同学出去找，然后没有回来。后来有人发现，除了我们班，其他教室完全没人，有几名同学觉得今天不是上课的日子，就像回去，我也想，所以边说笑边往校门走。然后我看着他们走出去，一下子消失在雾里，我突然意识到了什么，慢慢撤回了教室，想和大家说这个事，让他们别出校门。然后，教室里的人也不见了，我整个学校里跑来跑去，从一楼跑到天台，完全没人。于是我走上了操场，准备去看看宿舍楼的情况，路上却看到一个不认识的同学，失神地往教学楼走。和他说话他也不回答，我转而担心他，于是跟着他，怕他出什么事，然后，他不顾我阻拦出了校门，也消失了。
我回到教学楼，看着无人的教室，百无聊赖之下爬上了天台，上面的风很舒服。然后我想到了当时是在宿舍楼看到人的，所以我从天台上望向宿舍楼，然后……
我特么发现那个宿舍楼……是一块熟猪肝！
后面我逃到了操场上，一转身，发现教学楼也是一块猪肝，同学们都被困在猪肝表面的汁液里！我不知所措。
然后，天空中出现了一只巨手，一下子叉起教学楼猪肝，送进那个巨人嘴里，接着又吃掉了宿舍楼！之后我身处的操场也被叉起来了！这是一片白菜！被送到那巨口面前，我才看清那个巨人的脸，是一位我不认识的美丽女子，而在女子身后，还有近十名美貌女子，她们都在教室里品尝菜品。
然后她把我所在的操场菜叶送进嘴里，梦也醒了（一身冷汗）。 <title>7</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
