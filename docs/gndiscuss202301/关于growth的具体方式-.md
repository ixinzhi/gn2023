# 1

作为一个growth控 其实我还是蛮在意成长的方式的。
比起简单的突然莫名变大/随意变大
其实对各式各样的成长原因/方式更感兴趣呢
我个人的话 便喜欢中世纪魔法的设定 因为刚好又是残酷系
是很喜欢通过杀戮获取能量来成长的魔王设定就是了
偶尔也觉得通过现代化甚至未来的科技来达到growth也很酷
不知道别的growth控有没有别的什么有趣的想法呢 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 3

我曾經想像過在魔法的世界裡，有一隻特殊的史萊姆，通過吃「東西」，吸收消化後，變大而不是分裂成更多的史萊姆寶寶。
這隻史萊姆，擁有智慧，可化成人型，久而久之，通過捕捉，消化，自我成長，最後慢慢的變大，最後成為全國家的公敵。 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
