# 1

    昨天做了一个梦，故事发生在一个很大很大的大厅，这个大厅经常发生灵异诡异事件，其他事件先不提，直接主题，缩小事件
    第一次缩小事件：大厅里出现了一个一米高的透明实心“花瓶​”，这个花瓶在悄悄地移动，几乎没有人注意到它，我发现花瓶路过的区域很多人过了一小段时间后就悄悄缩小了（大概一毫米），被其他不知道的人路过踩死了。我全程远离这个花瓶（毕竟这里有很多男人不全是漂亮妹子，其实吧还是惜命，理性超过性癖，死得太快了没意思？），大概十分钟后，花瓶事件结束了，被缩小而没被踩死的人变了回来。
​    第二次缩小事件：大厅里排起了蛇形长队，好像是大家都要去办什么事，这时花瓶又出现了，移动的频率和速度好像都增加了，我从队伍中偏离了出来，打量了人群，悄悄往漂亮妹子多的地方凑一凑（到底是要性癖还是要命，而且瓶子过来的话妹子也在这个区域也可能缩小啊，梦里好像没想到这点），然后我看见了队伍里一个亲戚，她疑惑地忘向周围人群，说怎么感觉队伍里忽然有些人不见了，我走过去跟她说：“一会儿无论发生什么事，你只管找一个安全的地方躲着，躲上十分钟后就会结束。”（居然不直接说明白，是何居心）
    然后队伍里一个妹子可能也是第一次缩小​事件的目击者，她说出来了那件事，不过她说是会持续12分钟，恐慌在人群中蔓延，大家都散开了，警惕地看着四周。有人在哭泣，可能是朋友被踩死了。
    过了5分钟，组织排队的那个人喊了一句：“5分钟过了，已经结束了！”因为之前的队伍散了，现在没有人排队，于是所有人都蜂拥而上往那个窗口冲去。又有人大喊：“还没结束！别往前去！它在那里等你们呢！”果不其然，在之前排队办事的窗口侧面的阴暗墙角里，那个瓶子正在冷笑着注视着人们（瓶子当然不会冷笑注视，我写出来时强加的表情，如果它有表情一定是这样。）人们如坠冰窖，然后一哄而散，瓶子高速地追了过来(当然也可能只是随机移动)。
    所有人都在疯狂逃窜，我想告诉他们不要跑，只要所有人都不动，尽管被缩小了也不会有人被踩死，时间一到就会还原，但谁会听呢？我也只好跟着一起逃窜。
    有人发现了去二楼的楼梯，​有人喊着瓶子可能不去二楼呢，十几个人挤着往上爬，我也跟了过去，第一个到二楼的人大喊：“二楼是假的！是封死的！”
   没有二楼，只有楼梯，这时我正被堵在一楼和二楼直接折形楼梯的弯折处，一不小心失足跌落。
  与此同时，不知谁在一楼踢了瓶子一脚，瓶子飞了起来，飞到半空正好迎面撞我脸上？？？？避无可避。
   当时我内心可能有一万匹草泥马奔过​，然后万念俱灰，心想这回死定了。
