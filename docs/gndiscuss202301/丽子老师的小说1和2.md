# 1

在M兄的介绍下，看了这两部,感觉丽子老师系列的都很劲爆啊!给大家说说情节.
故事1:丽子老师在乘电车，这时,她的臀部被色情宇宙人摸了，愤怒的丽子老师马上巨大化100米，企图踩死藏在电车里的宇宙人。她先是踩了4号车厢，当宇宙人跑到其他车厢时丽子老师有用20米高的高跟鞋踩坏了其他几节车厢，当然里边的小人都无法幸免了.随后，丽子老师用巨手握着最后一节车厢握溃,企图把宇宙人和小人一起捏死.但最后，所有的小人死光后，宇宙人跑到了城市中心。丽子老师随后跟了过去,一路狂踩,造成了大量的地球人伤亡。最后，当宇宙人企图跑掉时,丽子老师用什么法术把宇宙人挡了回去。最后，用手捏着宇宙人在丽子的阴道里玩弄。宇宙人就这样被丽子老师的淫水所淹没最后死亡。而这次消灭宇宙人也付出了城市的毁灭和1500人的死亡.

故事2:丽子老师在一个珠宝店门口和男朋友约会。男朋友刚到，丽子老师就觉得这附近有怪兽出现要(ULTRAWOMAN的INSTINCT),于是，丽子老师马上巨大化,并且向在手心里的男朋友告别。男朋友走远后，丽子老师开始踩杀附近的一切可疑东西，包括人类和建筑。最后,她确定宇宙人和一个公交车附身了。宇宙,丽子老师企图踩坏那个公交车，而外星人控制的公交车则一路狂奔.最后，丽子老师一扑，用身体压住了公交车。这时,丽子老师本身的SADISTIC欲望大发，她把公交车放在引导里反复抽插.需要注意的是，公交车里有丽子的三个学生，这三个人根本不知道这个车被外星人操纵。随着丽子老师自慰的加剧,大量的淫水进去了车内，小人门的生命受到威胁。丽子自慰结束后，发现公交车里有人.倒出来一看，是她的三个学生。而这三个小男人也发现原来巨大化的女巨人,就是他们的丽子老师.对于发现自己真实身份的小人，丽子的办法一般是消除他们的记忆;如果是认识自己的人，丽子一般直接弄死他们。但是这三个人是自己的学生，所以丽子原谅了他们.但是威胁他们，不可以说出去，否则就要弄死他们.并保证以后可以缩小他们并且和他们玩SM,比如把他们放到自己的阴道里.
