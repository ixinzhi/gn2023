# 1

歸結起來，巨大娘的喜好其實可以歸結為對“巨大”或“強大”的喜好，除去性癖成分而言，實際上喜好或崇尚強大其實是群居動物的本性。
正如人類在遠古時期生產水平低下時的生殖崇拜——如極力刻畫女性的形象，這些女性一般有著巨大的胸部、腹部；如男性和女性在田野裡進行交配，以祈求來年豐收等。
喜好或崇拜巨大娘的行為，實際上也是出於一種對於“強大”和“巨大”概念的喜好，只不過摻雜與性癖後，性癖為女性的人更喜歡巨大娘，性癖為男性的人更喜歡巨大男，如果雙性戀則都有之，僅僅只是看你比較喜好男性還是女性。
之前我犯了形而上學的錯誤，我僅僅在意了“喜好性癖對象強大的人大多是M”而界定了gts的愛好者重女性不佔少數。
考慮了大部分群居動物對強大的崇拜特點後，我確認喜好巨大娘的人大多為男性，只有雙性戀或同性戀的女性會喜歡巨大娘。
所以從這個角度來說，綜合之前的帖子里多說過的，我認為把“強大”和“巨大”所體現出來的巨大娘作品才是優秀的作品，當然，我說的並不是越大越好，比如幾十米對幾萬米，我指的是能體現出巨大娘“巨大”和“強大”，能夠展現這兩點。
只要能夠體現出巨大娘“強大”和“巨大”的特點，無論巨大娘是什麼樣的大小，它都一定是優秀的作品，能夠引起大家共鳴的作品。
之前我看過由壇友xnr氏所提出的gts作品三要素論點（https://giantessnight.com/gnforum2012/forum.php?mod=viewthread&tid=8040&extra=page%3D9&mobile=2）
總的來說，我認為：
①美雖然是所有人都追求，每個人心裡也有自己對美的標準，但美並不是巨大娘作品所需要的，更不是評價巨大娘作品最重要的要素；如果巨大娘作品是一道菜，美只是這道菜的一個可有可無的調味料，比如加不加花椒粉除腥提香。
②正如我之前所說的“強大”與“巨大”，真實性也只是這道菜的一個配菜或者一道調味品，但是讓人有身臨其境的感覺確實是很重要的，良好的描寫能讓人感受到巨大娘的“強大”與“巨大”，所以我認為“真實性”這一點沒有必要從作品的“表現”中單獨提取出來。
③“互動性”同樣屬於作者需要考慮的作品中“表現”的一部分，所以不再做贅述。
之前xnr氏也發表過一篇討論gts屬性的文章，對我形成這篇文章的論點很有建設性意義，在此感謝。（https://giantessnight.com/gnforum2012/forum.php?mod=viewthread&tid=14643&extra=page%3D9&mobile=2）
因為出發點不對，即我認為巨大娘（giantess）不屬於性幻想，我認為巨大娘的喜好源自物種基因中對強大的崇拜，所以我不對這篇文章進行提出意見。
歡迎對我的論點提出任何意見和建議。
