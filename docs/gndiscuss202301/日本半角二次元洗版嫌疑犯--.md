# 1

嗯，因為我現在在等一件事情，必須坐在電腦前面，沒事幹卻又不能離開，無聊之下乾脆來跟大家分享八卦。

這是來自日本的消息。必須說明的是以下的指控並無百分之百的確切證據，不然就不叫八卦，叫公幹。

或許板上有人知道，日本著名的BBS：2ch上面有好幾串GTS相關的討論，當然都是在成人區。這些容量最大一千條留言的討論串內容不一，有的純聊天，有的偶爾會貼短文，有的則會有繪師進駐，三不五時來灌溉一下這貧瘠的GTS大地。照理來說，不管是寫文章的還是畫圖的，創作者當然是越多越好；因此今天會有這樣一個聲音、指名道姓衝著一個繪師問罪，在我看來是非常不可思議的一件事。然而不幸的是，這些指控似乎也非空穴來風。

這位遭受責難的繪師，就是以前曾經出沒在G-fork留言板、線條簡潔有力的NAO先生。老實說他是我很喜歡的繪師之一，背景不怎樣但女孩子畫得很甜(個人感覺)，而且有在持續創作。

那他到底是幹了什麼事情？

最近的狀況，要從九月二十一日前後算起。2ch有三個GTS圖畫相關的討論串，遭受了從過去就有的大量惡意洗版攻擊。由於2ch每個討論串固定容納一千則留言，所以一直用無意義的文字洗版，造成留言數超過一千條的話，該討論串就會關閉。雖說一些傳統討論串都會固定重開，但這樣的行為還是很惹人厭，畢竟這令討論串完全失去了功用。而九月二十一日的洗版攻擊更是達到了頂峰，連重開的新討論串都瞬間湧入了惡意的廢水。這下子終於讓2ch的使用者抓狂了，開始追查線索找兇手。當然這中間是有一些技術可以使用，不過光就廢水的內容來看，2ch的偵探們發現惡意灌水者留下的內容，是某個相當私密的繪畫chat中流出來的。簡單講就是，若不是參與過繪畫chat的人，基本上是不會知道那些事情的。

所以，在二十一日的當天晚上，該繪畫chat的成員們開始討論誰是內鬼。他們從廢水湧入的時間帶(早上七點到下午一點)來判斷，將嫌犯範圍縮小到包括NAO在內的數人，但其中又以NAO的嫌疑最大(據說他每天下午兩點固定出門，而他一出門就沒有廢水)。為了查明此事，2ch的偵探開始運用一些技巧(這裡我就不是很懂了)，將嫌疑犯一個一個測試ID(2ch留言都會有個人獨特的亂數ID，沒有人會一樣)，結果在試NAO的時候吻合了。

答案出來了，ID吻合，證明NAO就是大量灌水的兇手。

為了保險起見，繪畫chat的成員決定暫時封鎖NAO的進入權限一天。不過就在隔日的凌晨三點半，GTS討論串上又出現了一些怪模怪樣的留言，於是結果就很明顯了……

# 2

想想這主題也挺遜的，而且原本就只是為了打發時間才寫。本來想刪掉，可是打了一大串的文字，刪掉又太可惜，就乾脆轉到這人少的地方來。

那麼繼續話題吧！在繪畫chat封鎖了NAO的進入後，2ch的討論串上開始出現了密集的攻防戰：一個未具名的ID在耍白目(曰：歐耶，少了一隻NAO蛆虫，這個討論串一定會變得更加美好)，另一個掛著NAO名號的ID則在喊冤(曰：等等等等，上面在說什麼？)，問題是手法拙劣到讓人一看就知道是自導自演，令2ch的人們看了是又好氣又好笑。

另一方面，2ch扒糞的功力可也不容小覷；根據一份調查指出，過去兩年來偶爾會爆發的GTS串廢水湧入事件，其犯案的模式(留言時間帶、內容等)與本次事件幾乎相同。同時亦有人指出本次事件爆發前，NAO曾多次遭受「畫技沒有進步」的批評，進而研判這是一起典型的尼特族受辱報復事件。

注重「和諧」的日本人當然不會放過搗亂被抓包的傢伙。除了該繪畫chat日後將嚴密監控、防止NAO另外掛名偷摸回去之外，還大聲呼籲要各BBS管理員小心「幾乎不可能沒有」的惡意灌水報復，甚至還要求真實世界的主管機關小心預防衝動殺人等可能的犯罪案件……

Well，我已經說了，這只是一則八卦。當然2ch偵探那咄咄逼人的德性基本上也很欠揍，不過事出必有因，灌水是真的，自導自演的留言也都還在，連我一個外國人都能八九不離十猜得出可能性。只能說人在做、天在看吧！同時也證明日本的環境確實比較嚴格，能這樣大方把一個創作者給犧牲掉，雖說他畫得不是最好。

目前的最新狀況是NAO的親衛隊跳出來辯解了，強調ID還是有不同人但卻一樣的可能性，例如住同一棟樓、同樣使用一條線路之類的……不過2ch目前一律把這些聲音視為NAO的自導自演就是了。總之NAO似乎已經轉到GTS Shrinker Uploader貼圖，其實這樣就好了嘛，那種想創作給人看的心情我又不是不知道，何苦把自己在2ch的名聲搞臭呢？或者去Giantess City也不錯，他的程度在那裡可以排上級了。
