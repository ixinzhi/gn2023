話說，過去我曾經有在作夢途中、突然意識到自己是在夢境的這種體驗。
很多時候我是直接醒過來，也有時轉變身分，成為觀察者旁觀夢境。

但有少數幾次，我變得可以掌握自己的夢境，可以控制自己在夢中的行動。
那股感覺在當下確實是挺爽的，不過醒來之後能記住多少，就是個問題。
而且過度干涉夢境的話，結果很可能還是直接醒過來，論體驗的話還不如真正做一個好夢。
所以，試著在睡前構思一些美好的邂逅，或是看點書，
同時千萬避免在激烈運動或觀看電視後馬上睡覺，尤其禁止看動作片或恐怖片。

網路上也有很多方法幫人作夢，我覺得都是可以嘗試的，或許能幫助各位在晚上有理想的好夢。

还有三要素，凝缩、置换和表现力...还有梦大多是来自于过去的经验...
综合这些的话，随心所欲的梦还是会有些困难的...还有醒来之后还记得多少是个问题...
尝试过清明梦，大部分时候都失败了，成功只是偶然...不管怎样，梦还是梦而已，最终面对的还是现实。 <title>7</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

所谓清明梦，正确说法应该是【清醒梦】，是指人们在做梦的时候还保持意识的清醒。这时对梦境会有更加清楚的感觉，有时甚至可以直接控制梦的内容。完整的从开始到结束的这种过程就叫做清醒梦。但是我认为这有可能是人体的一种病症反应，是大脑皮层神经异常反应的结果，很多时候与催眠术有关。所以不建议大家试图自己练习这样做梦，如果你一定要体验，可以去买一种【清醒梦眼罩】的电子产品。
你们想要得到逼真刺激的GTS视觉效果，与其绞尽脑汁练习清醒梦体验，还不如练就你自己丰富的想象力，在睡前闭上眼睛让脑海中构建一幅美丽的场景，以自己的标准想象出一个最美丽动人的女巨人，这样岂不美哉！ <title>16</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
