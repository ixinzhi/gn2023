# 1

据说Enty是当月付款后只能看当月的作品，
那么Patreon的机制又是怎么样的呢？是只能看当月的，还是以前的都能看？
————————————————————————————————————————
问这些问题主要是因为想看寺田Patreon的图片，
但是注册了一个Patreon账号后不知所措。
（以为支付宝全世界通用的我真是个⑨）
因为没有境外支付的手段，
所以就想在论坛里找个志同道合（有境外支付手段）的人合购。
一份的钱，双份的快乐。
————————————————————————————————————————
寺田这个月更新了很多巨大男图片。
不过我对寺田的巨大男本（包括伪娘）没有任何兴趣。
所以，如果Patreon像Enty一样当月付款后只能看当月的作品的话，
我也就不打算合购了（当然主要是因为我比较穷）。
但是，如果以前的图片都能看的话，
我觉得这一点付出还是值得的。

# 2

Patreon要看收費依據，並不是說成為訂閱者就一定能看到全部的東西
不過基本上只要是同個模式的訂閱，不只當月，連同過去所有的作品都能看到
以寺田來說，他只有一種收費模式，所以基本上你只要成為訂閱者，所有他的作品都能看到

# 3

寺田的Patreon和Fantia两者内容是没区别的
但Patreon能看到以前的图
我先在Patreon上先赞助了寺田收了以前的图后来就转在Fantia上赞助了
（虽然5刀和500日元差距也不大
如果从现在开始订阅寺田的话Patreon应该最好的选择
