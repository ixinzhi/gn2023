# 5

合理黨嗎……？不錯的稱呼，我覺得我也應該是這一類的人。

不管故事中的GTS與小人之間有著怎樣的互動，「劇情合理」是身為作者所該秉持的最高指導原則。一個沒有說服力的故事只能是讓人看過就忘的隨筆或散文，但偏偏GTS界有很多作者太過於抓著自己的性幻想不放，讓筆下的GTS毫無理由地一登場就開始破壞、踐踏或吃人，或是被縮小的男人不管怎樣都只能遇到天生就有虐待狂的S女；當然這單刀直入的寫法可以是一種風格或共識，但看久了誰不會膩呢？

回到主題。GTS處於弱勢的一方當然沒有什麼不可以，要讓身高達數百米的女巨人可憐兮兮地躺在地上讓小男人輪X也很OK，問題是劇情要合理。我不希望看見一個超人在故事一開始的時候就把GTS打倒在地上讓大家開始玩病棟遊戲，但若把GTS的個性設定為原本就膽小懦弱或根本天生就是個M女，就我個人來說是可以接受的。

--
不過，一般來說表現得太囂張的主角（單靠著臉與性器官就可以征服天下的那種）對我來說是不具有閱讀吸引力的。即使是一篇主打弱勢GTS的故事，我也不希望在文中看到這種白長了腦袋與四肢的男主人公，像這樣的文章我自然是不會多花時間去看的了。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 6

但丁說的好啊，劇情才是王道！
我的希望是GTS的能融入正常向作品中，向超時空要塞致敬！
至於我個人的見解是，從人類放大的GTS必然會保持著人類的感情和價值觀，站在人類的角度思考，而不是瞬間變成壓路機

至於男角色，也不能僅是個符號或某器官，必須要有強烈的個人魅力點，和同樣明顯的缺陷（對於完美存在我很反感） ，武力值高低無所謂（個人喜歡高點） ，不過智力值不能低於平均，情商則要高 （超喜歡但大的法雷爾，柴可夫，莫扎特等人的）

至於弱勢的GTS的愛好，我認為我不算大男子主義者，但我是個人類沙文主義者，我不喜歡接受人類會全面不如某外族的設定（即使被征服，人類也不甘成為奴隸，總有領導者出現反抗） 。
