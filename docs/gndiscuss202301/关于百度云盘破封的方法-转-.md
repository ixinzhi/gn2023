# 1

百度云盘不用多说，除去封杀一些资源之外（绝大部分种子还是可以洗白再下载的），我觉得是目前最好用的网盘，没有之一。
但是现在很多资源连接点开就看到度娘一脸楚楚可怜的说：你来晚了.....
你愤怒了没
你砸键盘了没
你重新找资源了没
蛋是，网友的力量总是强大的，总能找到一些方法让双手穿过度娘的秀发，嗯，不是绝对的，方法只适用于文件在还分享列表但分享链接失效的文件。
下面说重点：
以下面链接为例：链接是[http://pan.baidu.com/share/link?shareid=76297&uk=3238350839](http://pan.baidu.com/share/link?shareid=76297&uk=3238350839)
打开看到的是什么？是不是度娘一贱贱的说，哎呦喂，小样来晚了吧，下次赶早哟~undefined（我了个去，这图片怎么那么大！）
重点不是连接失效了，重点是"uk="后边的数字3238350839
这是百度的默认网盘分享主页
[http://pan.baidu.com/share/home?uk=189904483](http://pan.baidu.com/share/home?uk=189904483) 把上面那个数字3238350839替换到尾部即可，由此得到。[http://pan.baidu.com/share/home?uk=3238350839](http://pan.baidu.com/share/home?uk=3238350839)
分享者的所有分享都在列表里了，慢慢找你需要的资源吧
括弧说明：不是所有文件都有效，反正看运气吧
以上的方法似乎只有对刚封不久的文件才有效，时间长了可就悬了。![](img/a1181086053f36e608ffedb11b712ecc.png)
