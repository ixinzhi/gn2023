# 1

虽然自己不知道准确的原因，而且也只是感性地感受而已，但是我觉得这对于以后论坛如何保持兴旺是一个用一定重要性的讨论。

我自己先来说一下自己的感想吧。
我最大的一个体会就是论坛的画风。
没有错，就是这么肤浅的东西。但就像是山寨手机和正版系统的区别一样，对于一个论坛而言，论坛活力与与论坛的颜色搭配，是否有广告，首页大图是否俗气，是密不可分的。
可以注意到我刚才用的都是几个贬义词，论坛经历的一次长时间的萧条，正是在升级了论坛服务器之后，一下子风格变得和国内杂乱的色情资源论坛，乌烟瘴气的QQ登录，水贴严重到几乎混沌的那些网站非常相似。可以说我那时候的感想就是，还不如不升级，尽管能有了许多新的功能，但是瞬间，从原来的“干净，整洁，世外桃源”的风格上跌落了。
欣喜的是，去年论坛总算换成了清新的蓝色底色。而且十分明显的，不再需要严刑峻法，不再血淋淋的需要水贴禁言公告，回帖的质量和创作的热情明显提高了。我不敢说这二者之间有必然的联系，但是对我自己而言，我感觉得到我在论坛的归属感一下子回来了。
已经很多年了，论坛伴随着我的青春一路长大，这里已经是除了我以外，很多人生活中重要的一个部分。不可避免的，老一代的创作者要把接力棒继续交给第二代了，我希望在这个关键的几年，借助这个帖子，各位老司机能在这里畅言自己对这些“因素”的见解，我想一定是很有裨益的。
