# 1

我最近在续写一篇老文，文章大体就是女生变大然后大搞破坏的压路机文，简单粗暴型，所有常见的爽文属性，我都尝试的写了一下。接下来我想向女主大规模破坏建筑开始着笔继续下去
但是我有点迷茫，我该以女主的视角描写她破坏建筑?
还是以一个幸存者视角来描写女主巨大化后破坏力有多惊人，就像观看灾难片那种

大家觉得，如果看女巨人破坏城市的文章，是希望以女主视角来代入，还是说以第一人称来感受女主带来的破坏力?
这两种哪个才能让你们有感觉呢?

# 2

我感觉只要内容充实，不同视角对文章质量的影响应该不太大。
要说差别的话，女主视角可以进行心理描写吧，比如觉得破坏毫不费力，以及对触感的一些奇妙比喻，像踩纸盒的感觉之类的；第一人称则可以关注一些女主无意识或不在乎的部分，比如头发里的碎屑、转身时的破坏等，而且在描写上也可以添加更多细节，毕竟是身临其境的视角。

似乎女主视角的文比较少，所以个人更倾向于女主视角一点。当然两种视角都有更好了![](img/292cac21d95f8816694afbc0a4294746.png) <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
