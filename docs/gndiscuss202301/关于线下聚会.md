# 1

不知道有没有人组织过线下聚会的，但是我觉得有一个方法可以组织

1.首先肯定是同城聚会，所以先要有信息发起人，考虑到这个爱好极其特殊的属性，建议在论坛等平台上选一个愿意做信息中间人的，然后通过邮件等一对一方式联系这个中间人发起在某个城市的线下聚会，而考虑到隐私问题，最好中间人不要是聚会本地人

2.场所要是公共的，私密的，所以可能KTV之类的地方会比较理想，当然本人没有什么社交方面的经历所以不确定具体哪里可以线下

3.人员聚集的隐蔽，应该设计各种人员见面对接的方法，比如设定一个主题，所有的对话在这个主题下展开，通过暗语交流对接

总之就是设计一套可以尽量避开信息泄露的方案来进行线下聚会
