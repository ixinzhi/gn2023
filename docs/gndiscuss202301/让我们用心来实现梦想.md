# 1

今天，我来告诉大家一个新概念，用这种概念可以实现我们每个人心中的潜在愿望，当然也包括GTS和SM范畴了！首先我用两个词语来表达核心思想，这就是【随想实相与道成肉身】。我并不主张唯心主义论，但是根据今天的自然科学研究成果和心理学、灵性研究的发展，这种新观念表现的越来越强烈了！下面我来为大家解释。
在当今物理学的前沿学科——量子力学的发展下，科学家发现物质的底层是能量，而能量的底层是精神(意识)。因此，一切物质在最底层都是同质相通的。人类目前掌握的【量子力学】，已经发现到物质的波粒二象性：物质同时具备波与粒这两种特性。当你放之任之，置之不理的时候，它是波；当你注意之聚焦之的时候，它就是粒。
从这个角度来看，意识所发出的意念波，能够启动能量的循环，从物质的根本层面(最底层)去影响它、改变它，这就变成很合理的事了。既然物质的底层是波，而意识又能够发出意念波，那只要我们能够转念一想，这一切的物质现象就不得不跟着解构重组、重新洗牌。这种随着你怎么想，你的处境就跟着变成什么样的情况，这就叫作“随想实相”。
你自己说过的话如果是表里如一，不是口是心非的，没有自我推翻的话，那你讲的话就是“真言”，这是一种会成真的预言当各种条件走到一起的时候它将会实现，这种情况就叫作“道成肉身”。“道”即是话语，肉身即是物质形式。道成肉身，不是以肉身得道、羽化登仙，而是你说的话最终会以物质的形式活生生的样子显化出来。
实际上，我们可以用心来创造我们自己的世界，如果你心中持续不断地想着、关注着、聚焦在女巨人或女神上面，当各种条件汇集到一起的时候，你心中所想的内容、形象就会显现出来在你面前，你说的话、写过的字都会成为现实生活中的一部分。因此，大家不要放弃和绝望，坚持自己的梦想把它贯彻到底，总有一天你会梦想成真的！ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
