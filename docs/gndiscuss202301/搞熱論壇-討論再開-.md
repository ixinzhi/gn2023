# 7

最近睡眠不足，雖然看見這個很有意思的主題，但竟然久久想不到什麼認真的回覆。下面所要講的東西有些雜亂無章，基本上是想到什麼就說什麼，請大家看看就好。

其實不光是GTS文，所有小說都需要有「衝突的場景」當作劇情高潮的一個重要環節。其中體術或運用兵器的戰鬥當然是比嘴砲或勾心鬥角來得更簡單明快，不過GTS文章因為本身先天設定的關係，讓這個戰鬥場面變得很難拿捏，畢竟女巨人光用看的感覺就很強，對吧？尤其對我們這些愛好GTS的人來說，難免會私心地賦予巨大女角色一點無敵的特質，所以戰鬥的平衡度就跟著被打亂了。

記得過去也有同好跟我抱怨過類似的問題，一個身高僅僅100英呎的女人若保持她生物學上的特性，那麼是不可能承受一輛裝甲車的砲擊的，但這些「小個頭」的GTS卻依然能在絕大多數的故事中視軍隊於無物，甚至推倒大樓的時候還能不被瓦礫割傷，當時我們的討論結果是這確實並不合理，但寫文章的作者顯然是故意如此偏心的，至少我也不會想去寫一個女巨人被人類軍隊合力打倒的文章，因為實在沒什麼樂趣。

不過，所謂幻想的世界是沒有界限的。只要表達得當，給GTS們安排一場刺激的戰鬥其實是很簡單的事，你可以試著解釋所有不合理的部份或甚至刻意忽略；一些格鬥遊戲中的戰鬥顯然就屬於後者，巨大化的女角色不過就是血條變長一點、攻擊方式多了踐踏與搥打等等，對原本就被設定有高超戰鬥能力的對手來說根本不是問題。對於不是GTS同好的遊戲製作者來說，巨大化只是一種賣點，就如simple 2000系列的雙葉理保一樣，他不需要顧慮角色的行為是否能滿足GTS性癖者的性慾，他只需要讓角色對一般人來說看起來很性感就好。但這樣的表現方式對同好們來說吸引力就不足了，甚至可以說，反而有些失望。

縱觀現有的華文GTS文章，有描述到GTS戰鬥行為的小說著實不多。其中「月之幻」是早期讓我印象很深的一篇好文，世界觀非常夠力，只可惜作者最終沒有把它完成。或許大多數的圈內作者並不認為需要在文章中添加如武俠小說一般的戰鬥場面來滿足幻想慾方面的需求，單刀直入對性挑逗的場面加以描寫顯然能更簡潔有力地生產一篇GTS文章。我無法評論將GTS小說的層次停留在這個層面是好是壞，但我自己是無法藉此就滿足創作慾的。

# 21

嗯，差不多可以說說我主要的看法了，不過在那之前我要先談點前提。![](img/ec942b6b7db075eaad5da7a4ad136d51.png)

GTS原本就是興趣、嗜好類的文化。想創造出怎樣的GTS作品，都隨作者的喜好而定。而就一個讀者的角度來說，通常大家希望看到怎樣的GTS作品呢？或許是能用最快速度挑起興致的那種、結構鬆散但相對實用的文章；一篇這樣的短文就如同一張單幅的畫作，只有單純設定上的呈現而沒有事件的前因後果，但大家都很喜歡這些畫，不是嗎？因此即使是因為一時衝動而寫的發洩作品，只要描寫得好，一樣能讓人有認同感並喜歡上。即使是看來需要大篇幅描寫前因後果與過程的戰鬥場面，憑作者本身的功力，一樣可以在短短的篇幅內搞定一切，並變成一篇讓人回味無窮的極短篇佳作。

然後是故事中戰鬥要素的問題。把戰鬥場面融入GTS作品，其實不是什麼很稀奇的點子。重點在於作者本身想創造出怎樣的場面，其次是讀者的一方想看見怎樣的場面。假如作者很重視取悅觀眾這一功能的話，兩者的重要性也可以倒過來，你甚至可以發一篇投票來試探大家的口味何在。經過一整晚的思考後，我相信一份創作的重點還是擺在作者本身的意願與熱情上，也就是所謂的「一愛天下無難事」。

好，那麼可以正式說說我對GTS創作中戰鬥場面的想法了。![](img/2a75436cdd82604688d40f3dbfcc31c0.png)

正如我所說的，任何一類GTS創作的主要素都只有GTS。當你想讓戰鬥場面成為排行第二的次要素時，我覺得身為作者其實應該要想清楚，你是想要描寫「戰鬥」，還是要描寫「破壞」？這個主題已有多人針對GTS與人類軍隊抗爭的問題發表了想法，大多數文章中的人類軍隊之所以一面倒的慘敗，原因其實很簡單，因為文章作者想描寫的是「破壞」而非「戰鬥」。被踩扁的軍人與坦克車不僅不是故事的主體，甚至也不是客體，而是背景。大部分殘酷系GTS文章的作者都喜歡讓軍隊慘遭屠殺來彰顯巨大少女的恐怖與強大，讓大家印象中很強的軍隊被打敗原本就是各種創作最常用來襯托主體的方法之一，這太稀鬆平常了，但有些人看到故事中描寫了軍隊就以為會看到戰鬥，這完全是錯誤的，請分清楚作者想寫的是「破壞」還是「戰鬥」，而身為作者則更應該要搞清楚這些事。

那麼，撇開那些被當成地面佈景的二等兵與M1戰車吧。當作者確切地認知到自己想寫的是「戰鬥」而非「破壞」的時候，站在讀者的立場，我會要求作者對戰鬥互動的描寫上盡量清楚且合理。我可以不了解這場戰鬥發生的原因，也可以接受你所設定出來的各種奇妙戰法，你大可以用一則極短篇讓戰鬥忽然開始並忽然結束；但當你描寫的主體是GTS的戰鬥的時候，無論如何我希望能看到你說故事的誠意，同時我也希望你的文章能脫離最基本的實用性的層次，讓我有真正在看一篇故事的感覺，並盡量把你想像中的場面闡述給我知道。簡單來說，你需要讓我認同你的世界觀。更高明的手法則當然是讓我融入你的小說世界並深深為之著迷，不過那是偏離主題的討論了，在此跳過不談。

那麼，身為作者的一方，該如何讓讀者認同自己的世界觀呢？其實這不是太難的事情，畢竟現代人想像力都很豐富；過去被老一代視為怪力亂神的東西，今天大家都看得很歡樂。尤其我們還是以想像力自豪的GTS愛好者一族。因此與其說要讀者認同你的世界觀，不如說你自己要認同自己的世界觀。你的設定越完備，就越沒有破綻；你帶著自信心在寫作，人家也能從文字上感受到你的自信。你唯一要做到的事情就是讓故事合理，不要顯得莫名其妙。我完全能接受一群石器時代的小野人打贏500米級的高科技女外星人，但這個戰鬥的過程請務必要有說服力。退一百步來說，你起碼要給我一個理由，例如將磨碎的毒草汁加進巨大少女的食物裡讓她腹瀉不止因而敗北，聽起來很破格且甚至不是戰鬥，但至少是一個理由，你必須要做到這些敘述才能讓文章完整。至於讀者們對文章的評價？那都是後話。這個圈子要是沒有創作，就什麼都不存在了，因此我完全不想討論如何寫出一篇好作品，現階段我只想討論如何寫出作品。

最後談談一個問題，也是我本人一直不斷在犯的一個問題，那就是文章的主體偶爾會從GTS漂移到別的地方。舉個例子的話，假如圈內有個業餘軍武研究者來創作一篇背景為近代的GTS小說，那他可能用三言兩語就把GTS的誕生場面帶了過去，卻花了超過千字的篇幅在描寫軍隊的出動，甚至可能把整個兵團的兵種組成與武裝細節都給寫了進去，然後對戰前夜的調度說明與戰術推演又花了好幾個段落，對GTS方的描寫卻僅有一句「巨大的少女仍在不停破壞城市……」。這樣的文章雖然有GTS的要素在裡頭，但請問版上有誰想看？一篇主體不在GTS上的GTS文章，除了用來滿足作者本身的創作慾之外，沒有別的功用了。假如你在意讀者的感想與文章的點閱率，那就別讓故事的主體從GTS的身上跑掉。

（不過，很明顯這是我近年來一直在幹的一件事，對各位還真是非常的抱歉![](img/6aa9b3e71108a323a0981d2b75ad6caa.png) ）
