# 4

「天頂星人」是大陸地區的慣用語，「傑特拉帝人」是台灣的慣用語，兩者指的是一樣的東西。
另，由於初代動畫有電視版與電影版兩個設定上有差異的版本，詳解如下：

電視版：男女巨人皆稱為「傑特拉帝人」，兩者共同合作、與「監察軍」作戰。

電影版：男巨人為「傑特拉帝人」，女巨人為「美特蘭帝人」，兩方互戰。無「監察軍」存在。

目前的設定是，電視版為正史，電影版則是劇中劇，是Macross世界拍出來的電影。
但是Macross對設定的綁定很寬鬆，基本上隨時可以改，所以一切都要等MF塵埃落定後才能定案。

--
M7未播放版「最強女子艦隊」中，大頭參謀有說出「美特蘭帝人」這個詞彙。
我覺得這並不違反初代電視版的設定。雖然美特蘭帝人是電影版發明的名詞，
但電影影響人類文化的例子不勝枚舉，因此Macross世界開始稱呼女巨人為「美特蘭帝人」是合理的。
