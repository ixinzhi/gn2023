# 1

吞食的喔？其實在國外GTS圈，吞食系的勢力可是很大的呦∼∼

不過這邊一直沒什麼人在討論，我想是因為缺乏領頭發聲的人物吧！如果時常有像樓主這樣的人出來貼主題討論、或是分享資源，我相信版上的吞食同好都會浮出水面的，數量一定不少。

而我本人雖然不是吞食本命，但我還是會看這類的東西，也一直很喜歡。我很欣賞的一點是：吞食系的劇情可以是溫柔的吞嚥、也可以是Hardcore的咀嚼，彈性頗大，不像crush難以擺脫暴力之嫌，這或許也是吞食同好常常自成一國的原因，幾乎能說是與GTS不同的類別。至少我在外國網站看得都是如此。

說穿了，GTS是一個非常廣域的名詞，裡頭包含的類別是極多、極細微的。我們同好都是在這樣一個大範濤下尋找自己的本命，那是可以很細、細到給人有龜毛感覺的。但吞食確實是一個能攀上這龐大項目頂端的類別，我相信樓主絕對不會是孤單的。^^

最近大陸有個新論壇「入腹」，樓主或許可以去看看？但依我個人的感覺，那邊似乎是專題討論吞食系中的某個分支……似乎非常專注在吞下去之後的場面描寫了。能專精到如此程度也不簡單就是∼∼ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
