# 1

昨晚塗潤膚霜的時候，注意到了3-甲基吲哚這個成分。
3-甲基吲哚又名糞臭素，當濃度高的時候會散發出動物糞便的味道，濃度較低會釋放出茉莉花香，再低一點就是水果的淡淡的清香味。
於是我想到一個關於某種成分的設定。
每個人體內都有不同程度的某種成分X，引劑N可以觸發這種成分，導致人變小或變大。
當濃度在0.1%以上時，被觸發的人會變小，濃度越接近0.1%就會越小。
當濃度在0.1%以下時就會變大，濃度越接近0.1%就會越大。
當濃度剛好不偏不倚為0.1%時，這個人可以自由掌握變大變小，而且想變多大就多大，想變多小就多小，沒有限制。
所有男性天生都具有大於0.1%濃度的X在體內，所以他們無法變大。
極大部分女性都小於0.1%，有一部分大於0.1%，而只有那麼幾個人剛好等於0.1%。
這是我昨晚想到的設定，也許可以應用到一些文章裡。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
