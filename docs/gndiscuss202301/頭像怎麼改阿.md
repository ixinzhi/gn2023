# 11

沒錯，不過我只是想解釋給他知道說「系統不可能因為你放上太大的頭像圖片而影響版面美觀」。
但我沒說到會自動調整確實是我的疏忽。

同時因為會自動調整頭像大小的關係，你放上太大或太細緻的圖片，細節一定會被壓縮。
之所以不建議使用有文字的圖片也是因為這樣，因為系統會自動幫你裁切或縮小，這樣文字根本就看不到了。

基本上是想阻止那種會拿桌布來當頭像的人…… <title>12</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
