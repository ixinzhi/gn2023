# 6

我以前试过几次清明梦，对我来说是在短时间的午睡中比较常见，大概就是那种想睡又不敢睡死的状态。可能会发现睡前一直在想的白日梦越来越真实；或者是刚睡着一下但立刻恢复到了半梦半醒的状态，可以意识到开始做梦了（有种白日梦被刷新的感觉？）。
我感觉清明梦和白日梦的区别只是清明梦没有身体的触感，像在飘着一样，实际体验不如平时随机到的真实梦境。因为我做清明梦时如果不断提醒自己这是个梦，容易醒过来（特征也许是身体的触感逐渐真实，意识到自己还在床上？）；不提醒又可能被潜意识牵着脱离控制，变成普通的梦。所以我觉得清明梦可能不太适合我，只是试过几次就算了。 <title>7</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 8

体验过的人在这里做个回答吧，
清明梦这种东西因人而异，每个人能做到的程度也是不同的，我确实是可以在**我自己**的清明梦里体验到GTS，但是**无法确定**其他人到底是否是可行的。

只要是我意识到了我是在梦里之后，可以自由行动和思考了以后，我的梦就会像是一个电脑游戏一样，每个区域都需要进行**渲染**和**加载**的，这个时候我在自己的**构建**出了GTS以后，通过打开一扇门，或者去另一片区域进入到了下一个场景之后，我所构建的GTS就会出现在那个场景里面。
不过这个东西还是有一些玄学的，有时候构建出来的可能并不是GTS，甚至会出现**巨大的异形一样的生物**，不过至少我还没有碰见过出现GT的情况，这可能也算是我自己大脑下的保护机制吧（……）。
关于触感的问题，以我在梦里的感觉是有的，不过肯定是不会有现实里这样的真实。

当然问题还是有很多的，第一个就是都梦到了这样的场景，自然会很难控制的**奖励**自己，但是奖励了自己以后就和小时候在梦里撒尿结果最后尿床的体验是一模一样的，所以我只是简单的体验了两三次。不过我个人的触发机制是，只要不碰到构建出来的GTS的话，就不会有任何问题。第二个问题就是不能构建太大的场景，也就是说超过50倍以上的GTS就不可能被我构建出来了，曾经有过一次出现到了城市的场景，但是下面的城市无论如何都渲染不出来，大概是大脑的算力支撑不起来了吧。
以上是我的个人体验，仅供参考，梦这种东西因人而异。如果想要体验的话可以去贴吧看一下，我是五六年前的时候在**清明梦/清醒梦**贴吧看到了一些教程，慢慢的开始有意识地锻炼，才能有现在的效果。
