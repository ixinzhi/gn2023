# 1

国内大部分其实都是恋足网站的工作室。。。好处在于对足控的人而言演员的足不会像一些欧美片十分粗糙。。。缺点就是，人貌似其实根本没懂gts和crush以及恋足的区别。。。至今为止好的作品基本没有，影像里有一部学生穿袜子的第一视角以及奴隶岛工作室的一部还有一个比较有名的婧儿，就这三部还勉强能看，但明显演员自己都不明所以，其他大部分就是坐着念台词，假装晃晃脚而已，和clip4sale上的工作室没法比。。。。
说了这么多，我就想说我鼓起勇气花了600定制了一部。。。剧情如下：服装：鞋子要求是白色帆布或者板鞋，白袜，其他服装只要是平常服装就好，不要化太夸张的妆。。。
剧情：类型是纯第一视角gts类型。开始是一个人缩小了自己躺在地上，女主经过时差点踩着他，女主反应过来，盯着小人，说：“这是什么，虫子？不对，是一个缩小人！”
女主好奇地看着小人，问道：“你在这里干什么，你知道这样多危险吗，如果我刚才不注意，你现在已经被我踩死了！你看看，你还没有我的脚一半大，如果被踩到，肯定就直接被踩扁了。”
“等等，我好像在哪里见过你，我知道了，你是隔壁班的，下课时总在楼道里偷偷盯着我的脚的那个男生，你居然缩小了自己跑到这里来了！”女主吃惊之余也很生气，“你偷偷跑进别人家里干什么！赶快从这里出去，不然我可真踩你了！”
但是没有料到小人继续躺在地上，反而很兴奋地看着她，原来小人希望能被女主踩死。女主对他的愿望十分震惊，因为那样他就真的死了而且会非常痛苦。挖苦讽刺了小人几句后，发现小人真的希望死在自己脚下，女主心生厌恶，决定满足小人下贱的愿望。
但女主觉得让小人这么容易实现愿望十分的不爽，她命令小人先舔干净自己的鞋底，袜子和玉足。在看着小人卖力地为自己舔鞋舔脚的同时，女主一直不往羞辱讽刺小人，告诉小人她早就注意到了他在学校里偷看自己的脚。
虽然自己不明白为什么有人会希望被踩死在女性的脚下，但对于像小人这种怪胎，被女生踩死也是十分适合的死法。同时还问小人是希望被赤脚踩死还是穿着鞋子踩死。最终，女主决定满足小人的愿望，用自己的脚踩死小人。一开始，女主觉得直接用脚后跟踩头就可以，
这样也基本不会有什么痛苦，但是小人拒绝了，小人希望被女主用全脚掌缓慢的碾踩至死。女主觉得十分鄙视，但也十分乐意。一开始用裸足踩在小人身上，慢慢用力，感受小人在自己脚下慢慢的粉身碎骨，可是一会后发现小人开始吐血，女主为了不弄脏自己的脚，
决定穿上鞋踩死小人。最终，女主穿上鞋，给了小人10秒钟时间欣赏他喜欢的女神的脚底，慢慢放下脚，将小人彻底踩成一滩烂泥。
备注：台词我也不知道怎么写，因为这是按照英文翻译过来的。
