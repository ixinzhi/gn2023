# 1

当时我开始手绘是受到SS的作品的启发，主要原因是他画的很多图都比较简单粗糙，但是效果却不错，这就激发了我创作的欲望。
因为我也是有一点绘画功底的，所以就开始手绘了。
从第一次开始画GTS图到现在有一年了，出的图也不算很多，但是积累了很多经验，今年肯定还会有新作品的。

所以在此想问问大家喜欢什么风格的图？

欧美的风格我就不说了，日本GTS界我个人认为画的好的除了白兔和SS外，还有Kemo，Teston等人，不过这些人貌似都没有什么新的创作，挺让人遗憾的。。。现在有在创作的新的GTS画家也都很少出精品。。。

希望大家都说说自己喜欢的GTS画家，最好能给出他的作品或者链接，也好让我有个参考和追赶目标啊。

本帖谢绝灌水，谢绝沙发板凳，希望能听到有价值的意见和建议。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 2

既然点菜就要认真点，画家我最喜欢的是HIK可能是黑白漫画看的比较舒服。

以下是我的个人建议

大小，100~10000倍最好，100~1000可以保证巨大女和小人的互动,1000~10000保证和城市的互动，小了像SM，大了没意义。

动作参考任何百合漫画和自慰动作，但一般都是很纯洁而不是很淫荡，毕竟女神是神圣的，女神只是在玩玩而已。

御姐，巫女，女仆，比较受欢迎，但傲骄或腹黑是必须地，否则没傲骄爱不上小人，没腹黑杀不死小人。

与其自己设计人物，同人会好得多。尤其是小和的胸，小澪的裤裤，614的飞机场，马上可以引起共鸣。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
