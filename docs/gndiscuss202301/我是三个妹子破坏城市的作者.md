# 1

可能论坛里有人看过这篇文章，是篇残忍系弃坑文，当时是在gtss吧里发布的，后来因为有事，停更了一段时间，等回来的时候发现贴吧已经被封了。
后来原本想在转移后的新吧里重新发布，但是之前发出去的字数已经很多，重开一贴就意味着要把之前的所有内容都再复制一遍。再加上贴吧对文字排版的不友好性和对帖子删除的不确定性，因此，每当我想在贴吧开贴续写的时候，这些问题就会把我的劲头压下去。
正因如此，我开始在文档里码字，断断续续持续到了现在。当初已经发布到贴吧里的内容如今也只占全文的前40%左右，整篇文章的字数也达到了10万5千。
而且最重要的是，我还没有写完....最近越写灵感越多，感觉有些刹不住笔，所以想把它发布到gn上持续更新。
但是10万的字数一点点的复制到这里也是个浩大的工程，所以我想问问有没有什么好办法能让我轻松一点。
还有如果有读过这篇的朋友，能否集思广益帮我想个名字。
这个账号是之前和一个网友公用的，之后可以的话我就再注册一个账号在文章区发帖。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
