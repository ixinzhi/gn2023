# 1

最近在翻译city上papayoya的长文《I，giantess》的时候遇到了比较纠结的情况

长话短说，在翻译的时候，我同时读了papayoya的《Side Effects》和另一个新晋写手aphrodite的《Yao-Jing》，感觉有一些带感的剧情简直可以无缝衔接，不串起来简直不过瘾。可是把不同作者的文章里的情节串起来的话，我又感觉不太尊重原作者（虽然我已经分别得到papayoya和aphrodite的首肯，可以翻译他们的作品），而且如果后来有人要开坑翻译这两篇文章（我是真没时间都翻译啊，这两篇都超长的）的话，也会比较尴尬。

我想请教论坛里的前辈们有没有遇见过类似的情况，同时问问各位读者老爷的意见。
