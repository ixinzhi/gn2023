# 11

从小学开始就一直在整了，虽然大多数设定现在都忘了……基本上新的想法如果不记录下来的话是会忘得很快的，平时正经的点子我会用专门的笔记本记下，但是性癖方面就不好意思记录了……
也总是想着“有机会的话就写出来吧”，但总是没有时间搁置了（也不完全是懒，本职工作内的写作一个接一个……虽然其实还是懒的）
另外我自己对自己写的东西很少有重新读的勇气（除非别人拿给我看），总觉得离开了自己的大脑以后那就不是自己的东西了，读着会有一种陌生感，所以点子写出来之后反倒失去了感觉……
事实上写和读的区别真的很大，对于个体而言，除非天赋异禀，或者自信心充沛，否则写作多半是一个把你脑海中华丽的设计图建成土房子的过程，这个过程是蛮痛苦的；因此论坛里分享作品的各位都是非常了不起的，大家都是为了其他人的快乐而消解了自身的困苦 <title>12</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
