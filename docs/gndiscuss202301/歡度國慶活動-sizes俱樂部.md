感嘆自己也已經到了這個了，以前總覺得商業資源跟商業本很貴，現在都是信用卡刷一下就過去的，以前買個本子都感覺痛不欲生，現在也只是半天左右的工資，自己稍微少喝個一兩杯飲料就能買的價格，以前總為了收集這些資源傷透了腦筋，現在卻幾乎無感，但失去的卻是更多的時間，以前總是每天準時到論壇更新報到，現在一個禮拜能看到四天的論壇就很了不起了...

只是感嘆即便是像我、Dante或是jitensha老師這樣子對size有熱情的人，總有一天也會被名為"現實"的潮流所擊敗...

好吧，廢話不多說，正文開始吧!

此次活動趁著國慶假日而舉辦的活動，在中國國慶是個連續幾天的長假，即便是在台灣也是個4天的長假，所以想說趁著長假的機會來舉辦個小小的活動

**活動題目:雙巨人+小人**

參加活動時間:10/4~11/11

主要發想的是因為jitensha老師在參加完國外的sizecons後，畫了一篇融合各種題材的漫畫，於是乎想說利用這本漫畫當作獎品而舉辦活動

(這件事情已經經過jitensha老師的同意了)

活動很簡單，就是雙巨人(大男大女)以及小人的故事以及互動(本來想說限制於雙巨人x縮小女，但是由於jitensha老師的新漫畫中有出現難見的縮小男，平時不喜歡縮小男的jitensha老師都如此犧牲了，想說就隨意吧)

不過幾個點注意:在文章中雙巨人必須都是加入正戲，不能讓女巨人或是男巨人變成路人(避免巨大男愛好者或是巨大女愛好者將其改編成單一屬性的文章)

由於這次活動的獎品只送出一本(漢化本一本以及英文本一本)所以最後是採取選拔制的

**活動需求:

5000字以上**

**加分題**

加分題並不會立刻讓您直接贏得冠軍，而是當參賽者的分數相當時，就會參考加分題

1.縮小(本來想說讓縮小變成固定項目，但是這些太限定思路了，於是改成加分題) 縮小項目的縮小人必須有一段正常生活的描寫之後才變縮小人，縮小人跟雙巨人(普通人體型)之間有所互動

2.縮小方注重在一個角色身上:這算是jitensha老師這次漫畫所注重的，在文章中可以有許多個小人，但小人中必須要有個"主角"所謂的主角意思並不是龍傲天，主角可以像小人一樣被玩弄、無力回天甚至死，但是必須要有一個有名字有設定且有"感覺"的"小人"

3.縮小女(私心的加分題，小人為縮小女)

活動獎項:

第一名:jitensha老師新作漫畫:size horrors漢化一本(由xfreex管理員負責漢化，之後將可以在jitensha老師的官方網站購買得到)
第二名:jitensha老師新作漫畫:size horrors原文本一本
第三名商業資源

參加獎(還在討論)
