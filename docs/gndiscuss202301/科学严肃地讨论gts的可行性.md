# 1

虽然大家开发出各种脑洞来想象gts的出现，但是实际上到底可不可能呢？简单来说，gts可以分成变大类和缩小类两种，我们分别进行讨论。
首先是放大类。加入人身体的结构不变而单纯按比例放大，那么就会出现严重的问题。因为我们的体重是按照尺度的立方变大的，而各种骨骼、肌肉的截面是按照尺度的平方变大的，这就导致压强随着尺度的变大而线性变大，最终人自身就无法负荷。这从巨人症患者往往短寿而且行动不便、心脏不好就可以得到验证。在不改变生理结构的情况下，放大的上限聚会超出两倍。

再来说说缩小类。物理学家曾经做过估算，如果宇宙的各种基本参数稍有变化，就会导致巨大的、严重的后果。从这些参数，我们可以计算出分子的大小，并进一步估算出细胞的大小。也就是说，细胞的大小可以看作是固定。在这基础上如果人变小了，那就意味着细胞变少了。但是完成人正常的生理活动所需的细胞数量不能太少，否则也会自然死亡。这从侏儒症患者多数智商低下就可以得到部分验证。在不损失生理机能的情况下，缩小的下限可以参见新生的婴儿。

也就是说，如果我们想要放大系的gts，女巨人的生理构造必然与我们不同；如果我们想要缩小的小人，那小人必须放弃身为正常人的某些机能。

那么有没有一种科学合理的方式实现大比例的gts呢？

事实上还真有，只不过这种方式似乎被大家所忽略，那就是考虑不同维度的人（如果有人有从这条道路出发的gts，请告诉我）。

为了方便说明，让我先举个简答的假象例子。假如我们有一张面积很大的纸，然后我们将它反复折叠，最后这张纸所占的空间体积可以很小。然后让某种“二维人”生活在这张纸上，对这些“二维人”来说，他们可以很大（即面积很大），但是站在我们这样的“三维人”的角度来看，这些“二维人”很小。在这个情况下，我们并没有违反任何的科学定律，也没有出现“不合理大小”的人类。

类似的，我们现在生活在三维的空间中，如果有“高维人”生活在更高维的空间中，那么对我们来说，这些“正常尺寸”的”高维人“也就成为了巨人。

其實不用太擔心科學的問題，因為對創作者來說，是沒有極限的。
或許這樣說有點不科學，但是科學這種東西就是建立在各種假設前提下的哲學問題不是嗎？
舉幾個比較容易理解的例子：
1.過去人類依物體會往下掉的生活經驗發展了「以地球為世界中心」的學說，畢竟以當時的科學技術只能得到這些結論，後來隨著科學進步逐漸推翻與修改，才有了今日的宇宙觀。
2.過去(甚至現在)很多人都認為時間長度是固定不變的，你的一分鐘與我的一分鐘完全相同，但是當愛因斯坦的相對論論證每個人的時間是相對的，會依移動速度而改變，而有了新的宇宙觀，甚至因為這個理論讓GPS技術變的可行(誤差得以控制)。

由此可知，我們所謂的科學就是在過去的經驗或流傳的觀念中用很複雜的交互驗證的方式建立起來的，如同拼圖一樣一片片拼起來，此外還有個特色是這個拼圖越中心的位置要被抽換改變，所影響的層面就會很大，如同地球是宇宙中心被推翻這點。

回到GTS的例子，我們堅信幾個物理的原則：原子或分子大小與質量不會被改變，因此太大的巨人會有無法支撐自身重量或過於脆弱的問題，過小的縮小人會有被物理上的各種不方便，例如血液的表面張力造成的毛細現象、甚至細胞數量的問題等等，是否也是可以被改變的呢？

如果今天換一個角度思考，如何創造一個可以兼容巨人與小人的世界，需要那些假設前提使其任何地方都可以合理被解釋，當這個世界觀的拼圖慢慢拼起來時，科學的解釋就說得通了。
