# 3

.
我個人的建議就是走馬看花，隨興而至，高興就好。

雖然我在台北住了一輩子，不過我也不是什麼愛到處玩的人，可能沒辦法給你很好的建議。
但如果是台北地下街那一帶，只要我有時間，我很樂意當地陪。
台北地下街是阿宅一定要去的地方！
其餘的景點要看你自己，台北的大眾運輸算很方便，只是資訊務必要先查清楚。

若還是想聽聽我的建議，我會說貓空纜車，便宜又能看風景。
西門町那邊我不知道現在還剩多少宅店。白天還行，晚上治安難免不好，自己小心。
如果對電腦等3C產品有興趣，台北車站前的NOVA廣場與光華商場新大樓，值得去看一看。

FF的話，在外面拍cosplayer之前要先禮貌講一下，或者直接混到一群正在拍的人裡面。
會場內部由於人潮眾多，不建議那種會阻礙通行的拍照行為，你如果有辦法不擋路就隨便你拍。
不過，原則上禁止在未徵求同意的情況下拍攝別人攤位。
（例外：你拍朋友時順便把攤位也一起拍進去就沒話說）

適合一個人的餐館...坦白說這就是勇氣問題。鼓起勇氣一個人走進去的話，通常不會被拒絕啦。
切記！不要難得出來玩，還一天到晚吃麥當勞。
除了這種連鎖店以外，吃什麼都值得！這是我的想法。

歡迎找我當地陪哦！

# 4

地下街如果你本身有在收集模型、電玩或是書籍之類的話，
其實台北Y區地下街就可以逛很久了，
西門町之前去逛的時候，
有間萬年大樓裏面是有賣些宅物，大多是模型，
現在還有沒有不確定，

FF方面，
建議去之前記得先去全家便利商店或是漫畫店買場刊，
就不用到那邊才要再買，
也可以先翻翻場刊，
裡面會有各個整個活動流程，
還有活動2天期間各個攤位社團的"簡單介紹"，
不過字很小就是了，而且是"非常簡單"的介紹，

COSER的拍照方面就如同樓上但大說的，
拍照的話禮貌性，點個頭或是開口問問，
對方有點頭或回應就可以拍，
會場內因為人非常多，而且夏天場會是滿滿的汗臭味，
一般而言COSER很少會在裡面出現，
就算出現也只是會在3樓的休息區，
或是體育館外圍的看台，
還有要注意的一點就是，
COSPLAY的淺規則就是你拍的照，
盡量不要公開或是外流，
這是對COSER隱私的一些尊重，

至於台北景點，
因為本身不是台北人，
不是很了解，
就交給其他版友了，
以上希望能幫到你，祝你玩得愉快 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

臺大附近吃的小店很多，各地方風味的料理都有，咖啡廳也不缺乏。

如果遠點，松江南京站（就是忠孝敦化附近）與八德路那邊蠻多有趣的店，不過那邊價位就比較高。

可以用這些地名空格後加上食記，搜尋有興趣的店家，例如：「忠孝敦化 食記」。

如果你有智慧型手機的話，當場找也不遲。TPE-FREE 蠻方便的。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
