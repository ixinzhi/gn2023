# 1

巨大娘A：“您好，这位先生，请问有什么可以帮你？”
小人：“我接下来要说的事，你们可千万别害怕。”
巨大娘A：“我们比宇宙还大，我们不会害怕的，你请说。”
小人：“刚才……刚才我遇到超性能娘了！”
巨大娘A：“超性能娘是哪一位？”
小人：“不是哪一位，是一拳能爆掉宇宙的那种超性能娘！”
巨大娘B：（在大陆上画画，画了一个琦玉）
小人：“不是一拳超人，她是女的，不是光头。”
巨大娘B：（用手抹平大陆，画了一个女超人）
小人：“不是的，超人没有她那么强。”
巨大娘B：（再次抹平大陆，画了一个阿拉蕾）
小人：“她不是个幼女机器人，是个少女！少女知道吗？”
巨大娘B：（又把大陆抹平，画了一个坂上智代）
小人：“这……”
巨大娘A：（把大陆抹平，画了一个巨大娘）
小人：“超性能娘啊！隔壁宇宙有没有去，就是那个你们也做的到，但是体型比你们小的超性能娘啊，明白吗？”
巨大娘A：“明白了，你请说。”
小人：“超性能娘说什么想发泄一下，可是地球太小没什么意思，试问谁不知道，然后她就用爱液把地球给淹了，就在CXN-N宇宙，整个宇宙全是她的体液，还有乳汁，白茫茫的乳汁啊！……我刚一探测那个宇宙，刚想看看人类科技怎么样，然后就看到她的体液瞬间充满整个宇宙，结果我的仪器就坏了，就像个……”
巨大娘B：（捂嘴偷笑）
小人：“你在笑什么！”
巨大娘B：“我想起高兴的事情。”
小人：“什么高兴的事情。”
巨大娘B：“我也用爱液淹了几个宇宙，一滴。”
巨大娘A：（笑）
小人：“你又在笑什么！”
巨大娘A：“我也用一滴乳汁淹了几个宇宙。”
小人：“你们淹的是同一个宇宙？”
巨大娘B：“对……对。”
巨大娘A&巨大娘B：（偷笑）
巨大娘A：“啊不，不是同几个。”
小人：“我再重申一遍，我没在开玩笑。”
巨大娘B：“对，对。”
巨大娘A&巨大娘B：（笑）
小人：“喂！”
巨大娘A：“我们言归正传，那个，你刚才说的这个，超性能娘，她厉害吗？”
小人：“她不是厉不厉害的问题，她真的是那种，很无敌的那种……她长得漂亮，身材也好，很可爱，只可惜宇宙被毁的太快，我没注意拍下她的样子……”
巨大娘B：（偷笑）
小人：“你欺人太甚！我忍你很久啦！”
巨大娘B：“我毁掉了很多宇宙。”
小人：“你明明在笑我，你都没停过。”
巨大娘B：“先生，我们见过很多宇宙被毁，无论多好笑，我们都不会笑，除非忍不住。”
巨大娘A：“不如这样，先生，您先回去继续看着，我们找到这个超性能娘，第一时间通知你。”
小人：“行，你们赶紧去找，好吧，很危险的，记得变得越大越好。”
小人离开，巨大娘A&巨大娘B狂笑。
小人闯进来。
巨大娘B：“先生您的东西忘带了吗？”
小人离开，巨大娘A&巨大娘B再次狂笑。
小人再次闯进来。
巨大娘A：“先生？”
