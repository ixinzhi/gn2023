# 1

看到近来一些朋友关注GTS各种身材大小的不同作品，心中有感特来发言几句。
我是一个对于各种大小身材体型女巨人上下通吃的爱好者，大多数情结还是针对10~100倍标准高度女巨人类型的，对于身高10米以下迷你女巨人的爱好并不很浓重，却也偶尔情有独钟，因为这个身材的女人和普通人类相差不大，文字描写起来显得有血有肉，读起来非常真实，所涉及场景空间和物件大小与现实生活较为接近，与正常人类的互动关系良好，不至于造成人类社会特大混乱，所以我们描写出来就能得心应手。因此，本人建议各位喜欢写作的朋友要想写的深入，不妨多写那些身材比人类差异不大的女巨人。
关于1000倍以上的超级女巨人(让我暂且这么称呼)题材故事，由于身材异常巨大，离我们日常生活的尺度太远，所以我们很难把她写活显得真实，其实我们忽略了一个问题，对于一个体型异常硕大又有着普通人类外表的巨大神女这样一种角色，我们在她们身上套用着太多人类的印迹了，我想作为一个那种巨大体型的生命体本身就已经超越了普通人类的形态，她不会再有人类的任何生活特性，作为女神她们应该超越了人类本身的局限而达到了生命的至高境界。而在我们大多作品当中却包含着人类自身许多污点和腐败的特质，以及仍然受限于普通女人生理现象引起的灵性缺失，实际上这是不可能的。我们的意识中还没有摆脱这种陈旧概念，即使向但丁版主这样的高级写手也未能完全体现出来，尽管他描写的情节非常精彩和优秀了。
我们主要还是注重身高30~150米这个范畴的女巨人，对于一个100倍以内大小体型的女巨人，我们还可以赋予她各种人类的特性，她的行为还可以像个人类而未达到女神的境地，不过我们也应该为她注入一些新奇和高级的特点，使其从本质上高于人类，可是我们除了让她们具备更强的杀戮能力和吃生肉的本领，要么就是叫她们发挥各种人类特有的坏心眼。无法超越我们人体自身的限制，我们就不可能写出非常精美的作品来。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 2

就我看來，LZ的意見就是，賦予GTS更多的神性而減少其人性，將其作為一種新的生命形態描繪

我個人是反對這樣做的。
首先，現在GTS文中，女巨人基本都是由正常人類變成的，不可能沒有人類社會的價值觀，由體型變化而突然神化是不現實的
其次，根據千年來的歷史可以得知，沒有人性存在的角色是不會受到歡迎的，人性的優點，人性的缺點，這些混合在一起才是真正有血有肉的角色，脫離“人”範疇的角色，即使再完美，也會因為其完美而被讀者敬而遠之——文字作品反映的本就該是人的事兒。古希臘神話之所以至今仍有著旺盛的生命力，其中一個原因就是其神非神，有著人的缺點，有著人的魅力
最後，LZ的觀點也有著許多支持者，即“女神系”的支持者，這類作品中GTS已經不僅是GTS，而是GODESS了，但我個人認為這種女神比較適合圖像作品而非文字，圖像化的女神能給人一種震撼，但文字中的女生卻更能給人以親切 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

萝卜白菜，各有所爱。在这个特殊的群体中，每个人都有自己独特的口味和喜好，谁也不能改变谁。

但是，我们可以通过自己的作品启发别人，给别人带来全新的灵感和不同的感受。
或者，看到别人的作品得来了启发和灵感，通过自己的进一步加工来修改和完善，达到符合自己口味的理想境界。

存在的就是合理的，不管你有什么喜好，各种口味都有那些赋有 极致魅力 的一面，优秀的GTS FANS兼幻想家，能把他心中最完美的GTS文化展现出来，这就是创作。

所以，创作出属于自己世界观和价值观的作品才是最有意义的事情，其他多说无益。
