# 1

楼主准备创作一篇校园故事，是很正常的故事，希望能和现实故事接近（所以不会有太多大小差距下的社会问题），故事的基础构思好了但总感觉缺些什么，所以希望能够征集一些素材。

希望您能够讲述您的真实经历(以能改成有意思的gts剧情优先)，或者您想象出的校园gts故事
不限悲喜舔虐
不一定要在校园发生，也可以是学生的家里故事，和同学在外面玩的故事。

比方说前几天和一个朋友聊天，提到他海王翻车的故事，我觉得这个就挺适合改编成gts故事的，海王小人同时面对多个对他质疑伤心失望的巨大娘（/狗头）
海王谁都知道，但可能接触太少，构思故事时就是没能想到。所以说您的故事不一定需要特别标新立异，普通常见的也行。

希望大家多多发言，即是楼主没能写出来，但您的故事会给其他看到的大佬提供思路呀，没准一篇佳作就能因此诞生了。
