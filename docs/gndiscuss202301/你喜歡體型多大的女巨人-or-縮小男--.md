# 1

先來解釋一下，投票選項的身高數值不只代表女巨人，同時也可視為男人縮小後與正常女人的體型差距。給一個最簡單的對照就是，假如縮小男的身高為一公分，則正常女人應等同於一位160m的女巨人，請各位照此標準投票，一個人可以選擇三個選項，投完票才可以看結果。

另外，在日本網站上也看見了有趣的分類法則，在這邊跟大家一起聊聊。
他們認為GTS有四種分野：

    ① Real Giantess…　真實系女巨人，身高3m~5m，是最接近真實人類的大小。
      　② Giantess…    即一般通稱的「女巨人」，身高15m~50m。　
      　③ Mega Giantess…　身高100m∼500ｍ，這個我不管怎麼翻都不對味，所以保留英文原文。
      　④ Giga Giantess…　身高1000m以上。

「Mega」與「Giga」都是用來形容巨大化女性的詞彙，大概不會有人將之使用在縮小男的故事上。不過「Mega」的能見度比「Giga」要高出不少。從原始意義來說，「Mega」在英語中代表一百萬倍，「Giga」則是十億倍。從這個角度來說，假如一個身高170的普通女性以Mega程度巨大化，那麼她將具有17000公里的身高，這似乎也太過頭了，圈內通稱的「Mega giantess」並沒有那樣大，所以「Mega」一詞應當只是誇飾用語，最為人接受的程度大致落在100m~1000m之間，但也有人認為可將範圍拉大到50m~10000m，甚至更大；這基本上是因為「Giga」一詞相當少用的原因，畢竟原文意義為十億倍，這也差太多了。

所以，我個人會把「倍率」當成一個分野；真實系的基本上定義很穩固，所以再往上看，巨大化程度100倍以下，也就是最多150∼180公尺高的女巨人，我將之稱為「最普通的Giantess」。超過一百倍高的，一直到身高10公里之內，我會歸類到「Mega giantess」。高過10公里的，就稱為「Giga giantess」。

关于这个主题其实有两个意思，一个是女巨人与正常人或缩小人的身材比例多少最合适？还一个是女巨人的极限身高多少合适？我们主要拘泥于第一个问题，对于第二个问题我可以简单回答，那就是没有限度。我们的GTS故事中要有人物互动，《格列佛游记》中描写的人物身高关系非常合适，那就是1:12，这个比例主要因为英国人的衡量单位是12进制，所以用英寸、英尺换算，我国使用的长度单位是十进制，所以对于我们说身材10倍差距非常合适。
我认为，女巨人的身高可以没有限制，可以用很多种类，但是必须有一个与正常人接近的高度，大概在8~12.5倍比较好，即10~16米左右。而上面还可以存在更高大的女巨人，下面也可以用矮小的女巨人，比如4~6米的高度，或者40~60米的高度、这样几百米、上千米都无所谓。总之，可以同时存在很多不同大小的女巨人，但是她们之间的身高比例始终保持在10倍以下就非常有意思了。 <title>28</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
