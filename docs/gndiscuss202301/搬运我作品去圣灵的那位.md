# 1

   我不知道你是谁，我也不可能知道也不想知道，不光是圣灵，还有搬到其他网站的某些人也一样，我说过我的作品可以转载，为什么？因为这个圈子非常小，我的作品虽比不上真正的3D大神们，但自认还看得过，所以我希望能带给更多的同好，要是有哪位受到启发激发了灵感那就更好了，这是我的初衷，但我有个要求就是转载要标明作者出处！另外，转载之前打声招呼这不难吧？这是个人就应该知道的基本礼貌吧？

    我知道你们搬去其他论坛为什么，为了收费购买其他资源，自己又不会做，只能偷，我没时间也没心情逐个论坛去找谁匿名转载发布了我的作品去收费，往后的新作品还可以转载，没问题，但我都会打上水印，当然是在尽量不碍眼的地方，如果还不行，那就只能在显眼的地方印了，影响观看我也没办法，谁让你们那么“懂事”呢，外网也有人在搬运，但至少标了作者。

    所以发这贴并不是想和你们这些人讲道理，懂道理就不会做这些事了，我就是想告诉你们这些人，现在原创作者并不多，这本就不是什么好事，如果再被你们搅得都收了山不出作品的话，到时大家都没得看没得玩！其实自己做做图自娱自乐一下也挺好，反正技术和想法在我手里我怕什么，所以我并不期望你们会改变，咱们走着瞧，继续下去的话很快我就让你什么也没得搬！
