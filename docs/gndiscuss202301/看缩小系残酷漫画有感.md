# 1

![](img/27533062db6ce4e24c3784cc8474f5af.png)在重翻自己在E站里的收藏的时候看到这一篇，
当时只觉得这漫画画风不错而且篇幅也不算短就点了一下收藏
今天从头到尾看了一遍，整个人都不好了..

从这里借个理由说说我自己本人为什么不喜欢缩小系的对比类女巨人作品
温柔系尚且不说，残酷系的残酷描写看的我浑身鸡皮疙瘩，就算不去代入也会感到作品内小人的痛苦
可能是因为缩小系里的人始终我还当作是人，看到她们对待那些缩小的小人怎么残忍以后，我居然会有一种恶心感
就好像看到小动物被虐待一样不忍，而看变大或者生长那种为什么不会呢，大概是因为我已经把她们看作是怪物了
即那些作恶的大怪兽，但是是人形而且还有女性身体的美型，所以看着会有很兴奋的感觉；
这里说回缩小，那些放进鞋子里或者放进衣物里的行为，如果小人是很享受的话我反而不会有什么感觉
但是故意使他们很痛苦，甚至接近死亡的话，我就会觉得很反胃，做出这种行为却还和周围的平凡人一样
我突然觉得很恐怖，这种人的内心很阴暗，阴暗面很大，会让我觉得这种人反而更不像是人，但她却很真实
因为怪物至少在现实是没有出现过的，而这种阴暗面的人却很可能就潜伏在我们的身边，
真的是想象就觉得害怕。

以上就是我的个人感受...![](img/2c20ff8d184f40a2bfe51032da3246e4.png)

# 15

因為這會觸動到人心一個脆弱的地方，就是如果其實我不被社會給需要怎麼辦
現在大家可以和平相處，但會不會扒下某種偽裝後，我們都是可以輕易傷害他人的怪物?
會不會因為力量不對等，會不會因為沒人能找到我的屍體，我的存在意義就被輕易抹滅?
原po可以試著理解自己的恐懼是什麼，畢竟恐懼來自未知，有點概念會稍微舒服一點

巨人感覺不可怕，我猜啦，可能是你沒意識到這件事對「秩序崩壞」的意義在哪
那就想像一群人被巨人困在大賣場裡，開始生成小型社會，不遵從規定者會被推出去......
等一下，那就迷霧驚魂(The Mist)啊，看一下就知道，恐怖的不一定是不存在的怪物本身喔
像死亡實驗(Das Experiment)，即便沒任何幻想生物也能帶給你類似的恐懼

講白了，我認為這種感覺就是一種社交恐懼
人類是群體動物，存活非常仰賴於他人的存在，基因上我們本來就該在乎彼此
但人對社會的期望越高，也會越害怕如果一切都是自己的幻覺該怎麼辦
所以當你看著一個小人能被毫不在乎的虐殺，也會想問自己：
「是否只要滿足某種條件，每個人都可以是巨人和小人?」
要想清楚這個問題，就得先自我反省，「社會」對你來說是什麼?應該是什麼?
藉由不人道作品的衝擊可以讓平時的思考逃避遭受正面衝撞，是個不可多得的好機會喔

我的作法則是把這種恐懼跟性欲混在一起，現在我看任何片都能想歪了，是不是很棒 <title>16</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
