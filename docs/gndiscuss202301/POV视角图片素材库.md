# 1

GTS手绘作品当中，最重要也是最困难的部分，主要就是POV视角的图了吧？（就是小人仰视GTS的视角）这里开个帖子希望大家有好的POV图片，都能贡献出来，给绘图作者们提供参考素材和灵感来源。

三次元、二次元皆可，具体怎样的表现形式都可以（比如不露脸，不完整亦可，欧美肌肉型的就算了吧）

唯一的要求就是平日不太常见到的视角（仰视为主，能凸显巨大感更好）。
