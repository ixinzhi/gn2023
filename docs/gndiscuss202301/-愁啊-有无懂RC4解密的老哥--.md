# 1

老坛友们应该都知道，i-raf-you的作品都非常好，无论是游戏还是漫画。现在来谈谈游戏的事。他的游戏大部分都是基于Adobe air做的，但是这种类型的游戏解密很困难（或许是我没找到方法），后来在论坛里翻了好久翻到了关于这一类游戏的解密方法，但是那位大佬提供的软件只能解密成惠的日记这个游戏，当然这游戏正在汉化中，因为每个游戏的密钥不一样，所以那位大佬提供的软件就不能用了。但是我想软件应该没问题，如果能找到密钥就好了，这种类型的游戏比如巨娘国，爱丽丝，成惠的日记都是非常好的作品，所以一希望论坛里的大佬能够在解密这类游戏的方法上指点一二，感激不尽。![](img/cdc51f89bb7d985db5e2efdb53ac7fc6.png)

# 2

啊，这话题难度太高了，我连怎么应一声都不知道……

不如说我竟然连RC4是什么都不知道……
Google了一下，看起来很难的样子。

而且我看懂的部分只有NOMORE进攻方法很有效，75小时就差不多可以破解一个密码了。

然而NOMORE只能对一个正在加密的网站有效，是通过注入一端信息，然后网站加密以后拦截这个信息，然后再解密的……

而我们要破解的这个东西，是没有办法加密新的内容的……

好艰难啊。

如果只需要几百或几千个小时破解的话，也许我们可以分流一下工程？每个人用电脑试一些？

但是我连怎么分流这个工作都不知道orz
