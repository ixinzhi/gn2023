# 1

饱经沧桑的大叔带了个女仆装的生角兽耳娘到沙漠中的城镇，然后到了大修道院见主教大人，白天玩火纹，晚上就梦见它的剧情，反正差不多都是身体里寄宿着XX之力，这种力量会让兽耳娘不断变大。

然后就是一个Grow场景切换，根据她长大的不同时候切一下场景和日常，对比下周围人事物什么的。

太真实太爽啦——这才是重点

最后是她穿着蕾丝裙，双腿穿着丝袜跪坐在地上（还是女仆装），但是旁边的主教没她大腿高，估计七八米？感觉棒呆了。

然后就是控制不住洪荒之力了，呼呼呼呼地就开始变大了。

为了不破坏城镇她费劲千辛万苦逃出了野外，因为她是那种人见人爱和街坊大妈阿叔和睦相处的那种，所以在过程中不小心破坏掉了城市道路和一脚也没遭到敌视，反而受到关心和帮助。

无论是肌肤的质感还是蕾丝的材质都爽死我了，第一人称视角是真的棒棒。

爬到郊外的时候大概是两百米的样子，还在不停变大，捧着主教——指蕾亚大人，一路回到那座沙漠中的城镇挖掘XX之力的秘密。

不幸的是，这时候我快醒了，我开始吐槽为什么女仆装会跟着她一起变大啊。

然后就是兽耳娘在变大的过程中各种迷茫，200米、300米、反正一直向上➕啦，走路过程中不小心踩死一头羊都会小小地心酸下给个快闪画面什么的，然后到沙漠边缘的时候是一千米了，等火伴做好进入沙漠准备时，和她一起进入沙漠。

最后抵达沙漠城镇时不知道，反正就是比那座城镇大很多啦，两条腿之间能夹住的那种，但是火办花了一个月还没找到关键道具。只能够在那座城镇人们的恐惧之下，双手撑在城镇两边低头寻找观察。

太真实了，因为我是上帝视角和主教视角切换着看的，主教视角也太恐怖真实了，是那种站在泰山山脚望山顶的感受，只能看见她身体的一部分，那种主教站在她手指上穿过云雾见到她的脸的感觉，如同窥见神明一样的感觉，简直无敌了，但是女仆装兽耳娘还是那种很温柔很舒服柔和的感觉，毕竟是温柔系的。

然后就这样过了一些时间

直到懈怠还是蓝胡子、随便啦反正形象就那样反派就出场了，我以他的视角见到了浮在天空中的兽耳娘的脸，以及她一根手指摧毁了反派所在的整个广场，只留下一个天坑一样的大洞，和她从广场大小的坑中取出手指上沾的少许肉酱。

以及最后视角转到坑那网上看去，那是一只大到遍满天空的通红眼睛。

然后我就她妈的醒了。

我我我我我我…………我好久没对这种级别的爽了好吧，就他妈这样结束了。

痛哭。
