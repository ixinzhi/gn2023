# 2

giga = 1000^3 通常指超过星球大小的GTS
mega = 1000^2 通常指超过大陆级别的GTS
micro = 10^-6 很少听到GTS用这个词描述
mini比较特别，通常是指身高容易与人正常互动，可同处一室的GTS，例如2~5倍
以上都是个人观感，或许不同人有不同想法 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
