# 1

樓主，我看了你在另一串討論的回覆，也找了一下過去您的創作，發覺其實您可能很適合寫短篇！

從您過去文的內容來看，有一種斷斷續續的感覺，或許是因為生硬的把不同屬性的爽文接起來造成的吧？如果不同屬性的play可以分開，那麼有趣的點子就不會打架了呀！
建議您可以試試看日更或周更短篇，有點類似一批大佬那個練筆串一樣，先讓每篇短文只有一種主題(例如赤足踩踏)，一種風格(例如殘忍)，別超過三種play，最重要的是要有起承轉合的劇情結構，然後稍微分段一下即可。

如此一來點子多多的樓主就能專心在較容易控制的短篇劇情設計上，而不用煩惱太多不同play銜接的問題。

所以別說什麼最後一篇了吧！快把第一個有趣的短篇故事寫出來吧！
