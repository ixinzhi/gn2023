# 1

最近几天涌现出很多百／千公里级别的文章，可谓正中本人下怀，非常激动。不过欣赏美文的同时想到一个问题，大家对于尺寸都是如何界定的？

我个人的看法，giga或者mega分别代表吉与兆两个数级，而这个数级指的是巨大妹子同常人的尺寸之比。例如giga妹子是常人身高的吉（10的9次方）倍，在百万千米级别，mega妹子则在千公里级别。

如此看来，论坛中的绝大多数giga文似乎都应该划分成mega分类。不过既然大家都如此划分，想必我的划分方式有错，不过也一定有某种力量／不成文的习俗影响着大家那样去约定俗成的区分。

我由此想到了论坛Boss但丁大大。但丁大大肯定算是gts界的元老级别人物，eoe和dgu对圈内文章有着很大的影响力，而每篇dgu文章前的关键字都会有出场妹子的尺寸。在那里，千公里级别似乎就可以算作giga了，而星体级别（希薇尔）更是被算作super giga（还是什么？我记不太清了）。这种分类方式是不是但丁大大推广的呢？

结果最后我又成自言自语了呢。笑。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
