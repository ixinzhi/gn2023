# 1

可能很多人都玩過 WAKU WAKU7  
大陸翻譯是 火熱火熱7 不知道臺灣怎么翻譯
打到最后一關，面對大boss的時候會，玩家和BOSS都被巨大化，于是便在大馬路上和boss單挑。
不過這個boss的難道有點BT，3拳打上就歇菜了，被打掉1/3的血還會暴走，不小心就掛了。
人物方面有口耐的女仆機械loli，出招喊的都是：“危險啊∼對不起啊∼”之類的，不過不是很好用，前+重拳 有的時候很好用，但是被敵人近身之后會很難看。
也有熱血少女，用叫獸的話來說就是和武藤蘭一樣動作相當靈活的女孩子。
也有做在大貓上的小女孩，基本戰斗都是靠大貓。
其他的，機器人..熱血少年、皮鞭壯年、精靈劍士......個人很喜歡用slash 這個角色。

多少年之后也沒有人記得這些街機游戲了，這個游戲也是，2D的街機時代的后期作品了，畫面還說的過去。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
