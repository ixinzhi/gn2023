# 2

并不是以前的作品实用性很高，而是你以前看的那几部作品很经典很实用对你幼小的心灵刺激度很高
我认真想了下我最喜欢的几部作品：
文：《神秘的药丸》
本子：《妹之足下》
视频：《女巨人折磨小囚犯》
图集：《被缩小了》
（这些都是六七年前初次在网络上找到它们时的名字）
好像确实都是老物了

但现在同样有我很喜欢的作品在更新：
纯Size Fetish向的游戏：《Resize Me》
永远的神：《Tower of Trample》

现在一样在诞生很实用很经典，让人看了会挖槽的作品，不能享受了这个圈子从有网络以来所有的存量后，奇怪怎么没有源源不断的作品继续冒出来了

有这种想法的人，我都希望狠狠的打他们的屁股，快去成为创作者让以后入坑的新人依然觉得我们这个小圈子有很牛逼的东西啊，魂淡

覺得和近年收費機制的蓬勃發展有關，應該說是實用性兩極化
有能力迎合觀眾胃口，強調實用性的作者，用收費機制的收益創作高實用性的作品，然而流通性比較低。

純粹用愛發電的，也不好要求什麼，創作的都是自己所愛，作品流通性高，但是實不實用要看創作者自身的口味到哪邊。
