# 8

什么图片的表现形式并不是最重要的，关键是图片的内容。

真人合成的如果做得好的话，很有真实感。
但是相对来说，好的素材比较少，不会有哪个女人专门为了让你做合成图而特意摆一些动作。（giantesscity上的部分西方女性同好除外）

卡通图主要就是手绘的了，这个资源相对来说比较多，但是做的“到位”的比较少，毕竟绘制原图的作者大多不是GTS同好，我们GTS同好中会绘图的人很少，愿意把自己的作品拿上来与大家分享的人更少，愿意分享又画得好，画得“到位”的人更是少之又少。

3Dposer在西方论坛比较多（giantesscity），但是大多是西方人的口味，做3Dposer对机器要求比较高，而且比较费时间，做得漂亮又到位的其实也不是特别多。
3Dposer的优势就在于真实感和美感可以同时具备，而且只要电脑好，愿意花时间投入进去的同好都能作出不错的作品。可是又有多少人愿意深入去研究呢？
而且我们论坛也发生过因为口味不同，把一个华人3D POSER作者气走的情况，这个大家要注意。

总之，真正优秀的GTS图片作品，总体数量是很少的，所以有不错的资源就已经很好了，大家如果不是有自我创作制图的能力，真的不必太过挑剔，这没有意义。
