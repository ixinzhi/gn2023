# 1

就，昨天下午，某非常好的朋友（同性）在聊天的途中，突然嘴里冒出了giantess这个词
第一反应就是非常意外！为什么这个词会从对方的嘴里冒出来呢？我们平时也没有讨论xp什么的……而且首先，为什么他会知道这个词呢？有那么一瞬间我还在想该不会其实是同好吧！不过现实中人的思路是没办法走那么快的，所以总而言之当时还是困惑的，不知道对方打算说什么！
结果对方说是因为自己在nga上看到某个帖子里提到了giantess这种xp，因为觉得非常好奇就去查了，然后觉得很意外。
他想要告诉我的主要是“居然还有这种东西啊，好意外啊”这样的感觉
我自然是受到了极大的冲击，真的有种冲动是想告诉对方“窝就是这个的爱好者啊……”
但是克制住了。毕竟是很重要的朋友，不论怎么样都不想因为这种事情被讨厌。嘛，当然也不是说giantess爱好者就会被讨厌，但是不管怎么说会有这种担心嘛……
于是只好旁敲侧击。“是什么呀”“惊讶？为什么呢”“这个词的意思到底是什么呀，你去查了吗”之类的。
他的回答是很好奇所以就去搜了。我问什么意思的时候他说“自己去搜不就知道了”
至于“那你是怎么看待的呢”，他的回答是“只是比较意外，因为之前没有听说”
甚至他还说“世界上居然有这样的xp，真好奇兴奋点到底在哪里呢”这样的话
我当时好紧张，所以也只是嗯嗯啊啊的，后来因为我开始晒游戏里抽到的卡，于是话题被终结了（物理意义上）
不过事后还是内心无法平静，老实说到现在已经过去12小时了，还是很在意这件事。
说到底为什么会突然讨论这个呢……
其实一开始还以为是我暴露了，不过这种想法未免自我意识过剩了，我们虽然关系很好但是也不是会搞hs的那种，从来也不会在聊天的时候聊到xp的。暴露什么的也无从谈起。
不过还是觉得他会突然提起这些事情，给人一种特别奇怪的感觉。我现在甚至产生了“该不会其实他真的是同好，想要试探我”这样的胡思乱想！
主要原因有：1、gts确实不算大众，可是也不至于让人这么意外，以至于意外到会特地和别人聊起来吧！我听说世界上就连对特定形状的树木发情的人都是存在的，相比之下gts又算什么呢。不过，我毕竟也是天生的giantess控，还在念小学的时候就对拇指姑娘一类的故事极度感兴趣，在我这种人眼里gts或许先天性地就被我认为是“没那么夸张”的事情了。圈外人究竟会有多意外，确实没办法知道呢。2、前后的话题都毫无关联，和xp也无关，总觉得对方提得比较刻意……只不过是提到nga而已，他就把话题带过去了。如果不是同好的话，那或许是……对这个爱好产生了相当程度的兴趣？3、最后那个“兴奋点在哪呢”的话题，总感觉诱导性特别强……
说真的我当时几乎已经决定缴械投降了。自己是gts控的事情一直是绝密，有生以来从未告诉任何人，但是第一次被人精确地点出，也是第一次在这类论坛交流场所以外的地方看见朋友说出这个词，加上又是非常非常重要的朋友（也是尊敬的朋友），以至于我在那一瞬间产生了被支配的感觉，忍不住要坦白自己就是这个的爱好者。如果不是因为那个时候抽卡欧气爆发，豹跳导致被放置，大概就说出来了……
可是说出来以后会变成什么样，老实说完全没有想过。这就是冲动吧。
不过心里还是有点期盼。比方说，如果他其实是同好的话……嗯，那也会看到这篇贴子吧。在这里被他看到这篇贴子的话，正经人的生涯就结束了。就算发出很大的声音掩盖过去……不过，某种意义上来说，如果是同好相认的话岂不是太好了？
又比如说，他正如自己所表现出的那样，对giantess其实抱有极大的兴趣。那么视成功率而定，或许我应该开始安利，尝试带他入坑……天啊，我从来没想过xp也有安利的说法！
最糟糕的结果是，他不是同好，但是单纯依靠巨大的好奇心找到了这里，看到了这篇贴子！那样的话，绝对马上就知道是我了。那就完蛋啦！——虽说这样，也请不要当做没看见，亲口辱骂我是个变态吧……！
不管是哪一种情况，都是前所未有的情况啊。我现在还是不知道该怎么办呢……如果明天没有什么改变……我也许会尝试继续和他提这个话题……
