# 1

目前找到幾個地方符合隱密、有食物、能夠長時間(這個應該沒有XD)等等的條件

**台中火車站**

**銀櫃**

就是上次去的地方，在某個外勞中心的頂樓ktv

好處:
價格便宜 330/1人
食物吃到飽
夠隱密

壞處:
噪音吵
燈光昏暗
沒有插座(基本上以下的地點都沒插座...可能要請要帶筆電的人要自行預備電池了)

**美樂地**

同樣在外勞中心的頂樓，是處在銀櫃對面的大樓

好處:
食物吃到飽
比較起銀櫃，食物較為優良"一點"
夠隱密

壞處:
噪音吵
燈光昏暗

比銀櫃的價錢稍微高一些(沒高多少，大概多50塊左右)

**動漫天堂**

上次本來要去的地方，結果因為沒位置，最後改到銀櫃

好處:
安靜
價錢非常低廉
夠隱密
因為是在漫畫店，所以有漫畫，還有電視，跟兩台電腦，要交流的話，方便當場上網取材

壞處:
因為太安靜了，又沒隔音...所以...
地方很窄(擠三個人就快不行了)
基本上沒有東西可以吃(只有餅乾跟茶類飲料)

**台中一中**

**好樂迪**

就是那個連鎖ktv...目前最近的只有台中一中店，要去的話可以座公車(反正我們三人也有機車，找一個人載站主以及外來的人就好)

好處:
隔音設備非常好
包廂非常乾淨整潔
除了拿食物外，基本上不用離開包廂(包廂裏頭有廁所)
比起上述的其他ktv，食物優良得多...
不用看見外勞
地方較為乾淨
一中街跟中友百貨就在旁邊

壞處:
價錢較貴
雖然說食物很優良，但是吃到飽的東西真的不多，只有少數幾樣而已，大多數食物都需要加費，飲料更慘，只有紅茶、綠茶跟開水，要喝奶茶以上的飲料，請自行加費XD

**逢甲商圈**

**銀櫃**

位於逢甲的銀櫃，比起火車站的銀櫃來說，算是比較好的

好處:
這個銀櫃是一整個建築的，所以不會碰到外勞
因為是一整個建築，所以食物是獨立樓層的(食物種類很多)
包廂整潔，隔音很好
價錢比較便宜(同樣330)

壞處:
很遠...台中火車站到逢甲其實還要一段距離，這家店其實也不再逢甲商圈內，要步行差不多30分鐘

目前整理就這幾個地點了，雖然還有查到一些地點...不過那還是算了，anyway，若是站主需要，可以在集合前先帶站主去看台中的三連星(宅界比較知道的地點)

若是上述地點都不怎麼好，那就去其他的地點吧(一些價格比較昂貴的茶店...)
