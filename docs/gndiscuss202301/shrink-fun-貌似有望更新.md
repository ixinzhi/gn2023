# 1

今天久违地点进了原在EKA上shrink fun的游戏原贴内
发现大半年没有半点动静的作者居然发表了一段话
我又往上翻了翻，发现是有个爱好者打算接手SF这个游戏打算对其进行改造和更新
也许是为了不让这个爱好者白忙活才出面的吧

总之这次作者发表的话的意思是，虽然大家很久都没看见他有动静，但是他还在更新，并且不会和之前一样由于各种原因拖延，会在近期更新。要接手这个游戏是没有问题的，因为这游戏所有的资源是全在网络上的。只不过关键还是在于制作这个游戏完全没有收入使得动力不足。

大概就是这么个意思
实质上我觉得作者和其团队完全可以开个patreon来进行收入……今年年初那会儿那波动静很大的无素质催更真的很厉害……
