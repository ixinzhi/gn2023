# 1

我在写文的时候纠结于能不能在考虑了熵的问题的情况下去写寒冷能力者
拿光，电，风，这类的能力者的剧情举例，她们的“能力”or“魔法”都会转换成热量最终散发出去
但是把物体变冷的化，我想不出这些热量会转换成什么能，不论是机械能（使用超能力的副作用是弹飞自己？）还是什么的，都想不出
而且内燃机原理是我唯一想到的可以转换一定熵情况下的热量变成机械能的方法，但是转换成光，电，化学键这些我都想不出来
我物理知识不好，对魔幻也很难，又在虚构的故事里执着于真实
