# 1

我就先聊聊我開始即興寫作的過程,為什麼說即興寫作,因為至今我才發表過3篇小說,應該有些人知道是哪3篇吧!篇數不多而且這裡還有位大師級的中文GTS小說作者但丁在這裡坐鎮,我可不敢貿然說我是位作者,在但丁面前可要謙虛一下!![](img/bfdf5aa3d92e3fa8fd0cb15ca27e1367.png)
 關於潛水一詞,我一開始接觸GTS的時候也是潛水客,我想每個人一開始也應該都是一樣,剛接觸GTS的網站就是日本巨大娘普及委會,裡頭作品蠻多的,後來就慢慢了解日本GTS文化,也看了很多作品,一開始也是都看圖片,慢慢的就開始接觸小說部分,但小說是日文的,所以就得靠翻譯網頁或軟體來翻譯,翻譯過的日文文章想必大家都知道內容如何吧!一開始看的時候一定眼花撩亂的,還要使用頭腦去理解這句話的意思,看了一篇都快暈了,慢慢了就習慣許多了!後來我在想如果有中文的GTS小說不知該有多好,但那時根本幾乎都找不到GTS中文的小說,根本連網站都沒有,所以當初我就先成立奇摩家族巨大少女的世界,就把一篇日文GTS小說改篇成中文的小說,也就是我第一部作品巨大少女安達祐實,也因此脫離了潛水客的身分!只是後來沒完成全部的改編就放在那裡爛,爛到連安達祐實也已經身為人妻啦!我真是超級不務正業沒有責任的人!
 好了不聊陳年往事啦!該回到正提了!我先談一下我寫作的方向及個人嗜好,我的作品類型都是巨大化的少女破壞城市然後踩死人類為主,幾乎都談到腳的部分,我比較偏好描述人類看待巨大少女的視野,所以我的作品就像我發表過的美少女超人裡頭的一個片段(這時候王瞳已經開始動作，第一步右腳一抬起，腳下的人類看見到其中一根粉紅色的柱子突然移動了，身旁都被陰影圍繞住，抬頭一看，天空整個都暗了，當然王瞳的右腳正在這些人類的頭頂上，這些人類眼裏看見了巨大的物體，表面上還有紋路，而且很快的逼近他們，見到這樣景象的每個人類幾乎都是用雙手想擋住的動作做出了最後的反應，但是這些人類對王瞳而言，就像是小蟲一樣的弱小，人類的這個動作，根本就是多此一舉。)這就是小說的片段,我的寫法就是喜歡加點這樣子的味道,我都是大量使用巨大少女腳這部分的動作來做殺戮人類的主要工具,當然也是有別的方式來當做巨大少女殺戮的工具,會以腳為主,是我比較喜歡人類對於巨大少女而言只是小蟲的身分!還有我也比較喜歡談到數據的部分,譬如身高多少,鞋號多少,踩死多少人,破壞幾棟建築物,我喜歡做這樣子的描寫,這大概是我本身的風格之一吧!
 接下來談談對於寫作的一些想法吧!我認為一篇小說應該分成3部分,開頭-內容-結尾,這三部分也是要多思考的!我遇到寫作的問題就是可能想寫譬如哪位女生巨大化怎樣破壞城市殺戮人類的劇情時就想到開頭該怎麼寫,當時要寫美少女超人時開頭我就想了一些時間,而結尾也是很難去想的,就像巨大少女安達祐實,說真的因為後面我不知道怎麼寫比較好於是這部作品就放到爛,這個問題我要改進才行!因為會提出寫作的想法也是因為wuliaojianke兄說想要寫作希望給的建議,所以我就用我的想法來給W兄建議,當然也希望各位同號一起討論,畢竟每個人的寫作方向不同!W兄有說過喜歡穿婚紗及穿高跟鞋的巨大少女,我就以這方面用我的想法來聊聊!
開頭:場景在禮堂,有新人要結婚,但新郎落跑或者不結婚了沒出現,新娘因為這樣的因素如何變大?還是說沉睡的力量覺醒?這些都可以!
內容:巨大化的新娘在尋找落跑新郎的期間如何破壞城市殺戮人類,這些可以靠自己的創意!
結尾:巨大新娘抓到烙跑新郎如何凌虐下殺死新郎,或者新郎沒找到把城市破壞掉在去別的地方尋找,當然靠自己的創意也比較好!
這些大概就是我的一些建議,我寫作大概都是這樣子思考的,有什麼好的想法歡迎大家討論唷!
