# 1

本区有教大家如何使用翻墙软件，如何自建服务器的帖子，这很好，翻墙是一项有用的技能。但就上GN这件事情来说，使用翻墙软件的费效比不高，改host更简单一些。翻墙本身不难，但要找到稳定有效的翻墙渠道还是有点麻烦的。

gn的真实ip地址为 198.252.106.188

针对小白的改host简易教程

1.找到hosts文件

如果你是windows用户：
windows的hosts文件在 C:\Windows\System32\drivers\etc这个位置（据我所知没有例外，如果你不是这个只能自己百度了）

把地址复制到文件管理器的地址栏回车即可。或者按照文件夹名字一个个点进去也行。

如果你是mac用户（我没用过）:文件在 /private/etc/

如果你是Linux用户：
蛐蛐DNS污染而已！这点小事你肯定能自己搞定的吧！![](img/710cdefcabfcfa19bf02f129764ecbd5.png)

2.修改hosts

windows用户：对hosts文件右键，打开方式->用记事本打开->加入一行：

198.252.106.188    giantessnight.com

反正加在哪里都行，加在末尾吧

然后需要管理员权限，确认

试试能不能登上gn

听说win10好像有各种杂七杂八的权限问题……我反正没遇到过

mac用户：复制hosts文件到桌面->用文字编辑打开->加入一行：

198.252.106.188    giantessnight.com

将文件拉回到 /private/etc/  文件夹中，选取代。（windows也可以用完全类似的办法）

试试能不能登上gn

技术方面的说明，可跳过：gn去年被墙的情况我不是非常了解，但这次应该是只有DNS污染，GN的网址被定位到了Facebook的服务器上，所以链接不上。在我的机子上改host直连无问题。根据站长工具，大陆大部分省份都能正常链接到真实的ip地址，有个别省份似乎不稳定（甘肃？四川？）
