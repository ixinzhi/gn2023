# 1

不知道大家有捷安特的自行車沒有呀？
其實，我在很早很早的時候就預謀著一個計畫只是——第一輛4位數的單車在我準備實施之前幾天，被萬惡的盜賊盜去——
第二輛3位數的腳踏車在我買回來僅僅5天不翼而飛——
第三輛現在，被我用3條鐵鏈所在樓梯間——至今還好好的。
…………
其實，大家看那商標 GIANT 其實是巨人的意思
也就是說，在後面加上尾碼 ESS
就成了 GIANTESS !
其實，英文裏很多，但是，我關注的就僅此而已。
拿出顏料，工工整整地仿者前者的字母，淡定地添上……
真的很顯眼。添上後，其實會出現以下幾種情形：

場景一：放學後在學校停車場取單車回家，然後突然有人在你後面輕輕地拍了一下，你回頭一看，是一位不認識的高中男生，用膽怯的語氣說：“同好麼……”，眼神停滯在車上的字母。

場景二：你在路上騎車，欸 ，一不小心撞見老師逍遙地開著他的轎車，然後第二天。你被莫名其妙地叫到辦公室。"高山流水——知音難覓——”

場景三：一日您泊車於某速食飯館門前，出來時見到一位紳士在車前停頓了一下，然後向前走去，你看著那似曾相識的背影，搶先走到他面前——你一定要抑制住內心的波瀾萬丈，或許啊你早已激動得淚流滿面，這長臉，和DANTE大人在論壇上的頭像酷肖焉……

有些事情，就這樣”明目張膽“地做出來，讓內心無所拘囿，熱愛自己的熱愛吧！

申明：當然，如果此貼侵犯了捷安特公司的權益，我在此道歉，並懇求版主立即刪除，以免造成不良影響，饒恕在下法律意識淡薄，畢竟，原本只是今天我們來談天而已——鞠躬。

最後，所有場景均為虛擬吐槽，特別是最後一個，DANTE大人一定海涵。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
