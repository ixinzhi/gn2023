以及最后是一些重要提醒（例如免责声明）：

*   因为不是正经注册的，所以难以遇见什么时候就翻了，例如管理员帐号被封或者一起报错什么的。所以不要用它放重要的文件，一定要放的话……记得多留几个备份。
*   以及，管理员是有方法看到用户的 OneDrive 的文件的。（那你去问 MS 啊）虽然我很想自诩保证说我不会那样做之类的，但你们应该也不信。所以别把你敏感的东西放上去，如果真想放的话……那我也拦不了你，记得加密。

以及第一次登录时要求修改密码、绑定验证器和手机号码或者电子邮箱那些步骤是微软要求的，和我没什么关系。

*   从这下载 Microsoft Authenticator: [https://support.microsoft.com/zh-cn/account-billing/%E4%B8%8B%E8%BD%BD%E5%B9%B6%E5%AE%89%E8%A3%85-microsoft-authenticator-%E5%BA%94%E7%94%A8-351498fc-850a-45da-b7b6-27e523b8702a](https://support.microsoft.com/zh-cn/account-billing/%E4%B8%8B%E8%BD%BD%E5%B9%B6%E5%AE%89%E8%A3%85-microsoft-authenticator-%E5%BA%94%E7%94%A8-351498fc-850a-45da-b7b6-27e523b8702a) ，如果你不是 iPhone 和用不了 Play 商店的话， 可以在这个链接看看有哪些国内商店有下载： [https://support.microsoft.com/zh-cn/account-billing/authenticator%E4%B8%AD%E5%9B%BD%E5%85%AC%E6%9C%89%E4%BA%91%E4%B8%AD%E7%9A%84-android-%E7%89%88-ebbef05c-a429-4236-8570-1bb1900fec35](https://support.microsoft.com/zh-cn/account-billing/authenticator%E4%B8%AD%E5%9B%BD%E5%85%AC%E6%9C%89%E4%BA%91%E4%B8%AD%E7%9A%84-android-%E7%89%88-ebbef05c-a429-4236-8570-1bb1900fec35) 。但是国内版不支持通知登录，也不能添加个人 Microsoft 账户。
