# 1

个人觉得Eka那种文章前序言大概介绍情节并且提示关键词的模式非常好啊，可以节省太多时间了，看两行就知道是不是自己喜欢的文，而没必要浪费时间往后读。这里倒不是说有人写得不好没读的价值，只是大家口味参差不齐，比如我就更喜欢min的，而且晕血……看得正爽突然giga一顿狂踩，弄得内脏、血浆横流的，马上就萎了。
所以呼吁大家发文都用个更专业、规范的格式，实在是懒，至少也加个高能提示，标注出scat 之类的敏感区域~~~ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
