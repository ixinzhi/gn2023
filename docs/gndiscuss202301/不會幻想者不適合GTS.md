# 1

GTS就是英文Giantess的簡寫,Giantess也代表是女巨人的意思!
所以算是虛擬的物件,既然是虛擬,大都是想像力才會有的!
如果不會幻想就不可能有女巨人的想法,那就不適合GTS!

GTS現今有分很多類型,以大部分來說,區分為:
縮小人:人類縮小化,普通的女生在縮小的人類眼中就像女巨人一樣!
大   女:身高比較高的女生,或者男生比女生很矮之類的!
巨大女:也就是巨大化的女生,有10多公尺高到100多,1000多公尺高,甚至宇宙級的身高,國外作品以幾百公尺身高的作品居多!

而作品內容就很多的情節了,就像常常看到的人類縮小後被女生踩死或者巨大女破壞城市殺戮人類...等等之類的情節
我也不多說這些,因為每個人喜歡的情節不同吧!

其餘就像是外表,有喜歡美女有喜歡可愛型的也有熟女類的,國外的我還看過有喜歡老女肥女的,但那些都是少數吧!
因為我看到就倒盡胃口了!

GTS屬於幻想領域,所以裡頭內容的分類其實很廣泛的,所以沒有全部人叫好的作品也沒有全部人喊爛的作品!
因為這個作品類型你不喜歡,但別人喜歡呀!

對了還有一種類型的我還沒提到,就是神社裡有個網站ねこのふぐり,這個網站是介紹巨大少年的,主人翁名字叫貓野風栗!
應該有些人知道,有人說我喜歡巨大少女但怎麼還喜歡巨大少年,說真的我也喜歡可愛系的GTS,而這個網站雖然是巨大少年的網站
但作者設計的腳色也都趨向可愛的巨大少年,這方面我就不排斥了!而且該作者也是會有巨大少女的構圖,所以這裡鈺我還是會接觸的!但是神社其他的巨大少年網站我看了看之後根本不想看,只有ねこのふぐり這個網站我會去瀏覽的!
