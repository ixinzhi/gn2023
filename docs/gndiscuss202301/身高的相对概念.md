# 2

身為一名足控，對於身高都是用足的大小來做思考
基本上一個人的足大約是身高的1/6-1/7附近
也就是說20米=2.8米的足長(超大)
一腳就能把你包覆
200米足長28米
坐在地上把腳掌立直，你從7樓就能舔到足尖
2000米足長280米
幾乎兩個半的足球場長度的巨足
光大拇指就有三輛轎車長 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
