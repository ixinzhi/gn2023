# 11

我是一个对于各种大小身材体型女巨人上下通吃的爱好者，对于迷你女巨人的爱好并不很浓重，却也情有独钟，因为这个身材的女人和普通人类相差不大，文字描写起来显得有血有肉，读起来非常真实，所涉及场景空间和物件大小与现实生活较为接近，所以我们描写出来就能得心应手。因此，本人建议各位喜欢写作的朋友要想写的深入，不妨多写那些身材比人类差异不大的女巨人。
关于超级女巨人(让我暂且这么称呼)题材故事，由于身材异常巨大，离我们日常生活的尺度太远，所以我们很难把她写活显得真实，其实我们忽略了一个问题，对于一个体型异常硕大又有着普通人类外表的巨大神女这样一种角色，我们在她们身上套用着太多人类的印迹了，我想作为一个那种巨大体型的生命体本身就已经超越了普通人类的形态，她不会再有人类的任何生活特性，作为女神她们应该超越了人类本身的局限而达到了生命的至高境界。而在我们大多作品当中却包含着人类自身许多污点和腐败的特质，以及仍然受限于普通女人生理现象引起的灵性缺失，实际上这是不可能的。我们的意识中还没有摆脱这种陈旧概念，即使向但丁版主这样的高级写手也未能完全体现出来，尽管他描写的情节非常精彩和优秀了！
