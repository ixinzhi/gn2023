# 1

本人做伸手党也是很多年了，毕竟学生身份嘛，各项花费肯定还是要在父母的同意下进行的。但是，鉴于有些时候钱不花不行，也还是会去购买一些不那么合理（家人认为）的东西，就比如某些游戏吧。从Tencent到steam，再到最近的patreon。嗯，就是像这样一步步入坑。
       我发这个贴也无非就是想鼓励大家（像我一样以前大多时候是伸手党的人）从伸手党转变为会去赞助自己喜欢的游戏创作者，毕竟gts是个小众群体，游戏视频之类的除了少数大咖外基本都是独立制作的。至于独立制作，肯定会遇上很多问题的，而很多甚至不是技术上的，而是像花费那么多时间精力，但是还是难于解决生活温饱的问题。人人都是追梦人，为了人家的梦想（竟是为自己的爱好做近乎无私奉献的行为），请你尽可能尽己微薄之力（当然巨款我想人家也不会拒绝）。或者，更务实一些，你要是不赞助人家拿头给你高产，肯定动力越大成果越好嘛，所以，有可能的话就用微薄金钱换自己所求的更新或新游戏更近一步，有何不可。
       这里我就顺带分享个国人在patreon上支付的方法吧，就是-某宝直接搜。嗯，万能的某宝，就是这么简单粗暴。
