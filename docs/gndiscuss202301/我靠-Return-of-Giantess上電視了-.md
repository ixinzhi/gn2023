來補充一下好了。這個節目是「中部日本放送」，簡稱CBC的電視台，在半夜播放的自製節目。以搞笑為走向是很正常的，因為節目的製作協力單位是知名的搞笑藝人公司「吉本興業」。

有趣的是這個節目的收視範圍。儘管CBC是TBS集團的成員之一，不過因為這個節目是CBC自製的，所以放送也僅限於CBC的負責區域，也就是愛知、歧阜、三重等縣。

講白一點，幾乎就是鄉下電視台啦。雖然播放區包含第三大城名古屋，不過跟全國範圍來比……真的差很多。 <title>10</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
