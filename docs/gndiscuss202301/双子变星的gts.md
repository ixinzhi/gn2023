# 1

首先解释一下双子变星，就是两颗星星的大小随着时间周期性的变大变小。

假如有两个人，每个人的大小都周期性的变大变小，这样巨人和小人的身份就会不断交换。一个巨人玩弄了另一个小人之后，就会被报复回来。这样不断交替，最好各种play也会不断升级。大家觉得最后会发展成怎么样呢？为了方便，假设不管怎么样都不会死亡吧。

不知道是不是有这一类的小说，如果有的话希望能帮忙推荐一下。

想过类似的，但是没写。
我想的是男主单向缩小，女主一直正常体型，然后还有一个陌生人（女）不断变大。
男主能让自己体型不断缩小（不能变大），陌生人（女）则联动地变大同等倍数。
明面上的主线：男主得知了自己能力后就慢慢缩小，用不同体型和女主玩耍，由于新奇有趣，二人并未考虑后果，发现自己不能恢复之后，男主这才慢慢控制体型。（从得到能力到意识到严重性大概两天）
暗地的主线：陌生人（女）开始不断变大，她的身高很快就突破了吉尼斯纪录，这让她在电视和报纸上成了头条，但是她不能控制自己的身体，她不知道自己什么时候会变大（男主一缩小她就变大）。
当男主缩小到十分之一，她就是16米高的女巨人了，这让民众惶恐，政府开始出手镇压（男主因为是缩小，所以并没有造成太大的影响，他和女主单方面获得了陌生人的情况）。
之后他们和那个陌生人（女）沟通了，三人成了一条绳上的蚂蚱。（巨人控制女主，女主控制男主，最小的男主却用缩小自我来控制巨人的行动。）
