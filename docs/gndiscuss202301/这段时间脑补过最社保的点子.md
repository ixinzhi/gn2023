# 1

米娜桑，又好久不见啊

因为最近都在看田龟源五郎的作品，所以真的很难重组灵魂回归gts呢(好孩子一定也要去看哦![](img/78d1f3cf3d1c68881c363e2c6014f483.png)

又浪费了一个午觉的时间来收拾这段时间意淫过的点子，不知道这算不算忙里偷闲呢

一个版本的背景是魔法学院，每个学生都有擅长的魔法。男主和女主作为班级的尖子生，擅长的分别是瞬移和缩小术。两个人喜欢在将自己和对方缩小后，再瞬移到女同学的靴子里野战。
今天上课的时候，她们发现一位不良女同学因为天气太热，就把鞋子脱了，光脚支在桌子横梁上，就分别借口不舒服和上厕所离开教室，然后故技重施瞬移到同学的鞋子里。
但是当她们风雨过后想在下课前变回来时，却发现自己用不了魔法了，这才记起来这个女同学虽然成绩很差不爱上课，但是却对沉默类魔法很擅长。
下课铃响，不良女同学尝试把脚伸进鞋子，但仅仅堵住了鞋子的入口并没有完全穿好，这时女主角发现自己的缩小魔法被解开可以用了。这种明明已经被发现但是没被揭穿的感觉，加上两人全裸在同学鞋子里野合的尴尬，让她乖巧的选择将两人继续缩小到大概蚂蚁的大小。而不良女同学在短暂的停顿后，又继续将鞋子穿好紧接着就去学生街吃个晚饭，去电玩城玩到很晚，紧接着就回家了。

中间穿插这男女主角各种心理活动。比如两个人躲在女同学的指缝中，但因为女同学的一次蹦跳，两人被抬起又落下的脚趾分开了，直到女同学吃完晚饭重新开始走路，才艰难的穿过有规律抬起的脚趾，重新拥抱在一起。
亦或者在女同学在电玩城消磨时光时，随着时光推移，又冷又饿又害怕的两人，开始正视自己的处境，难道要这样无声无息的小石头，从此成为同学鞋内饲养的小人吗。

深夜，女同学终于推开了家门，回到了冷清的家中，她一蹬脱下了一只鞋子，却专门坐下，小心的褪下了另一只鞋子，平稳的放到玄关前，然后就头也不回的去浴室洗澡了，而随着施法者远去，两人身上的沉默魔法也随之解开了…………

# 2

没想到把脑子里的设定以别人看得懂的程度写出来这么费事。另一个版本就以后写吧。

中间要套什么心理活动和趾下互动就按诸君的喜好吧。

我个人是很喜欢这个故事模板的。

首先我不喜欢完全无关的人被缩小然后支配，所以我的主角都是这种，有gts癖好，会因为被缩小被支配而兴奋的小变态。

但是完全让小人为所欲为又背离了gts的乐趣，所以我会想找点冲突，让小人在将自己缩小后，失去恢复自己的能力。

而加上一个也有gts癖的变态女友，是因为我总觉得走纯爱路线，最后巨大娘和小人一定会不受我控制的产生情愫，那干脆就来个臭味相投的女朋友吧。那就可以把巨大娘完全独立到一个支配的地位了。而且这种不是第三者插足，而是第三者主宰两人感情的感觉，试过后，就会发现前所未有的好用。

最后就是让我自己忍不住要写下来的属性:明明有意识的无意识。
无意识类文章是我gts癖的启蒙，但在后天发展过程中，我又成长成一个不和巨大娘的脚趾互动就无法施法的绅士，所以当沉默少女的想法窜入我的脑海，我就迅速抓住了她。

而且会很在意对吧，会的吧。
第二天上课的时候大家是当无事发生过吗？
如果这样的话，不良同学会故意脱鞋诱惑她们俩吗？
不良同学是因为寂寞还是单纯因为好奇戏弄她们呢？
