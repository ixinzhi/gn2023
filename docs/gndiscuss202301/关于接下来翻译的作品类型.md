# 1

由于不明原因，作者big-n-evil已经很久没更新了，他的存货基本要被我翻译完了，所以想询问一下大家最喜欢哪种类型的作品，我好从中选择合适的作品进行翻译。
由于我的权限不足（这论坛升级好慢）没法发起投票，所以请大家以回复的方式投票吧，就截止到我翻译完剩下的存货为止。
另外我想为我的翻译帖申精，希望哪位版主告知下流程，谢谢。

1.真人图片/拼接图片：这类最多，翻译内容主要靠作者自己编，文本量也比较大，缺点是模特参差不齐；
2.插画：这类最少，且基本很少有配文；
3.3D建模/MMD等建模图片：这类数量很多，但配文的不多，配的文也一般较短，但是图片质量最高。
