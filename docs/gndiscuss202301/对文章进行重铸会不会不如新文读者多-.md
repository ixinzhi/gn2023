# 1

*本帖最後由 西呱 於 2020-4-16 11:22 編輯*

之前写的甥女随笔文，很多人反映写的太粗糙了，于是我在考虑是再写一篇新的，还是把那篇文章大改重铸一下
我担心如果推出重铸版，会不会很多人没有再阅读的兴趣
外传的话，倒是没有这种顾虑，但是很多的内容我确实是需要重写一下，而且并不觉得这些内容是可以分割的 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 2

重构或新的都可以呀，总会有人没看过的，只要写的比以前好都不是问题，而且你还是原作者。

至于有没有在阅读兴趣的问题，我觉得可以不用想这么多，圈子小能满足自己创作欲望优先吧，而且重构有加剧情且描写细致的话，或许还有更多人看！ <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
