# 1

不知道论坛有没有追10月新番水星的魔女的，刚更新的最新话里女主驾驶的高达一掌把敌方士兵拍死了，血溅了女二一脸；但女主杀完人却还是一脸天真的笑容，吓得女二很害怕
本来应该是相当震撼的场景，但作为巨大娘作品里相当常见的剧情，我还是不厚道地走神了![](img/6aa9b3e71108a323a0981d2b75ad6caa.png)

这种巨大娘轻描淡写地抹杀小人，把旁边的小人吓得半死，但女巨人脸上还带着灿烂的笑容的这种桥段相当戳我![](img/cdc51f89bb7d985db5e2efdb53ac7fc6.png)

最后小小地推荐一下这部水星的魔女，一共13集，后半段会在4月放送。虽然是高达系列作品，但剧情相当有趣，毫不晦涩，即使是完全没接触过高达的观众而言也没有任何门槛，可以放心观看
