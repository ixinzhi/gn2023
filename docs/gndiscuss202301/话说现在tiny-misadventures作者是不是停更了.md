# 2

根据作者在2022年4月18日的发言，下一次更新在当时仅完成了四分之一左右。

关于更新速度缓慢的原因：
其一是因为现实问题，包括疫情影响使得原作者及团队的现实生活在一定程度上遭遇了困难，无法像以往那样高强度集中精力制作游戏。
其二是制作团队在0.3版本公布后，发生过一些争吵和不愉快，近半数开发者离开制作团队。虽然后来有过一些人员恢复，但是目前制作团队仍然处于人员危机中。
其三是在目前状态之上，作者担心提高更新速度很可能会导致质量下滑。
除此之外作者亦因为不少社群问题（例如催更、游戏在一些网站被卖钱、被更改游戏数据再发布之类）而焦头烂额。

基于以上原因，现阶段的游戏开发速度极其缓慢，作者也对更新速度大幅降低表示过歉意，但是多次强调过并没有停止游戏开发。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
