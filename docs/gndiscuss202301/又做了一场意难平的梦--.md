# 1

就在刚刚我从一场梦中醒过来，有种很委屈却找不到人可以诉说的空洞感（摆手），

**梦的大致内容是，**

我看到个给文章配图的视频，内容是相当戳我性癖的那种，讲的是主角一行人困在了女教师的直肠里。
我知道这么说可能有点像《文文》，但是区别在于这篇真实存在的文章是无意识，而梦里这篇是以“毕业探险”为由把学生们骗到一个“山洞”里。

印象很深刻的三段情节，具体描写不记得了，只能大概复述，
一是主角一行人刚进入“山洞”时，描述这片荒地竟没有杂草丛生，主角诧异前人走过地面后留下的脚印很快就会消失
二是文章过半主角返程的时候，惊觉“原来这个花瓣状的洞口是她的肛门”（而此时的配图也相当带感）
三是主角和女主独处的时候，发生了关系，这段没有写过程，就是讲女主已经提上了内裤，嗅了嗅手上黏的液体。

**这场梦如果到此结束，那就只是普通的xp春梦而已，重点是后面。**

梦里的我开始检索这篇文章的出处，方法很笨，就是把其中的片段截取下来google搜索，
一开始在某个野鸡资源网站搜到了，但是要收费下载（充值金币）

然后我又用工具把片段切换成繁体搜索（不知道什么原理，但有些网站的搜索功能简繁体不互通）
这次搜到了，但是不完整，却也找到了作者的名字（好像是一串英文还是中文来着，末尾有个“GTS”）

（这时我浅浅醒过来了，眼前的电脑和文档都消失不见，只有天花板，辗转两下很快我又重新回到了梦里）

然后我在论坛上搜索到了这个作者，但是主题和回复都是0，我私信他，他在梦里呈现的就是一个具象的人出现在我面前，
我告诉他，我想看他这篇文，但是一旦醒来我的搜索记录就没有了，
于是他从包里抽出一本打印装订的本子，就是他这篇文章，装帧很好看，里面还配上了很多插图，也得以看到这篇文章总共有12章节，而第12章又写了三个不同结局版本。

他把本子交给我，让我去打印，我想到其实也可以用手机把每页拍摄下来，可是就在要动身的时候梦醒了，手上的本子也没有了，只剩枕头旁边手机还微弱的亮着。

什么都带不出来。

这种空虚感缠绕着我，我不知道该和谁说起，只能记录下来当日记发到论坛上。

我经常会做有关圈内的梦，无论是看到现实不存在的作品也好，还是梦见自己参与到某种play当中也好，
但都有两个共同点，
一是每次醒来后，如果不及时记录下来，可能不出一个小时就全都忘光了，
二是这些梦都比现实好他妈太多倍，有太多现实里不存在，看不到也玩不到的。

梦本质就和AI绘画一样，只是把看过想过经历过的东西，重新糅杂组成全新的东西呈现出来，我们不难从中找到很多现实里出现过的影子。

可我还是更愿意相信现实和梦境是两个相互隔绝的世界，
从比现实好百倍的梦里醒来，却不回去的寂寥。

写到这里的时候已经没那么难过了，用文字也还原不出当时难过的心情。
这种感觉，大概只有自己知道，那，就自己知道吧。
