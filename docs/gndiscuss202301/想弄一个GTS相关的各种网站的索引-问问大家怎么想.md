# 1

本来是想写一个索引网站？想要实现一个能够供很多人查看、完善，并且还能实时更新的那种
但是发现并不是那块料，也就只能写个本地的，写得还非常繁杂赘余，更何况还需要花钱...所以就留着自用，就不献丑了...

后来想想，google的在线表格似乎不错，毕竟没有成本来着。不过可惜的是，有点卡。而且很多功能是实现不了。其实我是想要能展开/隐藏类似的功能的，不然整个表格实在是拉的有点宽了，在排版方面也十分乏力，我本来还想每一个都多一列用来标注描述和属性的。

[https://docs.google.com/spreadsh ... g71C1pSc/edit#gid=0](https://docs.google.com/spreadsheets/d/1eUpu-8AOAv_G99ZMqaLA2jHd48Pejm6LC6Bg71C1pSc/edit#gid=0)
（需要翻墙）

先试行一下吧，看看能不能行得通，当前的这些都是gn和其他地方大家的帖子里发出来的一些。稍微有点多，我还没有全放进去，准备慢慢来
编辑权限也不太能放开，肯定会有人乱搞

不过从某种意义上，我感觉这个还是有这个必要的。fantia的作者或许无须记录，搜属性就有了，但其他网站的相关作者搜索难度还是有一点高的。前几年我就还处于什么作者都不知道的状况。知道近两年才对各种平台、画师等有一定的了解。
希望大家有自己喜欢的作者，且刚好这里面没写的，可以回复一下，我好塞进去。
虽然作者多得跟海一样，但是有时候真就闻其名但找不到网址。

先看看大家的建议吧，如果确实大家的回复让我感觉很有道理的话，我正好也可以想想别的或者放弃这个打算。
可能有点吃力不讨好，或者说白费功夫，没什么用也说不准![](img/c21ba2c6b3682f8392e3b81ea4b59fd7.png)
