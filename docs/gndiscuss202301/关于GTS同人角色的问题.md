# 1

在我的这个帖子里谈到了关于同人角色的问题，这个问题有必要单独再开一帖。
[http://giantessnight.com/gnforum ... &extra=page%3D1](http://giantessnight.com/gnforum/viewthread.php?tid=6691&extra=page%3D1)

就是关于GTS手绘作品（包括漫画）当中采用同人角色的问题。
（补充一下，其实日本很多有名的GTS画家，白兔、SS、HIK、还有《缩小学园》都出过同人作品或采用过同人角色。）

优点：
1\. 同人角色设计成熟，服装搭配，性格特点都是现成的，很容易引起同好的共鸣，对画师来讲也方便易行。
2\. 很多GTS文学作品都带有明显的同人的痕迹，如果要以此来改编为漫画，则很难再去创造新的不同的角色。
3\. 自己设计角色很难达到同人角色那样的效果，而且要付出更高的成本（时间和精力）。
（超弩级少女就完全是一个原创的GTS角色，目前来看，还算是成功的）

缺点：
1\. 同人角色有其应有的世界观和故事背景，如果硬是加入到GTS世界中变成“巨人”与小人互动，会很不协调。
2\. 以同人角色来创作GTS的图画或者漫画，需要对同人角色有深入的了解，除了外表的长相服装等，还要把握住内在的性格特点，这一点很难做到，画虎不成反类犬。（在下不是宅男，这方面了解的确实不多）
3\. 如果把握不到位，会引起对同人角色有爱的同好的反感和反对，而且被好事者传出去，对我们GTS文化圈有很大威胁。

以上，想听听大家的看法。

本帖谢绝灌水，谢绝沙发板凳，希望能听到有价值的意见和建议。

# 3

我不是很喜歡同人，一般看H漫畫的時候都會直接shift+del 和諧掉纏在Cxx中同人作品。
因為每個人對作品中的人物都有自己的理解，而同人很大程度上就是“和你不同的理解”，看著摸樣一樣的主人公說出根本不對路的話出來，囧TL之意會不斷襲來，當柯南開始當海賊，路路飛始玩推理的時候整個作品就是亂七八糟了。續集都有很多人無法接受，所以別說同人了。
當然同人也有很好的作品，比如水滸傳的同人金瓶梅，和什麽西遊X傳之類的肥皂劇不同，沒有選擇松江 晁蓋這些鮮已經大紅大紫的主角，轉而選擇了西門慶 潘金蓮這些配角，擁有自己的中心思想，而又沒有距離原作中的情節太偏。而比較有爭議的“同人”：高鶚續寫的《紅樓夢》的后四十回，很多人就把此稱為垃圾。
雖然“縮”借用了初音未來這個角色，但是初音被我們熟知的時候沒有任何的故事，基本上算是一個背景為零的少女，所以“縮”借用的時候根本不用在乎這一點，而團長一行人基本上都是打醬油的角色，啊囧本身也沒有多少臺詞。
如果要創作的話，我比較推薦的同人是東方系列，因為東方系列本身就是同人的奇跡，其價值也在無數的同人上。 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
