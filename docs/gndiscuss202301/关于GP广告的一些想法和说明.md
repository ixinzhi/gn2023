# 1

最近我对GP网站添加了广告，有人反映广告很影响体验。前两天我由于设备，网络和软件的原因看不到新加入的广告，于是就没有正式回应

我今天重新调试了一下，一套体验下来广告确实有些偏多

我去广告联盟咨询了一下，广告频率应该是没法降低了，只有开和关两种状态

我在网站上添加了一个提示，大家意外点到广告跳转直接退出就可以了

我个人觉得这个广告频率虽然有点高但是应该也没有达到无法忍受的程度

我本来想使用更为友好的Google ADSance的，但是Google由于网站含有成人内容拒绝了申请

我接下来会更新一些自购资源上去，就作为大家看广告的补偿吧

其实就我个人而言，这个网站本身只是一个服务和方便我自己随时随地访问资源的网站，经过大规模爬虫，上级存储服务商撤服，被人攻击等等糟心事之后，我已经好几次想要关闭站点了，添加广告，也算是让这个网站继续存在下去的一种方法吧。

**总之，我会坚守不收费访问的底线**
