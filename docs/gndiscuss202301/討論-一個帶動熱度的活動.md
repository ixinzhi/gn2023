# 1

.
是這樣的，因為最近論壇缺乏活力與熱度，為了提振大家的士氣、吹散懶洋洋的氣息，
有人建議我說，應該設法辦些活動來帶動大家的熱情。
不覺得我們可以試試看嗎？

初步討論後的構想是，為了鼓勵新人創作（或勇於發表創作），我們可以採用有獎徵選的方式。
開放會員以創作圖片、文章或影片的方式，提交他們的作品，經評審審核後來決定名次、並給予獎品。
而獎品的部份，則可能要請常出沒在本版的各位當「贊助商」了。
一些付費的漫畫或影片，應該是適當且夠資格的獎品。

我希望這邊先不要討論「會否有人願意參加」的這個問題。
請各位在相信有人會參加的前提下，提出對這個活動的任何意見。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

我觉得这个提议非常好！

正如我前段时间所说过的，要尽量让更多的人参与进来，投入进去，才能让我们这个小圈子走向良性循环。

1\. 参与的人多了，有了更广泛的群众基础，就会有少数有心人想认真的创造精品，这部分人就能成为我们群体的精英和骨干。

2\. 投入进去的时间越长，就会越有爱，越会去不断的改进自己的作品，参与GTS文化的讨论。

在以上两点的基础上，同好们的整体素质也会得到极大的提高，灌水和伸手党将会成为众矢之的。

这半年来，我也一直在酝酿一个GTS作品（或资源）的基本标准（理论性比较强），原来觉得没有什么存在的意义，现在似乎能有一些价值，过两天我会先发到这里请元老们过目。

最后自曝一下，我在大学时期就担任过大学生海报评选的评委（大学三年级的时候）。 <title>8</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

我們要的是原創作品。隨便轉貼或挖墳的那種惡劣的人，我們絕對嚴懲。
當然，我們確實必須考慮到有人故意擾亂徵選作業的可能性。吃不到葡萄就砍了葡萄藤的人現在可有不少。

然後我們得想想這個活動舉辦的訴求與目的。
有些人，可能有那個心，但畫出來的東西看起來像火柴棒人，寫出來的東西像老外寫的中文。
遇到這種平心而論實在爛得可以的投稿作品，我們該怎麼辦？直接否定對方可能的努力似乎也不好。
因此，我比較傾向設置一個「參加獎」，獎品可以是外面那些付費網站中的一個迷你短片或漫畫。
至於這種獎品有沒有吸引力，那就再說。

我是覺得，這個活動的目的不該只是鼓勵那些有能力的人去創作作品，
應該也要使一般的同好「有勇氣」張貼自己的創作。
我一直相信，有嘗試創作的人應該不少，但礙於怕受到批評或其它怎樣的理由，始終不願發佈。
我覺得應該要設法給這些人勇氣，拉他們一把。
當然，這不是說我們不能給他們負面意見，只是我希望大家學會用建議代替批評。
還有就是，在你對對方這個人的性格並不了解的情況下，不要拿對方作品的內容來開玩笑。
有人就是不想這樣被你裝熟，甚至他不覺得這樣很有趣，我最近就接觸過這樣的一個例子。

至於其它方面，包括活動的舉辦細節與評審的評分標準，我大概已經有底，這等稍後再述。
活動的主策劃我想交給F 超版。當然一旦發起，就是全體評審、甚至全體同好都要參與，這應該沒什麼問題。 <title>15</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
