# 1

最近发现即便是m向作品也是有具体区分的，虽然行为上受与授是建立不对等的情况下，但精神上的受与授则根据不同的作者表达的内容也有所不同。
大白话来说就是，双方是否真的爱着对方而不是一方单纯向另一方献殷勤或者遭受暴力。
有些作者很享受精神上凌辱对方的快感，也至始至终不把对方当做人来看待，用之即来丢之即去，这是我无论如何都不能接受的，因为我觉得这对另一方来说是在太过残忍。那就不是爱了，只剩下虐了。也就是对象是不是你也无所谓，甚至不是人也行。
可能是我多想了吧，但这一类作品我始终都冲不起来，甚至会心里发寒人会郁闷半天。T_T
