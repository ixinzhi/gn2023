# 1

我们现在的作品主要目的是渲染GTS，实际上描写真正女巨人的少，缩小男性的大多数。这主要是他们总要在现实世界框架中来阐述故事，很多人要描写缩小男性作品而显有描写巨大化女性作品，因为要为女性安排一个自己的家作为基地，在这里巨大的女性可以有很多工具和物品可以使用，生活比较方便。如果是女性孤立于正常世界而巨型化，她将成为众矢之的-人类的攻击目标，这样她也无法利用人类的正常工具和物品，甚至连衣服也没有，如此一来女巨人们的举动只能是破坏环境的战斗与杀戮，或者被弱小的人类征服奴役。这就是格列佛的两种境遇，都是自己被孤立，所以说描写缩小男非常方便。
其实要构思就要不拘泥于现实世界，必须要建立自己的新世界，故事中的世界可以虚构但是逻辑关系必须符合现实。其实GTS作品的产生就是一些人心中对于现实世界不满的映射，我们是要按照自己的意识去改变或者重新排列世界秩序，依靠这种意图来满足我们自己的心理需求，达到一种快感。
所以，我认为这样的故事中不要一味地把女人放大或者把男人缩小，我们应该把女性分类定位，可以同时放大和缩小，而男性可以不改变身材或小幅度收缩身体，因为现实世界中就是男人为社会主体，在人们心目中地位基本上差不多，而女人就不同了，她们之间的反差非常大。 我们可以按照现实生活中不同女性的身份、地位、经济能力和相貌为蓝本，按照我们自己的意识形态和喜好给她们重新定位，在这个GTS世界中加入一些SW成分，这么着故事就更加丰满也充满情趣。
所以我建议新构思，在作品中可以描写女性巨大化，但是要为她们添置服装配备一些自己的物品和工具，最后给她们安个家。即女巨人可以孤立于人类世界，但一定要有自己的家，在那里她象正常人类一样的生活，然后她的家园和人类世界是并行存在的，即一个蓝天下生存两个不同大小的世界。我认为描绘女巨人赤裸着身体到处游弋，这样的情形有些亵渎成分。我们的描写除了模仿《格列佛游记》那样的形式，还可以按照《艾丽斯漫游奇境》里面的模式写作。一个是主人公身材不变化但有不同大小世界，一个世界不变化而是主角身材不断变化。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 3

　　這些理論道出了有些同好對於GTS的看法，尤其大陸女神的同好所發表的文章應該大多都趨向縮小男模式比較多，有巨大化的大多都大個幾倍或十倍左右，可能較為重視現實面吧！

　　我本身可能受到以前小叮噹的宇宙小戰爭和一些超人力霸王特攝片的影響，比較喜歡在城市中巨大化的ＧＴＳ劇情，而對於我以前寫過的一些文章來看，我比較趨向有穿著服裝的巨大少女，畢竟裸體跟素足踩死人之類的行為，會讓身體變得很髒吧！所以我所找所看的文章跟圖片影片比較少會去看素足裸體類的！

　　關於ＧＴＳ文章創作，畢竟ＧＴＳ就是空想的，幻想的，所以別太實際，就照想寫的就寫，之前看過一些討論，有些同好太過於計較，認為女生變巨大，衣服服飾怎麼可能變巨大，不和哩，但ＧＴＳ本身就是不合理的，所以你想怎樣，那故事就是怎樣，寫作就是發揮自己創照力，別因別人怎麼看而不敢寫，希望我們論壇的同好有人想要創作就創作吧！相信很多人會支持的唷！
