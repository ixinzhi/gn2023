# 27

从生物学的角度来讲：洗洗睡吧![](img/3233bd66294232d422ae83c7b6a9b621.png)这里有几个最基本的问题：
一，你怎么样才不会被自己体重压死？
最简单粗暴的就是像恐龙那样长成大粗腿（暴龙：我能怎么办我也很绝望啊.jpg），或者听起来更牛逼一点的就是人体改造（不要太指望基因改造了，毕竟恐龙说起来就是基因改造能达到的效果）。
二，你怎么样才能摄取足够的营养与能量以供生存？
这个都不用去研究恐龙，看看大象之类的就明白了，一天到晚主要活动就是进食。
三，回到第一个问题：你知道一个5m高的庞然大物摔一下是什么情况吗？
大概就是严重点可能能直接死亡吧，嗯。
所以说，就算科技发展也不会往这个方向进步，相反更可能会研究如何控制身高，毕竟制造这种巨人可谓一点实际意义都没有，还会造成严重浪费。
当然也不排除在未来的哪一天真的因为某些特殊需要而出现类似的生物，不过即便如此，GTS大概也是可能性最低的解决方案吧。个人猜想会有这种需求的地方应该是一些无法使用大型机械的特殊环境来用生物代替，但女性的特征并不适合进行这种需要工作（或者说进行这种工作的个体没有任何必要拥有女性特征）
最后，以上内容也仅限于5m以下，至于更高的······地球范围内还是洗洗睡吧。
