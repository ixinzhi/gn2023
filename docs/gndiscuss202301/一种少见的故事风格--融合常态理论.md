我最早接触到这种风格是在这个故事里[https://www.giantessworld.net/viewstory.php?sid=5579](https://www.giantessworld.net/viewstory.php?sid=5579)不断变化的现实
那时我没有看简介，就很明显的注意到了这一点。
故事讲述的是主角有改变现实的能力，于是他就到处乱逛，把见到的妹子变大或者一些人变小什么的，关键在于这些人并没有意识到自己的变化，仿佛天生就是这样的。
这种感觉很新奇很神奇，不是大家常见的那种互动，但仍然是一种撸点，如果大家有兴趣可以看看这个故事，我说是一回事，看过体会了是另一回事。（还是很建议看的，如果你只见过温柔残酷无意识这些常见的，那么它能给你带来船新体验）
后来看见作者的故事简介“Tom, a normal guy, gains the ability to change reality in whatever fashion he pleases. And whatever changes he makes becomes the norm with everyone acting as if the changes were always present. Even though many changes are possible the only changes Tom will be creating is size related”才发现作者就是明确地定义了，要和平常一样

对于这种有体型差距却和正常世界一样的风格，我愿称之为“融合常态”，意为把现实世界(正常的社会关系) 和gts世界(大小差异的设定) 融为一体，人物虽然有着巨大的差异，但人际关系却和平常世界一样。

当然，如果不是故意要写常态的话，基本上不会有故事是这种风格，哪怕故意去写，也很难达到真常态，因为大小差异的世界就是和正常世界不一样嘛，何况我们要的往往就是身体差距带来的差距性。但是我们可以在某些地方某种程度上利用这种风格，带来另类反差萌或增加故事的新异感，下面列举两个有一丝成分的：https://www.giantessworld.net/viewstory.php?sid=7009这个故事讲诉的是妹子带男主健身，做得不行就会被悄悄变小(好像是) ，但是妹子使男主忘记了自己的变小，男主：“我是不是变小了。”女主：“你一直是这个身高啊，你不记得了吗，上周你还被一只鞋子压倒了，如果你不是这个身高怎么会被鞋子压倒呢？”男主内心：哦对，我确实是这么小，然后脑海回忆起相关画面。这些不存在的画面出现在了他的记忆里(还不是被篡改，是他自己脑补的)，让他以为自己就是这么小，最后不知不觉变得越来越小……
这个故事本身就很新奇很美妙，当然不能算融合常态，而是一种温和的欺负，但还是有一丝丝的相似，这是因为加入了“当成原来就是这样”，增加了一些观赏性。

我写的小短篇 长度单位[https://giantessnight.com/gnforu ... ad&tid=30543&extra=](https://giantessnight.com/gnforum2012/forum.php?mod=viewthread&tid=30543&extra=)尺寸变化后大家都会以为本来就是那样，男生变小被当成正常状态，这种感觉就算是融合常态了，但后来女生变大时震惊到了城市便不是。
总结下来：
    1.当你遇到大小差异时，不要有震惊迷惑恐惧奇怪好奇什么的情绪，要习以为常。
    2.妹子不会故意利用巨大的身高帮助/欺负小人，因为这是身高改变带来的影响。具体怎样我硬说也说不清楚还是建议看一下最前面那篇文章，才能体会。

对于 我记不起来了啊 大佬写的 母女[https://giantessnight.com/gnforu ... tra=page%3D1&page=1](https://giantessnight.com/gnforum2012/forum.php?mod=viewthread&tid=30612&extra=page%3D1&page=1) 这篇故事，我对网课这个地方十分意动，小女孩上网课然而全世界都在她家里这个太妙了，于是跟作者申请，准备就此翻改一个，我想翻改的就是一种加入了类似于融合常态的东西，(以下为个人翻改，不是原文内容！)全世界都缩小到女孩子屋里了，妈妈却要求她继续好好学习，用的理由是：“好好学习长大了能有好工作，能回报社会。”女孩：哦。然后她还是一个需要读书写作业的普通小女孩，这样的融合常态。以我们正常逻辑来看，你说全世界都成你家的地板灰了你还好好学习回报社会呢？这种荒谬的反差，正是融合常态的魅力。
女孩知道全世界在她家且过正常生活→融合常态
不知道→无意识
知道且玩小人→残酷
知道且照顾小人→温柔
大概就是这种区别，类比一下，此处使用常态又有点像“憨憨无知少女对小人世界的错误认知并很傻很单纯地做事。”

另外说下我想翻写的内容1.融合常态2.母亲掌控世界而女儿不知道世界在她家里3.女儿不出门以为自己家里的是小人而外面是正常人4.“外界人”的世界地图是一个室内，女孩以为自己在一个巨人室内，不知道所谓“外界人”就是自己屋里的小人，巨人竟是她自己5\. 女孩会对网课的老师同学找到在自己家里的位置且互动。

这里有多个相互矛盾的内容，既知道又不知道，既如常又非常，但我还是决定写成一个故事，这并非逻辑错误(逻辑错误一般是作者失误造成的，而我这是故意的) ，这叫增加艺术性，当然，真正阅读起来并不会像逻辑错误那样违和

摘自度娘
      世界系是“主人公（我）与女主角（你）为中心的小小的关系性问题，中间没有夹杂具体设定，与世界的危机及世界的灭亡等抽象的大问题直接联系的作品群”。
但是非常渺小的“我和你”的纯爱世界和非常夸大的“世界危机”的同步进行是世界系的特征。 很多人认为这样的世界系是在《[新世纪福音战士](https://baike.baidu.com/item/%E6%96%B0%E4%B8%96%E7%BA%AA%E7%A6%8F%E9%9F%B3%E6%88%98%E5%A3%AB/55458)》的影响下出现的，但是也有人认为“我和你的世界”加上“世界的毁灭”的这种构造是galgame和成人游戏的特有方法。

世界系这个词据说最初出2002年10月下旬。因为没有明确定义就传播开了的原因，所以比狭义定义范围更广的用法也有。 比如，世界系是以“打算去控制世界的意思”和“对成长这个观念的拒绝意识”为两大基础的作品群。
有的看法认为清凉院流水的JDC系列及上远野浩平的ブギーポップ系列可以作为代表。清凉院在‘カーニバル・イヴ’(1997年)中有这样的话：“以创作与社会派小说不同的世界派大说为目标。”游戏撰稿人元长柾木认为这是“世界系宣言”。上远野同年发表的‘ブギーポップは笑わない’与前者合在一起被认为是世界系的出发点。在这两个作家的影响下出现了高桥しん、新海诚、秋山瑞人、木城ゆきと、川上稔、大暮维人、西尾维新、うえお久光等世界系作家。

现实扭曲的话，侧重的是“尺寸改变了，相应的历史扭曲了”。小的来说比如你的青梅竹马变大了，在这个世界线上，可能你受欺负的时候一直是你的青梅竹马在护着你，现在的她会认为一直是这样子，但你记得另一条你有一个娇小的青梅竹马的世界线。大而言之比如女性的身高整体增加了，这个世界的历史女性一直都是这么高，但主人公记得另一种历史。故事的看点之一就是不同历史的对比产生的奇妙错乱感。不同的尺寸应当有不同的历史和记忆。这也是为什么这种故事也叫”memory alteration“或者”history alteration“

至于你谈论的融合常态，大概是不同尺寸的人过着与之前相似的生活，重点只在此刻的整个世界的错乱感。和楼上说的”世界观本来就是女大男小“这种有点像，这倒也是一种很有趣的设定。当然我们也不必很细致地划分不同设定的种类，喜欢就好。
