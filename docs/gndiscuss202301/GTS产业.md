# 2

这个题目楼主发过很多次了。

还是认真回复下吧：

1\. A片和SM片是因为有大量的需求，GTS仍然是很小众的群体，你问问你身边的男人，知道A片和SM吗？他们都知道，你再问他们知道GTS是什么吗？不会有人知道，这就是群众基础的问题，这跟恋足和SM都差太远了。

2\. 资金、演员、技术，这三个要素，看似简单，但是实行起来都是登天一样的难题，为什么偌大的亚洲几十亿人口只有一个GEN先生？为什么他花费了自己人生大量的时间精力和金钱，才做出那样你认为“很假”的作品？背后的艰难可想而知。

3\. 中国人都是吃干饭的吗？我可以负责任的说，在世界GTS圈子内，大多数人都是吃干饭的，华人GTS圈子本来就是这个小众群体中的少数派，有少数几个创作者想做点什么贡献都会被人嘲讽，你还能指望我们创作出什么优秀的作品？

最后，楼主在这里所探讨的一个根本目的就是为了得到更多更优秀的GTS作品，那么只有两种方法了：

1\. 认清一个现实，我们是小众群体，是少数派，在世界范围内能够满足我们需求的资源极度匮乏，要想得到更多更好的资源，只有靠我们自己来创作，自给自足。这就需要鼓励大家多多创作，保护和激励更多的原创作者。

2\. 有钱能使鬼推磨，楼主要去赚很多钱，在社会上混到足够的地位，能走入演艺圈去认识自己喜爱的明星演员，让她们为你表演，此几率虽然不大，但是也不能排除今后会有我们的同好成为社会名流。到时候我们都能沾他的光。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
