# 8

个人认为论坛主要是提供一个让具有相同爱好的人一起讨论交流的平台，写文章以及转载资源之类的只是附加价值。因此，收取费用或者设置权限之类的行为可以说不是很符合这个论坛创办的根本目的，尤其在人流量大、论坛发展较好的时候，无疑是会遭到大部分人反对的，而且我个人也觉得隔壁商业化气息太过浓重；但是，就目前的情况来看，上述行为是利大于弊的。
弊：论坛惨遭DNS污染（或者其他什么原因导致部分用户无法正常访问），人流量急剧减少，此时采取收费模式很容易流失用户。
利：首先，同楼主所说，

> 如果价格合适收点费的话若是能促进更新也不错

；其次，我觉得更重要的是，现在查得那么紧，再加上gts圈内的过分图片（如寺田的重口味图）在网上被多次曝光，采取上述措施可以使论坛的隐私得到有效保护，毕竟愿意付费或者权限足够的用户肯定是真正喜欢gts的，而不会去傻傻地到处进行所谓的“宣传”。
以上仅为个人意见，欢迎指点。 <title>9</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

实际不妥，优先这是论坛，不是商业网站。其次gts不仅小众还是个灰色领域，还是不要那么张扬的好，最后，有很多人有储存文档的习惯，如果订阅制度，私下传阅怎么办。相对于免费文章的私下传阅，付费文章的私下传阅更为致命，现在很多作者不想接着写，无非两种，一种自身不想写了，一种没有人评论，讨论。如果变成订阅后。不会解决这两个问题，反而会出现第三个问题，因为我明明付费文章，但是很多人却免费看到了这种不舒服感。再者，很多伸手党常年一个评论都没有，如果付费他会看么？另外说下，付费并不会提高作家写作速度，因为谁也不差那几毛，中文文章群体那么小不可能实现盈利，不为盈利而写文又会回到那个老问题是。说实话文章不更新主要问题在于没有相关讨论度导致写手没有热情，虽然高付费可以一定程度提升写手的热情，但是实际上上面也讨论了高付费是不现实的。比如说国外patron一些著名的gts大佬起价1，3，5美元/月 这个程度写手要写多少字才能达到？还是说读者肯每月花费这些钱来固定观看。所以解决文章坑的主要因素不是付费而是如何保持高讨论度。这里有几个建议

专门来讨论文章的地方。
后续回复可见功能。
连载性质小说 只会回复后才能继续阅读。比如说第一章免费看 第二章需要回复第一章内容后再看以此类推。
