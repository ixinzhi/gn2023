# 1

今天在文章区翻了一会儿短篇。感觉大部分短篇与其说是短篇小说倒不如说是节选片段的样子，读起来有种没头没尾的感觉，好像有好多东西没交代清楚，更像是突然想出来了一个画面然后就那么记录下来了。咳咳，跑题了，下面回到主题 说实话文字在画面表现上是远远弱于图片和视频的，但小说作为一种能够长期在各种幻想领域都有所建树的表现手法，自然应该有其特色才对。只是本人一直感觉没能get到这个核心优势。
那么大家觉得小说的真正优势在哪里呢？

个人觉得小说的真正长处在整体世界观的交代上，小说可以用大量文字来说明一些用画面无法讲明白的东西，比方说大眼珠子的《希灵帝国》就用了大量的篇幅来构建起一个基本自洽的信息宇宙的世界观，之后甚至可以跟着这种。但这么一来又很容易陷入堆砌设定的陷阱之中，最后写成了一本大百科或者设定集而非小说。

# 2

我觉得文字的话比起视频漫画的优势就是心理细节方面的描写吧，现在的gts漫画视频感觉都比较偏向快餐式，不像语言文字能用大篇幅去描绘世界观和营造屈辱感。小说能给人更深的感受和印象 ，语言文字的魅力很多时候不是视频漫画能比的。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 3

小說=漫畫
散文=單圖
小說漫畫是一種各方面的功力集成，取其精髓去其糟粕
世界觀有強弱之分，人與人的故事的世界觀是不太重要，最多交代背景算了。強的世界觀是對角色有影響，世界變了人也要變，有這樣的世界才有這樣的人，空穴來風未必無因，完整交代佔的幅大（像叫人先看歷史再看故事），在故事中用間接表達較好但會很拖長度，不易短篇化。
市場定位也很重要，以我看法像打遊戲的方法最易入手，用遊戲手法打比喻學生都明，拿科打比喻學生不一定明。西遊記唐僧四師徒，用遊戲比喻的話，唐僧吸怪（不是坦），吳空打怪。 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 4

操作起来比较简单可能也是个关键因素吧。
绘画要有那个天赋，在此基础上还要再苦练上个几年才能拿得出手，而且还要制备不少绘画用的软硬件，不管是学习还是置备这些软硬件，都是要下不少成本的。（所以我特别佩服寺田老师他们，免费放出了那么多张优质图片，要知道那个水准的图片完全可以拿出去卖的，他们真的是在用爱发电啊。）
然后拍视频要考虑到请演员，要置备摄影器材和各种道具，以及天杀的后期处理，需要下的成本比起绘画来很可能只高不低。
反观文章…一支笔，一张纸，，一些时间和一颗想写文的心…必须要准备的东西可能就这么多?而且甭管写出来的东西优不优秀，就算再差，偏差值也不会像绘画与视频那么高。门槛低，再加上先期不用投入太多，所以更容易普及化与大众化吧。 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 12

用电影相关的理念来说，小说就相当于剧本，而漫画就相当于导演分镜，而视频就相当于发行的电影。作为一个优秀的剧本自然能够反映生动其中的人物的形象，漫画这种导演分镜就是在剧本上的提炼，相对于剧本，能更准确的抓住剧本中的重要情切以最适合的方式展现出来，而不像剧本那样晦涩。但是由于篇幅问题，所以在很多人物塑造上面只能通过对白等情况来把握，没有剧本的细腻。至于视频，就更加直接了，但是视频的好坏就和演员还有制作团队与投资等等有关系了，大体上，圈子内的视频因为种种原因，我觉得都没太多意思 <title>13</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 16

我非常认同小说的想象空间一说，也许寥寥数笔就有着数十年的画工积淀和正好碰上合胃口画风的运气的效果，特别是对我这种中二病晚期加二次元禁断综合征末期的处男来说，三次元的长相远不如漫画顺眼(尤其是这个相对小众的圈子)，而小说更是随心所欲的yy甚至可以自己瞎改，比如换个日系名字之类的。但我想小说上手虽易，但要想写的精彩却是很难的，需要的积淀甚至要远超于练画画的时间，不过好在文笔这个东西，日常其实就是不断在练习，想要写出好的gts文我觉得要本身文笔好再加上富有想象并且能对上读者胃口，当然有些大神总能打开我新世纪的大门(想想当年在gts吧里一点残酷都不能粘的我，再看看现在手中uru的扶她魔法少女本，我的节操早已蓬舟吹取三山去)，这样的文章才会精彩，但相对gts漫画而言，虽然上手不容易，但是只要成了uru那样的大神(不太会画崩)，他的随手草图对我来讲也很受用了，夸张一点即使Vincent来画gts图，也不会有天差地别，毕竟gts漫画不是很注重细节的刻画，意境可能更重要。另外就是由于gts的相对小众性，为了经济，漫画通常是直击要点，大量非gts的铺垫是比较没效率的，而文字的廉价使得可以展开更多世界观之类的背景，和心理描写的细节，当然图配文你问我滋补滋磁，我当然似字词的啦(笑)
