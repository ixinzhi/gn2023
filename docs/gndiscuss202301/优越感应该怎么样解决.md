# 1

因为工作的原因，我接触过很多心理疾病患者和各种各样的人。
其中有一类人，对外界事物始终抱有怀疑态度，对逊色于自己认知和能力的事物却看不起，有着认为自己凌驾于其上的想法，不能从事物中发现本质、学习。
我曾考虑过击破他们的优越感的资本，然而这些人大部分无法受到激将法或击败法等一类传统办法的影响，处于绝对的自我当中，油盐不进，同时极其爱表现自己。
这样就俗称为“优越感”。
然而优越感对于认知自己和使自己完善、进步来说是极其大的阻碍。
由于样本不足，再加上我并未有过类似的感情和想法，当面对因“优越感”带来的心理疾病时，我没办法及时去疏导和理解。
我在很多的地方都有探讨过类似的问题，但基本上无疾而终，最近浏览互联网时，我深感这个现象更加突出，并且还出现了“外种优越感”等新的病症。
集中表现就是认为别人的、别家、别国的东西比自己的、自己家的、自己国家的更好。
现在想和坛友就这个问题进行讨论。
本话题的第二个问题是——如果巨大少女（Giantess）没有对自己的优越感，那么她会伤害人类吗？ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
