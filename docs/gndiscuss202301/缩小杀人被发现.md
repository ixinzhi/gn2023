# 4

重点还是在于人能缩小这点大家知道不知道。

如果大家都知道缩小技术的话……那么玩儿蛋啦。能那么随便杀人就有鬼了。

不过总体来说有人没了还是会被调查的。而一般来讲，最容易出现破绽的应该是人。所以要看大概多少人知道相关的缩小技术。

比如，是女主的亲人有进行间断科技研究的话，一般很容易就可以查到。

如果女主有闺蜜或者团体行为的话，找到破绽的可能性就大的多了。

总之，我觉得这种案件如果是一两个人消失的话，跑掉比较容易。但如果开始连探查的人也消失的话，很快就会有更多的人开始注意了。

附加：作为我们圈内人，毕竟读了很多文什么的，所以很难准确理性的想象实际上的情况。作为思考，可以考虑“类似的奇幻杀人手法被发现的可能性”的问题。

比如说，假如有一个手法可以把人变成纯水，或者把人变成纯灰，被探查出来的可能性是怎样的？
