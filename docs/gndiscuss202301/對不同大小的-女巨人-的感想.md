# 1

<180cm
對比較高的女仕沒有甚麼感覺，反而很想高過她們
就是回歸了現實的心理，很矛盾的基本上就是踩生果那些

180cm-200cm
非常高的女仕
有很多幻想，都是睡房那些
大部份影片都是拍睡房那些...加上約會的景況
但這也是現實與幻想的掙紮，又是很矛盾的心理

200cm-10m
非現實非常高的女仕
不太流行，也沒有很大感覺

10m-100m
女巨人
最喜歡的是50m，不太大不太小，剛剛好
最好就以腳的大小來作比較
一隻腳跟一台車差不多大小就最適中
這也是最多幻想，最多互動的系
小弟試了幾次以文子表達夜中的幻想.....
還是留待文筆較好的人來寫吧![](img/849fd4931c60eeeeb1c5c9d1c3799899.png)

100m-500m
超巨大女巨人
已經大得人都看不到的大小
感覺已缺少了互動，變得跟踩生果的分別不大
只是會講'你城市怎麼怎麼'那些説話

>500m
極巨大女巨人
就是踩到小人在女巨人的腳指指紋中死去那些....
請理解小弟未能發掘到箇中的樂趣

知道這壇中有很多>500m的支持者
請發表你對不同大小的感想
甚至攻擊小弟喜款的大小(不過不要攻擊小弟)![](img/bd15f544951e88cbc9e70768366c9ed0.png)

個人所偏好的女巨人大小約略是在3~20公尺左右

畢竟我很喜歡想像各個角色之間的心境，行為模式和各個角色之間的互動

比方說尤菲並不只是一個靠著龐大身軀來欺負男朋友的小女生

她也想和堤洛變得更加要好，想穿著漂亮的婚紗和堤洛舉辦一場浪漫的婚禮

看見蟑螂的時候會嚇的尖聲怪叫，也會因為怕黑而在半夜偷偷溜進小男友的房間

就我而言，這才是最主要的樂趣所在

而不是單純地將女巨人作為一個洩慾用的題材

當然我也是會有精蟲上腦神智不清的時候

但即便如此

我仍舊無法從一腳踏平整個城市或是一口吞下整顆星球的超大型女巨人身上享受到什麼樂趣

在我看來，那不過就是一個夭壽大的女生而已
