# 2

其實你不用那麼辛苦啦!
反正看不懂的人自己在去翻譯就行啦!(比如說我啦TT)
因為也只有少數人看不懂
不過有個簡單的方法
你可以到google的首頁
在搜尋列的旁邊有三個選項,請點"語言選項"
在要翻譯的文字那欄你就把文字貼上去
點從"中文"到中文(繁體),再按"翻譯"就行了! <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
