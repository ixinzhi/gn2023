# 3

嘛，他以前的風格我個人認為比較死板、比較單調。但他會在每部新推出的影片中加入幾樣新要素，應該是為了嘗試技術或探探市場吧！總之我覺得GEN一直有在進步，這是件好事。

不過他影片中女巨人的行動一直有點少，這點倒是沒啥進化。美國Giantess Zone Studio的道具設備雖然遠遜於GEN，但女演員的動作是真的勝出不少，假如你不排斥西洋臉孔的話。

當然兩邊的規模也有點不能比啦。Giantess Zone已經是集團化經營了，產量大，又常固定跟幾位女性合作，因此女演員對拍這類片子都很有經驗，這在影片效果上真的是有差。當然也可以說，兩邊著眼的賣點不太一樣，就像Giantess Zone那幾部Mega片的道具真的是看到煩了。

最後，我說說我自己的想法：打死我都不信大陸沒人持有GEN的影片。不會說日文？GEN懂英文呀！不懂英文也未免太遜了吧(至少在台灣不懂英文大概就只能行乞了)，所以溝通不能當藉口；那購買呢？大陸現在經濟起飛，有錢人滿街走，會沒有信用卡好支付嗎？不可能嘛！那有溝通能力又有信用卡，基本上就已經具備當一個「消費者」的本事了。若真不想付那麼多，可以辦合購啊，台灣這邊都快要第四屆了。你們找台灣幫忙基本上是不可能的，因為沒有三通，也許明年可以，但今年真的不行。現階段大陸同好真的得靠自己，自立自強。
