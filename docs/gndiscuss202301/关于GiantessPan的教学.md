# 1

大家应该都知道，我架设了一个Giantess资源分享类的网站。
但是我本来以为大家应该都会使用这个网站，但是似乎有不少人不了解如何使用它
我在这里就简单的讲解一下网站的组成和使用

首先网站的网址是：
[https://share.giantess.eu.org/](https://share.giantess.eu.org/)
或者
[https://giantess.eu.org/](https://giantess.eu.org/)
这两个地址是等价的，喜欢哪个用那个即可

首先网站域名不是国内常用域名，网站内容也包含一些瑟瑟的内容，当然会被某些垃圾浏览器的“安全系统”所拦截。
所以第一个重点就是：
一定，**不要使用包括但不限于QQ/微信自带浏览器，手机系统默认浏览器，百度APP，360浏览器，搜狗浏览器等的垃圾国产浏览器访问本站**，推荐使用**Via(手机端)，火狐Firefox，Edge，Chrome等干净、快速、现代化的浏览器**访问！

然后进入本站前，会遇到一个几秒钟的中间页面。这个页面是一层安全保护页面，绝大多数时候会很快消失并让你进入主页。但是在某些及其特殊的情况下，你会被这一页面所阻拦。这个时候，你需要做的是关闭所有浏览器插件，关闭代理，清除浏览器缓存，重启路由器（如果有），然后再次尝试访问。如果依然不行，请联系我。

进入主页以后，页面上会出现4个框形元素。
第一个代表着当前所在的目录，你可以通过点击里面的按钮来返回你想要的上级目录
第二个是广告，忽略即可
第三个是公告栏。**在某些情况下，公告栏可能会报错，这一报错不代表文件有问题，忽略即可**。
第四个就是文件列表了。

除此以外，**右下****角****还有一个蓝色的圆形按钮**，它的作用有两个：
1.在目录页面下时，点击它切换列表和图案视图，并显示/隐藏目录下图片的预览。
2.在文件页面下时，它呈现一个下载图标，点击它就可以下载当前的图片了。
当你到达一个满是图片的文件夹是，点击右下角的按钮，就可以显示目录下所有图片的缩略图了

当你找到你想要的文件，点击它，你就会来到文件页面
如果这个文件是一个视频或者一个图片，又或者是一个PDF文件这种支持在浏览器上面预览的文件，稍等片刻，浏览器就会将文件的内容加载完成供你观看。
如果这个文件时一个压缩包或者是一个陌生类型的文件，页面就会提示“无法显示预览”。**这只是页面不支持展示这个文件而已，文件本身还是可以下载的**。你只需要单击之前说的**右下角的一个蓝色的圆形按钮**就可以下载这个文件了

特别的，如果预览的对象是视频，由于文件提供服务在海外，**视频加载速度可能不甚理想**。建议将视频下载后再观看。如果**视频在线播放时黑屏**，是网页端不支持视频编码的缘故，**视频本身是好的**，**下载后即可观看**。

**关于批量下载的问题**，**出于服务器压力的考虑**，**不提供官方的批量下载功能**。如有需要可以使用第三方工具或者自行编程解决。
如果使用了爬虫类工具，在访问过高的情况下，会暂时性的限流。如果使用过爬虫类工具然后被拒绝访问，稍等片刻即可。

**最后关于广告：**
**广告是支持站点继续运行，支付网络服务费用的基础。**
**本站不会拒绝屏蔽了广告的用户浏览，但是请明白：禁用广告的人越多，本站停止运行的可能性越大。**
**如果有更多的人愿意关闭广告拦截，广告收入高的话，我会在支付服务器费用之后用这笔资金购买资源以供分享。**

**安卓端强制跳出流氓页面的BUG已经修复，无良广告主的马已经在天上飞了，各位可以放心使用移动端访问。**
