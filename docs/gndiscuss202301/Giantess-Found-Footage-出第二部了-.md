# 1

在GtsTube上看到了免费资源，和第一部一样精彩。总体风格没有变，依然是残忍系，能体会到小人的绝望。总之看完以后愈发觉得GTS爱好确实挺变态hhh。

下含剧透！

和第一部不同的地方在于多了一些足控内容，自慰的视角更直接了，不过还是以吞食为主。另外这一部的“录像”画质好像还不如第一部……背景噪音也比第一部大。

大神还是牛的！

说一点有趣的地方，两部是串联的，第一部的女主在第二部结尾出现了（更好看了），连鞋都没换：粉色的高帮帆布鞋，但看上去没怎么变脏……第二部的时间在第一部的一年半以后，可以看出是一代女主在建立以虐杀小人为主题的小团体，二代女主结尾算是入伙了。另有一人之前没出现过，作者可能会在下一部用到她？

和一代女主的娴熟、残忍、冷血不同，二代女主明显是新手，作孽时和事后的心理都通过动作神态表现出来了（手抖、恍惚、颓然坐下、不处理现场），更加让我佩服作者的功力，同时这也是一个新视角。

不知道作者是不是有长期做系列的想法，但如果想做一定能做，毕竟剧情已经打好了基础，能发挥的空间太大了。期待他的新作品。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
