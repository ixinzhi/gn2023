# 1

嗯，講老實話，我也不是很清楚自己為什麼要寫這一封信，也覺得我寫這封信可能會引來一陣論戰，但是我心中還是隱約想寫。可能是因為暑假那陣子感覺上論壇文手井噴、現在卻又感覺上緩緩變少了吧？
Re-Fairyland的A君和深紅-Scarlet的皮帶、許久露天大坑的劍爺(遭毆)、還有許許多多發新文試手的新面孔，這些人在暑假結束之後或許有人還堅持著填坑、又或許就這樣消失了。就我的觀察，消失的屬於多數。
為什麼會這樣呢？消失的文手中不只有劇情派創作者(常見)；同時也有福利滿滿的福利派創作者。是因為開學了嗎？不，我個人認為原因不僅於此。
GTS是一個很棒的要素，對我來講，她在文章情節衝突上的棒，完全超越了單單當作一個「性偏好」的棒。因為這是一個在文章上可以造成很直觀衝擊的要素。不僅是力量差與體格差而造成的衝擊，更是有著情感的可鋪陳性。而且，這樣細膩的情感鋪陳空間，是一種專屬於GTS的特殊屬性。

等等！這裡好像有哪邊錯了，萌R你憑甚麼認為這僅屬於GTS？哥吉拉不行嗎？進擊的GT不行嗎？
不，哥吉拉不行，進擊的GT也沒有GTS在文章上那麼有可操縱性。
因為哥吉拉沒有感情，只有單方面的動物本能與破壞，這樣除了選擇背景要在東京還是紐約之外，一點可控性都沒有。
那進擊的GT呢？或許是我的私心、又或許是刻板的印象，男人在情感的表述上是有一定難度的。講白一點就是很ㄍ一ㄥ，比較放不開。GT最近的連載中萊納跟貝爾霍特處理得很好，跟同學之間的互相喊話也堪稱一絕，但是沒有了尤彌爾跟女神希斯特利亞的話一樣要失色不少不是嗎？(被打)
重點，就在女生給人印象的多變上。可以強氣(龍)可以狂(魅)可以傲(鷹)可以溫柔(燕)，而且這些特性都可以發揮得很極端，而且極端得讓人覺得很自然！因為女性就是這樣一個像水一般多變的生物。水可以洪大地破壞、可以溫濡地創造、可以溪流中柔軟、可以在兩極剛硬。在這麼多變的元素之間，如果能互相激盪，那麼，這篇文章能激盪出多少火花？

或許有人會說：我就是想寫福利懶得去想劇情，有差嗎？
當然是有啊！
你想想，一個人類型態的GTS走福利路線，可以應用的部分大概只有前後二陰、雙手雙腳再加上嘴吧？用排列組合的概念，大概是C7取1(面對GTS一個人頂多就被一個地方PLAY不然你要怎樣？)，頂多給個觸手再加上FUTA，讓你C9取1好不好？這樣也才九種變化。可是，如果在故事中加入感情，GTS跟人類之間的關係就不是C9取1了。因為GTS可以有許多情感模式：親情的互動、愛情的互動、相互景仰、互為仇敵、青梅竹馬、歡喜冤家、女王強氣、病嬌、傲嬌、天然呆萌等等。
而這些情感模式底下還有不同的七情：喜、怒、憂、思、悲、恐、驚。而且人與人之間的情感連結還不只一條，妳可以對我又愛又恨、妳可以是我青梅竹馬型的歡喜冤家，這麼多組合，那就不是C幾取幾可以說清的了。而且在體格與力量相差甚多的情況下，又要相戀相依、長相廝守，這樣的組合難道不會讓人覺得心弦觸動糾結、卻又心醉不已嗎？
再加上，如果你情感寫得好，自然有情感被啟迪的文章好朋友來跟你討論劇情、文中人物的情感是否合理、自己有沒有遇過相同的感情等等。
拋棄這些成分，純寫福利，那麼得到的回文大概也就只能期待是一群正在撸的宅男們用最短的時間給你的最短回復，因為你啟發不了他們心中的共鳴...你只是打中了他們小頭上的G點而已。
看到一堆「樓主淫力滿滿」或「好人一世平安」，看久了也會覺得自己沒有什麼成就感，不是嗎？可是，純寫福利，又能期待誰跟你煲QQ湯討論劇情呢？

對我來說，小說就是一個與自己對話的過程，是一種內心情感的解放...許多你渴望、卻不一定會經歷的濃郁情感，如狂喜或大悲、或是心被挖空的失落或是遭到背叛的情傷，全都可以在小說中模擬出來。GTS就是一個完美鏈結這些情感與我所愛的虛幻世界的璇璣星。

希望大家都好好想想這個問題，多多探討GTS千層酥般的內在。
共勉之！

R

首先声明：我喜欢高质量的福利文，也同样喜欢有内涵有深度的剧情文。
R版的建议确实出自心声，因为论坛剧情党实在是少。
但两者的存在对现在论坛的发展都做出了相应的贡献，且扎下根来，缺一不可。
它们虽然建立在一个大地基上，但定位实际上是不同的，因为两种作者想得到的东西不同，受众（读者）需求的也不尽相同。
这样一来，大概只能说：各有所好，强求不得。
但也不是说就这样把派系什么的东西给划分明确了，在不违背本人意愿的情况下，不断地寻求改变是提升自己的路径——当然，这明显有些扯了。
总之，要看自己的心态是什么样，究竟是怎么看待这些问题的。

当然R君也知道咱是属于那种对福利漠不关心的纯剧情党类型……所以对于R君的发言我还是深有同感的。
首先我也要承认，我确实是追求高精尖的人，对我个人来说是宁缺毋滥。但尽管如此，我还是不得不说。尽管我不喜欢纯粹的福利（卖肉）文，但这也只是我个人的喜好而已。正是萝卜白菜各有所爱，你喜欢吃辣，我喜欢吃酸。大家互不相干……
可是，真的互不相干吗？这是个论坛，是个小群体。没错，剧情党也好，福利党也好，大家都是各有各的想法。所以即使我不喜欢纯粹的福利，但我也不能说我不承认其存在意义。毕竟不同人的追求不同，不同类型的文字也自有不同的受用者。喜欢物质的追求物质，喜欢精神的追求精神不是吗。
我不觉得互提意见是什么错误的行为。剧情党可以对福利党说“为什么不试试强化剧情呢？”，福利党也可以对剧情党说“加点福利会不会更好呢？”。这样算是嘲讽吗？算是纯心引战吗？算是低级吗？
文字都有相通之处，大家只是侧重点不一样罢了。我不喜欢那么做，但自然我也不能说那么做是错的，因为我没有那个资格。
我想R君只是希望提一提建议，把自己的体会分享一下，纯粹是希望对福利党的文章的提高有所帮助而已。当然我并不是说福利党的文章不好，就像有时候剧情党的文章缺乏福利，福利党的文章缺乏剧情，都是可以互相补充的不是吗？
且不说别人怎么看待，至少我没看出R君有一丝嘲讽的意思。从头到尾，只是说“如果……会不会更好呢？”，归根究底也只是希望各位文手们都能体会到自己所体会到的喜悦而已。既然如此，又何尝谈得上“大肆”两个字。
的确，每个人所获得的喜悦是不同的。也许人类就是追求舒适感和愉悦感的动物。
食色性也，人之常情。
福利党和剧情党各有特色，各有各的优势？但是怎能说“顶多”就是相互包容？文字的交流本没有这么死板的界限，何必上纲上线的批评别人呢？非要弄得壁垒分明？弄得老死不相往来？
阁下用词颇为激烈，“根本”、“很”、“大肆”，但我并没有太批评阁下的意思。
我只是想说，请不要随随便便的把别人的意思理解成恶意，也不要把别人不同的意见当做嘲讽。总不能别人踏入你的领域，哪怕送块月饼过来你都要死战不休？
希望您冷静的思考一下。对别人的提议喜欢就接受，不喜欢就是不喜欢，这样就好。
否则总是对别人报以如此激烈的态度的话，第一个跳出来讨战的又会变成谁呢？
我只希望论坛的秩序能得到维持，大家都能得到一个更好的讨论环境而已。
