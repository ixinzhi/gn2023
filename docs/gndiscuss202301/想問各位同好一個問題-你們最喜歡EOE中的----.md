# 7

喔喔，我感受到樓主的召喚了∼∼∼![](img/5b7702fe88717bc5cc29f98f6a1fd730.png)

算來至今也寫了不少小說，每當看見有人願意討論相關內容，都讓我很感動，覺得自己至少不是在瞎忙。![](img/cdc51f89bb7d985db5e2efdb53ac7fc6.png)

至於我自己最喜歡的角色嘛，以EOE系列的登場人物為限，應該還是妮菲兒吧！能力強悍但性格其實很脆弱，不情願地被迫背負復仇的使命，同情弱者但面對敵人則毫不留情，這樣的設定會讓我覺得很舒服（不知道大家聽得聽不懂？）。而克勞蒂雅會這麼受歡迎倒讓我有些意外，因為我並沒有很刻意去描寫她的背景，戲份上她也是點到為止，難道是蘿莉吸血鬼+槍形的設定在發酵嗎？嗯，這樣的組合聽起來確實挺迷人的………

再說DGU系列，正如樓主所說的，目前的DGU故事還無法連成一氣，在主要的時間軸上支離破碎地分散著，因此多數角色都僅僅活躍在一個單篇當中，這樣確實比較難讓人下定論。不過就我來說，「天國覺醒」是我花費比較大心力去完成的一篇作品，因此裡面的角色也比較讓我喜愛。其次就是「冥神家的女兒們」當中的冥神夜，她的個性被我設定得與妮菲兒有點相似，應該說是命運更坎苛的版本吧！

其實身為作者，我腦海中最理想型的完美巨大少女，至今還沒有描寫出來。畢竟幻想可以是完美的，但我的寫作能力卻是有限，總覺得自己無法將那種悽美的個性完整地表達出來，因此一直沒敢去碰。 <title>8</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
