# 1

各位论坛同好大家好，DGU作为我们论坛殿堂级的作品，出场了众多人物，但是还没有一次像样的人物评选的活动，因此在这里举行燃王投票选举活动。
投票规则为每人5票，获得票数最多者将成为本次比赛的燃王。

这里面的角色中，可能有你喜欢的男主，可能有你敬佩的角色，也有可能有你的情敌，如果有因为各种原因让你着迷的角色，就请投票让他成为燃王吧！
