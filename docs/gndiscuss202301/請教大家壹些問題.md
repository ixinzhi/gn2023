# 2

嘛……簡體字通不通我不知道。我自己是看得懂，也沒聽誰說過看不懂簡字，但我相信看到簡體字就跳出主題不看的人應該不少。理由無它，看得懂跟讀得順是兩回事。

另外你說「貼吧」？這裡應該叫論壇吧！我確信這是兩岸共通詞彙，貼吧應該是專指百度那個留言板吧？

還有，你點出的問題我大概被問過十幾次了。光是看別人的帖子卻不肯回覆，這種人在台灣叫「潛水員」，明明是論壇會員卻不肯參與討論；除了你說的害羞因素之外，要注意的是本論壇600個會員中，真正是華人的可能只有三分之二，甚至更少。我這裡開站以來沒有關閉註冊過，美國人日本人澳洲人，甚至哈薩克人我都見過(去後台查一下IP就知道了)，你怎能指望這些不屬於中文圈的傢伙參與討論呢？他們有很大部份是註冊當天逛過後就再也不來了。像這樣的帳號我已經處理過一次，砍了大概一百五十個人，我預計是每三個月執行一次這類的大掃除，下次應該在九月。

最後關於成員流失的問題，嗯，有句話我現在想說，但我不記得過去有沒有說過。可能我很久之前就有那樣的概念，但因為還很模糊的關係，所以沒能好好闡述出來，不過現在我能說了：

「離開的人是因為他們的愛不夠」

以上。我找不到更好的說法了。這個站並不需要那些路過探頭的鄉民來充人數，我從來不會看著會員人數上升而沾沾自喜。這又不是營利網站。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
