# 2

我的合成圖片作法，可簡化為以下步驟：

1.找素材：所謂的素材，不外乎背景、主角與配件。配件就是一些能讓圖面更生動的小道具，例如車輛或小人，視情況可省略。

2.編劇：決定人物在圖上的位置、進行的活動，或甚至是要講的對話。

3.匯聚素材：將人物去背、深度化背景、同時針對配件製作特效。

4.在角落處貼上作者的Mark，完畢。

....感覺好像有少什麼步驟？但又好像沒有。

# 3

1）構思
先想好自己要做的情節等，但通常我都是看到自己看上的好圖後才開始想製作方法。
2）找素材……
本人認為最棘手的是這個……![](img/1751a37041f23d672f2071488c42b774.png)
要找主要人物，配件（符合圖中情節的，如飛機和船隻），以及其他符合現實的元素（如腳印，煙霧，廢墟，火焰等）
3）正式製作
反倒認為這個不太棘手…只要之前找素材這項工作做好，就萬事OK。但俗話說萬事開頭難…
4)正式添加劇情
作完以後,再看看作品,再確定主題.練習作有時便不定主題了.
