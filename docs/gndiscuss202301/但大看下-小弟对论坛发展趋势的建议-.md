# 1

首先自我介绍一下,并在此感谢但大创建这个论坛,以下内容纯属个人建议,绝无半点针对性。
小弟是大陆的GTS爱好者,从很小就对女巨人情有独钟,不夸张的说这也该算是一种天赋吧。
小弟也曾在女神论坛混过很长一段时间,担任过女神论坛游戏区版主,所以也该算踏出GTS新手圈了。

以上都是些废话,下面就但大论坛接下来的发展趋势提点建议。
对于GTS有关的Article和pictures以及videos,在全世界同好的共同努力下,其各类的数量已经不计其数,而其中从大的方面分类,无非就是残酷系的和温柔系的GTS,然后在这两个大的主题分类下,再联想出不同的情节以及人物的安排,然而我们在欣赏和享受这些的同时,是否也意识到一个问题？我们对于这些情节发展的趋势是否已经产生了审美疲劳了,我想很多同好也有相同的感觉吧,随着数量的剧增, 使得我们在不同的情节中潜意识的找到了共通性,就好比方说当你现在看一篇新的GTS文章时,是否看到一半就已经知道后面的结果了,也许只是在于你所想到的结果和作者的表现方式不同,时间,地点,人物的不同,一个很简单的例子反应出我们GTS爱好者都存在的一个可大可小的问题。

我们每次看到的,都是女巨人们如何的去摧毁一座城市, 或者怎么杀小人,玩弄小人,小人是怎么为女巨人服务,说这么多也就是想提下自己的一点点建议,我们现在可以在保留旧的GTS观念的同时,能否对GTS有一个更高层面的认识,能否从另外的角度去欣赏和品味GTS,在全世界的GTS爱好者都还没有着手去做这件事情的时候,我们论坛能提前创新出新的GTS概念,这也包括小弟所想到的,纯欣赏的角度去感受GTS,纯审美性质的去欣赏女巨人的容貌,姿态,性格或者心理变化,也许这几点在之前很多GTS有关的内容中有所体现,但是在我们欣赏的时候更多的会把侧重点放在其情节的发展之中,往往忽略了其他一些值得欣赏的东西,所以说我们的思想概念完全可以达到一个新的层面,从新的角度去分析和欣赏以及思考。

以上这么多纯属小弟对于GTS的一些感慨,同时也是对但大论坛发展下去的一点点建议,还希望各位大大能够发表一下各自的看法,也许每个人都有不同的新的看法,不妨说出来,大家一起交流一下。

小弟在此再次感谢但大创建这个论坛,并再次申明以上内容纯属个人建议,绝无半点针对性。

补充：看了很多同好的建议,感觉你们有些误解我的意思了,我提的这个建议只是想说明我们现在会很容易对GTS产生审美疲劳,其目的是为了让大家能够多多提议,在保留旧观念的同时,看能不能讨论出一个新的GTS发展方向~

先说下我对LZ的意思的理解：传统的GTS文章看腻了，想看更偏向正常小说的GTS文章。
我不知道这理解是否正确，因为LZ使用很多抽象概念性的词汇我看不懂……所以要是我的理解错误的话请54……

要说更偏向正常小说的话，我只能想到4个例子。
一、但丁个人故事集。
由于每篇都很长的关系，但丁可以游刃有余地插入非GTS情节（实际上，GTS情节才是小部分……）。结果往往是呼声很高（因为产量高），回复不一定多。（GTS情节少了或不合口味，回复就少了。当然，这也跟但丁把文章放在网页上，有人直接去网页上看然后懒得再点帖子回复了有关）或者是直接拖过剧情部分，进入GTS部分。我不是老人，EOE在连载的时候怎么样不清楚，但很不幸的，作为少数直接发在论坛上，以但大对某游戏的爱为主，GTS为辅的其平日风格不大相符的温柔系作品姬神系列在连载是是比较惨淡的情况……（看我定语这么多，就知道不确定要素也很多吧囧）

二、剑兄的短篇
剑兄是以写正常的带GTS情节的小说为目标努力奋斗的。他的作品中，以情节、对话和环境背景描写占绝大篇幅，相应GTS只是极小部分，而且本身篇幅也不长（这是跟长篇连载的比，跟坑们比还是比较长的），但每篇的回复数都极高，而且也广受好评。但回复这么多的一部分原因，我相信是因为他设置了后记的回复可见。这理由虽然很可笑，但我觉得，对于一个对资源渴求的人来说，只要有一个回复可见，他会毫不犹豫地回了再说，而不是看下LZ的说明文字而打消兴趣。至于这会起到多大作用，我也不好下定论。但如果不是这个所导致的话，我会很高兴，因为这是我所愿意看到的景象。

三、18X兄的同人
只有一篇作品而且是新出的，我也没什么好说的，不过有发展成中长篇的趋势，18X兄要努力填坑啊~

四、GTS-C兄的红唇狩猎场
由于被擅自转载导致作者发飙于是坑了的佳作。但由于是老物，所以现在没什么人提起也是正常的，但我的印象中大家对这篇作品还是很赞的。我甚至怀疑拿出去的话完全可以以假乱真，被人当成正常小说对待。（喂喂，说说而已，表真的拿出去啊，GTS-C大大会把我凌迟，然后吊在避雷针上等台风来）

总得来说由于剑兄的关系（虽然但大的作品也符合，但恐怕大部分读者都是为了实用才去看的），这种趋势应该不是没有，但是恐怕是小众吧
