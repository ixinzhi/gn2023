# 1

說起來好久沒來過閒聊區了（所以這裡真的叫閒聊區麼），又或者是高中生活與初中生活相差太遠，我現在想起初中竟是：欸？竟然是幾個月前的事情麼？感覺是幾年了耶？
  扯遠了，先拉回來吧。
  我特地查了一趟自己的發帖記錄，發現最早的貼是標記著2010-12-11的，比起自己ID的註冊時間，估計是耗上了不少時間混在當年的新手村吧？新手村的倒閉還令我惋惜了一陣，因為不得不出來打怪了，嗯。。。。。。總之呢，當時就在現在想來極其奇怪的環境寫出了第一篇文章——淩晨4~5點寫，然後再睡一個小時再去上學，每次開機都要小心翼翼的不被發現，那個該死的電源打開超大聲的，但現在聽不到了還真的蠻懷念的——新屋終究是沒有舊屋住得舒適。
  總的來說，除去那麼該死的幾天，我發跡的該是上年，也就是2011年了，從一個無名小卒混到現在還真是不容易了，儘管在多數人看來就是不斷挖坑棄坑的過程（多數人：不，事實就是不斷挖坑棄坑吧？！）。撇去什麼小小的東西不談，這一年讓我覺得很快樂，什麼什麼的電腦前鏈接一大堆奇怪的人，也讓我獲得了很多，認識了很多人：但大18E劍爺無知R還有很多寫不出的懶得寫的不知道該怎麼寫的人，一切一切的，我想我沒把它拋在2011年。
  屌，我自己都覺得前面寫得有點噁心的，那麼有心情看到這裡的，輕鬆點，放鬆點，對，先把你旁邊的那杯水喝掉再繼續看吧。
  我想，2011年應該不只是我的活躍年，大概是想作為開始活躍的一年，學業繁忙什麼的不必多說，大概有很多人也是這樣，要真想珍稀什麼的人，是不會害怕什麼困境的，我堅信這點，才會奮鬥至今的，儘管什麼眼前的成績不理想，那總會挨過去的，至少我這麼相信，反正。。。。。。就是這樣相信著傻事一邊努力也無妨嘛？
  唔，該是時候攤牌了，我感覺今年——2012年應該會較忙所以產量，唔，較低？該這麼說嗎？

  原創區的《ZUZ》（我必須得把名字改掉換個好名字啊=。=
  暫定是這三項必須有的，其他的也不太清楚吧，反正以上三個必須會填的，信我吧，最近也蠻好心填坑的啦，新開那個也會填掉的，假期裏。

  無知：《四次元內褲》（邊考試邊突然想到的東西，雖然是坑但還是毫不猶豫列出來了
