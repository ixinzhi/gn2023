# 1

日本那边有一个词叫超性能娘，这个百科我现在在写 ，国内目前还没人普及过这个概念，大体上类似女超人 无敌的力量然后刀枪不入并且是“女性优越”不是sm女超人的一种爱好
跟gts很类似，除了体型上不同基本上一样（破坏啊温柔啊等等）
跟怪力娘也不太一样，我前几天发现本论坛有人宣传过超性能娘这个新事物，但是他们群的超性能娘不太喜欢绝对无敌强大的，所以我自己另起个新群，专门玩这种无穷强的超性能娘
有兴趣的欢迎来玩，有看法也想来交流
