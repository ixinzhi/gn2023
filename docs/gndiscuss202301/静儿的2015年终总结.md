# 1

这是某静入坑的第二年
也许对于这个圈子来说，这个圈龄并不是很大，甚至还可以算新人的水平
论坛里很多前辈都已经入坑了数年，甚至十数年了，能够在圈里坚持这么久的各位，我真的很佩服
一个爱好如果能维持这么多年，是很了不起的一件事
今天突然发这个帖子，除了感慨一下，还是想检讨自己
这些日子偶然和一些人聊了不少，有的是论坛里的老前辈，也有的是还没接触过论坛的人，然而很多人都说到了一件事，【在这个圈子里想做些什么】
坦白讲，我以前并未考虑过这个话题，这在我看来是比较少有的
我是一个乐为人师的人（好听的说法，不好听的说法就是喜欢装B），每当我接触一个圈子之后，我总想去在这个圈子里做些什么
但是在GTS这个圈子里，我似乎没有这种冲动，也许写文算是一种，然而最近的产量惨不忍睹
回头看一下，入坑一年半多，我最满意的一篇作品，居然是刚入坑写的那篇数千字的短文
从那以后写的文都差强人意，往往是一些当时自己看着爽的东西，莫名其妙就发上来了（现在看看好想删了
感觉自己写文越来越拘束了，很多时候自己幻想的一些东西只能停留在幻想的阶段，感觉不可能写出来
也越来越在意别人对我的看法，如果文中的表现过于露骨，总是思前想后，然后要么砍掉要么干脆不发了
国外很流行New Year Resolution，国内这样的概念似乎没那么强，那正值2015年的最后一天，我也来列下明年的New Year Resolution好了
1-参加展望杯并上交让自己满意的作品
2-全年至少发五篇能让自己满意的单章作品
3-展开至少一个令自己满意的长篇作品
4-长篇作品维持定期更新，至少两周一更
嗯，大概是这样吧~
（感觉过年的气氛一年比一年差了，今天几乎什么都没做，还要去学校练球，也许是我的问题吧……
最后，祝GN的各位成员2016年新年快乐，希望在将来的这一年中，大家都能实现自己的梦想哦~ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 5

我一直不明白在这个越发开放的网络环境下，为什么GN的发展一直很平缓，各种方面都没有太大进展
论坛文章和图片的产量最近反而下降了
这很奇怪，非常奇怪，就像是无缘无故少了许多人一样
静姐这种，也在其他人见过，就是环境每况愈下的情况下更加失去动力
这样是不行的，必须要有更多新人加入我们的队伍中才行
我期待着新人，尤其是中小学生，就像我七年前第一次踏入GTS圈那样涌来，那么以后就会有更多新人成长起来。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 6

从某种意义上讲，我感觉也不能说是文章和图片什么的质量下降了，而是我们的口味越来越高了。之前某投票中的回复就能看出这一点。

我入圈比5楼更早些，当时也是中学生，由于处于萌芽阶段，不要说接触到正统的圈内作品，就是遇到擦边也会收藏起来视若珍宝。我最先接触的是百度的gts贴吧，贴吧的文笔都很幼稚，而且基本是快餐文，毫无情节推动可言，但是当时看的非常兴奋。后来接触到女神和但大的一些文章，作品也是少的可怜，更不要说如今文章，图片，视频，游戏百花齐放的状况。

说实话我并不想要中小学生加入到这个圈子，首先是圈内作品有很多色情元素，不适合未成年人观看（这绝不是我道貌岸然，这与早恋不一样），其次，中小学生的阅历和能力都比较低，很难为圈内做什么贡献，很容易就沦为伸手党。

圈子发展已经很迅速了，只是伸手党太多，包括我，在刚注册GN的时候也只是想来索取一些资源，后来账号被永久删除，重新申请了账号，到了现在才会尝试为圈内做些贡献，伸手党也会逐渐成长起来的。

最后我保证一定将我挖下的《安娜的复仇》坑完，之后可能会翻译其前作《偷窥者》，也有可能自己尝试做一个圈内游戏。总之，祝愿我们的圈子蓬勃发展吧。
