# 1

心心念念resize me已经很久了，以至于很长时间看到别人的录屏甚至都不舍得点开。去年疫情没怎么出门攒了一点钱，索性就开春之后借着“要学习unity的名义”买了一台台式机和一台最基本的occlus，然后当然是第一时间就下载了resize me，配置了链接模式，打开一个bonus scene就开始体验这到底是什么感觉。
我能想象刚才我自己坐在地上肯定是丑态百出，在贤者模式之后，我感觉我必须写下点什么。因为实在是，太棒了，太棒了，太让人感动了。
gts是我有了十多年的fetish，一直以来一大遗憾（抑或是幸运？不然我可能早就作死了）是这种爱好是不可能在现实中实现的。然而虚拟现实维他它打开了一扇前所未有的大门：这是可能的，你可以感受到一个可爱高中生一脸微笑着把你摁在脚底反复摩擦，你可以切身实地身临其境的去用五官感受这种幻想，而且在资本的加持下这项技术会越来越廉价，越来越精致，直到成为下一个主流。
所以我想说什么呢，快去买VR啊！！趁着有机会趁着年轻，早点去进入这个圈子，我也要学动画，做游戏，然后让全世界的gts肥宅被我的游戏掏空！！！ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
