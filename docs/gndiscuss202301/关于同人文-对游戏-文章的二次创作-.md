# 1

这两天在家将电脑上面的很多游戏又一次玩了一遍，还有很多论坛上大佬新分享的游戏（比如说大佬汉化的缩小日记）
都玩过一遍之后觉得还有很多的意犹未尽，于是就想要写一些同人的文章
将里面的一部分情节扩展成文章
（才不是因为我懒得想剧情）
不过有些担心这样写会不会接受，会不会觉得已经有游戏了所以进行二次的文章创作不太好？

# 2

二次同人的难点在于如何还原角色，架构自己的剧情的同时又不会影响原有的剧情，如果做不好的话就可能被喜欢该作品的人批评。
不过话说回来，二次创作也是最容易让自我满足的创作方式了，请多多加油吧。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 3

二次创造主要是切入点，既不要影响原文的内容和剧情走向，又要有新的故事。
对于没有完结的内容还是不要随意的进行二次创作，毕竟你的故事是会影响到原作者的思路，至少要征得原作者同意。

# 5

有人願意寫文，當然支持呀！

但是，不得不說二次創作風險很高，要掌握原作的細節，還要拿捏劇情與設定不會太偏離原作。就算一切都很完美，還是有可能遇到沒讀過原作的讀者，導致大量的梗與故事難以解釋清楚。

不知道你要想什麼，像東方Project我覺得就是個超難的二創題材，但縮學的話似乎就容易許多。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
