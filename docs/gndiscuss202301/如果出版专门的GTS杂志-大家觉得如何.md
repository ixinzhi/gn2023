# 8

說說我的想法吧。

綜觀世界各國的GTS社群，想要聚集一批同好參與團隊合作、搞個什麼影片或雜誌出來，這個前提一定是「成果能拿來賣錢」。

Giantess Zone 集團如此，I-Raf-you! 如此，過去一些合辦的雜誌（如Crush Magazine）也是如此。

以GTS這樣一個幻想為主題的創作，正如樓上有誰提到的，為了求得比較好的效果，我們需要招攬夠技術力的人才。要玩合成影像，我們要找懂得操作影像軟體的人；要製作遊戲，我們要找編劇與程式設計師；要出版雜誌，我們也得找人畫插圖、寫文章，還要排版、送印，跟外人打交道。

上述都不是輕鬆的事情，不是可以簡單一句話「我們有愛」然後就衝過去。在這個合作案的終點假如看不見利潤，那麼這個團體遲早都會四分五裂的。因為你投入在這個創作上面的時間，會壓縮到你真實世界的生活與事業，我相信大家應該都同意要把現實生活面先給顧好。

假如明白了這些、還想要搞下去，那欠缺的就是一個完整的企劃。我們需要一位有領導經驗的人來招募技術人員，並告訴他們各自負責怎樣的工作、說明工作中可能遇到的困難，最後要闡明事後的收益如何分配，而且他的說明要能夠服眾。簡單來說，我們需要一位懂得專案管理的人才。

可能扯得有點遠，但事實上，經由不久前的遊戲製作嘗試，我開始明白光有愛是不夠的。

什麼東西可以光靠一個人的愛就完成呢？很簡單，就是那些能夠靠一個人就獨立完成的產品。小說從寫作、排版到送印都可以靠作者自己去完成，畫漫畫也是一樣，所以同好自行創作的東西數量最多的就是這兩者。

當然，我也不是沒看過光靠一人之力就搞起來的合成影像，但那品質還達不到能商業化的水準。要說能夠看的，Giantess Zone 集團的影像合成技術在GTS圈內應該是第一名，但他們也磨練了很多年的時間，靠著科技的進步才終於有今天的水準。但他們的付出是有回報的，我相信他們應該賺了不少。

然後我又回想了一下各位在樓上的討論。其實沒錯，因為內地環境特殊，所以大家都還停留在鼓勵彼此「拿出勇氣」的階段。這樣很好，因為凡事都得起個頭，先把這個冒險精神培養起來比較重要。我不是要大家走上街頭到處跟人嚷嚷說我有GTS癖好，這可以是你一生的秘密，但若你想在這方面搞出什麼名堂來，那麼至少，你得先在圈內廣結同好吧？「信任」是一個合作案相當重要的因素，假如大家不了解彼此，那基本上說什麼也沒用了。
