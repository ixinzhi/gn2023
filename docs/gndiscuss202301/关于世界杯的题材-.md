# 1

我是一个球迷，虽然说算不上是专业！呵呵，刚刚高考完，也迎来了世界杯了，可以说这一段时间懒惰了好多啊！紧绷的神经松弛了下来，自己也开始堕落了。
所说的关于世界杯是我的一个设想，写GTS文的时候为什么不能够用世界杯这个题材呢？全世界的GTS在一起组织一场足球比赛，地点就是全世界，或者是一个星球的大陆之上，这样的话可以将男主当作是球或者是球鞋，在论坛当中不是有好多变物的文章么？虽然说我对这一类的文章不太感冒，而且其中一个作者周斌和我还是好朋友，从文章当中就知道了，嘻嘻~作者之间也是有文笔之上的交流的啊。
好长时间灭有写文章了，写文章只不过是一个爱好啦！对于我来说是很难忘的一段经历，我的高考成绩很不好，但是我并不后悔，我知道即便是后悔了，结果也和现在没有什么区别，现在这种心态是我宝贵的财富，我相信我以后的路会很好的。
这个文章设定是在QQ群当中侃世界杯的时候无意中想到的，希望有文笔好的人能够写出来，这也是我们所有人的财富。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
