# 3

这都是什么，本来就不合理的东西为啥非要用现实的合理性来解释嘛，
我的考据一般都是常识性的，比方说要写一个妹子比什么星系还大，我会先查一下那个星系有多大，写谁比细菌/发丝小，就先百度一下细菌/发丝大小。再就是如果是现实相关背景，会查一下政治经济文化，类似于这种的考据 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 4

对于写虚拟世界观我，大概会考虑大陆 气候 然后是种族类型 文化观 之类的之后构建道德体系 最后完成世界的架构然后再写文章剧情吧，不过一般来说这种过于复杂的也可以考虑用现实世界部分魔改
然后就是写人类的时候要考虑下心理学之类的吧，形象不能过于突变 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 5

我把虚构作品中的现实分为物理现实和逻辑现实。物理现实就是楼主所说的，体型巨大后受到引力的变化之类。但是写文毕竟是写文，又不是写科研报告，更何况这个圈子里大多数看文不就是图个爽么，光是巨大娘这种存在就违背了一堆科学原理了。还有许多文直接加入了魔法、黑科技等要素，所以根本没有考虑物理现实的必要。除非是写硬科幻小说，那是真的大佬。![](img/c21ba2c6b3682f8392e3b81ea4b59fd7.png)
但是逻辑不现实的话（比如“为了推动剧情朝作者想要的方向发展，而不得不让主角强行降智，明明有更好的做法却非要走歪路”）除非这篇文章已经摆明了可以不用带脑子看，不然对于稍微用心的读者来说都是折磨。要让主角逃出监狱，可以设计监狱里有内应的情节，可以设计监狱发生暴乱，主角趁乱溜走的情节，甚至说主角觉醒神力直接杀出去都行，但是直接说“主角趁看守们睡觉时偷偷从联邦级的监狱里跑出去了”，这就很不合逻辑，影响观感![](img/68c369ef297b289c8dfa168996e9b39a.png) <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 6

跳舞有一句话，我不是专业人士，我是作者，我只要他看起来是那么一回事就行了。 <title>7</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 7

其实蛮赞同7L的，我曾经很在意文中的物理法则和人物装束等细节，后来发现根本无法兼顾。只要写了细节就很难避免为了写福利或自己想要的发展而导致人物强行降智。倒不如留白让读者去琢磨这里的物理法则，在恰当的时机补点设定来的合适。 <title>8</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
