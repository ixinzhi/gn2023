# 1

　　最懶惰的小說寫作的人又出現啦！之前實在太多前科了，首先就先期許這次創作小說可不要像以前一樣寫了一半之後又完全消失不見啦！

　　這次的新作品我大概寫了一半了，我習慣寫完之後還會校正一次（大概每個寫小說的人都會這樣的吧！），所以應該還要幾天才會公佈吧！這次小說我是設定女主角為１５０公分嬌小型的女生，有時候想想這樣嬌小的女生平常看起來很矮小，但如果變巨大之後會怎樣了，我常常這樣子幻想著。

　　我這次小說一開始說一位可愛的嬌小型女生，雖然可愛，但是因為嬌小，所以會被人欺侮取笑矮子，所以往後獲得巨大化以及把人類縮小等等的魔法的劇情，可以說裡頭的巨大少女會有魔法的能力，可以說是類似魔法少女。

　　而當然魔法並不是只有巨大化或把人類縮小，可能有什麼類型的破壞魔法等等，當然如果有同好想要提供一些劇情或者什麼魔法也都可以聊聊唷！大家互相交流！

　　當然我寫ＧＴＳ小說的類型大致是巨大少女如何玩弄人類之類，當然玩弄不只是玩弄人類而已，那麼巨大的少女玩弄人類，當然人類是被巨大少女玩弄死的，可以說是巨大少女如何把人類給消滅掉，而腳踩的部份可能佔的比例較高，通常是有穿鞋子的，而這次小說所寫的不只是這些描述，會包含用手來玩弄，當然用屁股坐著壓死的，用胸部往地面上壓死，還是其他身體部位大致都會採納，而之前我所排斥的性這塊，應該是會涵蓋一些，但不會寫成像性愛小說那樣子，這樣子說應該知道我的意思吧！

　　當然吃掉人類這部分我這次會包含在內，吐露一些劇情架構，吃掉人類的用一是我在小說裡頭打算設定有魔法的女主角她們平常保養自己漂亮的肌膚身體等等之類的，簡單來說人類即將是女主角們的食物點心，吃掉人類可以當作食物止餓，吃掉人類可以當作點心解饞，重點人類有供給女主角們極高的營養成分，所以可以讓女主角們的肌膚變的水噹噹，更白皙柔韌，不輸給嬰兒的肌膚，而外表可以看起來更可愛漂亮，維持年輕的美貌............挖哩哩！說到這哩，感覺越來越邪惡啦！

　　當然不只是這樣子的設定，再吐露一下，人類被吃掉或踩死等，也就是死亡，當然靈魂就會脫離身體，而小說的設定人類的靈魂都會被女主角們吸收轉換成魔力，而魔力是可以累積的，殺死人類越多，魔力累積更多，魔法使出就更多，所以沒有什麼輪迴的情形。

　　那麼殺了那麼多人類，地球上的人類不就是５∼６０億嗎？那麼巨大少女如果巨大化一千倍或者一萬倍，很快不就都殺光了嗎？挖哩哩！拜託！這是小說，我不會設定女主角們所處在的地方是比地球還來得大以及更多城市，人類不會設定１０００億或一兆的人數，總之人類是殺不盡，還可以設定巨大化變成星球級大小，一手就捏碎星球，而且週遭的星球語言是共通的一樣都是人類居住的星球，放大到宇宙，那麼還有這個問題嗎？

　　最後我還想到一個設定，就是巨大少女踩死人類後，她們的鞋底不就沾黏到人類骯髒的屍醬，所以以往ＧＴＳ小說就會寫到巨大少女的腳底或鞋底沾黏相當多骯髒的屍醬，這樣子雖然可以突顯巨大少女踩死很多人類，但是想想女孩子的鞋底沾黏這些骯髒的東西，如果現實裡看到一個女孩子的鞋底是這樣子，那麼好嗎？所以我就設定女主角們變成魔法少女後踩死人類或身體沾黏到人類的屍醬，血肉，會被女主角們的身體或穿著的服飾給吸收，這些人類部分的血肉會轉化成微弱的魔力變成女主角們魔力的一部份，而女主角們的身體服飾鞋子永遠是光鮮亮麗的，最多鞋底會沾些地上的灰塵，這點是說如果鞋底也是新的，那麼感覺也不太好吧！

　　以上聽我描述，至少這次我要執筆寫新的小說可是花了很長的時間思考呢，雖然我很久沒寫了，但是平常還是會想想ＧＴＳ的劇情，還有上面我一直提到女主角?們?，這樣子凸顯，應該知道我的意思吧！簡單來說，就是魔法少女不只一個，以女主角為中心，我大概會設定出約４∼５個女主角，會有幾個是女主角的朋友，女主角設定１６歲一年級的高中生，還會有一個２０歲的少女，可以的話還打算設定出女主角的妹妹，約１０歲的小女生，但重點來說我選角大致都是可愛型的，所以平常的身高最高不超過１６０公分，最高的一個身高才１５７公分，所以這次都會以可愛嬌小型的女生為女主角，而黑澀會美眉裡頭的糖果會出現是女主角的同學朋友唷！

　　所以以上面所描述，這樣子的設定我想可以寫很多篇連續性的長篇小說，而且短篇小說也可以連結，至少看起來應該部會像以前中斷那麼多寫不太下去，而這次第一篇我會以三部份來貼出來，因為看起來第一篇可以寫到１萬字，不要一次寫到完再貼出來也比較好，因為這樣子不會因為為了要貼出來而寫那麼多，造成後來又懶惰寫不下去了。

　　說的那麼多，也花了很多的時間啦！總之如果各位同好有相關的建議及劇情的建議都可以提出來唷！如果不錯我會盡量採納的！但是女主角們的選角大致是這樣子了。
