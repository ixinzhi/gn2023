# 7

看了片子後就作一個小小的影評吧..

首先是N比例系影片的問題
那些車子跟本踏不扁！
而建築物是像踩紙一樣是意料之内吧
有一點不明白的是他們為甚麼要安排踏壞有摩打的火車車箱
調包不就行了嗎？明知那不可能被踏扁的..
而背景方面不像Attack by 800ft woman 1和2 那樣合成了一些圖片
這部片只有藍色的錄影廠布做背景
整體感覺上這部比上兩集差了(也頗多的説)..
有一幕是她把車子放進口的
但很可惜沒有像Return of giantess 4一樣咀嚼車子
至於那個演員，真的可説是我看過GEN的片子入面最美的那個..
而她好像很代入劇情一樣，表情上表現得冷皓之餘又不會不自然
(按：事實常她真人並不是那麼樣的，詳情可看G.T.S. 第一部
不過我聽不懂日文，不知她在訪問中説了甚麼)
最出眾的是動作，看了她4部片了(總共有5部)
行為上真的很有女巨人風範，慢動作的穩定程道實在讓我嘆為觀止
真的希望她可以繼續拍下去

今次的影片要評分的話.. 5/10吧 <title>8</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
