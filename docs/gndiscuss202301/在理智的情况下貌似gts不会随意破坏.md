# 10

樓主的這個題目，真要討論起來說大不大說小不小

首先，「理智」的定義是以"普通人類"為標準還是以"GTS"這個種族為標準？

如果是以GTS種族來說，基本上就是隨意設定了

但若是以普通人類為標準，又要牽涉到其他的設定
1.是先天還是後天的GTS
2.本身的"普通人個性"為何(每個人個性都不一樣不用多解釋吧？)
3.如果是後天形成，形成因素(藥品、儀器、電波)有否改變(扭曲)當事人精神狀態？
4.體積大小所形成的與社會互動(10倍和100倍生活對應方式完全不同)
5.GTS是極少(單一)現象還是在社會上已有一定比例？
6.GTS現象在社會上是從未發生還是已行之有年？
7.當事者的心理承受度或心理準備狀態
