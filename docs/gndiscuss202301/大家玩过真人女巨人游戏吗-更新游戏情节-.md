# 1

大家用比例很小的城市模型 沙盘，小人玩具和女孩玩过巨大女的游戏吗，女孩扮演威力无比的巨人统治世界，毁坏城市，踩踏车辆，捏起捉弄小人，最后吃光城市里的小人，应该很刺激的，我小学时候和亲戚家的女孩玩过，当时很兴奋的，不知道现在有没有玩这种游戏的，可以和女朋友玩吧，现在很多女巨人视频都是用比例很小的模型，如果真能临场体验就更好了。附件是一些视频的贴图，如果能玩就好了，呵呵。
我回忆了一下当时玩之前，也试探了一下她愿意不愿意玩女巨人，她愿意了，于是在桌子上摆放了几个当时塑料制的建筑物和一些小兵人（估计有1cm左右），一些配套的战车。
桌子旁边有一个差不多高的床，她就在床上跪着。当时她穿着小裙子，我从桌子上的小人的视点看她，能偷看到内裤。我说了一下场景内容，就是女巨人在城市的旁边准备攻击城市，她也准备好玩了，当时不知道这是gts,估计是我这辈子最亲密接触GTS了。我装作小人的口吻，下面是大概内容：
小人：太巨大了，你是巨人吗？
女巨人：对，我是世界上最大的人！
小人：你是女的哟。
女巨人：女的也可以是巨人呀。
小人：太可怕了，你这么巨大，你多重啊。
女巨人：一亿吨！
