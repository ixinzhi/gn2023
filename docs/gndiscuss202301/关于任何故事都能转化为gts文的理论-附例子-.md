# 1

世间大多数故事都是相通的，gts说起来是小众性癖，但实际上和正常世界只是差了一个大小关系罢了。

我以前混迹贴吧时，曾经用小号把一些小说改编成gts故事，但实际上改编一个故事耗时费力，大伙也就看个新鲜感就没了，远不如做一下引导，开拓大众的yy能力，让他们把自己喜欢的故事yy成gts故事。

基本原理：
1.有强弱区分的故事，强者全改成妹子，弱者全是小人，攻击方式变成踩压拍吞等。所以玄幻打斗类的故事相当好改。
2\. 普通的交互动作，强化妹子动作影响，弱化男性动作影响，妹子轻拍男性，好疼好疼，男性跳起来全身力气击打妹子，勉强察觉。用于修改无打斗的日常番。

修改的四种不同程度：

1.改设定：对于一些与政治经济文化类高度相关的故事，可以改成gts世界观下的政治经济文化，或者修改一些有趣的设定

2.高魔改：修改度最高的一种，高度魔改，和原文只有少量关联，一般设定和剧情全都变了。有些像谭里大佬的一些故事了。

3.原文改：剧情和原文几乎一模一样，只是把人物互动变成了gts互动，说白了就是大段照抄，少量修改。什么？你说这不是抄袭吗？我们gts二创的的事，能叫抄吗，这叫yy。  一般来说这种改法只改了互动而没增加细节会是比较无聊的(像我写的某几个故事一样无聊)，全依靠原著优秀剧情。

4.无意识：任何有人物动作的文章，都可以变成无意识文，也就是，妹子走路必踩到小人城市，妹子坐下凳子上必有小人城市，妹子喝水杯子里必有无数船只……好吧，我交代了，我们小人就是无处不在的细菌

举一个番剧和一个国外名著的例子：
《re：0》
改设定：
著名的“死亡回归”：主角死亡后会回档，只有主角自己记得死前发生的事。这回我们不仅回档，还让486的身体随着每次死亡变小一些，从齐胸、齐腰、齐臀逐渐缩小到膝高、脚踝、脚趾大小等……为了合理性可以添加“同时让女主们认为486本来就是这么高”的设定，当然，但随着剧情的推进主角太小了就需要编者自己来创造解决方案了或者修改危机的难度。这样一篇自创度高的伪同人就造出来了。
原文改：
所有白给路人变成小人，486穿过来就是小人，反正486也没啥战斗力，不会影响剧情，基本照抄。这就是没啥深度的抄袭性改编。
《鲁滨逊漂流记》
高魔改：
鲁滨逊（妹子）漂流到无人荒岛无法离开，这个岛上有很多小人，于是她便征服这些小人成了统治他们的女神。这样的魔改，真是只要把名字换一下的话基本上一点关系都没有了。
无意识：
老规矩，妹子走路必踩到小人城市，妹子坐下上地上必有小人城市，妹子喝水水里必有无数船只……就有点像这篇神作了：[https://giantessnight.com/gnforu ... iewthread&tid=21965](https://giantessnight.com/gnforum2012/forum.php?mod=viewthread&tid=21965)，妹子普通动作对小人来说是毁天灭地实在是棒极了。

其实我也这么改过，效果还是不错的，省去了很多工作，也产生了不少好的脑洞。但真要将原作改成gts文，要写出合理流畅的剧情仍然要花费大量的精力，原作通常又非常长，一路改下来一般作者没有这么多时间。我的看法是：

1.这种方法作为脑洞的发生器非常好用，有了脑洞再设计剧情会简单一点，但总的来说跟自己写差别不太大。
2.可以对原作进行裁剪，写出自己觉得想写的东西即可，过渡剧情用前情提要之类的跳过就行
3.直接截取片段改，前因后果全部忽略，当成短篇写，不失为一法

# 5

和楼主的部分想法相同。
以前写文时总在思考一个问题，就是感觉自己的文差了些什么。一开始，我猜测是自己描写的还不够细致，或是玩的还不够开放。所以有一段时间，我仔细研究相关的玩法，结果发现，除了玩的越来越变态，对于文章代入感的提升却微乎其微。
后来我才渐渐发现，自己喜欢的性癖有很大的局限性，能描写的部分总得来说只有那么几点，再开发下去，也不过是无数个套不同皮的人不停再做同样的事，并且做的越来越过分而已。
因此，我觉得在这个阶段，除了开辟新性癖的新鲜感，想要文章的代入感再次提升，只有对剧情设定进行更深层次思考和打磨，才能写出更有代入感的文章。
这可能也是普通故事改成gts故事很好看的原因，毕竟普通的小说，无论人物还是世界观，都有大量完善的设定。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
