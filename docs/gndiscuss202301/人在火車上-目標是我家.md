# 1

我終於脫離了當兵的生活
在最後一刻，大家在餐廳吃飯，在這邊的人和我一樣，都是時間到就退伍的
其他選擇分發或加入特戰以及去國外訓練的在其他大廳(不知道要做什麼)
而我們這一連的長官和弟兄輪流要去台上說自己在這期間的感想
有一些弟兄勇敢的說出某連長官的壞話，那位長官當場目瞪口呆，其他則是歡聲連連
我們還一起唱歌跳舞、在營地打打鬧鬧

直到長官出來，叫我們收拾行李離開營地

下午從班長那取回被沒收的手機後立刻飛奔到學長家拿筆電
(放假期間不是回家就是去學長家玩，為了方便，還把筆電寄放到學長家)

目前我人在火車上，剛剛才睡起來(蠻累的)
再過一小時就能回到家了

因為是對號列車，旁邊的位子沒人，我才上線來這裡

真的是太久沒上線了，從註冊帳號到現在，根本沒機會到這裡= =
畢竟擁有著與自己相同嗜好並且能夠交流的地方，我只知道這裡而已......
