# 1

都是和女孩子高跟鞋有關啦，我也不知道爲什麽，只要有相關的場景就容易引起GTS幻想。先說兩個。
   大學一年級時，一次課上，老師叫一個坐在我右前方的女生去前邊回答問題。我當時正在低頭看書，就聽得地板被震的“咚咚”，聲音很大，好像地震一樣。抬頭看去，是那個女生在朝著講臺上走。整個女生也是穿了一雙黑色的高跟鞋，而且下身是超短裙。整個美女每走一步，對地板都是强烈的震動。因爲聲音是在是太大了，我覺得好像地震一樣，甚至內心有點害怕。那個美女好像是故意一樣，每走一步都很用力。我認爲她就是故意使高跟鞋接觸地面發出巨大聲音的。
   還有一次，在一個樓里，樓上下來一個美女，穿高跟鞋下樓梯。她下樓的速度特別快，幾乎是跑的。而且每踩一步。。。。那種感覺就別說了，就像女巨人襲擊城市一樣。我當時覺得那個樓梯都快要塌了。因爲整個美女跺地的力量是在是太大了。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

在我的想法裡頭要寫作品時我會先考慮裡頭有什麼破壞劇情,因為這方面是要評估巨大少女的身高,其實也可以寫成身高大小自由控制,也就是巨大少女可以自己控制巨大化的身高,這些都是想法而已,像GEN先生賣的片子裡頭巨大少女大約都是40公尺的身高,這樣子可以多凌虐小人,有幾部就是巨大少女用高跟鞋的鞋跟刺死小人,如果是100倍身高以上的巨大少女,鞋跟大概寬度有1公尺以上,人類被鞋跟一踩就扁了一半以上,如何談刺穿呢?
其實很多GTS作品都大同小異吧!只是女主角不同,穿著不同,背景不同,通常都是喜歡看巨大少女怎樣破壞城市玩弄以凌虐小人,其他的就加上自己的想法吧!
