# 2

接着上面的回复讲的详细点吧，网站页面最下方有个支持的付款方式，有支付宝的，点进去后会给一个魔法集市的链接，在那个地方买点数，会给你一串激活码，回到fantia左侧个人资料有显示剩余点数的地方，点击充值后最下面的付款方式有激活码，把之前购买后获得的码输进去就行了。赞助的时候也是选最下面的付款方式用fantia点数付款。
一般都是一个月500日元约合35元，要注意的是这个和以前流量一样按月度结算，而且看不到之前的东西，赞助以后要及时备份看的东西，如果更换赞助方案的话之前的都不算数了的。
