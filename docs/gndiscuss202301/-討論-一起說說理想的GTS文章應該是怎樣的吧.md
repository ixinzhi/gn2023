# 3

呼……這兩天忙，還加上胃痛，一直沒辦法好好上來觀看文章，若有疏忽還真得先道歉。

理想的GTS文章？嗯，能讓人看得高興，就是好文章了。
娛樂用的東西可以沒有說服力，可以沒有前因後果，反正效果達到就好。
絕大多數的單張圖片都沒有交代前因後果，可大家還是看得很爽不是？
文章的道理也是一樣的。

但若作者自己不想停留在這個層次、而想往上提昇的話，就得下苦工了。
畢竟要編排出一個能讓人接受的全套劇本，並不是那樣容易的。
但只要習慣這種思考方式，靈感就會源源不絕的來，自然也能寫出讓人叫好的文章了。

--
還有，沒有前因後果與沒有文筆是兩回事。
在我看來，很多短文之所以閱讀起來沒有共鳴，與其說是劇情交代不清，不如說是敘事能力著實欠缺的緣故。
文章是作者在對讀者的說話。說話口齒不清的人，別人又怎能了解他的言語？換作任何人都是一樣的。 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 4

想了很久，还是理不太清- -
我试着叙述一下。
有两个问题需要弄清楚：
1.什么是GTS。GTS是否需要GTS进行特定的某个动作来使读者产生快感，才能称之为GTS？比如说，我可以从头到尾描述一个女巨人的活动，但不让她做出GTS范畴的活动，也就是像正常人一样活动，只不过做这些行为的人是个GTS，那么这是不是GTS？又比如说，我对吞食一点兴趣都没有（实际上也不是完全没兴趣啦……举例），那么只含有吞食要素的文章对我来说是不是就不是GTS文章？
2.GTS文章是以“文章”为主，还是“GTS”为主？如果GTS的部分占篇幅很小，甚至完全没有实用性，是否就不是好的GTS文章？

无论如何，我是希望GTS文章能向轻小说的方向前进，角色是鲜活的而不是满足读者的杀戮机器，比如说，开了一个好头的，EOE。 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
