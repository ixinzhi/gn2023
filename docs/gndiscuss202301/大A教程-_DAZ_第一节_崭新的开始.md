# 1

 相信大家都對Poser很了解吧，我的任務3d制作旅程也是從poser開始的，不過，有都一點我要向大家說明，poser是制作3d的二流軟件，為什麽這麽說呢？
 第一，poser的預覽速度超級慢，如果放入一個復雜一些的模型或者是高精度的貼圖，調整起來的速度實在令人崩潰（我的電腦是雙核，顯卡9650gt高端顯卡，都受不了那種速度。。。。），如果加入大的場景，再添加幾個人物，我保證你機器崩潰。
第二，poser的光源設置不好，設置方式怪怪的，與主流的軟件相差太大。
第三，界面很不專業，雖然poser的界面很漂亮，可是很不好用，沒有明確的坐標軸,很孩子氣。
第四，預覽不準確，不解釋了。。。
第五，軟件太大，占用資源太多，渲染超級慢。

好了，下面說說Poser的優點。

第一，poser渲染精度高，對模型庫的支持很到位。
第二，poser的渲染質感強烈，poser的材質技術源於maya，無可挑剔。
第三，poser的動力學比較完善，想要做人物動畫，非他莫屬。

       其實，一個好的作品不是一個3d軟件可以做出來的，還要有一定的圖像處理功底，也就是ps技術，例如的我第一步作品，小盈和晴晴做出的體內場景就是將二者結合制作出來的。

       想要做好的作品，必要的色彩處理技術和圖片疊加技術是不可缺少的，如果需要制作濕潤感，還需要學習更多，所以，如果有心的同好，希望不要半途而廢，堅持學習到底。
 **下面進入正題**
       我下面給大家介紹一款新的軟件，名字是：DAZ Studio
去哪裏下呢，答案是：電驢。軟件很小，不到100M。但是性能完勝poser6.0以前的版本，因為poser7.0和8.0的渲染比DAZ強出很多很多。
我推薦下3.0版本，這個版本是最新的，我用了很久了，不會出現問題。
 首先還是說說DAZ的優勢。
第一，速度快，無論是建模，渲染，還是就算導入大的場景，依然絲毫不卡。
第二，光源設置很方便，和諸多著名軟件相同。
第三，渲染引擎好，是3dlight引擎，渲染出來的人物CG感強烈。
第四，人物調整方便，有明確的方向軸，調整時不卡。
 再說說劣勢
第一，對動力學支持一般，不如poser。
第二，在高畫質的條件下渲染效果不如poser,皮膚質感不強，但是可以通過後期ps處理完成。
 這段考試比較忙，今天就先介紹一下界面吧。

# 10

其实poser的渲染质量是很优秀的 不知道你的poser版本 我用的是poser8.0 渲染效果很赞
至于你说的“我想实现那种小人坐在裙子上的质感还有丝袜的感觉”
要有两个前提
第一，你的裙子模型要有完整的动力学设置机构，因此，下载的裙子模型要精挑细选
第二，要有好的贴图，最好是DS的，另外，材质质感表现poser是完胜DAZ的，不信你把poser的渲染精度跳到最高试试，呵呵。DAZ的话你就要自己多打打光，再增加分辨率，好的质感还是可以实现的。 <title>11</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
