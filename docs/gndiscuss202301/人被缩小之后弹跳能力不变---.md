# 1

是这样的，最近突然发现谷歌和高盛在面试题库里都有这样一道题：假如你缩小到3cm或者硬币大小，质量变小密度不变，现在把你放到搅拌机里，搅拌机一分钟内就会开启，你应该怎么办？

有人回答找到搅拌机刀片的空隙躲在里面，有的人回答打电话喊朋友把自己弄出来（这个答案有点意思），有的人回答直接抵住刀片（因为电机开始转的时候在某个平衡态用一个特别小的力就能直接堵住，涉及磁通量）。

标准答案是直接跳出来，因为你密度没变。我刚开始没想明白，后来突然发现用很简单的力学知识就能解释。

是这样的，人等比例缩小之后肌肉横截面积是平方缩小（和弹跳力成正比），而质量是立方缩小（和重力成正比），所以弹跳高度是一直不变的。

我一直以为缩小过程是不科学的，缩小之后能呼吸、能看见东西是不科学的，没想到小人弱鸡的运动能力也是不科学的![](img/780f2fa7dfbf7f70153d10eeeb3e8bb1.png)

感觉自己一直在代入的那种无力感被物理定律直接否定了![](img/27533062db6ce4e24c3784cc8474f5af.png)

# 3

不要被所谓科学限制了想象力。
或许有物理法则不一样的宇宙呢。

这个弹跳的问题，要考虑空气阻力呢，力量平方缩小，质量/重力立方缩小，但是同速时空气阻力是平方缩小的啊！足够大可以不计空气阻力，足够小的话空气阻力还是有影响的。
