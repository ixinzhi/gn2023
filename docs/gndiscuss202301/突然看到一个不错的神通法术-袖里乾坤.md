# 1

刚刚在看玄幻小说，看到一个神通，叫袖里乾坤，用袖口一挥就可以把人缩小收进袖子里，想想都画面感满满。（毕竟缩小术都有了，袖里乾坤带上名正言顺的缩小功能也很能理解ww）
场景：
        一个女修大能飞过一个凡俗城镇，突然想起了似乎自己洞府里的玩具不多了，那漂亮的脸带上了微微的粉红色，于是手一挥，带动淡蓝色的袖子从眼前一遮而过，整个城镇慢慢飞起缩小，最后收进袖子里。
        城镇里人头汹涌，热闹非凡，正是闹市期间，忽然有人大叫一指
        “看！天上有一位仙女飞过！”
        “唉，真的唉，是一位仙女啊！”
        “这是飞仙！是祥瑞！我们小镇将有鸿福呀！”
        “老婆/老公！快出来看仙女啦！”
        顿时，城镇里各种杂音响起，抬头的，探头张望的，比比皆是。
        突然地面开始震动起来，每个人都来不及反应，大地似乎飞起来了，人人开始惊恐起来。也有人看着那越来越大的仙女，看到那副天仙之貌，不自禁的沉迷其中，忘记了周遭。
       最后，恍惚间，天地都变了，人们似乎还不明白发生了什么，但入眼的，除了城镇的本身和周围的一切，整个天空都变成了朦胧的淡蓝色，还有一根白里透红的晶莹梁柱，横在天空，遮断了中间的半片天空。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
