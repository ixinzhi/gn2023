# 1

GTS同好領域有在創作或提供作品的比例是蠻少的,大多都是觀看的心態吧!
我之前也是從觀刊的心態轉變至想創作的想法
雖然發表幾篇不太好的作品,但也有很多同好蠻支持的
現在小V也開始嘗試寫小說了,而且但丁也正宣布提供新作品
有可能是新的長篇的開頭,大家要多支持唷!不然熱火一冷變結冰,但丁可能又要消失一陣子啦!
現在也有注意到至圖大師大仁也歸隊了,大家掌聲鼓勵!熱烈歡迎!

現在我暫時可能步會發布新小說,短時間沒這個打算,還在思慮中,學業上工作上投資上這些我還在努力及思考
所以有時沒那麼多心思編寫新小說,所以我線在以手頭上有的作品做提供,應該有些人知道
就是那些小人國相片,我那可以長時間從網路上一一尋找的,我還有一些長的不錯的女生在小人國拍的照片
有些人蠻多張的,我有想過可以以故事方式提供出來,找時間在貼這些相片了!

我有購買過譯經8.0版中日雙向翻譯軟體,現在出到9.0版,我升級大概要花個1千多塊
我有打算購買此套軟體,然後把日本GTS網站一些文章用此套軟體翻譯後提供上來
算至少我也有做出點貢獻,等我用好軟體之後,如果有同好想要我幫忙翻譯日文文章
我可以幫忙,到時再把文字檔給我就行了!
