# 1

這是昨天我去台北西門町萌萌同人館無意間發現的書，是台灣於七月28日舉行之同人誌展「FF10」的新刊。書名「帶子西瓜」，文字為繁體中文(廢話)，內容是東方系列的一般向同人誌。

以為是槍書的人可以出去了(逃)

這邊是刊物作者的網站：[http://shinzerotype.blogspot.com/2007/06/blog-post_7442.html](http://shinzerotype.blogspot.com/2007/06/blog-post_7442.html)

從下面的圖你能知道，故事是嬰兒化的靈夢為了恢復原狀而被灌藥，偏偏藥又搞錯了…

順帶一提，東方系列的角色「西瓜」(好，我知道她本名叫伊吹萃香)似乎有巨大化的特殊能力？那是否有別的東方本有表現出這樣的一面呢？以後我是會特別注意西瓜本。

--
說起來，其實我對普通同人誌的興趣沒那麼大說。FF10我是有去，不過光看封面哪會知道這本有GTS內容？而且又是我很不熟悉的東方系列。要不是萌萌有試閱本，而我又剛好隨手拿起來翻，這條資訊大概一輩子不會在版上出現。

如果大家以後有類似資訊，歡迎貼上來彼此分享。一個人的力量畢竟是很有限的。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

老實講在接觸這本子前，我對東方是完全陌生的。我甚至是在看本子、看到有些橋段完全看不懂的時候，才跑去查的資料。不過我之前就有看過你貼的那個四格就是。

像這種材料，遊戲原作表現得如何倒不是重點，重點在有沒有社團看上西瓜(好，我知道是同音笑話)的這個設定來出本，或本子裡有拿西瓜這個設定來做文章。要打比方的話就像極限女孩(UG)，短短的動畫並沒表現出多少合我們胃口的feeling，但我們總可以期待有不錯的UG同人，有時這些同人誌帶給大家的happy time還更多哩。![](img/bfdf5aa3d92e3fa8fd0cb15ca27e1367.png)

另外回影武者，萌萌在台北西區的西門町，搭捷運到西門站下車。地址是台北市中華路一段114巷17號2樓。
我必須說明的是並不好找，所以你最好先上一下官網：[http://www.moe2.com.tw/](http://www.moe2.com.tw/)

老實說接觸東方這麼久 看過這麼多同人誌 ........

我壓根沒把東方跟GTS聯想在一起 Orz(別問我為什麼

巨大化只能算是翠香的能力 操控疏密的程度的表現之一(將身高聚集起來 所以比例不變XD

雖然說很多本子都畫出巨大化的場面 不過大多也只是帶過 沒有完全以這個為題的本子

畢竟以西瓜的人氣來說 雖然不低 但比起主流的紅魔 冥界 八雲家 來說還是低了一大截

而且以他為主角的本 大多走搞笑或四格風格

比較推薦徒步二分的リグルVS紅魔館 雖然翠香的戲份只有兩三頁

不過輾蟲那段 真的很有魄力 而且我想徒步這種狂氣1000%的劇情跟畫風
