# 19

.
其實很多時候，自己的反應過度驚慌，反而會讓別人起疑。

確實，我們都有曾經以為自己是天底下唯一一個有GTS喜好的人。
對於自己跟一般人有所不同的這個問題，讓我們打從小的時候開始，就已經習慣隱藏自己的喜好，
所以後來，當我們在處理豐富的網路收藏品時，也總是格外小心，深怕被人窺見。

把資料夾設置一層又一層，嫌不夠還得再多設置隱藏，這樣的事情我也作過。
可是，真正會讓你遇上收藏曝光這種危機的情況，多半都要怪罪自己不夠小心。

我來講個跟樓主的遭遇非常相近的例子。
當時的情況也是有朋友來住處拜訪，那時的我還不認識任何華人的GTS同好，接觸網路的GTS社群大概也只有兩個年頭。
不過那時的我已經不大在意收藏隱密度的事情，畢竟我還挺能肯定自己的東西不會被亂翻。
當然，身邊的朋友永遠會打破你這種自以為是的觀念。

當天，朋友來到我的住處。他那時到底來做什麼，我根本都忘記了。
我一如往常倒杯飲料給他，讓他在我房間稍等。
結果當我把飲料送進去的時候，他已經拿著我印下來的英文小說在盯著看了。

真他X愛亂翻啊，我至今仍這樣想。
我把印在A4紙上的文章放在風琴夾裡，並不是說很無防備的擺在桌上，結果這小子還是大大方方地從書架上拿起來看，嘖嘖。
（現在想想，應該是平常交情太好的關係，即使已是大學生卻仍沒什麼分寸）

好吧，被看到就被看到了。
如果是普通的英文文章，就算能力再怎麼強，要在十幾秒內看出這是在說什麼的文章，也不太容易，對吧？

偏偏好死不死，我印下來的是用翻譯軟體翻過的中英對照版。
更叫人哭笑不得的是，當時的我不知在想什麼，居然還自己把 giantess、feet、crush 之類的關鍵字上了特別的顏色。
大概是想說拿起來看的時候可以儘快找到重點吧？
可是，對我來說是如此，對別人來說卻也是一樣。
所以我當場就變得很慌亂，一把將風琴夾與印下來的文章從朋友手上搶走。

大家瞧，這個動作不正暗示對方你自己心虛嗎？這只是在徒增對方心中的疑慮。
原本覺得沒什麼的，都要變得懷疑起來了。
所以在此奉勸各位後進，當你真的遇上類似狀況的時候，裝傻或裝作若無其事是最好的。

總之，這樣一搞，變成我非跟那位朋友說明什麼是GTS不可了。
那場面實在有夠尷尬，處境比樓主悽慘多了，我也是有小嫩嫩的時期呢……
就這樣，那朋友成為第二個在現實生活中知道我這個性癖的人，不過他後來也是絕口不提此事。
我想，我那一天的反應，真的是太大了一點。
但人的行動在那種狀況下，會受到情緒很大的牽制。
那天的經驗讓我明白，自己並不是個意志堅強的人。
那種全身血液倒流的感覺，我算是體會過了，真強者應該永遠也體會不到那種感覺吧？呵呵。
