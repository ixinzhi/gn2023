# 1

很多潜水员手握每天免费可以送出的奖励点都懒的送，而很多创作者也很想得到奖励点（就比如我，穷的连银河都不舍得进）。

于是我们可以改革一下奖励点的用法吗。

比如一篇文章，超过多少字数就可以设置  奖励点（门票）：【想要看，先支付我5点奖励点】，或者10点具体看作者的设定，然而这些奖励点不需要从自己的账户里扣，可以直接用每天免费的20点扣，扣完每天的20点才接着从自己的账户里扣。

这样即使什么作品需要奖励点作为门票才能看每个人也有每天20点的票门。其实说白了既不用看官花费什么，作者还有收益。

具体多少字可以收多少奖励点的门票要求，影视图片游戏等怎么收费可以再商议，毕竟如果每个作品都这样以奖励点做为门票收费的话每天20点估计也不够用，我先提个意见，你们看看怎么样。

另外希望可以在文章区加上统计字数功能，在购买前看到整篇字数，这样买的也放心，难免出现付了钱发现只有1000个字很坑爹的情况。
