# 3

论坛里大部分MMD作者目前还是在K帧和改模这一块的吧，自己做模的大神好像……没有。
MMD和自己画漫画不同，绝大部分MMD模型的所有权还是归模型制作者所有，连背景甚至特效往往都不例外。你用别人的素材改模、做图、做视频，都不能算是完全原创，是有版权争议的。 
自娱自乐可以，分享视频网站也没问题，很多模型其实连R-18或者露点都不让，但你自己低调做不传播出去那别人也管不了那么宽，商业化那是绝对不行，让原作者乃至MMD圈内的人知道那肯定要来怼你没跑了。
不代表GN，个人立场，我不支持有版权问题的MMD作品用本论坛作为商业化的宣传平台。这事儿实在容易产生纠纷。

除非你的MMD作品从建模到完成都是自己原创（或者经过模型配布者的认可，同意商业化），没有任何版权问题。
