# 1

       我说说自己，我个人是很喜欢一部作品能够重点描绘一个普通人，突然发生一个意外或者是不可预知的情况导致变为了mega级或者之类的GTS，当情况发生后对于自己成为GTS是无意识的，慢慢的当她发现自己已经成为GTS后哪怕是一个脚步都会造成人类的大面积灾害时的内心刻画（就是类似那种内心非常纠结、痛苦和不知所措的情感刻画吧），我非常喜欢的作品例如但大的《文明吞噬者》、《两万三千俱乐部》、但大作品中的红狱神女的出现（但好像没有哪一部作品专门描绘红狱神女出现的）以及70英里詹妮（但这部作品中对内心情感并没有很细致的描述，而是相对简单的一笔带过）
       圈中关于此类的作品真的相当少，giga类的作品大部分都是普通人变成GTS后没多久就接受了自己已经是GTS的事实然后就和小人开始了互动。

       我自己很少在论坛里发文，相当多的时间都是在潜水~发此文主要想看看圈中是否有人和我一样喜爱此类作品~

  同非常喜欢这种，GTS一开始是个普通女生，从与小人们一样卑微瞬间转换成统治支配一切的存在，巨大的反差感与滋生的上位感，优越感是我非常中意巨大化要素的点。
  问题是，自己写这种文，巨大化后的女性的心理，作为，想法，最终完全会写成为自己癖好而服务的发展。比如自己想让妹子变大，踩踏城市和小人，征服世界，妹子就一定会喜欢变大（至少之后会喜欢），喜欢破坏与支配的感觉。而变大后的其它可能性，比如恐惧，惊恐，罪恶感，想回归普通生活，亦或是根本不想破坏，想温柔对待小人，想和平共处等等，因为我自己觉得不爽就几乎不会写触及这些可能性。把自己说的狠一些，其实根本就是把GTS当成自己泄欲的工具，GTS不像一个真实女生，更像是一个输入了指令的人形怪兽......
