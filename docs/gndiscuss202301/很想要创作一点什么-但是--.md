# 1

但是并不知道大家喜欢什么，从我的数据调查来看，通常带有如下标签的内容：
足控、vore、giga、mega、UB、futa
……我在说什么啊，这已经几乎囊括所有可能的性癖了吧。
所以我其实更多是想了解大家喜欢什么比较多一点。
比如说是喜欢剧情多一点还是破坏啊，黄色啊多一点之类的。
……算了感觉这样更扯不清，还是直接告诉我应该怎么做吧，医生最擅长对症下药，而不是理论救人。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
