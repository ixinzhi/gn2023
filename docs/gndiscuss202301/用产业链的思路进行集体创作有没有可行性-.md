# 1

众所周知，日本动画能行销全球，一个很重要的原因就在于他们有一条高效的产业链，也就是ACGN这四种形式的作品互相改编。一方面改编作可以吸收原作粉丝转化为观众，另一方面有更高曝光度接受度的改编形式(如动画和游戏)也拉动了原作的知名度和销量。

那么圈内有没有可能也采用这个形式？
以我个人的观察来说，圈内小说的受众比例其实是非常低的，远不如图片视频这种直观的媒体更能吸引受众。电子游戏同时可以承载图片和视频两种媒体，虽然不如那两种更直接，但电子游戏的互动性可以给予玩家远超其他表现形式的代入感。
如此便出现了非常不均衡的现象：小说的受众虽然不多，但因其极低的创作门槛，小说类型的作品反而是最多的。
自然，文字创作的低门槛导致一部分作品质量并不高，但也因文字表达的特性，使其中同样不乏有趣的设定&世界观、令人印象深刻的角色塑造、优秀的剧情剧本以及各种脑洞大开玩法play。
反观圈内图片视频和游戏…本人对于传统JRPG式的过于割裂的叙事手法一向不感冒所以暂且不讨论，至少视频方面确实充斥着大量OOC套皮压路机MMD，即使是山水辺这种做出个人特色的作者，情节也往往是相对单薄的。
目前圈内的各类创作基本是各玩各的，加上之前推《魅魔爱丽丝(サキュバスアリス)》([https://www.dlsite.com/maniax/work/=/product_id/RJ065081.html](https://www.dlsite.com/maniax/work/=/product_id/RJ065081.html))时几乎是圈内电子游戏/视觉小说最良好的阅读体验给了我比较深的印象(听说这一作就是改编自小说)，于是我就想，圈内能不能也像ACGN的产业链一样，多一些互相改编、弥补各自类型的短板、同时互相引流人气呢？可行性有多少？

# 7

产业链的形式啊，主要是上中下游的创作者（我暂且这样说）的合作存在问题。
并不是说不方便合作，而是说，是否存在合作成功的可能性。
以我自己的理解，上游是文案，中游是图片漫画制作者 下游是游戏/视频制作者，三者之间的分工明确。但是这三者的人数分布是不平衡的，上游的创作者相对来说比较多，下游的就非常少，从基础这一块来说说，就存在了一定的困难。
还有其他的困难就不说了，楼上已经有人说了。
当然，也不是不存在成功的可能性，就我而言，我自己会写文会做图会一点游戏制作（unity和RPGmaker），我自己是在等一个机会，等一个有魄力的人来召集起圈内的创作者共同创造一个事物

# 9

集体创作需要一个共同的世界观，这在上游阶段就很难实施了。不过格列佛的世界观，却意外的可以统一上游小说，中游图片，下游游戏的思路。最好的例子就是grimmGlitter，出现大人，普通人，和小人的世界体系。 <title>10</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 10

一个可行的方案就是，gn平时注重一个共同世界观的主题，比如格列佛世界。然后朝着这个主题，积累上游小说，中游图片和视频的素材。最后下游游戏制作时，gn就有一批格列佛的小说，图片和视频素材使用了。 <title>11</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 11

我认为gn集体创作应当是异步进行的，大家互不干扰，但都在一个世界观中前进。目前只有格列佛世界观能够统一上，中，下三游。实现这个步骤，只需要版主发布一个格列佛世界观创作的论坛活动即可。 <title>12</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 12

目前格列佛世界观其实在gn里不得人心，主要是大多数作品都是英文翻译过来的，与本土语言存在差异。我建议可以先从国外引进格列佛世界观的好文，在gn建立格列佛世界的体系，然后共同朝这个格列佛世界体系推进，才有可能实现上游，中游和下游的集体创作。 <title>13</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
