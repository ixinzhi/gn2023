# 3

稍微瞥一下教學，你使用官方的Bridge來做匯入啊……
這個正規手法可能會有的問題，就是你用了非官方的東西會匯不過去(像是Golden Palace的性器官)
第三方的Diffeomorphic可以處理那些，但是設定起來非常麻煩，我在學會前已經跳船了所以也不知道怎麼弄
有聽別人說過直接匯出FBX就好，不過那個匯出匯入也有設定要調就是

還有就是單純影片的問題，你的資訊實在給太慢太少了
請善用：
1.稿子
你要開在桌面上還是真的放本筆記在旁邊都好，就算寫不了逐字稿，先把重要的點寫下來能讓你錄影時省下一堆時間
2.剪輯
別跟我說不會，只是把不要的片段裁掉的話，連Blender都能剪好這種影片
3.字幕
直接解決錄音不清的問題，有忘記講的話還能附註上去，方便很多

看看YouTube上的人們在1 分鐘、30秒內把問題講完，你也該試試看
事先做好計畫的話，你錄影的20分鐘已經夠剪完上傳一隻資訊量相等的1分鐘影片了 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
