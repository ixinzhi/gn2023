# 1

    这是我入坑gts圈体会到很大的一点，最开始入坑的时候是在B站上看了“古唐寺”的视频，慢慢到后面加了各种资源群以及文游群（在文游群里还认了个主人），知道后面才知道，真正的大佬都在gn（比如说但丁），gw（比如说血蔷薇-满咲，大超萌的作者），giantesscity（当年用虚幻三制作了playingroom的作者），等等等等。而很多这些大佬的作品，都有一种命运。那就是被打包进压缩包发到那些xxs创建的资源群里，这就给了黄牛很大的可乘之机，天猎依稀记得当初有一个兄弟问我说要资源吗？内些资源全是他花钱买的，结果发过来的时候我一看（当场傻眼）这些都是免费的呀，谁把这些资源收集到一起拿来卖钱？这是对于创作者的不尊重，试问制作一张电脑算绘要多长的时间？一个游戏的制作要多长时间？这些创作者们花了大量的时间和精力做出来的东西，他们自己都没有要求收费或者是报酬，但是黄牛们却拿着别人的劳动成果去骗内些刚入圈的萌新的钱。（幸好我没受骗，但我现在血压已经升高了），举一个最例子，血蔷薇-满咲制作的大超萌领狱，在游戏地图的左上角，有一个区域没有开放（说是会在后续更新的时候开放），但是我们现在已经很难看到它开放了。血蔷薇在gw的大超萌官贴中说过，不要拿这个游戏去收费，不要把里面的cg提取出来（已经有人这么干了），否则他可能会放弃更新游戏。但事实就是黄牛们依然无法无天，让一个又一个的创作者心寒停止创作。。。。。在gw血蔷薇的主题当中，有一个板块是暂扣回收（也就是没过审）名字叫论坛隐退声明，由于权限问题我们不知道那个帖子里究竟写了什么。但绝对可以猜得出个一二。可能他真的已经寒心了吧，又或者是他已经累了。

ps:写了这么多，喷的是内些拿着免费资源去骗钱的人，不是普通的搬运人。也不是内些上当受骗的人，更不是那些qq资源区（把保护打在公屏上），如果我的言论有不对或者是冒犯到什么人（骗钱的人除外）可以留言指出。望管理不要封号（????）。

# 3

拿资源去贩卖几乎是所有圈子都会做的事儿了...... 忘了具体是啥时间了，反正那会儿gonzres的MMD视频在油管和gts圈子挺火的时候，就在贴吧上看到有人在转卖了，当时本人对于浏览器插件那一套几乎不了解（Tampermonkey），对于喜欢的外网视频一般都是弄电脑自带的录屏，就挺佩服他们能有很高清的视频版本。那会儿大概问过价格，具体记不清了，反正一部也得五块钱吧。其实说实话，几乎就是零成本，顶多付一些网费啥的，专卖给一些搭梯子费劲或者不懂得小白，分分钟入账一二百。gonzres大佬在视频发布的下方，也注明了“禁止转载”。后来看大佬也不在YouTube更新视频了，转到了一个网络挺卡的网站...（反正我觉得比油管卡的多多多）。
可能也就真的是创作者的无奈了...毕竟圈子比较小众，而且网络上也很难有效监管![](img/f8b075f16066c71dcec04987104add78.png) <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
