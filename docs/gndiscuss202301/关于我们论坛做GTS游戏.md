# 14

其实你说的我明白，玩家在通过一定的努力后才看到的CG，要比直接看到CG印象更深。但这种“努力”并不一定就是战斗，别的也行。当然，战斗系统是最普遍、最能让人接受的方式之一，但要求的技术含量非常高，一般人是做不来的。有战斗系统肯定最好，然而在做不出来战斗的情况下，解谜也是努力的方式之一。国外有很多游戏，都是通过解谜来完成的。
至于解谜的难度，你可以体会一下文曲星《搬运工》的165关。谜，不是那么好解的。
再说，在大地图中漫游、寻找隐藏的东西，也是一种乐趣。你不觉得《缩小学园》在走迷宫时不断遇到敌人的系统非常闹心吗？
也就是说从实际出发，既然我们缺少制作战斗系统的人才，这一部分“努力的过程”就只能靠解谜来弥补。但解谜毕竟有限，为了避免出现游戏过短的情况，就要适当增加游戏的长度，改良游戏的画面。这样，没有战斗系统也能做出一款好游戏来。
以上，我说的就是这个意思。

1、**恋足**。这个类别的人群最多。而且我发现不管残酷也好，玩弄也好，虐待也好，99%都是用脚的，踩踏就更不必说了。因此把它们全部归为这一类。
2、**入阴**。别看我设的投票少，喜欢这个类别的人也应该非常多——甚至比吞食系多。但华人论坛里相应的作品较少，我认为这是作者的水平有限，而不是他们不好这口。
3、**吞食**。这类人是希望和GTS同化的一部分人群，他们认为被吞食能够最大限度的与GTS的身体接触，这一类人群也相当多。
4、**纯爱**。这类人群也很多。基本上就是告白、温柔、胴体、乳房、爱抚、舔吻等淡口味情节。
5、**铁扇公主**。这是最特殊的一个人群。我之所以单独把它拿出来，是因为中国大陆同好90%以上都是这个系的，而且估计全世界也只有他们有这种癖好。他们喜欢在GTS的肚子里打闹来折磨GTS，让GTS痛苦。这一点和吞食系是格格不入的，因为他们并不想和GTS同化，而是处在一种“男王”的方式来折磨巨大“女奴”。这一大类，应该为中国大陆同好们单独开辟一块出来。
6、**其它**。包括另外那1%的残酷系、纯血腥、工具、触手、怪兽、屁股坐、排泄……等等折磨小人的其它方式，属于少数人的嗜好。

以上这六大类，均指的是在一部作品中**占比重较大**的部分。

![](img/ec942b6b7db075eaad5da7a4ad136d51.png) 虽然第三类和第五类表面上看起来都是“入腹”，但它们的本质是**相反**的。这两类人群的喜好也大都互相抵触：吞食系是GTS强大，忌讳折磨GTS；而铁扇公主系则是小人强大，忌讳消化。

另外，**第六类**不管用什么方法，则都是体现GTS的巨大形象，进而通过与强大的GTS互动来获得快感，通常是作为主线情节的**配菜**，渗透在前三大类中的，但有时也可以单独成为一类。第四类中立。而第五大类“铁扇公主系”则是小人变成了强大的一方，是如何折磨GTS的问题。
![](img/b1b9be6d3dedba0ae0ce05ca67385dab.png) 话说回来，“巨大女”和“缩小男”都是一回事吧？我觉得但丁应该重新划分一下类别，基本上是这六大类。并且建议设定“**同时选择多种类型**”，毕竟在很多作品中，这些类型是互相渗透的。
