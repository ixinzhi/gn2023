# 1

如题，我最近在构思千倍文，世界观和剧情方向有大致思路，但苦于找不到冲突。毕竟剧情要靠冲突推动，而冲突的前提是双方在博弈，博弈的前提是双方力量不能相差悬殊。然而，在Micro世界观下，GTS的力量过于强大，博弈甚至冲突就失去了基础。
所以想问问论坛同好有没有什么思路，因为我目前看到的Micro文以单方面虐杀比较多，或者是让弱势一方有超能力，但我不太想写魔法一类的东西。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

例如写gts如何艰难地不伤害小人/帮助小人/开发高难度play/...（他力本愿）
反正没有剧情矛盾，那就是没有困难也要制造困难，专门找giga+级别不容易做到的事情来做
例如举个例子，如何让tera级gts在和星球互动时，确保杀死小人的不是潮汐力？ <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
