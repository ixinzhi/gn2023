# 1

本来想搞个投票的，不过想想大家的理由可能都不一样，也很难概括，就算了

先说说我自己吧，我开始学日文的契机是那个FantasM，机翻效果不怎么样，平时上推看这么多日文也不懂，就觉得日文的环境已经在了，就开始自学了

后来学了几个月之后，大概是去年6月，搬了一张图到国内的QQ群，**机缘巧合**有人问里面几个字是什么意思，就顺手做了第一次翻译，后来又坚持下来了的原因，包括开始做英翻，也主要是**学习语言**吧
