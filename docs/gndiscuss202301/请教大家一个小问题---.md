# 5

简单来说是LCD的色位没有CRT的色位高，响应时间也比CRT长。
笔记本基本都是LCD的，而且因为体积和电源的问题背光灯管基本都只有1根，显色不均匀。
先天不足加上后天缺陷- -所以笔记本和台式看图区别很大
同一张32位图  台式看着正常，笔记本会失真
