# 3

想和做毕竟有所不同。我写文章的时候也是这样。
但与其精打细算，有的时候不如顺其自然。
构思是出来了，但感觉并不是什么时候都有的。
你的情况大概和我有时候缺乏写的感觉一样吧。
乱写果然还是不好的吧，这种时候就试着顺应自己的感觉去写写吧，不管结果怎么样。
把第一时间想到的写出来就好了吧。
我也没有什么更好的解决方法了。
希望不要误人子弟才好…… <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
