# 1

这贴本来前两天想发，都写差不多了想想还是算了不发了，结果今天上来逛一圈又有新发现，我只能用哭笑不得来形容，具体内容不详述了，就是个别老年小白（长期潜水的初级会员）实在是不懂事，也许是因为发文字而非对面交流产生的误会，但字里行间“我是伸手党我光荣”“我就先转载再询问你能怎么地”之类的节奏铁定是错不了。

所以无奈把曾经的多数作品都加了阅读权限，我的作品也许算不上多精品，但都是原创作品，而且即便有人认为作品这里那里不够细致，这个那个不够合理等等，然而每一个作品从构思到成品呈献给大家都要耗费非常多的精力和体力，转载去分享我没意见，但转载去收费就不合适了，所以对不住真心喜欢我作品的很多初级会员朋友了，权限设置并不高，真喜欢GN论坛，真尊重每一位作者的，这点权限很快就会突破，我们共同努力吧各位。

另外，正在构思新作品，但愿不会太久与大家见面。

再一次感谢一直支持我的朋友们，还是那句话，你们是我的原动力，谢谢。
