# 1

看一些老文都需要阅读权限255以上，怎么才能够255以上？ <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 2

不用管255的，你平时通过正常的发帖互动提升积分，举例10000积分就是70阅读权限，255的基本是发帖人那边出了什么问题把自己的帖子封了。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 3

由于论坛没有作者自己删帖的机制，所以有些人想删帖的话，只能通过将权限调为最高权限，也就是255来达成目的。 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
