# 1

今天读到了[Foobarz](https://giantessnight.com/gnforum2012/home.php?mod=space&uid=35201)的帖子，突然想起以前一直在想的一个想法。
这个倡议的名字叫做OpenGiantess，是受到的开源代码的启发。

下面规定OG1.0协议内容如下：
1、本协议可适用一切GTS相关的作品，不管是图像音视频或者文字。
2、源作品声明使用OG协议的，其衍生作品自动适用OG协议
3、作品适用OG协议必须由作者本人声明
4、作者自愿放弃作品适用OG协议以后的收益权。
5、作者可以声明本人适用OG协议，其本人一切GTS创作自动适用OG协议。
6、作者可以随时声明退出OG协议，但退出前的一切作品适用OG协议
7、作者允许适用OG协议的作品在互联网和线下自由传播。
8、适用OG协议的作品不得用作任何与现金交换或可兑换现金的代币交换相关的用途
9、适用OG协议的作品可以被自由的改写或续写，但必须声明为二次创作
10、适用OG协议的作品在传播中必须保留作者的相关信息
11、作品适用OG协议的声明不可以被撤回
12、不可以仅针对作品部分适用OG协议，如果作品未完成即声明适用OG协议，那么该作品的后续部分自动适用OG协议

我声明加入OG1.0协议
