# 14

從讀者角度，我之前買過數個外國創作者的3D漫畫，主要我身處香港，所以用paypal付費會比大部分人方便，paypal比起非官方信用卡付費(有被盜用信用卡的風險)和轉帳(實在麻煩) 方便了不少，但也代表更容易花錢(被巴)。以pateron為例，主要收費方法有兩種:每個月付費和按作品收費。相信大家都明白意思就不多解釋了。作為讀者相信會偏愛後者(按作品收費)，這種方法下，一般創作者會丟出一些預覽圖或文，讓各讀者按自己喜好購買，於是不喜歡的乾脆無視就好。而每個月付費對於讀者的風險則比較大，畢竟不知道接下來的一個月創作者到底會創作什麼內容(畢竟gts下還有一堆分類)，讓讀者有種上當或被騙的感覺。

但同時以patreon的創作者的角度來說(我真的沒在賣廣告, 真的) , 而且還要創作者有正職的情況下，這二選一的選擇題也不是那麼容易選擇。首先，按作品收費的方法來說，在前期並沒有多少人氣的情況來說，已經很少人會看到你的作品，更不用說要付錢買你的作品，從沒人氣撐到高人氣這過程實在艱難，而且當中幾乎沒有收入，一直無腦地創作實在有點難撐下去。而且當你高人氣後，也不一定是終於到天堂了，像一些大神SorenZer0 還有OOH，一些人買完後本著無私的"分享"精神放上熊貓等網站，結果一個人買，一百個人看完不買。這樣的結果任創作者有鋼鐵般的心神也會灰心吧。因此對於一般創作者而言，每個月收費確有其吸引之處。以我本來為例(不是賣廣告)，本來電腦配置不算太好，而且也不是土豪，要我一下子付錢升級電腦去做圖來賣，也算是一個有風險的做法，因此我在P站上目前都是以每個月收費，然後一邊做圖回饋支持者，同時儲錢。

好像說離題了![](img/5d52b431f57f09d2573252d9cbd9a925.png)
