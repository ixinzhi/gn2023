# 1

最近冒出很多與GTS非直接相關的CRUSH類影片主題……在本人的觀察中，支持者並不少。
考慮國外的情況，多數製作GTS影片的網站都有CRUSH的內容，而 giantess city 也有CRUSH的影片專板……
為了將這些與GTS非直接相關、但又有一定數量支持者想要的資源與普通的GTS內容分開，各位是否覺得應該另開這樣一個版面？

選擇「要」的人數若超過50人，我就會考慮開板。
另一方面，選擇「不要」的人數若是「要」的兩倍以上，我就不會開這個板。
選擇「沒意見」的人……啊就已經沒意見了，所以這選項當然不會起什麼作用。

我個人是認為有必要獨立出去開板，這可以增進板上的資源分配與瀏覽效率。
當然了，若真要開這板，我也會參考 giantess city 把板規定出來，至少不能出現踩貓、踩老鼠那種過度血腥的爭議影片。

那麼，就開放大家投票了，時限是兩週之內。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
