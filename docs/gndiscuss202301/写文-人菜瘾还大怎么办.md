# 7

写文的话，第一点还是要自己爽的（连自己都爽不了，读者也很难有啥共鸣）第二点的话，写完的话自己多读读自己写的文字，以读者角度审视下自己文章，看看假如自己以读者身份看这个作品会是什么阅读体验。
第三点的话，福利多些，太多剧情设定空洞或者剧情部分太多也不太会有人看的（毕竟都是上这里都是为了冲）当然纯福利的话也可，但节奏一定要把控好，有个轻重缓急。（这一点我也好犯这个错误，因为作者是知道剧情深意的，不自觉就自顾自想当然剧情这么走，但读者则一脸懵逼。凡事都是有铺垫的）
第四的话我觉得楼上哪位朋友说得对，既然设定是“公主”一定要好好利用这个身份，甚至可以大胆创新下——这个世界是公主的小世界，而人们爱戴的公主其实是一个普通的女高中生，整个世界只不过是公主也就是高中生女放学没事玩的玩具罢了之类的。
最后的话，写文这东西其实出发点还是自己开心就好，但他人的肯定也是不可或缺的动力，沉下心来，好好打磨下，回顾下自己的作品，相信楼主会为我们写出更好的作品。加油(ง •̀_•́)ง <title>8</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
