# 1

如果有人對物化沒概念，意思大概是說「將……視為物品」
像是女權講的「男性凝視」，就是以男性為出發點把女生當觀賞物
用大家比較習慣的講法可以說是「花瓶」，對故事沒幫助只是出來賣肉的橋段
通常這會讓角色平面化、缺乏靈魂，而進一步讓互動的說服力降低

不過最近翻本子，我覺得有些時候這種物化橋段反而對GTS劇情刻劃有幫助
說得更精確一點，要表達力量(Power)差距的時候

實踐上就像是，當女(大)要把男(小)抓來舔，將描寫全部集中在舌頭和身體接觸上
此時的小人並不是跟另一個「人」在互動，而是一個大肉塊，一個無法溝通的對象
但這個「物化女性」卻不是「依附男性」產生的，而是男性被拒絕在溝通橋梁外之後互相無法認知的結果
所以，物化女性部位的同時男性的存在整體也只剩一個「物」，所謂的「被物化」是「雙方無法心靈互動」的結果
當然，如果後設解析撰文者的心態那還是把女性當成男性的娛樂對象，這就要作者自己拿捏
......再講也沒人聽懂，就先這樣好了

P.S.物化單指物理上而言，意思是比起口白和心理描寫更側重肉體
用這種心態寫對話只會寫出垃圾

P.P.S.如果都看不懂，簡而言之就是多花點心力描寫肉肉，別管內心刻劃

“物化”是一种手段，也是一种态度，它把巨大娘当做服务小人与读者的工具，不管文中的巨大娘是什么性格设定，它的使用会让巨大娘发情，看到小人就无理由开展福利，文章立刻就变得“符合大众读者口味”，对应的，读者撸就完事了。
这样子的人物不会给读者留下很多印象，除非偶尔落到描写好的作者手里可能才会有“那个巨大娘h起来怎么怎么带劲”之类的印象。
描写剧情刻画人物，这是一部分作者的执着与追求，他们看着笔下的人物在自己设定的世界里随心畅游，心中产生的这种爽快感，是只顾着撸撸射的读者体会不到的，那些读者甚至不愿意静下心去读完这么一篇有血有肉的文章。不过反过来，不只是读者选择文章，文章也在选择读者，精心刻画人物，沉迷描写的文章会将大部分吃惯了快餐的读者拒之门外，迎来的却是能静心欣赏作品的好读者，所以还请剧情文作者们坚持下去！
“物化”不止是让文章变味，也能让作者读者跟着变味，曾遇见过只会说“666”，“大佬牛逼”，“快更啊，求后续”的张口要饭形读者，也遇见过“作品就是工具，让我在圈子里出名的工具”的作者，更可悲的是还有些作者把读者明显的不能再明显的敷衍当成真心的有价值的东西去喜欢，为了获得更多这种东西，就一次又一次执行着“把巨大娘当做讨好读者的工具”的操作。
发出来估计会招惹很多人，不过我已经习惯了，自己爽是最重要的。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

怪獸電影大概可以劃成兩種，怪獸有人性的和沒有的
有科學怪人怪獸這種幾乎是人類的，也有一團沒大腦的吃人果凍的故事
換句話說就是，你要不要把怪獸本身當一個角色來刻劃

所以套到你的問題，巨大娘沒人性並不必然導致一個乏味的作品
不然看看殭屍題材，內容重點就在把人性拔掉，這種題材怎麼能活這麼久?
因為描寫重點在他者的反應上
怪物本身只是一個契機，用來導引出一般人的內心衝擊
角色作為觀察者的內心變化才是核心，不然你拍個只有無腦殭屍沒人類的電影有誰想看

像是你提的，有人用高跟鞋把貓踩死，這件事本身的確完全無法做成像樣的劇本，但做為題材出發點絕對堪用
隨便掰，有個小明很愛貓，他覺得自己應該要為了貓貓盡最大的努力，成天領養並宣傳希望大家都去收容所領養
但是有一天，他看到了貓咪被踩死的影片，當下馬上作嘔關掉，但是畫面在腦袋裡抹滅不去
他開始也對親近的貓咪感到反胃，害怕自己的愛會不會也造成某種傷害，進而重新反省人與貓的關係
最後結論是什麼也不重要，只要沒有突然毀掉之前鋪陳起來的邏輯就好

不過你要說男女不可能又視對方為物品又視對方為人類，這個已經算初階邏輯謬誤了
首先是，你得靠慮到時間軸的存在
有種愛情就是一開始單純靠打炮建立起來，日子久了開始產生心理上的依靠
反過來也有可能，一開始賀爾蒙衝腦黏得要死，後期只剩空洞的洩慾最後分手的也一堆
就算沒有長期鋪陳，短期內發生會讓大腦狀況改變的事件(喝酒、睡眠被干擾)也可以馬上改變相處模式
這還是不代表你可以直接重寫人格，但是人是會變的，遇到什麼事件會有什麼反應還要把之前的事件考慮進來
