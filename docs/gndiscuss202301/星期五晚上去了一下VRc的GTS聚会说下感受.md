# 1

首先是电脑配置要求确实有。我是4800h 和1650ti
在20人的sansyo房间。基本没问题，不会卡掉，录像也行。
在40人的cafe屋。就必须开防暴死了，而且不能录像，一录像就会掉线。
_(:з」∠)_ 问了一下朋友。2060s（桌面端） 显卡 好像在40人的屋子里也会卡，但是掉线少。
虽然真的很有趣，但是确实要付出一定代价，现在已经买了r9p拯救者
希望两周后的聚会能好一吧！

然后是游戏体验，很新奇。虽然穿模，但是压迫感还是在的。
✧(≖ ◡ ≖✿)尤其是。感到了小人感知能力的局限性，1000倍的和1万倍的女巨人，小人根本无法判断大小。因为整个参照系都被对方笼罩了。
最后，日本地图那个图的烟雾效果，让巨大山椒姐从朦胧的阴影慢慢变为实体的感觉，太棒了。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
