# 1

        今年1月份的时候，看到resizeme这个游戏，我终于下定决心，想要自己动手，通过做图软件来达到自己动手丰衣足食的目的，首先，我在unity和blender间徘徊，到底要先学哪个，一直拿不定主意，于是两边都开始各学一点，unity的优点在于可以很好地制作游戏，可以做到以前那种第一人称视角类的游戏，把一个妹子模型放大100倍，然后就可以通过移动方向键来进行各种探索，可是局限性在于，相关的模型是fbx文件的，这里面贴图文件是一个很大的难题，百度了很久，结果才发现这是一个无解的问题，还有表情和动作都不好做，于是放弃。
       后来把目光放到了blender上面，有很多现成的模型的文件，但最核心的问题，表情和动作还是不好弄，也无法导入别人现成的文件，再次放弃。
       心灰意冷之际，由于在学习uniy的时候，教程教了我十几天的c#基础，我又把目光转移到做爬虫上面，目前P站，D站，booru类的，一些漫画网站，都成功做了爬虫，可是问题又来了，没有网站可以继续给我爬了，于是这门技术也又荒废了。
        这里不得不说一下，百度其实并不万能，很多问题，我百度了无数次，结果很多都是答非所问，或者并不适合初学者的，无数的坑在等着你，百度真的太累了。
        现在，终于发现原来mmd是一个非常好的软件，捏表情不要太简单，各种模型到处都是，求大佬能不能把做好的场景分享出来啊，我自己再琢磨一下，真心不想再百度了。
