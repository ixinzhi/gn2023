# 1

一般來說，文字遊戲可以促進多方交流，提高文筆和感情塑造水平，可是我沒見過gn群（或許那裡應該叫手游交流群）和論壇有人發起過類似的遊戲。
雖然我勢單力薄，沒有話語權，但我還是發起一個文字遊戲申請。
遊戲即滿足生存條件基礎上，在一定的規則、空間和時間的安排下，對自我進行滿足的行為。
文字遊戲就是主要以文字活動的方式進行遊戲。
我經常和朋友玩以下三個文字遊戲，不過因為我感情比較稀薄的關係，有些遊戲我玩的不是很好。
1.人物塑造：
當一個豐滿的、活靈活現的人設在作者的文章裡閃耀的時候，大家的腦海裡就能想到這個角色的形象、神態甚至是動作。
人物塑造遊戲，就是提供一個角色，書寫下人物的外貌、性格以及一段背景故事，看看大家能否在腦海裡浮現出這樣的一個角色。
2.故事接龍
每個人腦海中的思想幾乎都有不同，也就是所謂的“一千個人心中有一千個哈姆雷特”。
而當一大群人的思想以故事的方式鏈接在一起時，這個故事就會變得天馬行空，充滿每個人思想的光輝。
3.對聯
對聯不必多說，這是中國自古以來文人墨客必玩的文字遊戲，可高雅，可凡俗。
如今大部分地區的方言甚至是官方語言普通話，都早已缺失了古漢語的“平仄”，在對對聯時，除了押韻腳，還需要對平仄，但是大家都不知道平仄怎麼辦呢？
雖然很不尊重傳統文化，但是如果實在不知道平仄，那麼就用第一聲和第二聲代表平，用第三聲第四聲代表仄。
