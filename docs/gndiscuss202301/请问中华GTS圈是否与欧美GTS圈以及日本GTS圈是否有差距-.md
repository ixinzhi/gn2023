# 1

我混迹了论坛也快4年了，虽然很少发表作品，但是诸位的作品大部分是阅读过的。欧美GTS圈我也有涉及，而

且受他们的影响也很深，日本有不通日文所以无法涉及（希望懂日文的大大能分享）。

《蒙德的奋斗》，《平凡女孩》，《温韵》和《不能说的秘密》算是论坛的鼻祖，但大的故事也算是经典中的经

典，尤其是那篇讲述与魔鬼交易的那篇。其实中华GTS圈比欧美GTS圈更早发展，早在明代的《西游记》就已经

有GTS的存在，欧美在18世纪的《格列佛遊記》才正式开始。我不知道是否是什么原因，很多时候中华GTS圈似

乎不太有触碰到现实中的伦理以及人性的发掘。人性的发掘在《蒙德的奋斗中》算是最好挖掘得最好的，但大的

文章中也有不少类似的精髓，但社会的伦理似乎就比较少。欧美是比较多，但佳作也不是特别的多，Gheed大大

的Close算是其中一篇有关伦理的佳作。我认为只有不断的透过各种角度去挖掘，那么GTS才会的生命才会长久，

要不然只会停留在感官刺激上，刺激久了会很累。我们论坛中的大神大部分在塑造世界观似乎所日本的影响比较大

（如有错误请指正），这当然是和文化有直接的关系。个人觉得，我们还是有我们自己的特色，毕竟我们起步得

很早，而且神话体系也很成熟，但是在人性的发掘和社会伦理的探讨，我们应该向欧美多交流，这样才能将GTS

的生命继续下去。

我感觉天朝的g圈要比国外晚一些，所谓g圈应该是以gts为中心，对其的仰慕与迷恋之类的。西游记似乎没有这种氛围。再者我国g圈绝大部分同好还是以资源为主，热情很大程度依赖感官刺激，也就是所谓性欲。从其他角度，比如心理，人性，社会方面对gts进行分析的作品也是有的，但只是少数。国外也差不多，但是国外作品比国内多很多，各种创作起步也比国内要，所以g圈应该比较早。

1\. 全世界的GTS文化圈爱好者没有太大本质区别，无非就是审美观的问题而已。
2\. 中国人口多基数大，GTS同好数量不算少，不乏有很多优秀的GTS创作者人才。
3\. 混圈子的大多数是年轻人，思想不成熟，作品不成熟是很正常的现象，国外也有很多经久创作的作者依然“思路不够成熟”，相反，思想成熟的人大多成家立业很难再为这个圈子做贡献了，全世界都一样，所以好好珍惜还在创作的作者们吧。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
