# 1

gts和小人的关系最好是什么？这是一个我纠结很久，多方面思考的问题，
关系是什么，比如同学同事，兄妹，上下级之类的，采用什么互动，比如欺负还是照顾之类的。人物是几对几，年龄性别怎么办之类的，
最终的结局该是什么，这都是复杂的。
我思考的主要是在各种结局里选择哪种，以及为什么做出选择。
社畜和学生亲兄妹，社畜邻居，三个甥女和皇帝小姨，平民两个甥女和舅舅，皇帝和表妹，两国皇帝，我都试过了
结果，亲兄妹不方便欺负，邻居幼驯染太不真实，多个甥女我写不好，皇帝和表妹又写成了夺权宫斗，两国皇帝物理距离太远，简直头要想炸了。
虽然任何关系都可以写文，但是我一直在追求一个适合我的剧情，现在都半夜三点几了，
是现实中的普通人，还是异世界贵族，还是什么其他的，人物设定，人物关系，剧情发展，都很纠结，
gts和小人都满意的喜剧，温馨的，而不是残酷的，而且希望可以不留遗憾的，不出现上面我说的那么多的遗憾和问题的

# 5

个人看法是说如果楼主期待双方和谐关系的话，最好给男主一些超能力之类的或者是说男性在女方的成长中发挥过巨大作用（只是举例），目的是让男性达到和女方相对等的能力水平。

当然gts文何必过于追求完美和现实，只要作者写得高兴，读者读得开心岂不美哉(〜￣▽￣)〜，如果愿意听我胡扯请往下看：

只要是双方的实力（字面意思）存在较大差距的情况下都是很难最终达成真正和谐的关系的。即便是说女方非常照顾男方，实际上也只是体现了女方对男方的爱（以爱情关系为例）罢了，而男方如果没有与女方相对等的能力或者实力是很难赋予女方相对等的照顾或者爱的。在体型差距巨大的前提下男方的生活起居有时甚至全部托付给了女性，这样的关系长久发展下去往往是会慢慢变质的。我个人认为有这几点原因：

1.首先在女性方面：
（1）gts文里为了塑造出吸引人的女巨人形象一般只会赋予女巨人“神力”而非“神性”，女巨人本身的形象塑造就是需要体现女巨人和小人力量的差距，但这些女巨人的心智或者道德都是非常一般的也就是会处于正常人的水平上。

（2）这样就导致一种矛盾，如果把女巨人塑造成一般人的道德水平，那么她就不可能对一个渺小的人类持之以恒地保持着耐心和爱恋（想想日常生活就能明白）；如果把女巨人塑造成圣人一样的道德模范，那么故事就没法讲下去了，gts故事非常依赖双方的矛盾来推动，矛盾一产生，双方的实力差距一显现，关系中不和谐的因素也就在酝酿了。

2.其次在男性方面
（1）gts文章往往会低估处于弱势地位男性的意志力。许多情况下为了体现屈辱感和增加欺负的情节会把男性塑造的过于懦弱与无助。什么意思呢，就是说gts文在描写男性的时候其实更倾向于突出其“动物性”而非“人性”，不管是性欲也好恐惧也罢，男主人公在文中其实很少体现出其人性的一面，充其量更多是有智慧的善解人意的虫子，事件的进行和推动终究还得看女方。

（2）因此对男性的动物性的塑造也使得男女之间的和谐关系的保持会变得很困难。

3.总之，如果男性相比于女性来讲过于脆弱（不管是精神上还是肉体上），就不可避免地会给女方带来许多麻烦，在一般人的道德标准下，和谐关系可能可以存在一段时间，但是时间长了必然会起嫌隙的。我们日常生活里和等身的家人和恋人都时常有矛盾，何况一个脆弱的小人呢。

楼主想写出男女之间的和谐关系我作为读者是非常支持的，因为你愿意去琢磨人与人关系的磨合与改进，想必生活中也是个善良的人。但是gts故事要避免悲剧我个人感觉太困难了，这件事不能深究，很容易心里难过的，所以希望你也不必过于纠结这件事。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

上面一楼的super老兄说的也很有道理，男方对于女方的有特殊意义比较好。至于矛盾和冲突我觉得和温柔是并不冲突的，即便是温柔系带一点拌嘴和矛盾反而会更吸引人的。
至于剧情的发展，不知道楼主大大所期望的和谐关系是从一而终的还是循序渐进的。如果是从一而终的可以向前面朋友说的那样写成温馨的日常，用这种单元剧的形式来呈现，比较需要楼主你有丰富的生活经验和素材。

如果是后者，两人一开始有矛盾或者不认识，其实更方便写一个传统式的有剧情和主线的故事，来讲双方从不亲密到亲密的过程。例如女主开始是普通大小并且遇到困境，但是男主帮助女主解决了问题，代价却是自己变小；又例如女主一开始就是巨人，并且搞了一些破坏，但是有一定能力的男主通过某种手段教育或者感化了女主（胡乱编的几个例子）

总之，如果楼主大大想写好的是关系，那就不能局限只把一个人写好，至少要把gts和她的特殊对象给予丰富笔墨。女巨人即使力量超凡在这里也不能肆无忌惮，因为有那份和谐关系在束缚着她，她必须时刻考虑另一半的感受，反之亦然，而这本身就是两人相处的矛盾——他们是**两个人**，两人也许心有灵犀，但也不可能事事相通。另一方面，在温柔系的背景下，其实经历过冲突和矛盾的情感往往会更加和睦，因为总有一方会在反省中加深对对方的情感。

只是个人的浅薄见解，算不上什么建议，主要还是看楼主愿意具体写一个什么故事，安置何种生活场景，构建剧情服务于何种最终目的。
