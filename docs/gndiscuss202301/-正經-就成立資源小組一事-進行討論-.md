# 1

好的，這是yanmiaoyan先跟我提出的想法。他用大陸那些線上漫畫、影音網站為例，詢問我有關自行成立資源小組的可行性。

老實講，一開始我有點意外。我承認我自己從沒想過這一點。除了麻煩以外，我一直認為那可行性不高。在經由與yanmiaoyan的短訊彼此分析後，我發現倒也沒那麼不切實際就是。

首先，跟各位講解一下何謂「資源小組」。你可以把它當成類似動畫的字幕組、或漫畫的漢化(以台灣說法叫中譯)組。不過我們是有目的性的團體，只要內容有GTS，都算是我們偏好的資源，所以我們這個小組必須同時有處理影片與圖檔的能力。

然後，就來分析我們的優勢與劣勢：

與一般字幕組不同的是，GTS內容的資源久久才會出現一次，不像動畫或日劇那樣有週期性。相較之下，我們的資源小組會比較輕鬆，只需要在目標物出現後再做動員，不需周周待命，這算是一個能吸引人幫忙的地方。

但當然也有比普通字幕組更累人的。首先，GTS是一個屬性，她能出現在任何媒介上。從電影到漫畫、小說插圖甚至是報紙廣告，橫跨的領域就好比全國制霸那樣無所不及；因此這個資源小組必須有強大的「原始檔案搜尋班」。不論用任何方法，都必須把從各種管道得知的GTS媒介訊息加以確認、收集，然後丟給後續的處理組員。這裡是我相當陌生的區域，我連普通字幕組的RAW檔都不知道是從哪來的，因此這批「原始檔案搜尋班」，勢必要精通海外的各種檔案交流平台才行。別忘了，GTS也會出現在冷門作品上，普通管道有時根本派不上用場的。

那麼，分析過後，我來整理一下整個作業流程：

1.收集訊息：從類似G-ZONE或Giantess City的海外討論區上，擷取任何含有GTS屬性的媒介情報。

2.取得檔案：上游組員獲知情報後，不論用什麼方法，總之把含有GTS的對象物弄到手。

3.處理檔案：就是指對原始檔案的加工作業。是影片就上字幕，是漫畫就中譯；有時間的話還要把影片轉檔成較為通用的rmvb或avi檔，漫畫的話如果原始檔畫質不佳也請用ps修改得好看些……這部份會是需要最多人手、最多專業的地方。

4.發佈：這沒啥好講的吧？放出來就對了。報酬當然是零，古今中外的字幕組都是靠「愛」在運作下去的，而且要愛得夠深。免得遇上些挫折，整組就給我散掉。

以上，就是這樣。以成立一個免收費、純靠網路的小組來說，我們需要日翻中、英翻中(這可能少一些)的翻譯人才、懂得使用影像編輯軟體(photoshop、繪聲繪影…etc)的美工人才、還有時間或許沒多到能當翻譯，卻可以幫忙在海外網站搜尋資源的外文人才。這部份若翻譯有空也可兼任，普通同好更可以多多幫忙。

於是你可以看到，整個資源小組的核心，其實是在翻譯與美工兩項專業上面。那兩項都是可以賣錢的能力，而且基本上是靠時間在換錢；因此要他們加入當義工，真的要「愛」很夠才行。我本人的能力只到改改漫畫的文字框，影片就苦手了，對翻譯更是完全不行。

我的建議是兩項專業人才必須先找到，才能談後面的。在發展初期，我們可以用手上現有的資源做試驗，例如把一些漫畫漢化、或是把影片轉檔上字幕之類的。如果一直沒有新作出現，光是從古到今累積起來的資源，應該也夠大家忙了才是。

到此為止，我的討論都放在「免費」、「分享」的前提上，因此我一直拿字幕組當比較對象，而不是大陸那些在線漫畫網站。我相信後者其實還是商業網站。就算沒跟你使用者收錢，他一定還有別的營利管道，例如廣告收入、或燒成片拿去黑市賣之類的……所以我不把那些當成比較對象。

好了，這邊一直都是我在說，現在要換各位講講看法了。為了避免沒人理我，我要丟給大家一個問題：

「如果有東西怎麼找也找不到免費版，大家願意出點錢買回來，再請資源小組做中譯，分享給所有同好(包括根本沒出錢的)們嗎？若出錢的傢伙們不想讓肥水流入外人田，又該怎麼辦？」
