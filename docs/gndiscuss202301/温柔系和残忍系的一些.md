# 1

无大事件可讲，就是胡侃。
为了避免完全无感的情况，给出一个前提，那就是当一个人同时喜欢残忍系也喜欢温柔系。那么这个时候，两种类型的内容对于读者的享受而言是不是尝尝大相径庭。
对于我来说，温柔系本身真的会觉得，一点都不涩，唯一涩的就是巨娘本身了，而不是其行为等延伸。温柔系给我感觉是一个乌托邦一般梦幻的地方。巨大与渺小的和谐共存，携手同行，就好像步入金色梦境，散发的是理想的光辉。
而残忍系呢，真的很容易冲，非常容易（同样是于我而言）。碾碎，消化，残肢四落；屈辱，奴役，饭牛屠狗；破坏，绝望，毁灭一切。看到渺小的人们与文明，轻而易举的覆灭在巨娘哪怕最轻微的动作里。这种反差感，屈辱感，无力感，当然还有感官刺激等，应该就是对我来说的残忍系核心。然后就真的容易忍不住手。
就在想，这会不会也是其他一些人的想法。大家怎么看待这两个巨娘作品的基本类型和其间关系。
而且如果是对外宣传的话，营造一个伊甸园一般的另类世界。就像人们不会觉得人类精灵矮人生活在一起的世界观奇怪，那么巨娘物种和普通人和谐共处的世界观应该也不会容易被认为变态。那这就是温柔系了。残忍系，肯定相对来说，算是圈子的更深角落。
