# 1

会戳进这个帖子的朋友应该曾经玩过或是至少听说过潘德的预言。这是游戏骑马与砍杀——战团的一个玩家自制MOD。
好了不说废话了，玩过这个版本的人应该都知道里面是存在相当数量的女性NPC角色的，而且在新时代以后的版本里面是有相当多的高颜值妹子出现的（虽然还是有不少长得像僵尸的，不知道是代码的问题还是程序猿的恶意）。所以我想说的是如果我去写一篇污化版的《潘德的语言》的同人文，会不会被论坛里潘德的粉丝喷死？看过我之前文的读者大概能猜到我会将妹子弄成什么样的（污）形象，所以在这里征求一下大家的意见。无论同意还是反对，请诸位在回复里发下想法吧，或者说下想看谁的福利。

虽然这样说有些大言不惭的意思，但我还是想说这个游戏真的没那么难，即便开全伤，玩家技术足够好也能一杆骑枪捅死几十个人的，这不是在吹牛。至于剧情，其实已经提示很明显了，无论是根据好感度和拥有特殊物品后出现的隐藏任务，还是在诺多隐秘城堡里那大段的对话，那不都是很容易找到的剧情吗？
