# 1

目前只有火狐浏览器支持了此方法，而且很麻烦，不稳定。应该只能勉强做一个备用的办法
不是给火狐打广告！如果有任何支持DNS over HTTPS功能的浏览器也能同理使用
直接上教程
首先确认你的火狐并不是国版
打开设置-常规 往下翻找到更新一项

这一行如果有类似谋智火狐有限公司的字样就说明是国版
国际版下载地址是
确认是国际版后，再次回到设置界面，滑到最底下的网络设置一栏，点击设置

选择启用基于HTTPS的DNS，提供商自定义为 ，确定

在地址栏中输入about:config，回车，点接受风险并继续

在搜索首选项一栏中输入network.security.esni.enabled 如果你是一路看着教程过来的，此项应该为true，不用更改，搜下一项

再搜network.trr.mode，点右边的????形按钮，更改为2(自己的DNS作为备用 3全部使用基于HTTPS的DNS)

搜network.trr.bootstrapAddress，更改为1.0.0.1

重启浏览器即可享受不稳定的论坛浏览之旅
记住！用完后记得恢复！

刚刚更改的选项最右边都有恢复默认设置的箭头，至于基于HTTPS的DNS一项，取消勾选即可
自测对全部DNS污染网站（GITHUB CURSEFORGE等）基本都有效
