# 1

聚會這種事情，本來大家商量一下就可以舉行的。可是這論壇不一樣啦，大家都來自各地，要是沒個領頭羊，是很難收獲好的效果的！
我也想組織大家，可惜我說話的份量不能達到舉足輕重的地步![](img/38821495964f3f8ac52610181154b077.png) ，否則的話我願意出錢請客的。
就算不能參與大家，也熱烈希望大部分人不用總是戴著面具在論壇交流，通過一次聚會，捅破同好們之間的那層薄薄的隔膜。

我們相聚在此論壇的原因就是，擁有不被週邊人認可的的喜好。相信在現實中，我們能和一群有著相同愛好的朋友坐在同一個桌上，無所顧忌地暢言自己的想法；或者見識到更高的思想境界，一睹自己頂貼無數的作者們。對於我們將是極大的精神鼓舞。

在此希望各區的大大們趕緊站出來，趁著這次台北網聚的舉行，推波助瀾，在同好密集的地域進行動員。這可能對我們只是一場娛樂，但是我相信此聚會將意義非凡，成為人生中難忘的經驗。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

幅員廣大的地方光是交通費就不得了了，不搞成嘉年華會一樣的感覺是不會有人願意千里迢迢過去的吧？
搭八小時火車只為了跟同好見面兩小時，說真的有點好笑…

但就我所知，像廣州這個城市就有複數以上的同好存在。
我是鼓勵大家先從與自己同一個城市的朋友開始接觸，也歡迎大家用談天區來找人、揪團。

網聚這種事，務必心存善念，剩下我就不多提了。
雖然現在講還有點早，但如果有圈內人舉辦的網聚活動當中帶有可疑的性質，請提出來給各位作說明。
這種活動絕對不可以跟推銷、婚友社或者金錢扯上關係。
如果我是主辦者，我會請做了那些事的人馬上離開。
如果我不是主辦者，我會自己離開，這一點任何活動都一樣，我不是說笑的。
我某次的大學同學會就因為有人在拉信用卡業務導致我當場走人。
我講這些的用意是希望各位真的要心存善念，網聚不是生意場所更不是聲色場所，發生了麻煩也不是誰可以去擔責任的。
有人願意出來主持網聚是非常棒的一件事，但該講清楚的要先講清楚，完畢。 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
