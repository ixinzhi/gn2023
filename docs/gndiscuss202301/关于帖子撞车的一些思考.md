# 1

事情的起因是这样的
去年寺田C95本刚流出的时候，我顺手就汉化了，发布在GN和Microphiliafan上
在我发布的大概10天之后，隔壁的节操汉化组也发布了寺田C95本的汉化
后来，就在昨天晚上，隔壁有网友提醒我链接失效了，于是我重新补了
众所周知，隔壁编辑帖子需要审核（而且很慢），当然这不是重点
然后，刚才我重新看的时候，发现帖子被删了
（可以看看附件的图片，我确实发的比他们早）
应该是审核看到之后觉得是我撞车了（毕竟他们“日理万机”）
虽然很不是滋味，因为毕竟是我先发的，而且多少也算是我的劳动成果
但是我不想直接去隔壁论坛对线，我不太喜欢和人争论，删了就删了吧
再说了，我也没什么损失（这是最主要的）
以上大概就是这件事情的大概

我不是想在这里倒黑泥，也不是想说隔壁有多不好，更不是想在gn找公平
我只是产生了一些对于“撞车”这个事情的感想
这种事情在网络上是广泛存在的，哪怕是像我这样的反向撞车（对不起我想不出更好的词）也是有例子的
比如某知名up主借鉴某小up主的创意，结果前者的粉丝反倒去后者那里碰瓷（没有特指谁，只是举打个比方，因为实例太多）
但是说真的，在此之前我根本没有想过自己会碰上这样的事情，帖子里的撞车请删也感觉是约定俗成一样的话
这又让我想起之前比较尴尬的一次。有位坛友翻译了寺田某个套图（共三张）中的第一张，但当时我刚好也在翻这几张图片；后来看那位坛友迟迟不发后两张，我就把自己翻译的发了，没想到他也翻译好准备发了，这就很尴尬。不过幸运的是他很大度，没有追究我，在这里感谢一下。
撞车不是坏事，抓住这点不放去攻击别人才是不对的，因为双方同样付出了劳动，可能只是速度、空闲时间甚至网速的关系，可能事前压根不知道对方要干这件事情；尤其是对于我们这种单纯为了兴趣而不为牟利的人而言，因为既然只是因为兴趣，抄袭他人的成果是没有任何意义的。在这种情况下，在一定程度上，我觉得撞车的帖子是可以共存的，甚至可能对方的润色、嵌字技术比自己的还好，或是各有长处，可以比较学习；或是在某一方的资源因故丢失了之后另一方充当替代。当然，前提是得在帖子里注明除了自己还有别人也发了这个资源/干了这件事情。
当然，撞车也不是好事。无论如何，论坛里存在重复资源确实容易误导一部分网友，对双方可能有不太好的影响（比如一方的帖子底下门庭若市，另一方却门可罗雀，造成了心理上的不平衡），甚至有转载者和原作者之间的撞车（这种情况下我觉得只保留原作者的帖子应该是没有异议的）。这点我相信不需要多说了，大家都懂。
说到这里提一下，我上面说的被删的帖子阅读量和回复量是要比节操汉化组的帖子多一点的，可能是我卖得比较便宜的缘故吧。
总之，我有下面几点建议：
1、在帖子的标题尽可能的写出主要因素（如转载的作者、作品原名等等），方便检索查重
2、可以适当保留撞车帖子，如没有贩卖要素的翻译帖、转载帖、资源帖等，但要求楼主必须在一楼注明另外的帖子
3、如果是汉化作品的话，可以开一个告示的帖子防止他人撞车，也可以顺便找找可能存在的已经开始动工的人（这样一来就不用自己动手或是可以和别人合作了），或是直接在这个帖子里更新也可以（实际上已经有一些这样的帖子了）；但同时会有一个问题：万一有人占着茅坑不拉屎（对不起语言有些粗鄙，但我实在想不出更好的形容词了）怎么办？

才疏学浅，阅历不足，上面的话可能会有缺陷和不足，希望大家多多指正，多多讨论

就像動畫新番一樣，每個動畫都有不同字幕組在做，觀眾可以挑選自己喜歡的字幕組，

所以我認為撞車是正常的，是應該的。

應該刪帖的是同一資源同一種漢化例如2個人都發佈了A漢化組的作品，保留先發的
