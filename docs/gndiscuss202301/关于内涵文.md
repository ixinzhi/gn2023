# 4

所謂的"內涵文",其實在以前的一個吐槽帖都說過,
新手與神手寫文的分別就是,新手把重點放在GTS情節上,但神手則把重點放在鋪排,劇情上

舉個例,故事的第一章男主角就被縮小了 或 女主角變大了,然後就有一個美若天仙的女子路過,這種都是實用文的表現

不過,有不少人來看文都是要滿足生理需要,如果你給他一篇劇情文,他看到高潮時卻突然變劇情了,當然會不爽,會特別回帖要求作者少寫劇情文(雖然這從不奏效)

不同的作品對應著不同的人,劇情文當然是給優質讀者了,因為在回覆劇情文時有更多話題(有劇情嘛,隨便吐一下槽都夠一百字了),不會像快餐文一樣,除了"贊","頂"外甚麼都寫不出來

不過要寫劇情文要有心理準備,就是要一直把自己的構思砍倒重練和吐槽
除非你真的用了很多時間計劃整個劇情,再加上你有無比的寫文毅力,否則,還是去寫快餐文比較適合 <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
