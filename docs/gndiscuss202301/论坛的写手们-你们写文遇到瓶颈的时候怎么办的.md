# 11

嘿，我也和楼主一样，不过我内容上一般不会有瓶颈，都是水到渠成
阻碍我的主要是这些方面
1.写作水平不行，连让自己性奋都难以做到。或者说写的东西不符合自己性癖，只是把头脑中的故事拓印下来，构思故事是有意思的，构思完了把它抄下来是枯燥的。
2.会累，这几天平均每天两三千累炸了都快，何况是写gts文，写普通网文日更两章的那种写手岂不是更累。
3\. 在写文的日子里，总会要打开各种gts网站吧，然后就射爆了，身体严重损伤并开始贤者。

我的动力也是，一个完整的故事总不能就扔个开头搁那儿吧。所以说只要是想写，就继续写，不管是什么样的阻力都总能克服的，没时间就一天几十个字，一年下来也上万字了，比某些一坑就坑好几年的大佬强多了（对某大佬抱歉，说着玩的不是嘲讽您）

# 12

我没遇到过，或者每天都遇到，只是偶尔有闪光，大概2-3天，就会遇到一个特别特别特别想写的日子，要么是某种play想的人浑身发烫，要么就是我写的某个人物，让我脑子里忍不住去思考他或她或它的故事，总在想它们在某段时间，某个背景下会做什么，总在想我去过设置这样那样的舞台会怎样。当然，毫无疑问，我是自嗨型写手，我从不夹杂私货，因为我写的全部都是私货，所以对我写文开始，是一种想象力的发泄，也是对许多传统故事不尽兴的发泄，比如大多数某个类型的故事是这样，我就忍不住逆反，比如某个角色一般这种情况下回得到谅解而活，我就忍不住，我偏要她死，再比如有时候，主角应该信守诺言，不做某些事情，不去出格，但我也忍不住，我就偏要让他出格，所以对我来说，文章的走向并不是我能控制的，我是在下意识的去写，有时候写的就是我觉得就要这么写，甚至是只能这么写，要说我遇到瓶颈，大概就是一堆想法黏在一起相互矛盾需要时间理清，或者只是单纯的懒了罢 <title>13</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
