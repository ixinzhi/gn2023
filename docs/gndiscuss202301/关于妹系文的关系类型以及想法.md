# 1

　　感觉妹系文在圈内还是挺受欢迎的（本人也是没妹妹的妹控），我感觉根据关系大致可以分一下几种吧。
　　1.兄妹关系很好，一般是温柔系，也可能是个小恶魔妹妹，不停欺负捉弄哥哥但基本上是有分寸的在玩，这种基本上会是好结局。
　　2.兄妹关系一般，可以是意外缩小后对哥哥各种欺负，也可以是故意把哥哥缩小作为对某件事的惩罚。这里可以是好感度降或者作死低导致GG，也可以是好感度提升进入好结局。不过感觉一般人都会写好结局，坏结局的文好像不多（也可能是我没看过）
　　3.妹妹是个傲娇/病娇，可以是故意缩小想让哥哥隶属自己，或者各种意义上独占哥哥，也可以是面对意外缩小的哥哥，渐渐不再傲娇的妹妹。
　　4.兄妹关系极差的（好像没见过），我想的是意外或故意把哥哥缩小后对哥哥进行虐待，也可以是缩小交流后消除误会、关系渐渐改善？可以是哥哥最终沦为妹妹的玩物或者被处死，或者改善关系后进入好结局。
　　5.无意识，感觉无意识的话关系好像不是那么重要？
　　6.瓜文
　　考虑新文的时候主要是在想，不管关系怎样，除了深仇大恨（兄妹之间一般不会有吧）或者病娇，妹妹故意把哥哥弄死是不是太过无情？虽说gts文里一般会给女方绝对支配权，如女神一般随意决定生死，但妹系文里哥哥沦为和陌生小人同等的地位或结局，会不会导致妹妹的设定失去意义？

# 5

作品的现实分为物理现实和逻辑现实。物理现实相对来说并没有那么重要，更何况gts圈本来就带有奇幻性质。所以重点自然要放在逻辑现实上。
大多数圈内文都是图个爽就完事，但我觉得爽文也不能单纯为了爽，完全不考虑逻辑关系。在毫无理由的情况下，妹妹仅仅因为哥哥被缩小，就随意处置他的性命。这与其说是为了表现缩小人的无力和巨大娘的绝对力量，不如说是完美衬托出了作者文笔水平的惨不忍睹。要想知道这会不会导致妹妹的设定失去意义的话，只要把妹妹换成一个意外获得缩小能力的抖S路人即可。故事不仅不会串戏，还不会有逻辑漏洞。
虽然我不喜欢残忍系，但如果作者对情节的设计恰到好处，完美刻画了人物心理变化的过程，使得原本非常温柔体贴，很照顾哥哥的妹妹最后亲手残忍杀死了哥哥，那么我就觉得这是一篇好文。
也不是说作品的逻辑得有多严密，但是不能太没有底线。圈内作品不符合逻辑的桥段本来就挺多的。举个例子，巨大娘一边说着小人真恶心，一边用鞋子，袜足甚至裸足踩死小人。这是很常见的情节吧?但是小人的内脏和血液黏在鞋子袜子足底上不是更恶心么![](img/f86b7f0892ce62ea45ff58e417f53dfa.png)有的作者会交代一下反正马上要洗澡了，所以黏在脚底上也没关系。但稍微想一下，你会因为反正马上就要洗鞋子洗袜子就直接把一只蟑螂踩死，而不是用面纸夹住扔到窗户外面么……但是这种逻辑错误并没有离谱到哪里去，读者肯定也更想看到小人被踩死而不是被夹在面纸里吧。 <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

        事实上变小后的结局和变小以后的时间、经历关系也很大。
        比如妹妹一开始只是想用神奇方式获得的缩小器和哥哥开一个玩笑，结果导致哥哥失踪、警察出动，这种时候不敢把哥哥变回去了（怕被问责），最后哥哥被当成了失踪人口，而哥哥的情绪也渐渐失控崩溃，最后哥哥求助/逃亡的警要关头迫使妹妹杀了哥哥；又比如哥哥小时候曾经欺负过妹妹，但是长大后兄妹关系很融洽，在哥哥得病缩小以后，妹妹才逐渐展示出真面目（一直耿耿于怀），在日积月累中越来越占据主导地位；等等等等......（可以想出来的例子实在太多了）总之静止、刻板的兄妹关系也不是不行，但正常来说人与人之间的关系不会是静止的吧，兄妹一开始关系和睦、缩小之后破裂，兄妹一开始关系不和睦、缩小以后关系变好了这都是可能的。一篇文章的play围绕着缩小来写是合格，人物心理、关系都能随着缩小渐变我觉得才是应题。上述几种选项完全可以形成大量的全新选项，除了瓜文。
        至于妹妹设定和男主被残忍杀害是否冲突，当然是否定，楼主思维相当局限于温柔系的妹妹，大概是个人偏爱的原因了。现实的来说，完全不把哥哥当哥哥、认为哥哥分摊走了父母的爱、认为父母偏袒哥哥等都可以成为厌恶哥哥的原因。即使不厌恶也有可能是很冷漠的关系、那和普通小人就没有区别了，也就可以产生哥哥意外缩小求助妹妹，千辛万苦被注意到后妹妹却厌恶的把哥哥用面纸丢进垃圾桶那样的play。
