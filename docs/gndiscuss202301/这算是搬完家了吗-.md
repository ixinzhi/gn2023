GFW 常用的一種技術是網域伺服器快取汙染（DNS cache pollution）
所以就算換了個 IP 不換網址還是會被牆掉

簡單來講，當你打上一條網址要連上去時
實際上是連到其對應的 IP
如果該網址對應的 IP 沒有在 host 檔中（本機端的一種快取檔）
你的瀏覽器和底層的系統
要先去問問外面有沒有人知道該網址對應到的 IP 是多少
問到了之後再透過該 IP 進入網站中

域名汙染則是透過給你一個假的 IP
讓你永遠也連不到你想要的網站
所以如果要解決掉被牆的問題
就要像某些論壇一樣
擁有五、六個甚至十幾個不同網址
而且還要時常砍掉被牆掉的 / 新增新申請的 / 重新發布新版的論壇網址列表
會變得非常麻煩……
