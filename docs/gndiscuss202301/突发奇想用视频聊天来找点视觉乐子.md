# 1

一直很喜欢这个版块，喜欢带一点现实感的，圈外人朦胧模糊的擦边球。想去看看埃姆斯房间，想结婚时也去拍大小人的婚纱照，想去亲临巨大的人物海报、雕像或者人体主题乐园...然后今天跟一个关系很好的女孩视频时，她一直在玩QQ视频自带的动画效果，感觉挺好玩的（平时很少和人视频，这几天她突然特别想见我）。我看到有个边框上面有人在钓鱼，就把右下角的企鹅宠物拖过去截图，看着就像是钓上来的一样，然后我突然想到能不能把自己窗口缩小类比呢，于是跟她商量了下试了试...
唔，然后怎么说呢，虽然效果很粗糙，虽然演员长相平平，但一看到里面参演的是自己，莫名地有了些代入感呢...
反正这边比较冷清，我也不怕丢人了，就发张截图来看看吧。大家谁还有更好的现实玩法，不妨一起分享。
