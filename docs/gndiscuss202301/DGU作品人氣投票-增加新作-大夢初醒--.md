# 6

dgu的人气文章投票，我觉得大部分是根据自己厨的角色而进行的选择吧，至少我是因此选的雪佩雅丝，这是在dgu众多角色里面最戳我的角色![](img/7c971c00da9ff5ac272c2bc53420f473.png)然后剩下的，我必须给一票给世界扑灭三人，这一篇是我最早读到的dgu文章，是我接触dgu世界的开始吧，而且无论是故事剧情，或者是giga级的小可怜，也都是我很喜欢的。然后就是云端战略了，这一篇关于姜饼人的故事，也着实让我印象深刻，该说的姜饼人的设定和我中二时期的一些脑补刚好吻合吗？所以也必须给一票。
再来就是哀想音符（下），这一篇的故事也是很让我印象深刻，特别是结尾处莫扎特的一句话“象征着时间的尽头”，让我一度以为dgu是不是要完结了，震惊了好一段时间![](img/f8b075f16066c71dcec04987104add78.png)
最后就是最终神子了![](img/78d1f3cf3d1c68881c363e2c6014f483.png)大家好，我是仲宇桓吹！ <title>7</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
