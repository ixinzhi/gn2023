# 1

**如题，设一巨大娘，原高为** h_0**，****放大倍率为** n**，问之直立时能看到的处于地面的最小物体宽度？**

**错误**解答过程：
如题，查表，得人眼的最小辨角约为 1'（1/60 度），则 l_min=h tan 1'=h_0*n*1'=2.91*10^(-4)*h_0*n。
如取 h=5000，则 l_min=**1.45m** ，此时显然，巨大娘**观察不到**小人。

**正确**解答过程：
由光的衍射得，光学仪器的最小辨角 θ=1.22*λ/D。设放大前最小辨角为 θ_0，则放大后的为 θ=θ_0/n。
因而，l_min=θ*h=θ_0/n*h_0*n=θ_0*h_0
取 h_0=1.6，得 l_min=0.00047m=**4.7mm**

由于 l_min 是一个与放大倍率 n 无关的数，因而无论放大倍率 **n 有多大**，之**总能**观察到于其身上的**大于 5mm** 的物体。
