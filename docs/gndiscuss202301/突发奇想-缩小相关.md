# 1

**“小即是重，大即是轻”**
我们想象空间中有一个**方形盒子**，这个盒子里面有**若干个光子**，显然，光子的波长要**小于**盒子的边长，不然会因为衍射而莫名其妙跑到盒子的外面。接下来，用一种方式**使得盒子缩小**，缩小到原来光子的波长这么小，这个时候光子会跑出来么？
我们来思考一下，如何使得这个盒子缩小？肯定需要一种作用，也就是**要给盒子一个力**，使得这个盒子**压缩**。虽然盒子壁的质量为零，但由于盒子之中的光子在不停地运动，光子碰到盒子壁反弹**产生压强**，需要一直给盒子外**施加一个力**使得盒子能保持原来的形态，而盒子的缩小意味着这个力向空间中做出了积累，**做了功**，给这个盒子**提供了能量**，而我们都知道，一个光子具有的能量越大，这个光子的波长越短，因而盒子被压缩到原来波长的时候，光子们的波长也在变小。同时，由于光子们的能量增加了，这个盒子的**惯性也随之增加**，也就是说**这个盒子**及其里面的东西**变重了**。
这样缩小的时候可能可以玩出什么新的玩法，那就留给大家自己想象了。
