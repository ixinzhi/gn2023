# 5

无论残酷系温柔系其实都非常好写出来，尤其是残酷系也常常幻想于大家脑海中

但是一旦真正写实文章就有不少难点

1。宏观的背景环境衬托出GTS应有的能力

当然可以一句话（拿出缩小射线/天空一道闪雷劈下。。。）代过能力而完全忽视背景环境，短篇固然可以这样，但长篇难免让人觉得不协调，有着一种宏观背景也可以使文章显得很有深邃感

2。残酷系的性格

实话残酷系的性格有违人之常情，较难把握。在没完善性格描绘之前刻意表现内心那种幻想就越易暴露出做作之嫌，轻则让人觉得很假，重则让人觉得恶心。

《one piece》没看过的人都觉得不是一般的幼稚，看过的人都觉得是部神作，性格刻画的深浅决定了成败
