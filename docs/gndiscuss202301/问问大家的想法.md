# 1

    14日《女王》上线，15日已经被偷去一个叫烽火书城的论坛卖钱了，那个网站充值比例1：10，十元起充，显示会员四万多人，黑心网站赚了多少自己算，我《玩物》卖杯咖啡钱还多少位跟我这说道，甚至教育我作品就应该让人白嫖等等

    那么问题来了，往后我再发作品，不论长短篇，发了就肯定会被偷走卖钱，我等于在给那些个B网站提供素材，让他们大把大把赚钱，我不是他们的爹，没这义务，《女王》还加了阅读权限就这么快被偷了，往后总不能一直加高权限吧，这不是办法，而且对于那些贼来说，加了权限也总会有办法，就像锁，永远是锁得住君子锁不住小人

    我之前说过，GN是为数不多的一块净土，我想把好作品发在这里呈现给大家，但发了就被偷走让那些B人赚钱搁谁心里也不会平衡，而版权问题普遍且永久性的存在于我们生活的这片大地上，正经出书写歌出剧本的尚解决不了这个问题，更别提咱们这个特殊嗜好的小圈子了

    所以问下大家，我并不是没有想法，最简单收手不做了，对于有品位有素质有良知的同好来说，可能是挺遗憾失去一位原创作者，但对于那些贼来说不会有丝毫影响，你不出了偷别人的去，无所谓的。虽然这种事让人很恶心，但目前还不至于影响到我的创作热情，我会继续创作，只是再这么发了就被偷肯定不行，怎么办，我想听听大家的想法。那些要劝我释然且应该继续发作品让人随意白嫖的请闭嘴并GUN蛋，你们就是国内这圈子发展不好的一部分原因。

盜圖固然可惡，但賊人逍遙法外無法可管也是事實。
外在環境無法改變的情況下，建議樓主先想想自己創作的定位在哪裡，有幾條思路供您參考：

1.如果是希望在GN免費發布，又不希望別人拿去營利，就再調高閱讀權限吧，或是把重要的幾張圖改成付費內容，等到一段時間後再降低權限或開放免費。此作法雖然不能根除盜圖，但至少GN老用戶能先享用資源，也達到樓主分享目的。

2.如果1的作法不甚滿意，那麼可以改用更極端一點的手段，將重要的幾張圖加密處理，具體做法可以上傳網盤或是使用加密壓縮包，通過樓主您審核的用戶才給密碼。審核的方式可依該用戶在您之前創作(例如《女王》)的心得評論決定是否給予。

3.樹大必有枯枝，眼不見為淨，與其花時間精力去跟那些惡劣的盜圖者鬥智，不如還是專注在自己有興趣的事上，若是這種情況讓自己受挫，就慢慢調整創作的投入量，想辦法讓自己適應吧。俗話說替別人生氣就是懲罰自己，最好的懲罰就是不理對方，只要想著自己依然有許多良好的支持者在期盼樓主創作，那些花錢去買免費資源的都是蠢貨，或許就能稍微釋懷吧。

不論如何，想清楚自己為何創作，為誰創作，慢慢調適過來就會好些了。
改變大環境很難，只能想辦法適應；改變人的思想很難，只能期盼對方自己想通。 <title>10</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

因為我覺得樓主的文跟底下回文有蠻多議題的，那我覺得我也分幾個部分來分別說看法好了

1.首先是盜版的問題，盜版這東西就是防不住的，這種私人作品要跟盜版那些人認真真的太累了，舉個例子，minecraft 正版一款就900NT左右，就還是有人搞盜版伺服器還收一堆錢，然後就還是有一群傻嗨花了幾萬NT去玩那個盜版，這能說甚麼？我覺得如果是自己想做作品與同好分享，那就注意同好這區就好了，不要在意那群缺德跟弱智

2.再來是作品收費問題，我是絕對支持作品根據作者意願可以收取費用的，你花時間的東西，除非你公開不然沒人有資格可以說你只能免費分享(盜版那群就不提了)，至於品質是否達到那費用的標準就讓大家自由判斷就好了，時間會證明一切

收費與否，都無法杜絕盜版的傳播。

這些事情的解決方法，只能從社會對虛擬內容的付費意願程度下手，其他方法都無法治本。
大中華區，起碼以人口最多的大陸區域來說，目前更熱衷於IAP的模式，而非一次性付費，但這和本話題無關。
