# 7

拿著湯匙打在焦糖雞蛋布丁上，可以聽到啪啪啪 (X) 啪咑啪咑的聲音

不過撞到柔軟的東西然後又彈起來，通常是不太會發出什麼很明顯的聲音，大多都是用誇飾 + 腦補出來的
用注音的「ㄉㄨㄞ」（轉成拼音是 duai）或是英、日文的狀聲詞，會比較有感覺
硬要用中文字的話，拿鼓聲來比感覺比較適合，像是撲咚、撲通、咚咚、砰砰，雖然都有點怪 <title>8</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

*本帖最後由 琉璃 於 2018-8-19 09:02 編輯*

如果你会其他的语言也可以从其他语言汉化而来，比如说我就经常用日语的叹词。包括一些自创词比如说 噼哟（piu）啊嘞嘛（あれま）当然尽可能是吧读音注明，因为有些音因为方言的问题并未有完全相同的读音，比如说pia飞的pia，公认比较多的是（口扁）虽然现在打不出来来，你也可以自己翻阅词典，推荐康熙字典里面有很多生词 。比如我自己创的啊唡唡（a lia lia ）根据现代汉语读音是（a liang liang） 但因为俩这个字可以使用（lia或liang）所以我私自借代唡字也可以读（lia），如果是凭喜好写问完全可以自定自己的写文习惯依旧专属语气词 我是这样认为的。
