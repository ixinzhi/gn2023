# 1

跑團是很久很久之前就接觸的遊戲了，這幾年斷斷續續的跑了很多團，不過今天終於忍不住了，各位寫文章的作者，開團吧？
—————我是瞭解到各位對“跑團”兩字聽不懂而出現的分割線————————

首先說一下，什麽叫跑團，跑團說簡單點，就是幾個人在一起YY。幾個人在一個規則的制定者的制定的世界中冒險，生存，來滿足在電影，遊戲，漫畫等等中無法得到的滿足感。
比如說，我想痛扁一頓霸天虎，保護一下地球的和平。遊戲的制定者會利用一個規則（通常是DND規則），創造一個世界：一望無際的平原上立著兩個人：一個我，一個霸天虎。然後世界的創造者，簡稱DM，會問我：現在霸天虎氣勢洶洶的站在你面前，你要做什麽呢？
我：拿住朗基努斯槍直接扔過去。
DM：CUT
這個時候，那套規則就開始發揮作用了，那套規則可以判斷我可不可以投擲朗基努斯槍，而且有一個“比較明確”的判定來判斷這槍能造成多少傷害，可不可以打到霸天虎。
DM翻閱規則后，發現準確無誤，於是描述：朗基努斯槍把霸天虎擊飛出了銀河系，你保護了地球的和平，THE END。

簡單的說，跑團就是這樣，不過在世界的設定方面，DM要考慮到很多很多的地方，及時應變能力當然也要非常強的了。越是強力的DM便可以製造出強力的劇情，不過在複雜的構造，最終也要通過DM的語言來描述，所以，擅長文字的人，會比較適合當DM哦。
![](img/ec942b6b7db075eaad5da7a4ad136d51.png) 說了這麼多，步入正題，既然是構造世界，爲什麽不製造一個GTS的世界呢？對於放大和縮小這樣的設定，在DND規則早就具備了。讓我們看文的時候 不必再為主角不怎麼怎麼樣而擔心，自己就是主角，可以再這個世界的規則允許之下做任何的事情，代入感也比較強一些。

其實在我剛剛接觸跑團的時候，就想把跑團引入GTS中，不過，跑團不是一件簡單的事情，害怕很多人看到就放棄了，100個看帖的，可能只有10個看看DND，這其中可以還只有一個人原因嘗試跑團。但是我如果不說，不是一個也沒有了么？

![](img/786678b1db56b6b43edf881e00dc8dff.png)  但大怎麼能說逃跑啊，進入挽留的解釋模式。

我來解釋幾個屬於就明白了。
DM：地下城主的意思，其實就是遊戲的主持人。m是master的意思。
PC：就是玩家。
TRPG：RPG大家都知道，就是角色扮演，TRPG就是桌上角色扮演的意思。
D6 D10 D20：簡單的說就是骰子，比如我要跳高，在現實就這麼跳，但是在TRPG中，對於有難度有技巧的東西，必須要投擲骰子，D6什麽都是骰子的類型，D6就是6面的骰子，最大是6，D10就是10面，以此類推。 DM決定一個數值，達到或超過DM決定的數值，自然成功，反之就是失敗。骰子是爭端的最好辦法，要不然PC回同DM因為成功與失敗問題爭論不休的。不同的規則喜歡用不同的骰子，比如DND就喜歡用D20的。
