# 6

这个不能一概而论。
"看过"之类的回复 究竟对不对，还要看具体情况。

如果回复者仅仅是因为 自己以前在其他地方看过，就随意发出“看过”这种容易打击发帖者积极性的回复。那当然不对，这是对自身言语不负责任的行为，应该批评。
你看过不代表别人看过啊，非要说出来打击别人干什么？

但是，如果 问题来自于帖子主题本身，比如，出口转内销、火星帖等，情况就不一样了。
因为无论是谁，遇到这种贴的感受都只会是“看过”，很难去苛求别人还能表达出其他什么感受之类的。

呃，老實說我跟那個人（帳號三個中文字的對吧？當然也還是有別人作這種事啦，但他的口氣特別不好對吧）
算是有長期保持接觸的，他的個性大概就是直了一點沒錯。
不過他也不是故意要用那種語氣留言的，實在是因為他先天上有不及普通人的一些缺陷……
我不是很想講他這方面的問題，但以後大家若看到他的留言而覺得不妥，是可以去提醒他的。
他不是那種不會學習的人。

OK，回到正題上來。
發一些很多老人已經看過的資源，這種事情其實也沒什麼大不了的。
有人跟著大家在網路GTS社群上打滾七年，也有人才七天。
這兩者的資訊質量是不可能相比的。後者見了感動萬分而貼上來的東西，老人卻可能已經看到不想再看了。
遇上這種情況，作為老人的如果想表達意見，
最好的方法是告訴他更多有關這些資源的訊息，例如作者是誰、可以上哪找到更多……等等，
相信新人會非常感激你的，而且這也絕對比留下一句「早就看過了啦」的回覆更具意義。

如果只想留言抱怨已經看過，那真的不如不要回覆的好。
