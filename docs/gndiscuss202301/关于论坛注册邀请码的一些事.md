# 1

不久前论坛刚刚开始邀请码注册。经常潜水，上线也只是看文发文的我一直都没留意这件事。后来知道这事也是在加了一个QQ的gtsvore交流群和群友聊天才知道。
当时那个群友希望能买一个邀请码进行注册，我想着能为gn这个大家庭多招个新人，叮嘱他遵守论坛规则后，就花了自己积攒不多的积分送了他一个。（万一是个vore创作者）
但直到不久前，我看完了管理发的关于邀请码注册帖子的信息，我才知道论坛此举的目的。
老实说，我是支持这项举措的，因为能有效防止不少灌水搞事的人。但这2000积分的邀请码是不是门槛太高了。如果一个新人想入站，普通用户想要帮忙，就我自己这个开个漫画徽章都会肉疼的萌新来说（万年银卡）这就是一比不小的开支，不知道写上几篇渣文才能补回邀请一个人花费的积分。（就现在攒的这点积分其中不少还是大佬们打点的）只是一个人还可以，如果人数多了，花完积分想要再攒就不知道需要多久了。
虽然这些积分对高级用户和管理们来说都不是事，但对我们大多数普通用户来说，这也太难了吧。
不知道在这段特殊时间，这个门槛能不能尽量降低一些。虽然论坛大佬巨佬云集，不缺资源，但本来没有多少新鲜的血液注入了（墙），再加上这雪上加霜的邀请码积分数量，这不就很不妙了吗...
