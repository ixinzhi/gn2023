# 11

Gn确实存在过一段非常红火的黄金年代，
那个时候大佬横行，人才辈出。
几乎每天刷新帖子都会有惊喜，同时因为管理严格，连无意义的灌水都会被严肃处理，那时候的gn非常的纯净，环境也很好。
但是后来确实发生了很多各种各样的事情，
一度因为经费问题面临关停，后来因为猫大慷慨解囊才度过危机
后来随着政策原因，相关产业收的越来越紧，被打压也很严重
再加上当时为了保护创作者的利益，gn重拳打击资源搬运行为得罪了一部分人
后来gn被墙了，隔壁一些收费论坛也因为利益关系资源也更加丰富，所以开始迅速发展起来，这也加速了gn人员的流失
随着时间的推移，比较早一批的大神也开始有了自己的家庭和事业，渐渐淡出了创作的圈子，同时为了防止政策上的风险，很多佳作都在那个时期被删除了，其中的很多佳作现在都已经失传了，外部论坛都没有搬运版本的。
再往后，一直被重拳打击（连qq群都被炸了），导致人员很零散，用爱发电总是有限的把。。。。 <title>12</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
