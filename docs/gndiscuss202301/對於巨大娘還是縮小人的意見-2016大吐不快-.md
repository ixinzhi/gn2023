# 1

超級討厭縮小人啊rrrrrrrrrrrrr
沒錯
我超級超級討厭有關縮小的作品
每次看到都不禁白眼這裡是Giantess night 還是 Miniguy night

尤其是對於我本身讀了這麼多的作品
縮小人的作品不外乎是小人第一人稱
而且對於創作來說
小人的要求太多了阿!!!!!
巨大娘就是要享受她的力量
無人可以約束
沒有人拿她有辦法
為甚麼劇情主軸要繞著小人轉啊!!!

而且縮小人完全沒辦法體會巨大娘的破壞力好嗎!!!!!
我當初就是憧憬那份力量才會對這類的作品如此著迷
但是卻看到了這麼多小人的作品實在是..........(翻桌

雖然我並不要求要巨大娘的第一人稱
即使是第三人稱我也覺得還可以接受
畢竟巨大娘她依然有本身的破壞力存在
那種任意宰割的心態

但是縮小完全沒有阿
只能殺人而已
一些場景物建完全不能破壞阿!!

退一百步說好了
即使巨大的微小是一體兩面的相對情況
但是就物理上來說
越大能造成的物理效果越巨大這是不可否認的
而單純只是縮小人的話
是無法造成大量震動的物理效果的

我是不知道男性的想法到底是甚麼
可能是基於M屬性+上足控等等的因素
但是我不明白為何巨大娘的作品如此少見
就真的是因為自己不是主角嗎?

還是說無法從巨大娘的視角出發
我真的不懂

以上就是2016的大吐不快
順便預告一下
大概過年的時候吧
我會再更一下八呎大人的故事
但是實在沒什麼梗
可能就把後面鋪完
再把前面改精緻吧
