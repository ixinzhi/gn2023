# 1

*本文章最後由 karlchen0920 於 2014-1-22 12:24 編輯*

奖励点权限的新政出来也有两个多月了。
我想新政的主要目的，是在于鼓励“评论”，让GN“论坛”发挥应有的作用——交流。
至于创作，说实话GN的创作者够多了……甚至可以说，比评论者还要多（当然来一发就走的“创作者”也大有人在）。
所以以某K的视角，新政没有起到应有的效果。

当然，某K就是潜水过来的人，应该算是理解沉默时的心态。所以即使自己的作品无人问津也勉强可以淡定（另一方面是没有自信的缘故）。
但即使如此我也深信。**并不是人人都能够当创作者，但是只要学会思考和表达，人人都能当评论者。**论坛里总是有一批默默无闻的人，内心却是渴望表达的。

**所以，这篇教程的对象，是那些希望给论坛做贡献，却又苦于找不到方法的朋友。**
这篇教程的目的，是创造一个基本的思维和写作模板，方便**愿意思考但**缺乏语言功底的的会员表达自己的观点**。**
也因此，我把丑话说到前面：如果你只是想找个新方法骗奖励点的话，那么请赶紧点击右上角的小叉。

**一，基本原理和准备工作。** 
**１.1，选取对象**
评论的本质，其实并不是“评价”，而是表达。
评论的过程，也并非只是写或说，更是思考的过程。
**所以在写评论之前，你必须是能被评论的对象所触动的。**
这种触动，可以是愉悦也可以是伤感，可以是激动也可以是平静。可以是赞成也可以是反对……总之如果你是被迫找一个自己不熟悉的作品来写评论的话，几乎必然是废话+干巴巴的。
（当然，作品和思考能力是双向互动的。如果你的思考和敏锐，那么任何作品都可以找到评论点。）

所以，首先请选择真正让自己有评论冲动的作品。

**１.2，情感与逻辑**
语言是表达思想的工具。善用语言的人，必然也是善于逻辑的。（这一点可以看出高考作文是多么的失败）
在找到让自己有“冲动”的作品之后，你需要做的是将“情感”转化成逻辑。

“存在即有因”。如果你被一个作品触动，肯定是有原因的。
将这个原因发掘并整理出来，就是思考。

如果你"喜欢"，那么究竟是为何喜欢？因为让你性兴奋了？因为让你感同身受，还是因为让你对某种事物有了新的看法？
如果你"讨厌"，那么究竟是在讨厌什么?因为生理上的不良反应？因为触到了自己的痛处，还是因为破坏了自己的某种观念？

甚至，如果你只是觉得“平淡”，也还是可以找到原因的。这也是那些书评和影评“高手”为什么无论对任何座屏能洋洋洒洒写出理由123来的原因。因为思考。

所以，当你真正试图思考“为什么”的时候，就已经得到评论的契机了。

**二，评论的具****体化**
正如之前所说，评论的过程就是思考过程。
具体如何思考，每个人的习惯都是不一样的。有的人习惯先站立场再说话，有的人习惯由外到内剥洋葱，最后才得出结论。
所以这里只是分享一下我个人的手法。以评论一篇文章为例：

**２.1，表态**
我对这篇作品的看法是：非常喜欢？喜欢？一般？不喜欢？厌恶？
————————
**↓****↓****↓****↓****↓****↓****↓****↓****具体化**
————————
***你的“感觉”是怎样的程度？***
【这篇文章是我见过的**最好**（程度）的**福利文**（范围）了！】
——在具体范围内，以客观好→坏进行“垂直”的评价。比较的对象是其他文章。
【这篇文章的**构思**（范围）真的是**别出心裁**（评价），让我**大开眼界**（评价）。】
——在具体的某个方面，对特征进行主观的“水平”的评价。比较的对象是自己的标准。

评论是为了表达自己想法，表达是为了让别人能听到。所以在一开始就表态是最方便让人理解的方式。
事实上，无论是文章还是段落，把关键部位放在最开头都是良好的习惯。没有什么花哨的地方，读者一目了然，你的写作就达到目的了。

**２.2，论述逻辑／理由**
我已经表态了，那么为了让对方信服，必须列举理由。

**注意：即使是本意完全为了赞美，或者完全为了吐槽，你的评价也必须包涵“正面”和“反面”。**
道理很简单，你的逻辑必须是完整的。完整的逻辑必须包含正面和反面才能够让人信服。
————————
**↓****↓****↓****↓****↓****↓****↓****↓****具体化**
————————
***ａ，我认为这篇文章，在那些方面做得最好？** **ｂ，我认为这篇文章，在那些方面做得不尽人意？***

分析文章是一件很复杂的事情。这件事情就算是学术界也没有定论。这里还是某Ｋ以自己的习惯来举例。

**2.2.1, 通常用于论述的切入点可以分为：**
文笔/文风/措辞造句（华丽，朴实，利索，干瘪，粗糙等等）
设定/背景（是否完整合理，哥特，废土，现代，都市，中二，校园等等）
剧情/布局（是否波折丰富，吐槽，推理，惊悚，热血，卖萌，严肃等等）
人物形象（是否塑造丰满，三无，御姐，废柴，吐槽狂，女王受等等）
福利/互动（轻重口味，场景，氛围等等）
其他（独特的排版，标题，更新方式等，其他你觉得值得评价的特征等等）

**通过文章各个部分进行评价和描述，给出与“表态”相符合的理由。**
【这篇文章在**设定***（分类）*上真是我的菜！**废土的设定***（描述）*真的做的很完整，而且**找不到一点牵强的地方**（描述）。】
【我尤其喜欢**芙兰这个形象***（分类）*！**哥特萝莉***（描述）*什么最赞啦！尤其是**外表冷艳，内心善良的“表里不一”型***（描述）。*】
【楼主写**践踏**（分类）真是一绝！不仅仅是**互动写的很细腻***（描述）*，而且对GTS那种**超然的气质***（描述）*也渲染的很棒！】
【作者的在**语言****（**分类）上还有待加强。有**大量错别字和病句**（分类），而且**口语化的描述**（分类）让也人很不舒服。】

**2.2.2，用于辅助论述的手法。**
仅仅是评价和描述，不仅仅字数上会受限，而且表述上很苍白。这个时候需要一些技巧来让你的评价显得更加有说服力。
**A，引用。**
直接引用文中的内容，作为自己评价的佐证。例如：

> “在虚拟的世界中寻找真实感的人脑子一定有问题。”

【这句话真实一语道破天机，作者的神吐槽表现的淋漓尽致。】

**B，对比。**
对比是非常有说服力的一种技巧。不仅仅可以与同类作品横向对比，还可以与自己标准，理解纵向对比。例如：
（注意！此操作有一定的危险性，请谨慎使用）

【我一直以为**剑爷的文章**够热血了，没想到此文也是如此的激情啊！】
——横向对比。（剑爷磨刀霍霍）
【这方面的感觉还是欠缺了些。作者可以参考一下D先生吐槽文，或许会在吐槽方面有所领悟吧。】
——横向对比。
【说实话，原本我对哥特风格无感的……但是本文开始让我重新认识哥特风格了。】
——纵向对比。
【我心目中的GTS，不仅仅肉体上强大，精神上也要有超然的气场。作者对前者的描述非常细腻，但是于后者却没有太多建树。】
——纵向对比。
【我本是残酷系爱好者，最爱但大那种严峻冷艳的福利。而本文作者却用温存的互动打动了我。】
——综合对比。

**C，举例/假设/建议**
这种手法通常用于论述反面的观点。
**而且在很多时候是必须的。**一方面是出于礼貌，另一方面也是评论的核心功能之一——交流建议。

假设和建议是相对复杂的论述，但也是成为优秀评论者的必须掌握的方式。如果不能设身处地的在作者角度思考问题，那么你的评论不过是自己一厢情愿的偏执罢了。

例如：
【作者在这段场景上似乎花费了太多笔墨。在我看来，这场战斗无论对剧情推进还是人物塑造都没有太多价值。我的建议是尽快将其结束，尽快进入两人的感情互动。】
——对剧情布局的批评和建议。
【楼主非常执着于描述虐杀的技巧，实际上GTS的性情也是福利的一环。如果能尝试将人物塑造的更加复杂，比如描述一下强大GTS脆弱的黑历史？我相信故事会显得更加丰富圆满。】
——对于人物性格塑造的建议。
【第三章里对于设定的描述非常简略。或许可以理解为作者并不想直接插入硬邦邦的设定解说？那样的话，将A设定融入男女主角的对话或许是个好办法，还有增加情感互动的效果。】
——对于设定的建议。

**注意：**
**a,通常情况下，批评建议最好是能在“反面观点”之后，集中形成一个独立的部分，以此来突出其重要性和独立性。**
**b,批评建议最好是具体的，可操作的。例如“增加XXX戏份”，“在某个时候稍微加入写严肃风格”等，而不是“要更努力地塑造人物形象”这种笼统的目标。**

**２.3，检查，整理与总结。**
如果你能够认真思考并论述，那么恭喜你你的评论已经基本上要完成了。
接下来要做的事情，是让你的论述显得更加清晰易懂。

**2.3.1 检查**
检查的内容主要是：
a，表述错误。例如如错字漏字，不完整的句子等等。
b,  论述是否完全。是否还有想到但是忘记写的东西。
c,  论述是否有矛盾，硬伤。例如前面赞扬了作者的剧情，后面缺又批评了剧情。再例如错误的引用等等。

**2.3.2 整理**
要铭记，你的评论是要给别人看的。所以为了让文字更清晰易懂，任何组织排版上的技巧都是非常有价值的。
这其中主要是一些零碎的技巧。
a，将你的观点尽量分类，整理并编号。例如第一层【1，优点】，【2，缺点】，【第一条优点1.1】，【第二条优点1.2】等等。
b，规则的排版，尤其是空行。所有文字全部挤在一段肯定不行，胡乱空行写成梨花体也会弄巧成拙。最好有自己的一套标准。例如连续段落之间不空行，不连续的段落则空行示意等等。
c，善于利用各种标记工具，强调自己重要的论点。调整字体大小和颜色是最简单的方式，各种分割线，以及制表符也推荐使用。
d,  除了文字方式，非文字的方式也可以表意。最常见的是图片和表情（不过需要慎用）。如果有兴趣，表格，mind map之类的高端货色也可以尝试。

**2.3.3 总结**
总结不是必须的，但是如果有余力的话，总结会极有效地清晰你的表达。
总结最大的用处就是避免冗长论述中可能产生的误解。一段简单易懂的总结可以弥补你论述中存在的缺失。
同样，写总结的过程也是对自己思路的整理。算是二次检查。

如果你完成了前面的内容，总结其实并不难写。因为它几乎就是你整个评论的微缩版本：

（整体）总而言之，我非常___作者的这篇文章！
（优点）这篇文章的____，_____，_____给我留下了____的印象。最让我欣喜的是这篇文章的_____，______。
（弱点）同时，以我个人的观点，本文在____,______,_____方面还有改进的余地。尤其是_____，_______。
（建议）我个人的建议是______,________。或许可以让本文______。
（客套+祝福+催更）无论如何，这篇文章都是不可多得的_____，也可以看出作者的____, ______。希望能尽快看到下文！

**（注意：实际操作中请不要套用此模板，否则会有被判作弊得零分的可能。）**

**三，评论的节操**

**3.1,礼貌**
首先学会做人，然后再去做男人/女人/学生/职员/领导，然后再学会做评论者。
**礼貌并不仅仅指的是语言上的谨慎，更重要的是发自内心的尊重。**
蛮横，自我，固执的不成熟者，即使一句脏话也没有，也是会遭人唾弃的。在二次元里，你没有任何资格看不起别人。
所以，无论发生什么，先尝试倾听他人的想法，尝试站在他人的角度思考，然后再做出决定，**这是礼貌的基本原则。**

**3.2,题外话**
虽然偶尔的客套，打趣和吐槽可以让评论更有趣。但是请谨记：
作为评论者，你的目的是表达自己的思想，其载体是是作品和产出的评论。因此，任何无关紧要的话题都请尽量限制篇幅。
拿几个错别字或者计算错误涮上几百字并不是什么有趣的事情。好几个段落的神棍式膜拜也不会让作者感激你。
**你要做的，就是评论文章，鼓励和建议作者。其他事情最多是锦上添花，更多是画蛇添足。**

**3.3,心态**
评论的用途是和作者交流。作为评论者，唯一所求的就是作者的回应——直接的回复，或是采纳你的批评建议。
如果你指望用评论来获取其他的东西（你们知道我指的的是什么），那么你还是放弃吧。GN不适合你。
**不真诚的评论，一眼就看得出来。自欺欺人是没用的。**

另外，不是所有人都能当圣母，如果作者给出相当消极的回应，你完全可以有自己的处置方式（你们懂）。只要——只要心态摆正就好。

**四，总结**

凌晨四点……是时候给我自己总结一下了。

某K是去年2012年四月份开始脱水并尝试写文的。说来惭愧，断断续续坚持到现在也没完成什么有价值的东西……
不过，我现在还会时不时地将当时但大，十八爷等人给我的评论拿出来翻看。
回想起来，没有他们的鼓励，我可能连两个月都坚持不下来，很快又会滚回去潜水了吧。
我从未自视能写出什么好东西来过。只是一直很珍重这种幸运的“共鸣”。或许正是人类群居的本性？
所以看到论坛的现状，看到不少新人因为没有人鼓励而浅尝辄止，看到老人因为水贴泛滥而愤怒和放弃，我也只能说尽量（装作？）热情满满的样子，尽量（装作？）把这种“共鸣”传递给别人吧？

至少我自己依旧是会为有人能评价我的文字而欣喜的。所以我也坚持认为，自己花两个小时洋洋洒洒写篇长评，也可以让别人欣喜。

只可惜这种欣喜在整个论坛都越来越少了。

我相信肯定还有像当年的我一样的人，只是他们缺乏那幸运的机缘而已。
所以，鉴于我的水平连“写文教程”的门都进不了，于是只能废话一通写了这个教程。希望能给他们有所启发。
