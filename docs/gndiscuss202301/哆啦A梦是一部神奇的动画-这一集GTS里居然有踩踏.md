# 4

其實舊版的GTS幅度更大.像是模型小汽車那段最後是靜香裙底風光被大雄看光光.然後一腳把大雄開的迷你車踩爛
其他還有搭縮小的潛水艇結果傳送到靜香家浴缸(剛好他在洗澡).進入靜香胃裡拿不小心吞下鑽石的體內大冒險也是經典
還有一集是大雄想要大螢幕電視.結果那台電視是可以進入的.進入後巨大化的偶像踩著不小心掉進來的大雄

感覺以前這種健全向動畫.幅度大所以會有蠻多女角的福利畫面.但新版的哆啦A夢即使難得有GTS相關的集數
福利擔當的靜香總是一起縮小冒險.然後整集就是隔鞋搔癢.摸不到我們這些人想看的地方![](img/27533062db6ce4e24c3784cc8474f5af.png)
