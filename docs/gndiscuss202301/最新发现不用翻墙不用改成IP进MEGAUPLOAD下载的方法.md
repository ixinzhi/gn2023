下面把方法发出来：
  众所周知，因为某些原因，MEGAUPLOAD这个网盘已经打不开了，我们不得不把网址改成IP或者借助一些软件才能进入，现在这个问题已经解决！

方法：

只要在?和d中间加一个“&p=1&”就可以下载了

例如以下这个链接：

[http://www.megaupload.com/?d=KBZVR567](http://www.megaupload.com/?d=KBZVR567)

正常是肯定开不了的

但是，把它改成[http://www.megaupload.com/?&p=1&d=KBZVR567](http://www.megaupload.com/?&p=1&d=KBZVR567)就能安全进入，无需任何手段！

还在用IE6的朋友去试一下吧！
