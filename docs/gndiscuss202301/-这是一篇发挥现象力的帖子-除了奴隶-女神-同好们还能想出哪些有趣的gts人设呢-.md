# 3

可以，将小人看成百姓，把巨大娘看成国家。
因，国家的意识是统治者意志的集中体现，当统治者的数量趋向无限大时（适用范围：非独裁），集中出的群体意志——国家意识必然是绝对理性绝对残忍的，即在能使自己取得最大利益的前提下，尽量减少对手的数量。
这样来看，小人也是有价值的。可能，其相对于巨大娘来说，是小量，但小量与大数加和并不一定能被忽略。比如 e=(1+dx)^(1/dx)，这里 1+dx 明显就不能省略 dx 这个小量，否则 e 就等于 1 而非 2.707多。
至于发福利…这楼主自己看着办吧。毕竟文章中所有的情节都是为了主旨服务的，对于“凭空发福利”，本人向来十分抵触。并且为了塑造人物，突出矛盾突出主题反应问题又富有文学色彩的福利本来就是为大众所接受的，是可以放到台面上来而不需要藏着独自观赏的。
额，偏题了，说回来。前文所述设定是从“意识”角度讨论，对于“物质”方面，对到个体上，也就是生产力，个人与国家的对应关系是非常弱的，因为照理来说，国家代表的生产力是从个人生产力中抽离出来的（税或者啥啥啥的），也就是说个人们的生产力和要大于国家的，与巨大娘可以吊打小人明显不符，目前尚未找到合理的解决方法…
