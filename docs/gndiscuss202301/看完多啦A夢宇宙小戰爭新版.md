# 1

看完新版的宇宙小戰爭，可能以前GTS啟蒙在於舊版宇宙小戰爭靜香巨大化那時候的場景
這次感覺沒那麼震撼...

個人認為差異在於新版在靜香巨大化時是用撐破戰車後出現，舊版是先遊出去後才出現，塑造了大小差異化
戰車在海底的那一幕其實有點像，但感覺新版節奏比較快，以及新舊版靜香姿態描述比較不同，在這幕戰車到海底，靜香出現
新版顯得頭大身小，比較符合兒童身材
舊版顯得比較像身材較好（笑）????

出現港口的差異在於
舊版我是蠻喜歡靜香先在港口海浪衝散船舶出現，然後雙腳踏上港口
新版是靜香跟小夫一起出現，接下來靜香一腳踏上港口就換畫面了，顯得節奏太快

但新版值得稱讚是靜香巨大化那時頭手拋出海上時，這點比較有特攝片感覺

可能新版製作上注重傷及無辜，所以太花時間強調無人戰隊

新版其他劇情內容就不加以描述了
只能說經典不好超越????
舊版畫面精緻化就很讚了????
