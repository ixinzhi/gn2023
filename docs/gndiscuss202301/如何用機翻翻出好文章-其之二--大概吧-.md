# 1

接續[that123](http://giantessnight.com/gnforum2012/forum.php?mod=viewthread&tid=19241)之前的文章我想這位大姊過去已經分享了大部分的機翻技巧，所以這邊只稍微提一下我個人覺得特別需要注意的部分。1\. 將句子拆解成字詞，不要整段話直接丟機翻。知道如何將一連串的句子拆解成個別的字詞是很重要的。句子拆解得越零散，機翻時出錯的機會也越小。同時也比較容易從前後文判斷出該怎麼翻譯才最為恰當。以『黒いストッキングをはいた足で履く厚み２ｃｍほどの白いサンダルを振りかざした』為例。這段話依序可以切割成『黒いストッキングを』、『はいた足で履く』、『厚み２ｃｍほどの』、『白いサンダルを』以及『振りかざした』。丟進百度翻譯後分別得到『黑色丝袜』、『用脚穿的脚穿』、『厚度约2厘米』、『白色的凉鞋』、『挥舞着』等結果。再來只要把凌亂的字詞重新組合成符合中文語法的編排方式即可，及『用穿黑色絲襪的腳揮舞約2厘米厚的白色涼鞋』。2\. 習慣日文語句的慣用表達方式。比方說『片假名+と+動詞』，通常代表在做某個動作的同時發出什麼聲音。例如『ザブザブ(形容聲音)と(地)波立て(濺起水花)』即意味著『在做出濺起水花的這個動作同時發出ザブザブ的聲音』。又好比『体育座』，中文裡原本並沒有這樣子說法，google後才會知道『体育座』其實是指某種容易讓女孩子春光外洩的坐姿。再者，『ショート』雖然表示短(英文Short)，但在日文中通常指的是『頭髮短』而非『身材矮小』。 3\. 一些常用字詞能記則記。機翻的次數久了，總有一部份的翻譯會自然而然地印在腦海裡。尤其一些經常出現的字詞更應該如此。比方說『ように=像』、『まるで=彷彿、宛若』。這兩個幾乎是所有日系巨大娘文章都會出現的字眼，要說完全不認識根本就不可能。其他還有像是『れば=如果、的話』、『され=被動語句』等總之留在腦袋裡的日文的慣用語越多，翻譯出來的文章就會更通順。同時翻譯的速度也會更快，因為你不再需要花那麼多時間一行一行的轉貼到各個機翻網站上去等結果。 4.採用互相對應的翻譯方式看起來大概會像這樣**全長３００ｍを超える空母も、女子から見れば３０ｃｍほどの大きさでしかない。****全长超过300米的航母，从女生的角度来看只有30厘米左右的大小。**這樣比較容易發現哪一部分的翻譯出了錯，也能避免翻譯時毫無節制地延伸到與原文完全無關的領域。我個人在翻譯的時候，一段句子通常會出現三次甚至更多，第一行日文原文，第二行中文機翻拼貼，第三行中文潤飾修正這麼一來即使沒有辦法百分之百完全符合日文原意，通常也不至於錯得太離譜。

    第二点的话。。。其实日语的拟声词很多都是假的，根本就是副词。之前有翻到用ポチョっ    pochyo表示撞在大腿上。。。为毛撞到软的东西会发出P和Ch爆破音=_=你要拟声好歹是个ぼん、ぼよん什么的吧，从拟声来讲根本意味不明。应该是因为ぽちゃぽちゃ可以表示肉乎乎、有肉感，然后ぽちゃ变成ぽちょ。。。
    总的来说日语这个拟声词和副词的界限很模糊，不应该都当作拟声词理解。毕竟现实中大多声效本来就不是人类用喉咙能模仿的，所以日语很多所谓的拟声词都是约定俗成的口语化副词。那些ABAB的词，如果很难搜到，又很难理解是什么声音，可以试着搜ABと、ABりと、ABっりと、ABっと之类的，看它作为副词到底是什么意思。
