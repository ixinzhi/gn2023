# 1

记得以前看过一篇鸡汤文，标题好像叫《不努力，我们用什么去竞争》，然后大概就是讲男主依靠努力成为公司正式员工的故事。

现在的文章创作难度尚且门槛比图片要低，但是写文的人经常就会为各种描写而苦恼，可以说写文章简单，而写好文章难，维持对好文章的更新就更难，为此我决定向各位学习各位大佬描写的看家本领，加强自己的描写水准。

我越来越觉得自己写的东西很尴尬，描写手法换来换去就那么几种，若不是雪风大佬接济过我，恐怕我早就江郎才尽了吧。
