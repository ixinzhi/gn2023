# 10

剧情文的关键点在于要素和剧情的统一，比如我想写这个剧情，为什么要以巨大娘作为主导，为什么一定要强调这个有巨大娘存在的世界。
相较于巨大娘或成长系世界观，很明显缩小系会更容易塑造剧情而不会产生太强的割裂感，就好像实验那一派的作品，以人物的情感作为剧情推进的线索，可以创作出非常吸引人的福利和剧情。
但如果你说的跌宕起伏的剧情就要更难一点，那种剧情所需要的足够的冲突，有冲突才有看点，可巨大娘如果不是无敌的话爽点很明显会下一点档次（对于我这种常看成长文的人来说）
所以综上所述，楼主如果想写出剧情文的话，要协调好要素和剧情之间的比例，不然很容易变成挂羊头卖狗肉，两边都不讨好，我好像以前看过类似的文章，支持率的确比较惨淡 <title>11</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 11

我个人而言，写剧情文是很痛苦的，一天能写出两千字很好了。不是想到哪写到哪，首先要搭框架，然后要铺线索，还要设置人物形象，还要注意节奏……就和正常写小说一样，更何况还要有gts元素。所以我觉得，一篇好的剧情文创作难度大于普通小说，而多数人没有写完的能力或者毅力，所以数量少很正常。
