# 1

果然像FGO这种文字剧情为主的游戏，拿出来做成动画比游戏本体要好上太多，原本算是预料之中的某巨型女神登场集会特意加重巨大感和威压感，但真的这集一出来后正式登场的威压感真的是意外的足，正脸的细节加了很多，特别是正脸的头发画的特别厚重与细致，把气质给确实地做出来了，意外真的把“随手碾死虫子碾死人类”的感觉给做出来了，估计不少爱好者看的时候相当觉着吃着香吧。

但正因如此我总感觉这集后半集莫名的像个大型传教现场……
