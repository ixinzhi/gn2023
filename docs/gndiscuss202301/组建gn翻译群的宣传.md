# 1

因为最近火山姥爷的寺田汉化群停更，所以我试着联系了论坛中几位翻译寺田图的大佬意图组建一个汉化寺田图片的群。不过在组建的过程中产生了一个想法，论坛一直以来不乏优秀的文章以及图片的翻译者，但是一直没有一个平台供大家讨论，撞车、错译这样的惨剧经常出现，同时一些对翻译内容的纠错只能通过论坛进行具有延时性的沟通。
因此我斗胆拉上长久以来认识的各位论坛中大佬，建成了一个“gn翻译群”，因为我个人的力量还是有限，所以在此发帖希望论坛中众多优秀的参与者以及未来有志于翻译的人都可以进群一起交流，同时也可以相互交换许多作者的赞助图和文，让论坛中各位大佬的能力得以充分发挥，论坛中优质资源也能越来越丰富。
