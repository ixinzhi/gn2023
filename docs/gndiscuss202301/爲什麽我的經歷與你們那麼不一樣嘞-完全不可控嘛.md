# 1

新版入壇已經3、4年了……入圈子也差不多15年了~想想覺得都有點不可思議

小學一年級，去書店挑光盤，當時看到93版 50尺女巨人的封面，抓著一輛車。

和很早以前的怪獸不同，我就覺得女怪獸有了智慧，人類會很無助吧（其實那電影漢娜根本沒有發揮）

然後3年我從來沒有過任何一次想法

知道小學5年級，某天晚上居然做了一個夢（完全不可控的好伐！）

夢到我在超市選購，喜歡的女孩子突然掀開了屋頂把我抓走了……

然後就驚醒了，然後就對女巨人這個詞突然產生了興趣。

剛開始是翻來覆去看《50尺女巨人》這部電影，後來就利用搜索引擎
“女巨人”

無意搜索到一個 GEN的影片，天哪，從哪開始我很驚訝居然有這樣的圈子。
後來一遍一遍看，一遍一遍搜索
“高跟鞋女巨人”“女巨人踩踏”“美女巨人”“巨大少女”

再往後，我搜到了但大的一篇Paradox2 番外 七個白雪公主與小矮人（標題記不清了）
是初中時候，給我看得那個舒服！OMG。

緊接著的是加勒比海盜3、哈利波特與火焰杯的連番轟炸

偶然的機會搜索女巨人的翻譯，懂得了GIANTESS這個字

順著就發現了gts吧這個風水寶地（現在已經沒了）

讀了J戀舞和其他大神的很多作品，越來越融入這個圈子了。

順理成章就到了這裡，（始終錯過了女神網站）
高中喜歡PS那一段，經常來這裡發PS圖

我怎麼覺得我到這個圈子充滿了偶然性啊~不過我感覺我這個愛好還是很自豪的，也很痛苦。
因為是個永遠的秘密。
