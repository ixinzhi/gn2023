# 1

某日和十幾個朋友聚餐，某位先前不認識的男性友人對我的畫作很有興趣，想要看看
於是我拿出最近的在畫的素描本，
因為朋友圈內有些熟人知道我的興趣，所以我不會特別把巨大娘的作品另本創作，直到把本子遞給他之後我才想到這一點。
然而來不及挽回，只能臨機應變了。

那位朋友接過畫冊來翻了翻，翻到巨大娘那個區塊(大概是地球儀的那個梗那邊，而且還是彩圖)時頓了一下

友人：「嚯，這就是**巨大娘**嗎?真是少見的屬性啊！」
無某：「嗯，是啊，的確是滿少人...」(OS:WHAT!!兄弟您知道什麼是巨大娘!!)
友人：「不過這一類的尺寸太大，有點難以接受啊..」
無某：「是啊，呵呵...」(冒著冷汗目送友人沒事般的翻到一般向的作品)
(fin)

在場友人(含女性)知道我會畫圖的不多，知道我好這口更少，所以巨大娘這詞無預警冒出來的時候真讓人心臟病發...
好在其他人沒什麼特別的反應，預估也是皺了皺眉的程度，後來也沒人多問
倒是我在白緊張啊!

嘛，也有可能是我的畫風不怎麼重口的關係?
