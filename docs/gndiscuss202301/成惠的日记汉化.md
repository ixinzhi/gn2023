# 3

考虑到这游戏也发布很多年了，我就直接把拆包软件提供给你们吧。下面的内容我直接从多年以前的邮件稍加修改拿来了。

大家可能已经注意到了，游戏的主目录里有很多文件的拓展名末尾是_，比如an_，jpg_之类的。其实这些文件是普通的an，jpg之类的文件被游戏制作者加密了。

附件里的就是用来给这些文件解密的软件，**我在试玩版/完整版的目录里都测试过**。

先说一下**功能**：把一个文件夹里你能看见的（也就是**不包括**子目录）所有拓展名末尾是_解密，并且**删除原来被加密的文件**。可能各位对此会感到疑惑，认为删除原来的文件是毫无必要的。事实上，假如游戏本体发现了face.an和face.an_这两个文件，它便会报错，因为它只允许这两个文件中的一个存在。

下面介绍**如何使用**：
1.首先，这个附件名称是ExtractDoN.exe。
2.请各位操作系统是windows vista以上的用户，把整个游戏的安装目录（比如C:\Program Files (x86)\DiaryOfNarue）复制到其他地方，因为程序无法直接读写C:\Program Files (x86)\下的文件。当然我还是**建议**各位把整个游戏复制一遍，**防止解密器有问题**。
3.把ExtractDoN.exe复制到游戏目录的某个文件夹下面，然后双击运行即可（你应该会看见一个黑框，然后自动退出了）。
4.最后一点，的也很重要：假如你解密了scenario/main.an_文件，请用附件里的config.anprj文件替换游戏主目录里的config.anprj文件。

最后简介一下这个游戏使用的技术：
1.游戏本身使用基于Adobe Air的AirNovel框架做的，Airnovel开发者的博客是[http://famibee.blog38.fc2.com/blog-category-2.html](http://famibee.blog38.fc2.com/blog-category-2.html)。
2.加密和解密用的是RC4算法，密钥在源程序main.cpp里有。
3.其中的an文件是游戏脚本，所有的游戏文本以及各方面的控制也在其中，并且修改后仍能正常运行。
