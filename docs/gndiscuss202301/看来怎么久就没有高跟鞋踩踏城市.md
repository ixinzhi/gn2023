# 4

巨大化+践踏的死忠恐怕最多……但写起来又是最难就是了……![](img/ec942b6b7db075eaad5da7a4ad136d51.png)

比起缩小男，要描绘巨大化践踏城市是最困难的。
因为互动的对象不仅是小人，还包括整个环境。

身高尺寸和力量的拿捏，建筑崩坏的细节，还有践踏的花样…
这些都是现实中不存在而且无从模拟的。需要超常的想象力和对文笔的掌握能力。![](img/4cd144b1e80a429896922ff5e1245b21.png)

更何况是高跟鞋，相比裸足更是缺乏灵活，基本的动作只有踩+碾。
要写出精彩的段落都会很伤脑筋，更何况全篇巨大化……

因此大部分中文系的作品，即使是巨大化的情节，也往往会安置在一个限制的场景中，避免过多的互动…![](img/51c9d25d5c87303019ce0d6f9fff3a3c.png)
中文系这个领域除了但大恐怕无出其右了…质高量足呢！咳咳……![](img/afaf29bab8e0191f31485625eda6c801.png)
