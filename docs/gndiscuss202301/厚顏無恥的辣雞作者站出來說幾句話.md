# 1

各位觀眾老爺先別急，請先把帖子看完。

最近gn的風氣，或者說gts圈子裡的風氣變得很差，伸手黨和白嫖党越來越多，這是很多作者不願意繼續創作的原因之一。試想一下，你的作品被白嫖党一份百份的擴散出去，你的人氣也許上去了，但是你的錢包當然會受到損失。

一些作者的免費公開作品以及他們幸苦創作出來免費供人觀看的作品，甚至會在某些人手中二次專賣出去——這是最糟糕的事情，也是當原作者發現之後，最讓他心寒的事情。我的話，已經在很多地方發現類似情況了，某站的話……不管他。

另外就是帖子下方回復問題，這個問題並不嚴重，因為每個非灌水回復一定都能激勵作者繼續創作。不過，我相信任何作者都是希望他們作品的回復下有建設性評論的，我相信每個大佬都是在罵聲里鑽出來，而不是在掌聲里跑上頒獎台的——當然必要的鼓勵是肯定的，因為不是誰都跟我一樣沒心沒肺的XD。

最後祝gn越來越好，gts圈永遠輝煌，謝謝。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
