Project gpla
Gpla——gtspla——giantess plastic model巨大娘塑料模型        由gandai财团于新世纪之初（2101年）开发并售卖的划时代的新型小尺寸人型塑料模型。素体模型的初始比例为10:1（人类的十分之一），经过拓展后，在现实世界中通常可以达到1:1（等身大），在虚拟空间中则可以拓展到近乎无限大（会受CPU机能限制）。        模型具有精细的骨架和內构，外部由兼具强度和延展性的材料包覆。同时还搭载有相当程度的ai。由于ai是学习型，所以实际上为玩家留出了相当大的调整空间。        Gpla素体制作完成后会被安放在特殊的扭蛋基地中，通贩版扭蛋内含gpla的素体一个，以及对应的基础装备。扭蛋本身可以为gpla提供最基本的住宿、修复、改造、数据备份和专属虚拟空间等功能。        Gpla素体可由玩家自行改造，改造大体可分为两个阶段。第一阶段为启动前改造，即在gpla初次启动前通过扭蛋基地对其进行初始设置，主要涉及ai和初始外观方面的调整。第二阶段为启动后改造，这个阶段中玩家可自由的对模型素体进行各种各样的改造。但必须要指出的是，在这个阶段中，玩家持有的扭蛋基地的权限已经不可以直接重新设置gpla搭载的ai，如果玩家想对ai进行调整，则需要善用学习机能，逐步诱使gpla产生改变。        公司会定期举办gpla联赛，符合参赛条件的gpla可与其搭档（玩家）一同参赛。基础比赛模式分为单打对战（1vs1），双打对战（2vs2），小队对战（3vs3）和多人混战（大乱斗）。Gpla可凭借扭蛋基地将自己的数据上传至云端以参加联赛。每场联赛的胜利方会获得对战点数和评级积分，失败方只会扣除评级积分，对战点数不受影响。对战点数可用来从g财团的官方店换取稀有或强力的限定装备。        玩家可以通过扭蛋基地自带的屏幕观察身处虚拟空间内的gpla，也可以将全意识潜入器连接到扭蛋基地上，从而直接与身处虚拟空间中的gpla互动。        PS：虚拟空间分两种，一种是基于财团服务器的云端版，一种是基于扭蛋基地服务器的本地版。云端版主要用于对战等较为公开的活动，本地版则更为私密。

这一部分我还没整理好所以就没放上来，不过设定上局域网联机是没问题的，玩家通过全意识潜入也是可以参与到对战中的。线上对战的内容……基本就是很正常的机娘对打。但是在本地服务器里……怎么玩就随玩家的便了不是？
然后关于ai的水平，基本可以认为初始状态就是fa-girl的那种等级，而且这种等级的高智能的ai在那个世界里已经比较普遍了，只是以前一直都装在跟人差不多大的机器人身上，比方说机械护士，机械女仆，机械老婆这样的。只是在这么小的人偶上使用还是头一遭。另外gpla在现实世界里可以拓展到人类大小这点上，与其说是拓展，不如说是还原，因为gpla实际上可以说就是上一世代高智能机器人的换代版本，把身体做小的理由则是为了减少作为玩具购置时的成本而已，换个正常款（r18款）的大号身体根本就是分分钟搞定。
另外学习型ai（参考阿童木起源里的a106）的实用化也是头一遭。通过学习机能可以教给她们的则是真正的感情那类的东西，不是初始设定的时候强行模拟出来的那种（老款的非学习型ai都是模拟出来的）。想让自己的gpla更接近真人，是需要自己费时间一点点去教的。
然后失去ai什么的……太残忍了点，所以还是算了-_-||
另外ai是有数据备份的，素体本体、扭蛋基地和云端服务器一式三份，基本可以说是不会丢失，就算是素体因为什么不可抗力被毁了，也能从基地里或者云端服务器里把自己的gpla接回来的……就是又得掏钱买个新素体就是了(›´ω`‹ )

通贩版就是很普通的通贩版，限定版则是……额……骗钱版？比方说……换个色啦～还原tv版的特殊武装啦～附带个除了好看以外没啥卵用的限定武装啦～让人看着就想社保的轻飘飘的限定服装啦什么的。简直赚豹有木有。

然而光是想想就很难下笔了，这剧情给我写的话预计写到首战打完要拖一年……
首先这个gpla不都是用来r18，这就有不同人，而且是不同性别的人对这个gpla的不同态度，可以考虑一上来就介绍这点，也可以考虑时候遇到的时候再介绍，我觉得你是会看重这方面的人，态度不同肯定会引发矛盾冲突推进剧情发展，而且主角若是在获得gpla后遇到不同的人，逐渐改变了自己对她的态度，这部分会越来越难写。
其次，战斗部分，人类和ai如何战斗？这是个大难题，大家都坐在ai身体的同一位置还是随意行动，人和ai是否能远距离通信，是否可以人和ai分离，ai阵亡人是否投降，当多名玩家选择在战斗场地里不进行战斗（比如互相鉴赏对方ai身体）胜负如何，战斗以后奖励是什么，战斗过程中是否能强制下线，可以下线的话有什么后果，诸如此类……
再次，线上战斗，线下就是玩手办的日常吗？主角所处的环境也是比较难处理。
最后，希望真的猛士，开坑敢填。

附上我自己脑补的剧情：少年a得到了gpla，少年明白了操作步骤，他把gpla当做朋友。
少年b和他是朋友，他们一起买的gpla，然后因为年少，两人都捏脸捏的一般般。
少女c是少年a喜欢的对象兼同学，但他不敢说，她也买了gpla，而且是土豪大佬级别的玩家。
少年a和少年b约着上线玩耍，二人ai都还很不成熟，最终连对战都没点开就。
少年a学习好，空闲时间少，但是全都用来和他的gpla处朋友关系，他把gpla当做了少女c的代替品，但又不知不觉和她诉说自己对少女c的暗恋。
少年b学习差，空余时间多，上线基本上就是调教gpla。
少女c无意得知少年a和少年b在玩gpla，于是主动加入进来装萌新，结果带他们飞，拿了首胜。
少女c注意到了少年b的gpla的异常，看穿了他对gpla平常的所作所为，于是尽量避开二人。
少年a注意到了少女c的异样，但他误认为这是少女c瞧不起他们的战斗水平，于是拉上少年b刻苦练习。
少年b在一次战斗中看到少年a对对面的多金大叔调教gpla不耻，于是更想把自己的小秘密藏得更深，和少年a逐渐疏远。
少年a察觉到他们的疏远，内心痛苦找gpla倾诉，gpla表示还有她会一直陪在他左右。
另一方面，他们面临毕业，少女c因为一次战败被人嘲讽，约人出来打架反被黑涩会羞辱，少女c的gpla向少年a和少年b的gpla求助，少年b懦弱，表示自己也被那些有名的高级玩家在游戏和现实中欺负，不敢出手，少年a则在gpla的ai一阵嘴炮下，挺身而出。
黑涩会吃了苦头之后决心报复，但是他们毕业了，不好下手，于是在现实生活中威逼少年b约少年a出来，同时在线上不断羞辱少年a的ai。
少年b的gpla原本继承了他的奴性与受气，但当自己主人被找上门，也忍不了，偷偷联系了少女c和少年a，然后被黑涩会发现，素体被黑涩会（或者说黑涩会的素体，这样避免gt）毁坏。

少年a带上gpla如约而至，人和gpla都被羞辱之后，黑涩会（的素体）毁坏了他的素体，还准备揍他。
