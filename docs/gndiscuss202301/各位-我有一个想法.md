# 1

人眼的焦距是50mm左右，现在主流的低焦距镜头一般只是用来拍摄鱼眼或拉腿效果，而实际上低焦距视角通过裁切得到的就是我们所谓的“小人视角”
例如前段时间某音上的insta360所拍摄的擦边视频 就很好的利用了这一点
所以我认为 将两个低焦距迷你镜头并列排放，辅以VR设备，加上遥控的小机器人之类，可以很好的模拟我们的幻想 至少10倍左右是完全可以实现的
当然 触觉是不可能的了 触觉只能等脑机接口了 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
