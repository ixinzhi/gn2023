# 1

很多像我一样的GTS爱好者都会觉得缩小是好但是太危险，然而实际地分析一下就会觉得事实可能并非如此，下面逐步解释：

1。人如果被缩到蚂蚁那么小，会比蚂蚁还弱吗？ 答案是不会

<1> 蚂蚁从桌子上掉下来是摔不死的，而把等比例放大1000倍的蚂蚁从楼上推下去它就会摔死。这是因为，假设身体密度不变，外骨骼厚度只增加到1000倍，而其身体质量则有1000000000倍，落地一瞬间，动量变为0，外骨骼的强度明显不够抵抗这一瞬间巨大的力。实际上昆虫长不大也就是因为随着尺寸变大，外骨骼的强度越来越无法支持其身体。也就更别说搬运自身重量级几十倍的物体了。
<2>老鼠如果变得和大象一样大，会无法站立，因为腿的截面积是2次增长，质量却是3次增长，结果是如果放大1000倍，脚部压强就增大1000倍，相当于又背了999个自己。
就是因为这样，大象的腿才会长那么粗。即使缩到和老鼠一样小，大象的腿还是比老鼠粗得多。
<3>为什么能飞的动物体型都不是很大，原因与上面很类似，昆虫翅膀这种结构也就只能支持昆虫大小的生物正常飞行，鸟的翅膀也只能支持鸟或者比鸟更小的生物飞行，翅膀能够提供的升力赶不上质量的三次方增长，再大的话就飞不起来了。所以正常大小的人靠和鸟或者虫相同结构/材料的翅膀是没法飞的，真要飞的话翅膀就一定要有更好的结构，用更好的材料制成。

从以上几点可以看出：
1。蚂蚁的身体结构所能够支持的体形就是那么大，体形小的生物即使放大也不会变得很强，因为身体负荷始终是三次方增长，比身体大多数机能增长得快
2。同样的身体结构，尺寸越小相对的性能就越好。一般情况下，体形越大的生物越是具有更高级的能源系统和身体的相对强度，如果被缩小，也一定比原本体形就那么小的生物强得多。

换句话说，大的生物缩小了也比小的生物强，而小的生物即使放大了也还是弱。

假如人身体密度保持不变，如果人被等比例缩为1000分之一，人整体以及各部分质量就变成1/1000000000，胳膊和腿的截面积却是1/1000000，脚部压强就是1/1000，结果就是人会感觉自己的负载轻了很多，运动/跳跃等机能也会大幅提升。           

人的肺泡形状不变，体积缩小，与空气的相对接触面积变大，肺的有氧呼吸能力也会大幅加强，理论上可以连续加速跑至少好几分钟。

所以，和蚂蚁一样大的缩小男大概应该是这样的：

(1)非常大的力量，如果蚂蚁能搬起自身重量10倍的物体，缩小男就至少还要再乘个几倍，移动一些（对GTS而言的）小物体也是可以做到的。

(2)弹跳高度大幅增加，至少跳上桌子没有问题，也不会被很小的障碍物挡住。

(3)很难摔死。

(4)用不完的体力

(5)干掉一个比蚂蚁大得多的虫子应该不成问题

话说超人基本上也就这么回事了吧。。。。。。。

2。缩小男是否能被GTS抓住？

<1> 缩小男的神经回路大幅缩短，反应时间也随之缩短，所以不会存在由于GTS动作太快而反应不过来的问题。想想为什么人没法看清蜜蜂扇翅膀，而蜜蜂自己却能够自由地控制飞行。事实上缩小男就应该能够看清蜜蜂扇翅膀的动作，稍加学习自己拿个蜜蜂翅膀也能飞。

<2> 暂且假设缩小男依然采取直立行的方式，加速跑，靠脚的抓地静摩擦力前进并增加速度，每跑一步，身体重心都和一般人一样，依靠静摩擦力前进1/5的路程，其余路程靠惯性。静摩擦力正比于压力，假设缩小比例为1/1000，则对地压力缩小为1/1000000000，但是质量也同样是1/1000000000，一步的长度缩为1/1000，假设一步的功全部用来提升速度，末速度就是一般人跑一步的1/sqrt(1000)。缩小男的相对加速能力比正常人提升了30多倍。

繁琐的计算省略，结论是保守估计，1/1000的缩小男跑1米只需要0.2秒不到的时间。

虽然不是完全没可能被抓住，但绝对是很难被抓住，而且只可能是在离GTS非常近的地方。

再加上之前所说的反应速度的提升，眼看着脚落下来却来不及躲这种事情基本上是不会发生的。

反倒是缩小男会觉得GTS的行动非常缓慢。

综上，缩小男想要被GTS杀死，特别是无意识地杀死其实是很困难的。

第1点
关于重力加速度的问题
[1/1000的人则会感觉自己在以9800米每秒的加速度极速下坠]
这个类比实际上是有问题的
因为人身体的加速度感受器就是一个单纯的线性加速度感受器，所能感受到的是加速度本身，而不会随着人的大小而改变

因为g=GM/R^2，所以人在太阳上巨大的引力感只是由于太阳巨大的质量，而与人本身的质量是没有关系的，并不是由于人和太阳质量的差异。
所以，缩小男在地球上的引力感和一般人无异。

综上，缩小男在这方面不会有什么夸张的感觉。

第2点嘛
暂且假设细胞大小不变是可行的，血液能够正常流动，并且各器官也都能够按照原先的模式工作

首先，昆虫和细菌没有智慧是因为它们没有进化出脑的结构。
还有很多动物脑很大却不聪明。
人的大脑处理能力很强是因为大脑皮层的表面积大，大脑皮层勾回的复杂度高。
脑力是由大脑的表面积而不是体积决定的。

所以，人脑的处理能力为二次倍率减小。

大脑的处理能力主要被用于：
1。处理各感受器送来的信息
2。控制效应器作出反应
3。思考
4。记忆

信息来自感受器，然而由于神经末梢分布密度不变，感觉器官传来的信息总量随人的体积同以三次倍率减小。

1。2   神经末梢减少导致处理信息/驱动效应器需要的处理能力随之减小，由于是三次倍率减小，相比较而言大脑的处理能力减小得更慢，也就是说实际上会显得更聪明。
3。人的思考主要是利用镜反射的机制伪造出声音信号，与耳接受的声音信号同理呈三次倍率减小
4。人的记忆同样是储存在大脑皮层，记忆力依然正比于大脑皮层的表面积，而记忆的内容还是串成链表的伪信号，所以记忆力相对来说也是提升了

也就是说，缩小男的思考和记忆也完全不是问题。

思维的复杂度不会下降，严重下降的只有听力/视力这一类的。

但是要保存原先的记忆就不够了，所以缩小男可能会失忆，而且是失去绝大部分的记忆，但是大脑对于记忆的存放是有优先的，实际上最后留下的一定是最重要的，关系到生存的重要记忆。

[关于大脑]

比如“细胞大小不变，按照同样的组分形成可以正常工作的组织和器官”
这是不可能的，器官的大小和形状也是由细胞的性质决定的，直接减少细胞的话无法组成相应尺寸的器官，还有些能再生的器官会逐渐变回原大小。。。。

[压缩]

首先，原子对外表现为势函数，之所以能形成物体是因为很多的势函数相互叠加并且相互作用，影响各自源点的位置，最终达到动态的平衡。

形成大的物体靠的是先形成微结构，微结构作为新的势函数源继续相互作用形成更大的结构，这样一个迭代的过程。

原子的大小一般是定义为函数值大于某个阈值的范围，实际上原子不存在真正的边界。

基于以上几条，原子的大小直接反映其物理性质，也就实说原子的大小决定了任意一个多原子系统动态平衡状态下原子的位置分布，称为系统的平衡态。

如果只缩小距离而不改变原子大小，平衡态一定会被打破。

根据缩小的程度，两种情况：
1。缩小程度不太大的话会自行在原来的位置附近达到新的平衡态
2。缩小程度太大的话也会直接离子化并解体，回不到平衡态

第2种显然是不行的
第1种由于是回到原来的平衡态所以并没有缩小。

只缩小原子大小的话：

根据缩小的程度，同样是两种情况：
1。缩小程度不太大的话会自行达到新的平衡态
2。缩小程度太大的话也会直接离子化并解体，回不到平衡态

第二种显然是不行的，而第一种看似可行实际上也不可行，因为每一层形成的平衡态都与之前有一定差异，而每层的平衡态差异会导致这一层之上的层产生更大的差异，也就是说新的平衡态并不是之前平衡态的等比例缩小版，出来一定是和人完全不同的东西。

综上，为了保持平衡而不使人的身体自行分解，必须让整个势函数的图像等比例缩小。只有这样整个结构的平衡态才会等比例缩小，也就是说，**更小的原子半径必然导致更小的原子间距离**。

原子的形状和物理性质有直接关系，所以缩小的原子意味着：
1。外在表现为等比例缩小的势函数
2。更小的电荷

为了获得比现有夸克更小的电荷，只有一个办法就是通过粒子对撞来制造自然界中原本不存在的基本粒子，可能会得到一套用来产生缩小原子的新夸克集

但是这样的话就又变成了替换而不是压缩身体了

[关于反应]
即使可以做到整个势函数的图像等比例缩小，与外界的反应依然是问题：

首先，更弱的势函数意味着更小的电荷
由于电荷不够大，相比于一般的原子，与任何其他原子间的力都不够，所以一切来自缩小男身体的分子在与一般的分子反应时都会由于电荷过小而无力拆开原先分子的键能，也就是无法反应。
即使靠更多数量的缩小原子能够拆开键能，由于电荷量变少，反应的平衡点一定是缩小男的身体提供比正常反应更多的原子，化学式也会改变。
人身体所有的反应都需要外部环境输入的物料，这也就意味着所有反应都有一个和之前不同的化学式，生成不同的产物。

综上，这样的情况下呼吸和代谢都是无法完成的。
