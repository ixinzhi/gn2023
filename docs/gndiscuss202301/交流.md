# 2

大陸地方大，每個城市或許情況會不一樣吧？

沒有正版漫畫還挺正常的，台灣也是盜版盜很久後，才被外國施壓要求改善。例如荻野真的作品早期在台也都是盜版的。

不過，四拼一我就有點無法忍受……這樣只能給想打發時間的人看吧？想收藏的人根本沒法接受吧！

而且說真的，台灣漫畫進入正版授權時代後，書本的品質確實比以前好很多，不過還是比不上日本國內的原產漫畫(連最基本的紙質都比不上，台灣漫畫放一陣子就發黃)。

短篇也是gts漫畫的一個重要特色。基本上很少人會從頭到尾主打gts這個要素，多半只是像to love出包王女那樣偶爾惡搞一下。當一個GTS迷，這點覺悟恐怕不能少。像超級巨鳳那樣整本以巨大少女當主角的漫畫，至今有第二個人畫嗎？(同人創作除外)

電影方面，講真的我看得較少，回答起來恐怕也沒啥內容，就先跳過吧！

美國英雄漫畫……因為不喜歡肌肉畫風，就更不知道怎麼講了。不過我記得似乎有女巨人的英雄角色存在，應該是Marvel旗下的人物，最近好像還有出本跟蜘蛛人與浩克對打，giantess city上有貼。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
