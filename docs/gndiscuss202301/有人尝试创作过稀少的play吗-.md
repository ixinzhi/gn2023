# 6

我看過的大多是進入某一個洞的插入play與夾在人體某處的擠壓play，大概90%都是以上兩種吧！

還有一種較少見但也很有趣的：放置play
也就是把小人放置在GTS身體上既不是夾擠也不是入穴的某處，小人可能要自己抓穩或有某種因素使其黏在上面，感受GTS活動時的各種震撼或H情節。

幾種經典的情景：
小人黏在GTS手上，與她一起洗澡
小人乘坐在GTS的鞋子上，與她一同散步
小人抓著GTS的頭髮，隨風飄蕩

人體還有很多部位可以放置，相信樓主一定可以想到其他更有趣的位置擺放小人。 <title>7</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
