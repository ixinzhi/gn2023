# 1

自认为原来的投票选项并不能反映同好真实的口味取向，所以特开此调查，想了解下同好喜欢的GTS体形大小。
每个投票都可以选，如果你觉得只要是GTS都通吃的话，就都选上吧。
在此为GTS体形的大小做个说明，如果说明不完整的话，请大家留言补充，谢谢！
1.mini-gts：就是一些同好喜欢的高大女，身高差，tall woman，现实中可以找得到，也包括幻想身高2-4米的。缩小男则是属于侏儒的范畴。
2.小女巨人：参照美国电影《50公尺女巨人》和日本的macross巨人族，也就是人的10倍大，大约15-17米。缩小男则是芭比娃娃那样的人偶大小，GTS的手刚好可以握住SM（shrink man）的腰部。
3.大女巨人：楼房大小的女巨人，穿梭在城市的高楼之间，一脚可以踩扁一辆汽车，大约是人的100倍大小，100米上下。缩小男则是几厘米大小，捏在手中或者扔进高跟鞋里完全无法逃脱。
4.mega级女巨人：城市级别的女巨人，可以踩扁一片房屋，将高楼大厦连根拔起。缩小男已经属于micro级别了，差不多1厘米以下，手指脚趾可以直接捏碎。
5.giga级女巨人：大洲或者星球级女巨人，虽然与人不会有多少互动了，但是在部分同好里还是有市场的。这也是女神的终极体现，小一点的可以一手捧起一座城市，大一点的捏碎地球。缩小男嘛，细菌级别了，躲藏在GTS肌肤的纹理里面。

楼主此次设定的项目比以前清晰多了，也具体形象，我自己兴趣通吃，不过本人觉得还是稍欠精确。女巨人和缩小男名称和身高定义失准，比如【小女巨人】和迷你女巨人之间不好界定，而且说的是身材小还是年龄小定义模糊，我建议改为【标准女巨人】身材在10~50米之间。对于迷你型GTS身高也应在3米以上，而最好不要用现实的人类，否则没有戏剧性效果。在标准女巨人之上可以定义为【大型女巨人】和【超级女巨人】两类，她们的身材可以从100米直到5000米，再往上的级别最好不要设想在地球上了。
实际上，楼主的意思是按照标准女巨人和普通人类体型的比例关系，沿用在更高级的女巨人身上，这样的分层会很多(至少有10级)，可楼主的后两个选项太笼统包含过于广泛了，应该再精细一些。我设想GTS世界是分层分级的，在同一个层面下同时存在几个不同级别的女巨人，而在这个层面之外还有着更高层面的世界，她们就可以在地球之外的星球或太空中，而不同的世界又有好多层，高层世界统治着低层世界。一个层面内的世界女巨人的身高就可以相距上千倍，比如最接近我们现实的层面，女巨人就是从几米高直到几千米高。 <title>12</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
