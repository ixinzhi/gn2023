# 1

  自从上次发帖已经不知过了多长时间（喂喂，你认为会有人认识你吗![](img/6aa9b3e71108a323a0981d2b75ad6caa.png)），最近闲来无事翻了翻漫画分享区，偶然看到一片帖子：[[女巨人漫畫] 杰西卡和维基](http://giantessnight.com/gnforum2012/forum.php?mod=viewthread&tid=19200&extra=page%3D2)，针对提到的问题还是有些自己的思考![](img/46b114ca741ad5aafabac9efb3e801b8.png)

  关于之中提到汉化的问题，个人感觉还是可以进行的。![](img/efe6b63080909e5d63c4a6156fcfe80e.png)
  之前做M大的另一部作品的汉化时有就汉化的这个问题咨询过对方，M大大方的说汉化什么的当然大丈夫，所以当时也就僭越的翻译了M大的作品。
如果您担心此章节没有完结的相关问题，M大也有相关解释，作者近期已经很长一段时间不会针对此作品进行更新，而且复更到时再续上也就好了。![](img/efe6b63080909e5d63c4a6156fcfe80e.png)
  综上，汉化可以搞！![](img/ec942b6b7db075eaad5da7a4ad136d51.png)
  不过最关键的问题还是出在工作量上，就M大的风格而言，文字确实有些多，将工作交给个人来做确实有些吃不消！![](img/5d52b431f57f09d2573252d9cbd9a925.png)
因此在下提议不如成立一个相关的仅对此作品的一个工作组，一起来解决本作的汉化问题！![](img/bab821aad2ab4901d94c40b39c29b350.png)造福大家！

   以上就是鄙人对此作品的一些不成熟的思考！
    ps.如果已经有大大针对此作品进行了汉化，那以上的话就当听听得了！
    pss.请大家广泛留言，毕竟一人力小，众人力大！
