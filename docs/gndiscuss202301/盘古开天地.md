# 4

很有意思的设定啊
正好最近在读山海经，关于女娲，突然想到这一段：
《山海经·大荒西经》：“有神十人,名曰女娲之肠,化为神,处栗广之野,横道而处.”

那么把思维推移到山海经中，这里有蠃鱼、鵸鵌、胜遇、鸾鸟、玃如等异兽；有夸父、西王母、女娲、大小人国等神灵/人：又有小人国、君子国、魁拔等一系列拟人化虚构对象，加上书中所介绍的各种山、海的详细概念，这本书本身就已构成了一个庞大的背景啊！

既然如此，我们不妨发散一下思维：（娘化的前提下）
1.添加穿越者，挑选出一些异兽出来进行娘化，由于异兽的大小在书中未有过多阐述，大小的随意拟定也就有了其合理性。在与异兽交涉的过程中有太多东西可以写了，最后成功地回到原世界作为结局，甚至可以继续下去把蓬莱当中的生物不小心也带了回来，在现实世界掀起一阵腥风血雨亦不失为一种剧情的推进；
2.以一只无名/落魄的青鸟作主角（在《山海经》中青鸟是西王母的信使），像武侠小说一般一路战胜其他异兽（当然其中的打斗福利场景就有很多发挥空间了），最终见到西王母，成为她的信使一员完成梦想/回到西王母的怀抱；
3.回到刚刚的女娲之肠，将女娲设定为最终的大神，10个由肠所变得神是为小神，魁拔、凤凰、狍鸮等小有名气的异兽则可成为小boss，这样一个RPG设定似乎也有很多发挥空间？

当然以上只是文渣的一些幼稚的观点，但也侧面证明了中国古籍（《禹共》《淮南子》《吕氏春秋》等）在gts领域当中极大的挖掘可能性，写出拙见一二，以候抛砖引玉。希望能有大神来试一试！
