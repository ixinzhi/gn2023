# 3

一般來說創作者都會在twitter / deviantart 上面簡介都有寫會不會接受約稿(commission), 大部分都會按完成品的要求(線稿/上色, 半身/全身)寫出價錢供參考，不過外國一般收錢渠道多為paypal, 所以付款方法也是考慮因素。加上你需要盡量給予具體的圖片內容(角色動作, 角度等等) 所以你和創作者的語言也要考慮在內。

又或者簡單點你可能在gn圖片版看到喜歡的創作者畫風，也可以私人聯絡看看對方有沒有興趣接約稿這樣。
