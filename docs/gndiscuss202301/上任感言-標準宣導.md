# 1

大家好，我是疑難雜症區的新版主賊寇，來這裡說些上任感言還有對我的管理標準說明一下。

當初來GN只是純粹的想找縮學的遊戲，沒想到在這裡發現了一個新天地。

在當了一年多的伸手黨後，覺得是時候該自己回饋給他人了，於是開始寫文及分享情報和資源，看到不少同好支持我的文感到非常高興。

聽到但大說缺人幫忙時，認為自己可以以這樣的方式為論壇盡更多的心力，就向但大毛遂自薦，接著就上任了。

現在開始說明我的管理標準。

基本上就是照版規來，重複或是太過空泛的問題都會刪掉。

至於舊文嘛，只要是一年以上或是問題已解決的文就先刪掉，除非裡面有實用的資訊，而這樣的文就會加精華。

最後，這是我第一次協助管理論壇，還請各位GTS同好和版主們多多指教。

有什麼意見的話歡迎告訴我，另外我也不會忘記寫文的老本行。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
