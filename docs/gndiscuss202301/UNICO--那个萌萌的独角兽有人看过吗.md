# 1

花园里以前下的一个视频，老惯例，没字幕的日文原版...短片名就叫UNICO，里面的主角是一个像匹诺曹一样的木头人和独角兽。
刚开始看的时候就被萌翻了，但是没怎么注意剧情。后来仔细看才发现，里面的那个巨大兽娘其实是斯芬克斯...她遇到了流浪中的木头人和独角兽，问了他们那个经典的问题，还很调皮地调戏他们来着0-0  不过后来木头人好像歪打正着地说了人类这个词，他应该也是被什么变成木偶的，于是回到主题...
当时就看出来这又是手冢大师的作品，强调环保和人性，萌萌的画风配上比较深刻的内容那种。百度了很久，好像没有我要的答案，不知道这段出自哪里（有个八十多分钟的剧场版就是讲巫女把人变成木头傀儡的，可是好像不是）。顺便说下，里面还有在云层中被拉下来的阿童木（?），所以我有点怀疑那是不是同人作品...看画质什么的肯定不是啦，估计是不同的版本吧。就像逗啦A梦里的一集，我看过两个不同版本的，至今也不知道哪个在前。
大概就是这么多资料了，希望有直到真相的人提个醒。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
