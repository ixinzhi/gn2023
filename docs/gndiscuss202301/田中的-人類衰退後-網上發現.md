# 4

把這本當成ＧＴＳ小說的人可能會失望吧！但若只是喜歡故事的設定本身，也許還算值得。

看了大概一半，發覺作者的筆觸相當隨意，說故事的方法看似天南地北亂轉一通，卻還是有些許脈絡可循。不過這樣的敘事手法從以前到現在都不是很合我的味。當然用這樣的方式來營造一個輕鬆的世界觀是非常適合的，怎奈我個人就是偏好有些沉重的故事。

我雖寫文章，但卻很少看別人的小說。不是很能給出什麼中肯的評語，況且我覺得評語本身就是主觀的。基本上這部小說相當適合讀者用來打發時間，但卻不適合喜歡把腦子放空才來讀小說的人。儘管是輕鬆的故事，但仍有需要思考的餘地存在。

最後下結論：對於想要看到精彩的ＧＴＳ小說的人，不推薦。
