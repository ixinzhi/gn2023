吐槽是好事，更是一門學問，這對吐槽者來說是一種腦力的鍛煉和語言水平的體現，要想著怎樣吐槽，怎樣吐好槽，做到既能達到自己吐槽的目的，而又不至於讓被吐槽者太難堪，而被吐槽者則是鍛煉自己的氣量了，被別人吐槽后能做到氣定神閑一笑置之，甚至目無表情繼續流覽，這就解釋了將軍額上能跑馬，宰相肚裡能撐船的原因了，因為中華民族，或者準確的點，漢族，是一個很喜歡吐槽的民族，這來源于古代學究的高嚴格的一字不差標準和讀書人的咬文嚼字，要是不小心開罪這兩種人，前者肯定痛斥你不好好用功，不好好學習，要打你板子，後者雖然不會有武力行動，但是那張嘴絕對會讓你即可想變成兔子挖個坑鉆進去，被人搞了，自己肯定不爽，於是你，爲了心裡平衡，就跑去吐別人槽，久而久之，漢民族的吐槽行為就基本上是見怪不怪，家常便飯了。平常人也是如此，更何況在中國政治鬥爭激烈的朝廷中的一人之下的大將軍和宰相呢？他們當然不可能把那些吐過自己槽的人全部XXOO，暗中除掉一兩個也沒必要，更不會跟那些對吐，那樣多沒品，這樣子根本成為不了朝廷的支柱！所以，就不管他吧，於是，他們就養成了橫面冷對千夫吐得好氣量了，受到這些人的感召，我們尋常百姓和各階層人士也就積極響應了，這樣子，我們中的一大批人就變得很有氣量了。所以，你還再為自己的小肚雞腸所煩惱嗎？那麼，就全力以赴的被人吐槽吧！

當然，反吐槽也是一門大學問，如何針對對方的吐點進行一一的駁斥，很好的駁斥，一次就駁的對方什麽都說不出來，這可是寫論文的基礎啊，對日後一些要讀大學的人來說是很好的經驗，是一筆寶貴的財富啊！所以，大家都來吐下槽，被人吐槽的也都來反吐槽或者不吐槽。這是為日後的生活打基礎啊！！ <title>17</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
