# 1

原创文章是这个圈子最活跃的部分之一了
常说文无第一武无第二，但因为是入坑作使然，或是确实最对我胃口
《神秘的药丸》是我心目中当之无愧的第一文
无论是缩小的过程还是与几位女巨人的互动都是经典中的经典
与妻子、秘书莎莎、高中生妮可、小侄女的互动各有不同，但每部分都突出年龄、性格、体型差方面的特色，称得上引人入胜
喂奶缩小的设定没有展开描写但极具潜力
可以说文中大部分内容不突显淫秽色情，却让人遐想连篇，是注重故事性和情节合理性的爽文的代表
有趣的是，本文写到一半原作者就太监了，但续写者写的一样出彩，甚至更胜一筹
鉴于此文太过经典应该都看过了，就不贴链接了
