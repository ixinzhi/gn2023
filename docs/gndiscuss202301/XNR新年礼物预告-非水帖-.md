# 1

因为工作好久不做图的我终于要放出新作品了。
也算是新年的礼物吧。

礼物有两个：
1.《缩小学院》同人图补完。
2\. 真人合成超巨大GTS。

关于《缩小学院》同人图：
跟上次比有了很大进步，不论是质量还是数量。（我就不放上次的链接了，右上角可以搜索）
相对这次的来说，上次的同人图除了中间两张有保留价值，其他都不值一提。

对了，补充下：现在论坛内大多数人都应该通关了吧，CG插图也都收集齐了吧？大家不妨猜猜看我还为这个游戏加入了哪些插图呢？

上次的同人图当中，有一张有一个很明显的“漏洞”，遗憾的是12页一百多个回复当中，没有一个同好看出来。（这样如何能精益求精推出更多精品呢？）
这次我重新加工给补上了，在发布之前，如果现在还有人能给我指出来，我会非常开心，因为这说明真的有人在认真欣赏我的作品了。

我个人认为这些图已经接近甚至达到了SS的标准，离我的目标又近了一步。（不知道SS是谁的，上sizebooru去搜索，不知道网址的，去看置顶的红色帖子）
当然了，好不好还是大家说了算，也欢迎本论坛内的职业画家做专业点评，谢谢。

有人建议我多画草稿，每日练习，这个意见很好，但对我不适用，因为我有工作，与绘画和艺术设计无关。
所以我在这里说明下：
在我有限的时间和精力的条件下，我要么不画，要画就画出成品拿出来与大家分享，我不画草图之类的半成品。
一切进步都通过一次一次的成品来做到。（给别人的创意画同人图，本身也就是一种练笔）

这些图都是我在工作之余挤出时间来做的，一点一点的做，从画草稿开始到完成有四个月的时间了，也反复修改过。（工作真的很忙）
手绘图比真人合成图做的时间要慢，但是自由度高，可以完全按照自己的想法去画，是我以后作图的首选。

关于真人合成图：
一贯坚持我XNR的作图原则：
1.主角漂亮，清纯有气质。
2.高清大图，让你看得爽。
3.尽量做到“真实、自然、有互动效果”。

这次的图虽然没用什么新技术，但是依然注重细节。（不过抠图还是最让我头疼的，这方面做得不够完美，请大家见谅）
值得一提的是，在这组图中，我XNR还破天荒的突破了我以往作品的界限。。。让我心中很有负罪感。。。
--------------------------------------------------------------------------------------
最后，对于mini-gts控，恋足踩踏控，吞食控，我的作品可能不对你们的胃口，很抱歉。
预告完毕，今晚放图，说到做到。
