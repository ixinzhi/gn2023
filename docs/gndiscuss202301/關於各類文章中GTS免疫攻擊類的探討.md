好吧，所以針對這個問題，我也給予我這邊的回覆。
這些都是在私訊裡對原提問者解答過的：

----------
在我的「期望」裡，只要能夠被稱為GTS，就是無敵的。
當然，也有一部分同好喜歡偏向真實的設定，即便幾百米高的女巨人也會被炸彈打倒之類的。
我是不喜歡女巨人有弱點呢。

此外，我喜歡的比例是單向量一百倍，也就是一米六的女孩子巨大化變成一百六十米。
低於一百倍比例，我會變得沒什麼興趣。
----------

被問到對其他作品中的女巨人是否無敵的看法時：
----------
看作者的設定，我不會去干涉其他創作者的作品世界觀。
我要說的一樣只有「只要能夠被稱為GTS，就是無敵的」。

基本上連一百倍巨大都沒有的妹子，我很難相信用砲彈也打不倒。
除非作者還有其他設定，像我就是只要進入故事的世界觀，就沒有多大=無敵的這種事。
一切都為了劇情需要而可以隨意更動。
因為我並不只是依照自己的「期望」而寫故事。
----------

嘛，這就是我的回應。

# 4

嘛「巨大化的同時有魔法將她的皮膚變成刀槍不入」不就行了?
![](img/b8b9f48f20d11df08323242ccd08fd94.png) <title>5</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 5

理論上，若是真的考究物理因素，能夠巨大化到奧特曼等級而不被自身重力壓死的生物，其骨骼肌體強度顯然不是區區單兵武器能夠傷害的。畢竟連大象都不是一梭子子彈能幹掉的，肌體厚度和骨骼硬度太高了。
不過如果不考慮異能的話，純生物其實還是很脆弱的，輻射、毒氣、高溫、低溫都可以造成有效殺傷，無論體積。另外，人造真空環境可以導致窒息。

簡單地說，如果沒有異能，純粹生物體，那麼再大也是白給。舉個例子，螞蟻VS人類，不需要科技武器，非洲行軍蟻表示有肉的生物都是渣渣，上億螞蟻大軍可以在六分鐘內吃光一頭象。
但如果人類裝備全密閉式機動鎧，並且使用火焰噴射器，再多螞蟻也是渣渣
就是這個道理，科學技術才是第一武力（當然，魔法也是） <title>6</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
