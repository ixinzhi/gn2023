# 1

我實在太常看到讓人頭痛的角色描寫了，圈內外都是
所以我在這邊提供非常簡單的指南，讓大家都能寫出比較不會出戲的劇本

廢話不多說，直接講重點

當你的角色面對一個情況時，怎樣的反應算合理?

要做這種推想，一個很關鍵的要素一定得考慮

**這個情況，他/她有多少經驗****?**

如果你今天寫得是一個**每天**都縮小別人、已經**半年**內換過十個玩具的角色，

那她應該會很熟悉對方會有什麼反應、可能的逃跑方法、被做什麼質疑時能有什麼應對等等，

讓整個角色產生**從容不迫**的態度 (不是從容不迫沒關係，重點是先前的類似經驗會怎麼改變行為)

反過來說，如果今天你的角色是一個**沒經驗**的普通人，然後走在路上**突然**被縮小傳送到女孩子房間，

那他絕對不可能在五秒鐘之內跪地磕頭求饒，因為他的大腦此時極度可能還處在「載入中……」的情況

瘋狂飆髒話、或是嘴巴開開站在原地發呆，這才是突然被擺到陌生環境的巨人前時會有的反應

好啦，先確保目前為止都有看懂再往下喔～

人會從自己的經驗來尋找當下的對策

所以，上述第一段的便是「對於當下有充分的經驗做參考」，

而第二段則是「對於當下絲毫沒有經驗做參考」 (「絲毫」只是誇飾法，你還是會知道要離火遠一點)

不過當情況是「一部分有經驗，一部分沒經驗」呢?

那人就會把有經驗的部分作為思考的起點

舉例來說，在學校被縮小，「被縮小」可能是沒經驗的事，但「學校」卻是很熟悉的場所，

那麼在聽到下課鈴聲時便會知道自己應該要特別謹慎了，因為「下課鐘後的校園人來人往」是經驗，「被重物擠壓會死」是一般世界的常識

與熟悉的朋友到了陌生的環境呢? 「這個人可以信任」是經驗的話，也會大幅改變當下的行為對策

我還有很多能寫的，但目前為止應該已經寫太多了，所以......有問題就問吧

人被突然缩小，如果没有经验，应该会有一种认知到认知的过程，是一种从未知到可能已知。大多数人在接触未知的时候一般都会慌吧，恐惧会产生，之后是不断发展或者被强行克服两种。认知方面，关于人会先四处粗略的看，还是会盯着一个东西仔细的看呢，可以确定的是会把大多数注意力放在熟悉的东西上吧，基于克服未知的恐惧。假设小人能够认出gts是人类，当然这么大的体型差距会把对方和自己当作一个物种也是很奇怪的，除非是特别面善大程度缓解了体型的压迫，才会尝试有些交流吧。想想一般看到大一点的熊都是直接跑吧，原地发呆或者骂列都有些。。。人太直接了什么的。
结论就是没经验的人被缩小了，应该首先是寻找熟悉的东西，和远离会动的东西（gts）这样吧。 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

# 8

一开始我会觉得，要让自己笔下的人物“活过来”，慢慢的发现这个要求太高了，就降到只要他们的行事符合逻辑就好了。我以为最后会到只要他们能不出乱子的把我设定的剧情演完就可以了，但其实在“堕落”到这步之前，我自己就已经推翻了我原本认为的人物塑造的手法和过程。

在我这，其实一切的核心就是作者自身的文笔和控制表现张力的能力。你想好一个大纲，结果一个不注意，比如原本要踩死的小人，因为笔下女巨人觉得他还挺有意思的，饶了他一命。你后面构思的一切都被打乱了，而出现这种意外到底是好是坏呢？往好的方面说，这说明至少角色在你笔下不是单纯的工具人，她们有自己的性格，会因为铺垫和当时语境的氛围做出你设定以外的事情。往坏的方面说，其实就是你控制剧情的能力还不够，如果你顺着你塑造人物的性格走，那就会把剧情弄得一团糟，如果你强行把一切掰回正轨上，那就会有强烈的“违和感”。

那解决的办法就是:要么停下来，好好把目前的人物关系理顺了，想好了再往下写。要么就创造一个剧情的爆点，把注意力移到这个情节上，把“违和感”盖过去。
那说到底还是回到了原点，一切问题起源于你的笔力和对剧情表现力的掌控能力。一切问题又还是要用提升自己的写作能力和构思能力来解决。

当然我觉得楼主说的很好啊，有些文章它就没有思考过这些最基本的问题，这时候不改变自己的思维方式，就很容易写出那种“没有灵魂”的文章。
据说这种文章常常源自作者想把自己脑内的意淫付诸笔下，但因为真到动笔的时候发现，原本自己想好的人物关系和铺垫，用别人能看懂的描述写出来这么费时间啊，于是就变成只要能完成剧情就好的任务驱动型文章或者直接太监了。

但解决方法除了花更多的经历和时间去思考和丰富自己的作品，也可以选择用更有冲击力的情节和表达方式来解决，比如同样一个让小人屈服的情节，可能会出现:
①一上来就叫小人舔脚，小人就乖乖的舔了
②用一大段细腻的心理描写和pov来充分体现女巨人可以支配一切的压迫感和小人通过和女巨人各个部位的对比接受自己如此渺小现实，最后跪倒在女巨人脚下。
③用很有表现力的情节和合理的背景故事来直接推动人物关系的变化，比如如果要表现女巨人的残酷，直接碾死一个配角然后叫剩下的小人清理惨死同伴的尸体就好了。

所以一切的源头还是文笔和对情节的掌控能力，否则笔下的女巨人和小人们，永远都不会听你的话，或者写着写着，你就收不住了，会陷入不断刻画一些细枝末节无法进入自己喜欢的主线的窘境哦

你說的「不知道要不要強行拉回正軌」上就是所謂「角色優先」和「情節優先」的差別了
(關鍵字：Character Driven和Plot Driven，我不知道這些詞有沒有正式翻譯就先用白話講了)
所謂的角色優先，就是依照各個角色的個人抉擇作為劇情的發展標準，像是進擊的巨人
情節優先就是字面上的意思......像是JoJo的奇幻冒險

不過其實有時候重點不是二選一，而是你必須在一條路上一直顧慮另一邊的事

你的故事是角色行為當發揮基礎的話，你得為了觀眾把時間刻劃在有趣的情節上
這就像是你有一連串不能干涉的連續事件場景，你還是可以決定要把攝影機放在哪、要給哪一段更多更細的描寫
當然這個實際做起來很困難......因為大部分的人都是一邊寫一邊想劇情的

而你要塑造出一個特定情節的話，就必須一直顧慮「角色怎樣才會做出這樣的場景」
這不需要讓角色突然出現人格大轉彎，你只需要在場景中投放事件就能改變角色行動模式了
至於什麼事件才有效嘛，就是在不破壞角色的前提下一直試誤囉

不過沒錯，一切的源頭還是作者的功力
什麼叫合理的角色、什麼是有趣的情節......
品味不去培養的話沒救的就是沒救啊(笑)
