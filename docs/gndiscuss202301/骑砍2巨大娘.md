# 1

在骑马与砍杀2中可以通过skins.xml文件改变角色大小，方法其实和之前骑砍1的方法比较像，不过路径稍微有点不同，需要修改的文件是Modules\Native\ModuleData\skin.xml,
<skin id="skin_default_id"
          gender="1"
          name="woman"
          mesh_maturity_type ="adult"
          morph_key="1"
          uses_stitching="true"
          body_mesh_suffix="_fem"
min_scale ="1.06" #这是控制角色大小的变量，在文件的第1988行
          skeleton="human_skeleton"
          body_meta_mesh="body_female_a"
          body_meta_mesh_shoulders="body_female_a_sh" 

骑砍2中可以的人物皮肤是按照年龄和性别划分的，上面的该法会改变所有女性成人角色的身高，但是不会影响小女孩
我目前还不会在骑砍2中改武器和动作，如果只把角色改大会导致角色在近战中根本打不中别人，就像这样

顺便一提，骑砍2自带的捏脸是可以捏出可以接受的人物的。
