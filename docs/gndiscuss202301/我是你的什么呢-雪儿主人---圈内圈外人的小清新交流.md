# 1

      嗯...在论坛潜水五年多了，很少发什么帖子（似乎完整的作品只有一首诗）。即使是打酱油的积分也足够买勋章下漫画了。今天赶上学期最后一门课上完，心情比较轻松，又找了个能够用校园网上论坛的好教室，于是就想起了之前跟女网友聊天时提到的一些GTS方面的内容。那个时候刚聊完很顺畅，无论身心都得到了解放（不要在意如何解放的细节），当时是很有冲动把聊天记录发到论坛来当成小说素材的。
      可是后来想起了在这个板块看到的，类似的帖子，充满了男方引导性的话语，一大串YY换来女神一个“哦”，“额”...这样的效果似乎不是我想要的。我不得不承认，自己很喜欢这个板块，自己在看相关视频或者漫画时也很喜欢看圈外人制作的，没有刻意迎合同好要求进行互动的作品。这也许就是所谓的真实性吧?一点点互动都能如同画龙点睛一般耀眼。而在网络交流中，这样让自己沉浸的东西同样会让我忍不住，像捕捉一只野猫或者麻雀一样，小心翼翼如履薄冰地接近并没有虐待天性的女生。那么，各位就凑合着看看吧，让我看看用怎样的方式把我和一个女孩真实的聊天记录，用带有点艺术手法的方式展现出来。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
