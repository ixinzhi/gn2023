# 1

最近我在写一篇名叫：《宠物乐园》的文章，已经写了2章了，我想写完了一起传上了，免得以后填坑。

传统gts的元素有：袜子，踩踏，入阴，吞食，等等。

有没有比较新的游戏玩法，老是这几种看了感觉每篇都差不多，在少数文章中，还发现少数有时间停止，定身等构想。

希望大家把想到的有趣的游戏、蹂躏方法或者是道具功能介绍一下。

不仅是给我，还是给每一个写文章的同党一点构思，以你的灵感来创造属于大家的文章，在文章的末尾可以署名此构想由***提供。

文章应该属于多万字以上的，所以要些时间，大家到时候看吧。

# 2

我个人觉得：
GTS文章的精髓不在于多换几种玩法或花样，而在于怎样把简单的互动描绘的最细腻、最到位、最有文采，故事简单但是过程精彩。

之所以很多GTS文章看上去都差不多，是因为有些作者只求数量不求质量，看似洋洋洒洒一大篇，实则经不起读者们细细品味。

有没有深入研究文章的写法，有没有深入发掘属性的特点，有没有照顾到读者角度的感受？这就是快餐文和精品文的本质区别。 <title>3</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
