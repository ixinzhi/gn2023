# 1

最近在写文章时，需要一些小人用来狗带，虽然每个小人的剧情不多，但一群来构成故事的主体。在这种情况下，请问大家一般喜欢看哪种类型的名字？
①每一个小人都有独特且正常的名字
②小人A、小人B、小人C……
③小明、小华、小张……
总觉得选择不同类型，会对文章观感造成一定影响，特来问问大家，如果有其他建议也请提出，谢谢
