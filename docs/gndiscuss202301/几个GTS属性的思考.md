# 1

  入圈有一阵子了，很久没有发帖和大家交流，谈谈对GTS入圈多年的感受吧，捎带用几篇文章做点评，除了我自己的文章之外（自夸），最喜欢的几个作者，LUOLUO，正念，最近的正念同人、廖雪松，比较喜欢《履鸟传》、《女神觉醒启示录》、《完美星球》、《试验品成为神》、《三国无双外传》等几个作品，如果有遗漏的作品的作者请指正。
  跟圈内论坛同好有交流，熟悉的大概知道我的风格，几个关键词：吐火（或者口吐其他能够有力量感的东西，比如狂风、冰雪、激光）、高跟鞋踩踏、漂亮、珠宝、晚装，握溃，其实我到不是特别在意女巨人的体型，主要跟依据剧情发展变化吧，可以是试验品成为神里妈妈的无限巨大，比如连续光速变大一个月（自己想象脑补），也可以是一般的几百米几千你，不过女巨人的几个关键词，就是要威严、漂亮、压迫感、绝对的实力和美貌还有气质，女巨人应该是外表+身高+气质+容颜+力量的结合体，所以我幻想的女巨人除了自己交往过的漂亮女友之外，还有一些知名女明星、女模特、女主持人，无一例外都是各种高贵气质。
   关于为什么喜欢女巨人吐火或者口吐其他力量感东西，并不是说类似于哥斯拉，而是觉得口吐烈焰的那一刹那有一种力量感吧，主要是小时候看葫芦娃红唇吐火的那种感觉，虽然葫芦娃是男性。珠宝体现女人的力量和权力，握溃主要是来自以前看大话西游里的观音姐姐捏住孙悟空时猴子吱吱的叫声，大话西游3块上映了，预告里说有至尊宝和观音姐姐的对战，希望有一张精彩的GTS观世音对战孙悟空，再次捏住渺小的孙悟空。
   女巨人的几个关键词：
  1、对待小人的不屑一顾，巨大化以后对力量的充分展现。这种荷尔蒙大概来自于男性内心的恋母情结吧，或者说俄狄浦斯情结，毕竟很多GTS同好男性都是SM爱好者里的M，人们从阅读或者观看GTS作品里或者这种对女性权力的追求。举例，LUOLUO的完美星球里，里边的雪雯娘娘在完美星球是正派，严格禁止手下弟子杀害小人或者普通人，也曾经当面取消过一个杀害绑架过小人的女性的魔法，使她丧失了巨大化能力。但是当雪雯和燕玲两位娘娘自己通过获取转世神书拥有了女生化能力之后，便无限巨大自己的身体，然后到地球上来考察自己的能力。雪雯娘娘在地球上先是巨大化到100万米，然后又利用小人来发泄自己内心的淫欲，那次地球之行至少造成了几千万地球人的死亡，而且让多个大陆处于女神淫液之下，还口吐神火炼化了一个小女神接管地球。在没有获得无限能力之前，雪雯对小人还是有保护欲的，但当以无限巨大的体型接近地球时，这种S欲望便升起了，再以100万米降临地球大肆玩闹时，丝毫不顾自己订立的规章，结果造成了地球的悲剧。也难怪，女神就是无限力量的，可以约束别人，但却可以展示自己，不允许别人侵犯小人但却自己可以无意有意虐杀小人，体现了女巨人温柔中的力量感。履鸟转里女主角里在地球的高速公路上行走，无意中踩碎了很多小汽车，里边描写女巨人才不管脚下小人的死活呢，也是这种感觉。
  2、正派女巨人施展能力惩罚邪恶时连同无辜小人的一同消灭。女神觉醒系列的第二部，小星修炼后返回木星复仇，面对魔女统治的一座城市（这时候小星很大了，城市就在她脚下）喷出火焰，殊不知女神的火焰在在消灭城市里反派的同时也连同无辜的小人一起烧成了灰烬，更别说那些普通的城市建筑了。城市除了魔女，还有无辜的小人，也许还有恰好在那里避难的难民，也许他们也对反派的统治心存不满，但却成了女神复仇的牺牲品。这也体现了GTS的力量，尤其是这种故事情节的正派GTS，复仇时也不会在乎那些小人的死活，因为只想着消灭那些反派，所以就连同无辜的小人一起虐杀了。这种力量，也是GTS威力的一种展现吧，很多故事情节里都有，大家可以观察。
  3、GTS伦理。这个有些抽象，简单就是原先是人类或者能力一般的女人活得能力无限巨大化成为女神后，面对自己曾经居住/生活的城市、星球中里边有自己的亲朋好友时的态度。具体而言，可以是这种感觉，女巨人是妈妈，小人里有儿子，女巨人是姐姐妹妹，小人有弟弟哥哥，或者女巨人和小人是情侣夫妻，当然GTS未必能意识到城市或者星球上小人的存在。读过几个日文文章，比如有个女性跟上位统治者签署合约成为女神，回到自己的城市接自己老公到上位世界时是以巨大化形象出现的，当然也会踩碎小人、建筑，那些小人、建筑里如果有自己从小到大朝夕相处的好朋友甚至亲戚，GTS会纠结吗，也许会，但GTS也许成为女神以后就不会在乎小人了，毕竟神不在乎小人，就如同人不在乎蚂蚁蚊子一样。其实GTS伦理是很好的切入点，比如正派GTS想消灭反派，但反派窃据的城市有自己认识的人，如果不连城市一起毁掉，也无法消灭反派，此时对反派的痛恨和统治欲超过了对小人的同情，也就只好一起了。比如女神觉醒骑士录3里星雪无限巨大化以后面对几个被魔女占领的星球，在搜罗一番以后口吐超级烈焰毁掉了几个星球，虽然用魔法搜寻上边没有小人了，但谁有保证这些星球上没有其他声明呢，比如各种动植物花花草草，或者残存未被搜罗到的人类，就这么被女神当成和反派一起的余孽给消灭了。
  4、地球女性巨大化以后的黑化。比如试验品成为神里的妈妈，本身是个大学老师，很普通也漂亮的一个女性，但成为外星人试验品里意外的成为了女神，获得了超能力，不仅一口气吹飞了将自己当成试验品外星人居住的星系，更是统治了宇宙，而且在自己长大的地球啥虐玩闹也是毫不手软。虽然后来依靠女儿救出了地球上的一些亲戚，但无法保证所有沾亲带故的人都救出来吧，试想当自己的闺蜜看到自己成为女神，以8万米的身高和成为男巨人的老公一起空降地球时，那些小人会做何感受，当妈妈为了报复女儿被QJ，用手抠出一个城市，然后把城市放入避孕套里将城市作为和男巨人老公的性爱工具，就没考虑过城市里自己认识的几百万人里有自己的亲朋好友吗，也许，很多女人内心就有统治欲吧，这也是Gts圈子里女性大多幻想自己巨大化的原因。其实这一点和GTS伦理是交叉的。
入圈多年，随意总结，再次向奉献佳作的几位作者感谢，也感谢各位论坛好友。
