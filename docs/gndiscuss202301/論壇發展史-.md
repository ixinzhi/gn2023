# 1

突然想整理一下論壇的發展。多數的資料多來自yahoo家族的文獻。一方面讓自已溫習一下，也順便讓新人/舊人暸解論壇的變遷~~

一、yahoo家族時期：那時候可在家族那邊找到二個與GTS有關的家族，分別為：「巨大少女的世界」與「Vision821228創造GTS女巨人欲望園」 (V姐建立，旦丁繼承)。我印象很深刻，那時候V姐曾請「巨大少女的世界」的族長要勤快的更新家族資訊，一同為華語gts界，貢獻一份心力XD。

二、 Gehenna-GTS.com時期：在2004年左右，旦丁很熱血的建立華人地區第一個GTS論壇。

三、後Gehenna-GTS.com時期：Gehenna-GTS.com於2006年一直受到土耳其人的攻擊，導致論壇結束營運。有一部份的成員重新歸位於yahoo家族，當然也流失了許多同好。而在這種情況下，我們仍在家族辨了第三次的GEN合購(為了看影片，大家都超熱血的)!!同時，旦丁經過數個月的努力以及b兄等人的贊助之下，我們再次重建論壇。

關站事由，引自yahoo家族，旦丁：
『因為我發覺那些突厥駭客很纏人，他們又入侵了第二次，這次把整個站的資料都砍掉了= =+
當然我自己都有備份，不過論壇資料比較不幸，我手邊只有去年十月的資料。而且我也不打算恢復這些資料了，因為舊版論壇確實有安全上的漏洞，我想趁機換成新版的。 總之這些零零總總的工程加起來，也是要花時間的。這讓我很困擾，不知道是該給大家一個怎樣的時間表才行。目前就姑且先暫定為七月復站吧，「Gehenna-GTS」的名稱不變，不過後面可能會換成.net，可能的話我會把.com買回來。』

四、giantess-night時期：
應該於2007年左右建立，根據旦丁大大的說法，由於無法買回舊的域名並將舊資料備份回去，因此決定忍痛重新註冊新的網域名，另起爐灶!!而在重建期前後，女神論壇興起。此時，在華語圈，同時存在二個GTS論壇。

域名變更與經過，引自yahoo家族，旦丁：
1.『現在讓我還有猶豫的就只剩這件事了 由於以前的論壇裝了太多外掛(什麼金錢啦,快速回覆啦,暱稱啦...)導致舊資料過於混亂無法備分*所以新論壇將會從一片空白重新起家，大家的用戶帳號與舊文章都不會存在 在這樣的情況下，我感覺是很多人會離開。畢竟過去的心血全都消失不見實在不是件令人愉快的事。面對這種可能的情況，我個人很無奈，不過或許也可以因此而認識新的朋友們。 我想我會把網域名稱整個換掉，這樣比較有從新開始的感覺。 不過，網頁唯讀資料的部分應該是不會重做。』
2.『經過一番與FTP的纏鬥後…所有檔案總算是於昨天晚上傳輸完畢了。論壇方面也大致上有了架構。
目前網頁部分與過去一樣，只有logo更換； 論壇則請各位多多協助測試使用，如有發現問題可來信或直接在版面上報告。測試期間的資料並不打算保留，且預計將於三月25日之前結束測試期間，並在48小時內重新建立正式的論壇。 測試期間沒有所謂的版規，請各位不必顧慮流量與系統負荷，用力衝刺。 新網站網址如下：[http://giantess-night.com](http://giantess-night.com)』

五、giantess-night後期：關閉的原因有二，
1.主機代理商禁止在其空間上放任何有關18X的圖片與文字。
2.內地和諧giantess-night之域名
迫於無奈，只於2009年搬家並更換域名(旦丁口述)。

六、giantessnight時期至今：建立於2009年初期。在2010年年中，代管主機通知，本論壇佔用主機太多的資源，並進而強制關閉論壇之營運。原本以為又要回歸YAHOO家族。所幸3個月後，同樣在b兄等人的熱血贊助下，論壇於12月左右原封不動搬至新的代管主機。

從論壇整個發展史可知，要維特論壇正常運作並不容易，請大家要珍惜這得來不易的園地。最後，也感謝旦大、b兄以及許多支持論壇營運的同好~~此外，我也不想有第七點可寫![](img/ec942b6b7db075eaad5da7a4ad136d51.png)
