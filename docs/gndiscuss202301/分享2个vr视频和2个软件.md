# 1

首先先说下vr设备分两种，一种是基于cardboard的，国产的各种魔镜基本都是，这些设备依附于手机。它们根本不是什么电子设备，就是两组透镜。
第二种自然是vive，oculus，psvr为代表的真vr，价钱都比较贵，电脑也要gtx970+。
其实第二种也有便宜的，比如大朋vr，还有各种一体机算是介于第一种和第二种之间吧，不多介绍了，网上文章多的是。

第一种就可以简单体验gts了，淘宝cardboard或者各种墨镜。

介绍我一直用的两软件
第一个软件是trinusgyre，可以把电脑上的vr视频或游戏投到手机上。
第二个软件是UtoVR，个人感觉在iOS系统上算是非常不错的vr播放器了。

游戏的话去city上找，都是u3d做的，还可以！我特早也分享过一个。
