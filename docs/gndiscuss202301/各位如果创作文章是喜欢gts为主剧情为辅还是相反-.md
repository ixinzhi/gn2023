# 4

在我看來，GTS文章目前的主要作用還是“擼”這個字。
因為講直白點，大多數讀者看GTS文章的時候注意的還是GTS這個關鍵要素。
如果人家想看劇情文，網絡上的小說已經氾濫成災了…
當下只要比較劇情文和福利文的回復，雖然不是非常有效的對比，可是福利文的回復還是較多。
拿更加適合做調查的貼吧環境來看，絕大多數文章都是GTS為主的，而因為沒有那麼嚴厲的規矩，貼吧的環境更好的反映“我們想看福利”的這個思想，福利文如果坑掉就會有很多“快點更新”“等不及了”的回復，但是劇情文如果坑掉，收到的回復很少，甚至就這樣沉下去了。
至少可以說在現在的環境下，福利文還是主流，而在我看來，GTS這個圈子在短期內也不會出現劇情文>福利文的現象。
所以現在寫劇情文的幾位，兔球、劍爺，請務必撐住啊！
