# 1

我22岁刚刚大学毕业准备踏入工作，回国的机票在这周四，但我非常想要结束一切

一直以来我是一个严重的边缘型人格“患者”和社交恐惧症患者，现在我有很大的压力，处理不好身边的琐事可能是一个导火索。总之在自杀边缘我才发现，这里是我唯一敢于表达而不必承担未遂后果的community，所以抱歉了 如果打扰到各位 我觉得耻于和认识我的朋友求助 耻于和未遂的自己继续相处

想要跳湖可能原因有很多吧，“异常”的爱好和自闭导致的孤独和缺乏基本的社交技能。我常认为自己是一个笑柄和一直被消费的物品.. 有一部分生活是不属于我的 我也不敢去追求美好 我也没有能力为自己打抱不平 绝大多数时候我只是在日复一日地在网络世界做另一个我自己，一个我可以抛开现实世界定义我一切要素的我自己。但无济于事的，现实现在已经展开对我的惩戒。

我对世界充满恶意吧。憎恶自己原生家庭和所到的每一个环境。很多人说或者觉得我是一个只会赖和甩锅的人。这也是我最觉得无力辩解的命题

我也不知道我为什么要发出这段文字，要disturb更多和我素不相识的人，可能我就是想要伤害一下别人得到一点小小的满足。也可能我对这个世界还抱有那么一丝希望？其实不是的。一，我改变不了任何事情；二，我在给自己错误的希望。失败的自杀只会让我更加讨厌我自己吧。

如果我的身体活下来了 这个马甲可能也不会活下来吧
辜负了这么多人，即使是在GN这里，我无以回馈我曾索取到的宁静与快乐

推一部漫畫，《因為太寂寞了而叫了百合風俗小姐》，作者也是憂鬱症
我自己是沒憂鬱症，應該說我還有點對大眾情緒的認知障礙
但根據你的文字，我相信你最大的問題在於「太在乎別人的看法」
有時候人越是在乎別人的看法，越沒辦法看出與提供別人真正的需求，
製造出一種 想要認同→被別人討厭→更想追求認同→更被別人討厭 的迴圈
依據上面那部漫畫，作者驚醒的過程是「發現了自己不是想討好父母，而是想討好那個想討好父母的自己」
要取得別人的認同不是看向別人，而是要先讓自己認同自己
如果連自己都不想聽自己的感受，別人怎麼會在乎呢?
先別管別人要什麼，找找看自己想要的是什麼，時間久了一定能找到與你有相同需求的人
當找到了自己的同溫層，那也就能找到適合自己的工作了
