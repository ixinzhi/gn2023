# 1

从另一个帖子里转过来的，我认为可以讨论下，所以新开一帖。。。

我们这个文化就是需要靠不断的幻想和创作。。。没有他人创造出来的作品，我们只能通过自我的幻想得到性满足。。。

现在我们同好的创作完全是自发的，零散的，无组织的，自然创作量和作品的质量无法有保证，但是要把大家动员起来做一部大作，或者高质量的GTS作品，目前还是不现实的，从最近的游戏制作就可以看出来。。。
首先需要有足够多的人才，这些人才又有足够的时间和精力去创作，去互相配合。。。然后还要有完善的支付机制去让贡献者有所得，那样我们华人GTS圈子才会有更多更好的GTS作品

我有个提议，其实我们现阶段可以从最基本最简单的一步开始，那就是模仿HIK的运营机制
现在论坛里的画家和作家比较多，其他方面的人才比较少，我们就出一本网上支付下载的《华人GTS刊物》
说白了，就是每个画家画几页插画或者漫画，作家写几篇文章，或者更有才的人可以文章配插图等等
然后在一个封闭的圈子内制作（不给其他GTS同好看）最后整合在一起，做成一本网上支付下载的电子杂志
然后每个画家和作家通过自己的作品的质量和数量按比例拿稿费，同样都是网上交易，这样能者多劳，多劳多得，创作者更有积极性，其他大部分没有才能的同好也能得到更好的作品。
就目前情况来看，论坛有一千多人了，真正有能力创作的人并不多，所以创作者的收入应该是比较可观的
还有为了避免鱼龙混杂，可以请论坛的人来进行筛选，请大家真正认可的人来进行电子杂志的创作，这样也能确保公平性。。。
以上，是我认为最可行最有效率也是最公平的方案。。。如果成行，将极大的促进华人GTS圈子的发展，激发所有同好的热情和才华，未来做个什么游戏的也不是梦。。。

抱歉，仔細想想，我好像太激動了一點。

在圈內搞一本付費刊物，說起來是不難，但實際上有很多障礙要跨越。

首先得整合各位創作者，這已經不是件容易的事，畢竟大家的熱情有差。

再來你得決定要出實體書還是電子書。電子書容易被盜版，實體書遇上掃描器也是沒輒。

還有就是銷售方式。以中國幅員如此之大的情況來說，叫客人搭車上門來買是不可能的。所以這銷售管道就只剩下用網路，看有沒有適合的電子商務平台，也可以僅僅用電子郵件聯絡，然後請對方匯款，等收到錢後再寄出給買方。這在台灣是輕而易舉的事情（每年合購都這樣搞的），但內地的銀行連線狀況我就不清楚了……

最後，報一個小道消息：HIK的某一本GTS漫畫目前還有庫存，而且正低調的在招單。我不知道他是印太多本還是怎樣，他的新書都賣完了，舊書卻堆到現在。（他媽的，那一本我為了捧場，還買了兩本，分觀賞用與收藏用，結果居然到現在都還有貨）

可見就算是漫畫市場，只要估錯一步，也會讓你慘賠。
