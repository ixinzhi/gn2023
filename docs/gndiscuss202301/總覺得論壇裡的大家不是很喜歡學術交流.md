# 1

我並不是非常理解“生氣”這種情感，但當我的人格被侮辱或者我的職業被侮辱時，我會感覺到一種無法形容的感覺，也不是什麼想要怒罵或者說痛擊對方的感覺，只是一種很奇怪的感覺，應該是和“生氣”這種感情扯不上邊的。
大學裡和別人因為學術問題吵來吵去我也並不會出現這種感覺。
非常感謝之前在我的帖子里各抒己見，讓我感受到不同思想的網友們，多虧了你們，我的思路開闊了不少。
不過我也發現一個很奇怪的現象，那就是部分網友發表意見時，有一股直衝著我的“戾氣”。
可能是我的學識太淺顯，這部分網友對我感覺到幼稚，所以對我不滿吧。
又或者是像網絡上所謂的“我生活裡遭受了很多不公平，我很生氣，這裡有一個XX帖子，我要罵一頓樓主發洩一下”這樣的行為。
就像我之前說的，我不太能理解“生氣”或者因為別的什麼事導致了“煩躁”是什麼感覺。
如果我有讓這部分網友感到了不適，在這裡我要對這部分網友進行道歉。我以後不會再問一些讓你們感覺到不適的問題。
