# 3

從.de來看，[http://www.giantessgarden.de/](http://www.giantessgarden.de/)是德國網站。在上次的事件中，I-Raf-You!曾經用英文發表過抗議聲明，不知道是否因為語言不通的問題而被無視。歐美人對於瀏覽日本網站其實興趣缺缺，因為難度太高了，大概就跟我之前企圖要瀏覽一個韓文網站那樣困難。

所以，若有人能幫忙貼出日本那邊的圖片，他們是非常歡迎的。至於轉貼者與收圖者知不知道這些圖片來自於一個付費軟體？那就很難說啦。

不過，至少在這個案例上，I-Raf-You!我記得有在Giantess City發表過聲明才是。儘管Giantess City的文章流量大到足以在三天內把新主題沖到第二頁，但我想也不至於完全沒人看到吧。

因此，這次Giantess Garden貼出的I-Raf-You圖片，不是來自於Giantess City，而是來自於一個類似雙葉貼圖區的[http://orly.yi.org/7898/size/](http://orly.yi.org/7898/size/)，通稱7898chan。那裡是無記名貼圖版，想也知道不可能會有啥秩序，轉貼起來更是肆無忌憚，最近的I-Raf-You圖片多半都是從這被放出來。至於Giantess Garden？他從頭到尾做的就只有一件事，就是把四處搜來的圖片或影音檔放上去。那為什麼砲轟的第一目標總是指向Giantess Garden？一來是他有前科，二來也就是因為那次前科，使得日本GTS圈對Giantess Garden很熟悉，放的注意力也比較多。

講真的，英語系跟日語系兩個地球上最大的GTS文化圈，雙方其實是有點不相往來的味道。不過由於文化接受度有所差異的關係，英語GTS圈比較歡迎日本的創作，反過來歐美的創作在日本就被嫌得很厲害，尤其那種美漫的畫風根本不符合亞洲人的審美觀，就連我們版上也有很多很多人無法接受。

在這種差異性存在之下，就變成了歐美方面一面倒地吃日本豆腐。

當然網路分享的觀念，雙方也是天差地遠。Giantess city常可看見有人放了整本Marvel的漫畫，也沒聽說過有人拿著作權來說嘴；但G-ZONE就從來沒敢把整本漫畫放上去，要有也得在類似雙葉貼圖的那個無記名貼圖版低調進行，而且就算這樣也還是會有日本人反對。

連不是自己的創作物都這麼謹慎，輪到自己的創作被輪姦的時候，想也知道日本人會有多生氣。

不過，講真的，這種風波我看不會是最後一次。美日雙方的認知存在太大的差異，Giantess Garden也從沒理會過I-Raf-You!的任何一次抗議，我看我們就安靜地在旁邊看戲吧，畢竟我們實在沒理由要特地支持哪一方，儘管我們心態上總是偏向日本一點(畢竟日本的創作較合我們口味)。 <title>4</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
