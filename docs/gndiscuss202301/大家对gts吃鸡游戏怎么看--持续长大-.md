# 1

大吉大利，今晚吃鸡！（持续长大！）
神明大人随手制作的多人在线求生游戏，玩家上限100名男性，开局随机一个（不同职业的）妹子（大小随机，最小为正常人十分之一，最大为正常人体型），得到妹子的时候有且仅有一次机会修改妹子的外观（不能修改既定身高），妹子认对应的玩家为主人，性格默认加入淫荡。
妹子用嘴巴或者屁股，x道之类的地方吞食x液就会得到充能，充能足够就会变大，每10倍为一个阶段，一个阶段有九星，对应的体型也就是一倍到九倍。
每个玩家每次射x值设定为一万，冷却为四小时，可以使用特殊的道具（每个玩家开场两只）瞬间回复精力重置射x冷却。举例：15cm的小妹子升级到30cm需要数值5的x液，从30cm升到45cm需要10，从45cm升到60cm需要20，也就是每一星之间所需要x液为2倍。
但是从一个阶段到下一个阶段，都是瓶颈期，比如存在135cm的妹子（一阶段九星）想升150cm，则需要10倍于之前的精液量。
而150cm（指的差不多是正常人体型，每个妹子会略微有些差异）升二阶段2星，需要高达5000点的x液，这个值看起来比突破瓶颈需要的少，但是后面的升星过程会非常艰难。
当某个玩家的妹子到了三阶段，也就是正常人的十倍之后，她可以吃人，一个男性npc就价值两万，玩家更是高达五万！同时，玩家此时获得一个技能：持续冥想，可以与妹子互通五感，同时可以与妹子心灵对话。
到达第四阶段（百倍），则每天可以开启一次吸收模式（这个技能可以被最开始送的特殊道具重置冷却），将周围所有的npc与玩家吸入。当然，足够巨大的妹子是可以保护玩家的，不够巨大的只能看着自己的主人被吸上天空。
神明大人预计第五阶段以后这场游戏就将走向终末，因此箱庭的就设计为亚洲东部地区。
即使玩家之间达成某种停战协议，神明大人的真言会不停折磨巨大妹子，逼迫其继续战斗！
神明大人认为这场战斗对于最先达到第四阶段，或者说拥有最大的妹子的玩家优势过大，于是增加了一些隐藏的战术：比如可以让缩小的妹子被吸入胃中，在胃里捕食未被消化的小人，或者偷偷摸进x道，搜刮巨大妹子残留的x液，从内部强制长大击杀巨大妹子；或者数人合作，击败最强者。
另外注明：1.女性npc没有分数，但是可以玩弄，打发时间。
2.即使没有给妹子x液，每天晚上12点整，三阶段以下的妹子也会自动升一星（但是九星不会自动突破瓶颈）
3.此游戏介绍掌握在神明大人手中，各项数据及调整，玩家，妹子，npc皆不知晓。
4.玩家与妹子的关系：玩家死亡，妹子失去意识；妹子死亡，玩家除了头痛，并无大事。妹子默认玩家为主人，听从一切命令，不反抗心中的淫乱。
5.由于某些妹子身份特殊，可能开场具有很大优势，神明大人已自动修正为：千金大小姐（及其落魄的）；格斗高手（重伤退役的）之类的。
5（新）.上面一条划掉，所有妹子基础能力波动相差不大。
6.神明大人会通过电子音在恰当的时间给予恰当的提示，比如在开场会告诉你特殊道具的用处，玩家与妹子之间的关系，诸如此类。
7.获胜玩家会和妹子一同见证神明大人的降临，届时，神明大人的体型会与妹子联动，始终保持为妹子的10倍大小。（想要趁胜出之时偷偷变大干掉本神？还是太天真了！）
8.上面一条可以修改。（怕输的姐姐，哼！）
