# 1

作为一个历史很久远的同好论坛，我其实是很不想看到他走下坡路的。但是事实就是这样，不知道怎么回事，或许是制度问题或是别的什么因素，gn好像越来越不活跃了。也不知道什么时候开始，气氛就冷清了下来，至少从最近几个月来说，我们的活跃度远远不如隔壁。大家都乐意潜水，但是这其实是我们最不愿意看到的。我认为我们需要改革一下现有的制度。至少要稍加限制一下只伸手并一直潜水的人，或许要让他们不要那么容易拿到资源（比如说我这种人），或许我们也可以效仿一下隔壁的制度，让气氛再次活跃起来。免得慢性死亡，淹没在时间的长河里。以上。如有不同看法，请轻喷，理性讨论 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
