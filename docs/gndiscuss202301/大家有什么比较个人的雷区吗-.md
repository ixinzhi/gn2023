# 1

突然很好奇大家在看作品的时候有什么比较个人的雷区。或者说有什么要素是你觉得没办法接受的？如果可以的话能不能说一说为什么你会觉得接受不了？

这里的雷区指的不是特殊领域的那种本来就非常小众的癖好，而是一些更常见的、可能很多作者都会有意识甚至无意识地加进去的要素，但你就是觉得这种要素很碍眼？

举个例子，我个人很讨厌作品中出现缺乏背景铺垫的侏儒，尤其是在mini作品中常见的那种明明早就成年了但身高只有一米（夸张说法，领会精神）的侏儒角色，让我有一种“为了和后辈产生反差而强行搞反差”的感觉，反而很破坏代入感，远没有一个正常人与GTS之间的对比来得爽快。
