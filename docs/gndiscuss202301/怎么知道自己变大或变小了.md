# 4

事實上
在生活中就有這樣的案例了
例如小體型動物從自己身長好幾倍的高度掉落
並不會受到致命傷
但相同的比例
越大型的動物會受到越大的傷害
因為力的公式
F(力)=M(質量)A(加速度)
所以在大家的加速度A(地心引力)為相同的前提下
質量所影響的力就差了很多

測試的方法也很簡單
就是垂直的從較高的地方跳下
感受一下與先前的跳躍感差距如何
如果變重則明顯變大
如果變輕就變小
