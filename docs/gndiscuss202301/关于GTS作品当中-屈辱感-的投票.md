# 19

我觉得和性欲有关的，都有一定相通的地方。没有屈辱感，就单纯描写“虐”，也肯定可以达到刺激的目的，读者会多偏向于第三视角，看发生到简单刺激的画面时只想到刺激，过瘾，然后达到满足！ 我觉得形容的话有点像看了美女，然后想象单纯的活塞，就可以开始自慰了。
而有屈辱感，我觉得侧重的是心理层面，这个给读者达到刺激的效果更像是慢慢的累积，可以撸，也可以不撸，可以看剧情的发展，“虐”作为一种辅料，也可以两者连在一起，产生种种屈辱妙想。而且侮辱，是需要其他事物来衬托出来的，就涉及到多样性。

有时我也只想看没有屈辱感的作品，简单知了，而且看得是画面，只需要作者能表达出来，我就可以满足了。对作者的感觉，只要他写出我想看的情节画面就可以了。
很实在。老实话这样的作品对创作要求并不高。
想看屈辱感的作品，老实说我会选择在一个安静的时间，静下心去读。要有铺垫，要有剧情，要又一定结实的框架，老实说这样的作品对作者要求会更高，情节要引人入胜。我觉得这样的作品能让读者欲罢不能，不达到刺激的快感，总是在到达之前那一点点就收回去，然后如此反复，让读者读的事后一直处在想出不出，浑身难受的状态，这样的作品质量肯定更高！
