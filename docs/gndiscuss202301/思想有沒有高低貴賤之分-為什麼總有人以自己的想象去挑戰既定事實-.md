# 1

A認為人要把自己最好的那一面表現出來，而且A的好勝心很強。
所以當同事的女兒說“達成自己的目標就夠了，其餘的努力只是白費時間”的想法之後，A嘲笑同事女兒的思想並覺得她的思想深度太低。
具體的方式就是：
同事的女兒說自己能夠考上更好的大學（實際上確實如此，她在模擬考中的成績遠遠超出自己的目標），可是自己的目標是大學A而不是那些更好的大學，所以認為將自己的努力放在考高分刷題上是浪費時間。
而A便說“說大話誰不會說”諸如此類的話，然後說“人往高處走，水往低處流”和諸如此類的話。
同事的女兒說不過對方，便回了家。
今天大約中午12：47左右，同事的女兒來到醫院找他，然後和他說了這一件事。
我作為醫生，我的職業要做的就是救死扶傷，可是我這樣的思想仍然受到諸多挑戰：
比如說有人會說醫生低賤，或是在不了解醫生的情況下貶低醫生。
比如有的人連外科內科都搞不懂是什麼意思，就對外科醫生大加批判。
這裡也略微科普一下通常意義上的外科和內科是什麼意思：
外科主要以手術為治療手段，如切除、修復、填補等。
內科主要則是以藥物、行動、觀察或介入性治療等為治療手段。
同一個疾病，可能當內科診斷無法醫治，就需要送入外科進行手術治療，所以內科學的大量知識實際上也是外科醫生需要掌握的基礎知識。
而我遭遇的問題是一個網友對我的說法進行反對，並用其以經驗為主，想象出來的知識進行反駁。
在這裡我綜合這兩個事例想要詢問的問題就是：
思想有沒有高低貴賤？
為什麼總有人用想象出來的知識去挑戰現實實踐？

我认为，思想只有各自的自由，高低贵贱的标准全在于每个人自己判定，或许大多数人的标准由雷同而趋同，取得了所谓高低贵贱的“社会共识”，但如果他们同时也有一种彼此尊重的共识，那么我想您所列举的两个例子中，他们俩的态度就不会那么傲慢而引起您的不快了。
我心中也有一杆高低贵贱的秤，但我只认为那只是事物本身的高低贵贱，与人本身关联不大。别人考取更高等级的学校，或者比自己的专业能力强，但聪颖是天生的，后天努力也有各自不同的境遇，并不是不尊重不如自己或嫉妒强于自己的人的理由。
真正的低贱恰恰是人格的体现，或许那两个例子，让您对那二人的人格有个直观的判断，前者嫉妒，后者无知。您或许可以换个角度理解，前者本身就是个自卑的人，每天和人比个不停不光丢自己的人，还特别累。后者也许就是个喷子，要么在网上赚发帖费，要么就是在现实中就是个没有存在感的人。都是可怜人。。。。。。
小结下，如果非说思想有高低贵贱之分，真正为其敏感且苦恼不止的，应该是那二位，而不该是您。无知到挑战既定事实这种事，品性良善的您是无法要求某类人也同样尊重自己的，谁选择了什么道路，谁就自己承担什么后果，各自有各自的福报和苦果，不理就是了。

我虽然觉得思想有高低贵贱之分，但是更多的是把思想从容易想到的和不容易想到的上面来分。

就是说，如果有人想出来新的数学理论，经济学模型，或者新的计算机算法的计算方式，又或者一种新的哲学问题的切入方法的话，我会觉得这个思想很厉害。

而你说的这个例子……怎么说呢，感觉更多只是话语权的问题……

就是说，这里不是思想的高低贵贱，只是话语权的高低贵贱，如果有更强的话语权，很容易就可以随便嘲笑别人的思想嘛。

这其实从一开始就是伪命题

因为思想这个概念涵盖太模糊，等量代换成类似的词语：生命
也能起到差不多的意思。

生命有没有高低贵贱之分？

这点《拯救大兵瑞恩》里面已经把事情放在台面上说了。

生命本身没有高低贵贱吧，但是工具人和大企业家的命怎么就有差别了呢？

是先天形成的差距还是后天的呢？

生命没有高低贵贱之分，但人有。

同理：思想没有高低贵贱之分，但人有。
