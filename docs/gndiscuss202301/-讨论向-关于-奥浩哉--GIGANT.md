# 4

奧浩哉的作品腦洞其實滿大的
但他作品裡曾經有巨大娘跟GT的社會出現
所以這次不知道會以怎麼樣的形式展現出GTS
上一部完結的犬屋敷腦洞也滿大的
反正奧浩哉的作品大部分都是科幻偏寫實、偏血腥、真實感暴力
而且看到目前劇情之後應該會有大事件
以前的作品大約劇情就是先用高科技道具玩遊戲玩生存遊戲殺奇怪的物種
然後出現越來越扯的怪物
之後出現10倍(吧)大的外星人
還有外星小蘿莉
而且還有超大型機器人(因為外星人很大 反正想像超大外星人又駕駛了鋼彈那種概念就對了)
反正腦洞超級大
