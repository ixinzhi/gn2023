# 1

**年初去某個縣城的醫院進行學習的時候，醫院裡有一個長得非常好看的護士小姐（我個人並不知道好不好看，我算是臉盲，是醫院裡的其他人和我說的），因為她護理能力強、聲音甜美、大家都喜歡接受她的護理。**
**但是問題來了，儘管她的個人能力非常強，但是大部分人卻只注意到她長得漂亮這點，完全忽視了她個人能力很強的優點，提起她，大家會說：“那個護士長得真好看。”卻不會說：“她給我扎針扎的真準。”或“她的護理真是無微不至。”。**
**因此在我看來，這位護士過於漂亮的美貌掩蓋了其身上具有的其他特點，雖然外表要讓患者覺得舒心也是護士的職業天賦技能，但顯然因為這位護士的天賦技能等級過高，而導致大家看不到她其他的技能。**
**另外就是對於一些明星偶像的思考：**
**某位女星，雖然一直有很好的資源，演技也不差，長相甜美，但是卻一直不溫不火，後來和某位混時尚圈的朋友討論后，得到了她的見解：**
**這位女星並不是因為長得差或者是演技不到位而不溫不火，恰恰就是因為她自身的氣質過於穩定而且偏向甜美，當套用到不合適的角色身上時，她身上的氣質會讓人難以入戲，而且剛好她並沒有飾演到一個合適自己氣質的角色，再加上這樣的氣質在生活中很常見，所以才會這樣……**

**雖然只有這樣的兩例，但我得出一個結論：**
**過於強大的某種屬性會掩蓋某種事物身上的其他特點。**
**這讓我考慮到巨大少女在實際上的特點——**
**會不會有因為太漂亮而讓你注意不到她是一個幾百米高的少女呢？**
**或者是因為氣質太弱氣而讓人認為這根本不是一個百米高的少女。**
**或者換一個問題，一個幾百米高的女孩，氣質/外貌要有多弱氣/漂亮才能讓人忽略她有著百米身高的這點呢？**
