# 4

这个问题细究起来，其实还是挺值得一说的

首先我们得接受一个客观事实：大多能够流出的资源（如e-h等），往往只是整个圈子的冰山一角，
许多赞助作者才能看到的资源，以及部分小众网站上的付费资源，通常不可能被广泛流出。
因此很多资源可能几年了也等不来，还是得自掏腰包去赞助一下才行。

在决定购买后，还会遇到两个问题：**价格**和**付费方式**，

许多资源以美元标价，经过汇率换算往往不便宜（**例如某fan一个月会员要140人民币**），并且不是所有人都拥有paypal，
所以私底下也参与过一些合购资源的小团体，通常成员只有很少几个，也都能保证不会流出。
但团购也算是可遇不可求的了，
基本和小白无缘，这也是许多二道贩子猖獗的原因所在。

买完资源最怕的两种情况，1是购买后资源被流出，2是不符合心理预期/性价比不够。
而这两种情况偏偏都遇到过，前者是合购了某合同志后，不到一周就在eh上看到了这个本子；后者是花大价钱买的本子只有寥寥几张图，甚至还有封面党欺诈嫌疑。
但我个人觉得，在对作者/内容足够喜爱的情况下，其实这都不是什么大问题，权当是赞助作者也好。
就好比市面上可以找到很多盗版游戏，但我们还是愿意上steam支持正版。
