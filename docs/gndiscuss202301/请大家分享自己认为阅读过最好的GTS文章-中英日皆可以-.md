# 1

不才在GTS圈已经有最少七年了，七年间阅读过不少好文，今日就逐一来分享阅读心得，也希望各位也能不啬分享各位的高见

1.温韵-算是很早期的中华GTS作品，子一算是中华GTS圈里最有个性的女神之一，表面很会作弄主角，但内心确十分疼惜主角，个人觉得其实有点像《恐怖童话》里的玡子，只不过她比较理智一些。很可惜最近作者好像已经退隐江湖了，很少看到有作品了。

2.平凡女孩- 此作最后是蛮触动人心的，只不过作者花了不少的时间来铺垫故事，导致男主戏份不是非常多，不过让人高兴的是最后是完美结局了

3\. 朋友- 本坛少见的GC文，无助感和屈辱感写的非常好，虽然后来的《缩小射线》的确在情节塑造上已经超越了他，但是作品之所以经典，原因包括人性的扭曲描写的非常的好，很可惜的是作者好像很久没有大作出现了。

4\. Closed- GIANTESSWORLD上的作品，此大作虽然阅读量不高，可是他描写的继母与继子之间的情谊，是目前GTS圈很罕见的，那位大大如果能翻译，那么我们论坛的各位坛友就有福了。不才暂时没有空翻译此大作。 <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">

因为个人有些抖M癖好的原因，分享的文章也会密切结合所控的属性，若有不适，还请有选择性地赏读了：

偏好属性组合如下：
巨大倍率（1000倍恰到好处，300、500、2500倍等等同样接受）
足控（并不恋臭，单纯鞋袜裸足等等动作造成的视觉效果）
优越感（如女主人公对所造成灾难的毫不关心、公开自摸自慰、言语嘲讽等等行为）
临场感（第一人称，女主人公的日记、自传形式，或者特定一人的第三人称，比如身为小人的目击者等）
幼龄少龄（10-30）
以上便是个人控的属性的从高到低的排序了，对倍率十分特别相当的敏感。也许有说过的一句类似话，可以总结Size Fetish这一性癖的本能吧…那就是：
**“一个1000倍大小的少女只是单纯的站在那里，她的存在就如同一个神迹那样，足以令人想入非非难以自拔。”**

**唐変木 先生** [gs-uploader.jinja-modoki.com/upld-index.php?uname=touhenboku](http://gs-uploader.jinja-modoki.com/upld-index.php?uname=touhenboku)

推荐他的：
**《飲み過ぎには注意しましょう》**很有趣的一篇短文，讲的是体质娇弱控制不好自己巨大化的社交黑洞Office Lady，在入职第一天的欢迎会上就喝的酩酊大醉，睡梦中巨大化到1000倍，醒来后因为迟到而手忙脚乱赶去上班的故事。有赶地铁却一脚踩平了车站广场、拔起公司大楼来向领导谢罪的桥段。
