# 1

感觉作为理科生，脑子里经常能把大致的剧情想出来，也可以想象出不同的Play的方法。但是在想把这些内容转换成文字的时候就死机了。。。

发现对白一点都没有巨人跟小人的感觉。。。简直是跟面试和论文一样的气场。。。

女 “可恶的诚，看我变小你。”
男 “啊，被变小了，好可怕啊。”
女 “哈哈哈。你看看你变小的跟虫子一样的可怜。哈哈哈。”
男 “啊啊啊。好可怕啊。”

（笑）

总之大概是这种感觉的问题。。。
