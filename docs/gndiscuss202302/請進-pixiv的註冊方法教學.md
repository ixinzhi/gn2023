# 1

首先請到pixiv的主頁:
[http://www.pixiv.net/](http://www.pixiv.net/)
然後按最上方的"新規登錄"
在下一個畫面的空間裡輸入你的電郵並按下面的按鈕
再按右面的按鈕

接著就登入你剛輸入的電郵,會有封新電郵
總之按那個連結就是了
然後就是真正的註冊畫面
只要填好打星的項目就可以了

pixiv id: 用戶名,登入用

パスワード: 密碼, 6-32字元

ニックネーム: 暱稱, 15字以下

然後性別,住所等不用說了吧![](img/e165e72abfe717aae40508dc244b1023.png)
一切弄好就按最面的按鈕
於是大功告成!!!!![](img/bd15f544951e88cbc9e70768366c9ed0.png) <title>2</title> <link href="../Styles/Style.css" type="text/css" rel="stylesheet">
